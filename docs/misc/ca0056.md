# CA0056
Un archivo de proyecto tiene una versión incorrecta de la herramienta de análisis.  
  
 Esta advertencia se produce si FxCopCmd.exe se invoca manualmente con un archivo de proyecto cuya versión de la herramienta de análisis de código es incorrecta.
