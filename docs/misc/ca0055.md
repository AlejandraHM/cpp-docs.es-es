# CA0055
No pudo cargar file:path\<\>  
  
 La herramienta de análisis de código no pudo cargar el archivo de destino que se especificó para el análisis.  Esto ocurre si no se encuentra el archivo o se deniega el acceso a él.  Normalmente, sucede si el usuario decide reemplazar la propiedad de ensamblado de entrada en [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], o invoca manualmente fxcopcmd.exe.
