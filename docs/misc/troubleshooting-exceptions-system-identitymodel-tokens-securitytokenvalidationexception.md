# Soluci&#243;n de problemas de excepciones: System.IdentityModel.Tokens.SecurityTokenValidationException
Se produce una excepción <xref:System.IdentityModel.Tokens.SecurityTokenValidationException> cuando un token de seguridad recibido no es válido. El método <xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken%2A> produce normalmente la excepción.
