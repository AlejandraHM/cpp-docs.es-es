# Error de MSBuild MSB3176
**MSB3176: La versión mínima necesaria especificada es más reciente que la versión de publicación actual.  Especifique una versión anterior o igual a la versión de publicación actual.**  
  
 Se genera este error cuando la versión mínima necesaria de la aplicación es mayor que la versión actual del manifiesto de implementación.  
  
### Para corregir este error  
  
-   Especifique una versión mínima necesaria menor o igual que la actual versión de publicación.
