# Soluci&#243;n de problemas de excepciones: System.DirectoryServices.Protocols.DsmlInvalidDocumentException
La excepción que se produce cuando una solicitud del DSML \(Directory Services Markup Language\) o documento de respuesta no es un XML correctamente formado, o no se puede validar con el esquema de DSMLV2.
