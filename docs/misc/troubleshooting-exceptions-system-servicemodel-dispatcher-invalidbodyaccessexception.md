# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Dispatcher.InvalidBodyAccessException
Una excepción <xref:System.ServiceModel.Dispatcher.InvalidBodyAccessException> representa la clase base abstracta de las excepciones que se producen si se intenta obtener acceso al cuerpo de un mensaje cuando no se permite.
