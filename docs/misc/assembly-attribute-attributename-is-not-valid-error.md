# El atributo &#39;&lt;attributename&gt;&#39; del ensamblado no es v&#225;lido: &lt;error&gt;
Common Language Runtime no acepta el atributo en un ensamblado.  
  
 **Id. de error:** BC30129  
  
### Para corregir este error  
  
1.  Quite el atributo de la declaración o reemplácelo por un atributo que sea válido para el contexto.  
  
2.  Examine el mensaje de error indicado y tome las medidas adecuadas.
