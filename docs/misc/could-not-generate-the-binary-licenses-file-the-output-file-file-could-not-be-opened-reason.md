# No se pudo generar el archivo de licencias binario. No se pudo abrir el archivo de salida &#39;file&#39;. &lt;motivo&gt;
El sistema del proyecto no pudo eliminar una copia obsoleta del archivo de licencias. Como parte de la preparación de licencias, el sistema del proyecto transformará archivos .licx de texto en archivos de licencias binarios que el sistema de licencias de .NET pueda comprender.  
  
 **Para corregir este error**  
  
-   Reinicie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     El proceso de compilación no se realizará correctamente si se produce este error.
