# Soluci&#243;n de problemas de excepciones: System.ServiceModel.CommunicationObjectFaultedException
Se produce una excepción <xref:System.ServiceModel.CommunicationObjectFaultedException> cuando se realiza una llamada a un objeto de comunicación que ha generado un error.
