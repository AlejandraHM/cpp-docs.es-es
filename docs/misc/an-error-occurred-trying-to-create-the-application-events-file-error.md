# Error al intentar crear el archivo de eventos de la aplicaci&#243;n. &#39;&lt;error&gt;&#39;
Este error se produce cuando se crea una aplicación de explorador de WPF y Visual Studio no genera el archivo de eventos de la aplicación \(Application.xaml.vb o App.xaml.cs\).  
  
### Para corregir este error  
  
-   Guarde y cierre la solución.
