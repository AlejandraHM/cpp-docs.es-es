# Error PRJ0012 al compilar el proyecto
Debido a que los recursos del sistema son muy escasos, No se pudo crear un subproceso necesario para iniciar una compilación.  
  
 Si los recursos del sistema se encuentran bajos:  
  
-   Cierre algunas de las aplicaciones en ejecución.  
  
-   Reinicie Visual Studio para liberar recursos.  
  
-   Reinicie el equipo.
