# Cannot find component &#39;file&#39; of multifile assembly &#39;assembly&#39;
No se puede encontrar un archivo de un ensamblado de múltiples archivos.  Un ensamblado de múltiples archivos hace referencia a uno o más archivos, como los archivos de recursos o módulos.  
  
 El ensamblado que se está copiando puede ser una referencia directa, visible en el nodo **Referencias** del Explorador de soluciones, o un ensamblado que sea una dependencia de una referencia directa.  
  
 **Para corregir este error**  
  
-   Instale de nuevo el componente.  
  
     El proyecto puede compilarse, pero podría no ejecutarse.
