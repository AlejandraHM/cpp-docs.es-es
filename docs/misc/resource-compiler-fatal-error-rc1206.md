# Error irrecuperable del compilador de recursos RC1206
La página de códigos especificada en la línea cmd no existe en el registro.  
  
 La opción para especificar la página de código \(\/c\) se ha seguido por una página de códigos cuya ubicación no se especifica en el registro. Ejecute el programa de configuración para instalar la página de códigos especificada.
