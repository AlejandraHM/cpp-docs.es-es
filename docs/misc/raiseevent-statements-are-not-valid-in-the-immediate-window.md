# Las instrucciones &#39;RaiseEvent&#39; no son v&#225;lidas en la ventana Inmediato.
Las instrucciones `RaiseEvent` no se pueden usar en la ventana **Inmediato**.  
  
 **Identificador de error:** BC30739  
  
### Para corregir este error  
  
-   Pruebe las instrucciones `RaiseEvent` desde el código del **Editor de código**.
