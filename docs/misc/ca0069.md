# CA0069
UsingAlternatePlatform.  No se encontró *FrameworkVersion1*.  Se está usando *FrameworkVersion2* en su lugar.  Para obtener los mejores resultados del análisis compruebe que está instalada la versión correcta de .NET Framework.
