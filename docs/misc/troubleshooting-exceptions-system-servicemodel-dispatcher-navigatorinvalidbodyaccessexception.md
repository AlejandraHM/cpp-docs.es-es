# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException
Se produce una excepción <xref:System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException> cuando un objeto <xref:System.Xml.XPath.XPathNavigator> se dirige a examinar el cuerpo de un mensaje no almacenado en búfer.
