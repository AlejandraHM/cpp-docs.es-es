# Error del compilador de recursos RC2138
cierre de comentario '\/\*' desemparejado  
  
 Una secuencia de apertura de comentario \(`/*`\) no tiene una secuencia de cierre de comentario \(`*/`\) emparejada. Este error puede deberse al anidamiento de comentarios.
