# Soluci&#243;n de problemas de excepciones: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
Se produce una excepción <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> cuando el código intenta obtener acceso a un control que no está en el documento. Puede que el usuario final o algún otro código haya quitado el control. No existe ninguna forma de comprobar si existe el control antes de intentar obtener acceso a él; si se ha quitado, la solución no funcionará.
