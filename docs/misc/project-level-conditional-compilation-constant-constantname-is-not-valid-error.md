# La constante de compilaci&#243;n condicional de nivel de proyecto &#39;&lt;nombreDeConstante&gt;&#39; no es v&#225;lida: &lt;error&gt;
Se especificó una opción de compilación condicional no válida para el proyecto.  
  
 **Identificador de error:** BC31030  
  
### Para corregir este error  
  
1.  Compruebe las constantes de compilación condicional usadas en este proyecto.
