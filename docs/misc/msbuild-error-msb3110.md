# Error de MSBuild MSB3110
**MSB3110: El ensamblado '\<ensamblado\>' tiene una identidad '\<identidad del ensamblado\>' que no coincide; se esperaba el nombre de archivo: '\<archivo\>'.**  
  
 Este error se genera cuando el nombre de archivo del ensamblado obtenido de la identidad del ensamblado no coincide con el nombre de archivo real.  
  
### Para corregir este error  
  
-   Corrija el nombre de archivo del ensamblado para que coincida con la identidad del ensamblado.
