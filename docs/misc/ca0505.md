# CA0505
La propiedad desusada se omitirá porque se define la propiedad que la sustituye.  
  
 Advertencia: CA505 se genera cuando las propiedades de análisis de código desusada y sustitutiva contienen valores en un proyecto o archivo de configuración de compilación.  Para resolver este problema, quite la propiedad desusada.
