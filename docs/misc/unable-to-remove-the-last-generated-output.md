# Unable to remove the last generated output
El último resultado generado conocido no se ha quitado del disco ni del proyecto.  
  
 Si una herramienta personalizada devuelve un error, el último resultado generado \(si existe\) se quita del disco y del proyecto.  
  
 Esto puede suceder si el resultado generado está bloqueado en el disco.  
  
 **Para corregir este error**  
  
-   Cierre y reinicie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].
