# Soluci&#243;n de problemas de excepciones: System.Workflow.Runtime.WorkflowOwnershipException
Se produce una excepción <xref:System.Workflow.Runtime.WorkflowOwnershipException> cuando el motor en tiempo de ejecución de flujo de trabajo intenta cargar una instancia de flujo de trabajo cargada actualmente por otra instancia de dicho motor. Además, esta excepción se produce cuando el motor en tiempo de ejecución de flujo de trabajo intenta guardar un flujo de trabajo después de haber expirado el tiempo de espera de propiedad que se especificó cuando se cargaba ese flujo.
