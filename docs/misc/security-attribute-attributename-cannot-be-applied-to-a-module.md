# El atributo &#39;&lt;attributename&gt;&#39; no se puede aplicar a un m&#243;dulo
Common Language Runtime no acepta el atributo de seguridad en un módulo.  
  
 **Identificador de error:** BC30131  
  
### Para corregir este error  
  
1.  Quite el atributo de la declaración.
