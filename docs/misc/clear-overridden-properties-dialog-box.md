# Borrar propiedades reemplazadas (Cuadro de di&#225;logo)
> [!NOTE]
>  Este cuadro de diálogo no se admite en [!INCLUDE[vs_dev10_long](../build/includes/vs_dev10_long_md.md)].  
  
 Cuando agrega una hoja de propiedades definida por el usuario \(archivo .props\) a una configuración de proyecto, esa configuración de proyecto hereda de forma predeterminada todas las propiedades definidas en la hoja de propiedades.  Utilice este cuadro de diálogo para desactivar las propiedades heredadas no deseadas de una configuración del proyecto.
