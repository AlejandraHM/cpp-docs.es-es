# Type library &lt;name&gt; was not found.
Visual Studio no se puede ejecutar sin las bibliotecas de tipos vsext.tlb y DTE.olb.  Este error ocurre normalmente cuando se quitan o eliminan estos dos archivos.  
  
### Para corregir este error  
  
1.  Vuelva a instalar Visual Studio.
