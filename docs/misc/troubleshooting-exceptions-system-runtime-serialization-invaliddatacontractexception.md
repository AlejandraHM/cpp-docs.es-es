# Soluci&#243;n de problemas de excepciones: System.Runtime.Serialization.InvalidDataContractException
Se produce una excepción <xref:System.Runtime.Serialization.InvalidDataContractException> cuando <xref:System.Runtime.Serialization.DataContractSerializer> o <xref:System.Runtime.Serialization.NetDataContractSerializer> detecta un contrato de datos no válido durante la serialización y deserialización.
