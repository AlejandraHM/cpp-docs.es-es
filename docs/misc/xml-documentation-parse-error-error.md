# Error de an&#225;lisis de la documentaci&#243;n XML: &lt;error&gt;
Se pueden mostrar los mensajes siguientes con este error: Consulte el tema correspondiente para obtener más información:  
  
-   [La etiqueta de comentario XML '\<tag\>' aparece con atributos idénticos más de una vez en el mismo bloque de comentario XML](../misc/xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block.md)  
  
-   [No se permite la etiqueta de comentario '\<tag\>' en un elemento de idioma '\<element\>'](../misc/xml-comment-tag-tag-is-not-permitted-on-a-element-language-element.md)  
  
-   [El parámetro '\<parámetro\>' de comentario XML no coincide con un parámetro en la instrucción \<palabraClave\> correspondiente](../misc/xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement.md)  
  
-   [El parámetro de comentario XML debe tener un atributo 'name'](../misc/xml-comment-parameter-must-have-a-name-attribute.md)  
  
-   [El comentario XML tiene una etiqueta con un atributo 'cref' '\<attribute\>' que no se pudo resolver](../misc/xml-comment-has-a-tag-with-a-cref-attribute-attribute-that-could-not-be-resolved.md)  
  
-   [La etiqueta de comentario XML 'include' debe tener atributo '\<atributo\>'.](../misc/xml-comment-tag-include-must-have-a-attribute-attribute.md)  
  
 **Identificador de error:** BC42304
