# Soluci&#243;n de problemas de excepciones: System.IdentityModel.Selectors.UntrustedRecipientException
Se produce una excepción <xref:System.IdentityModel.Selectors.UntrustedRecipientException> cuando un usuario revisa la información de una entidad que solicita un token y decide no confiar en la entidad. El usuario hace clic en **Deshabilitar Windows CardSpace** en la interfaz de usuario de CardSpace.
