# Error de MSBuild MSB3178
**MSB3178: El ensamblado '\<ensamblado\>' se especificó de manera incorrecta como un archivo.**  
  
 Esta advertencia se genera durante la generación del manifiesto de aplicación cuando el proceso de compilación detecta que una referencia de archivo realmente es un ensamblado \(administrado o nativo\).  
  
### Para corregir este error  
  
-   Para obtener más información sobre la causa de un error de MSBuild y las posibles soluciones, vea [Recursos adicionales](../Topic/Additional%20MSBuild%20Resources.md).
