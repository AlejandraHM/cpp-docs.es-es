# Advertencia del compilador (nivel 1) CS2029
Valor no válido para '\/define'; 'identificador' no es un identificador válido.  
  
 Esta advertencia se produce si el valor que se usa en la opción [\/define](../Topic/-define%20\(C%23%20Compiler%20Options\).md) tiene algunos caracteres no válidos.  
  
 No se puede suprimir esta advertencia con la opción [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md).
