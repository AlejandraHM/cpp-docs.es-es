# override
La palabra clave `override` tiene básicamente el mismo significado y la misma sintaxis en las extensiones de componentes de ISO C\+\+ y de C\+\+ \(C\+\+\/CLI y C\+\+\/CX\).  Sin embargo, hay algunas diferencias en cuanto a las implicaciones de su uso.  Para obtener información detallada, vea estos artículos:  
  
 En ISO C\+\+:  
  
-   ISO C\+\+ [override](../misc/override.md)  
  
-   ISO C\+\+ [Invalidaciones explícitas \(C\+\+\)](../cpp/explicit-overrides-cpp.md)  
  
-   \(Aplicaciones de la Tienda Windows\) [Clases y structs ref \(C\+\+\/CX\)](../Topic/Ref%20classes%20and%20structs%20\(C++-CX\).md)  
  
-   [override](../windows/override-cpp-component-extensions.md)  
  
-   [Invalidaciones explícitas \(extensiones componentes de C\+\+\)](../windows/explicit-overrides-cpp-component-extensions.md)
