# Error de MSBuild MSB3168
**MSB3168: Se omitirá el elemento duplicado '\<paquete\>'.**  
  
 Esta advertencia se produce si se ha especificado que el arranque debe instalar dos paquetes de arranque idénticos.  En este caso, el arranque instalará sólo una instancia.  
  
### Para corregir este error  
  
-   Para obtener más información sobre la causa de un error de MSBuild y las posibles soluciones, vea [Recursos adicionales](../Topic/Additional%20MSBuild%20Resources.md).
