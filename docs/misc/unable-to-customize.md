# Unable to customize.
Este error aparece cuando intenta personalizar menús o barras de herramientas mientras está depurando una aplicación.  
  
### Para corregir este error  
  
-   Espere hasta que haya finalizado la depuración para realizar personalizaciones de barras de herramientas o de menús.  
  
     \-O bien\-  
  
     Detenga la depuración y realice las personalizaciones de barras de herramientas o de menús.
