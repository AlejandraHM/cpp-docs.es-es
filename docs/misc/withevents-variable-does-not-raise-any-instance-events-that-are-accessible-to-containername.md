# La variable &#39;WithEvents&#39; no genera eventos de instancia que sean accesible para &#39;&lt;nombreContenedor&gt;&#39;.
El objeto al que hace referencia no genera eventos accesibles.  
  
 **Identificador de error:** BC30600  
  
### Para corregir este error  
  
1.  Modifique los modificadores de acceso para el evento o elija un evento que sea accesible en el contexto actual.
