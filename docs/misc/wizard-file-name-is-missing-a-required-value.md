# Wizard file &lt;name&gt; is missing a required value.
El archivo .vsz del asistente no tiene el formato correcto.  Este error ocurre cuando un archivo .vsz está dañado o cuando el archivo proviene de un producto ajeno a Visual Studio.  
  
### Para corregir este error  
  
1.  Compruebe que el asistente sea un asistente para Visual Studio e inténtelo de nuevo.  
  
     \-O bien\-  
  
     Vuelva a instalar Visual Studio para reemplazar el archivo dañado.
