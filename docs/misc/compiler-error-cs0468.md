# Error del compilador CS0468
Ambigüedad entre el tipo 'type1' y el tipo 'type2'  
  
 Este error se genera cuando se están compilando dos tipos en el ensamblado que tienen el mismo nombre completo. Esto puede ocurrir si ambos están en módulos agregados o uno está en un módulo agregado y el otro en el código fuente.
