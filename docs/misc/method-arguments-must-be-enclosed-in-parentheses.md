# Los argumentos de m&#233;todo se deben incluir entre par&#233;ntesis
Las reglas que rigen las llamadas a procedimiento son más sencillas en las versiones más recientes de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)]. Todos los argumentos se deben incluir entre paréntesis.  
  
 **Identificador de error:** BC30800  
  
### Para corregir este error  
  
-   Incluya la lista de argumentos entre paréntesis; por ejemplo:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
