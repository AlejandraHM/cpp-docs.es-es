# No se pudo determinar si &#39;ensamblado&#39; es un ensamblado de m&#250;ltiples archivos. El manifiesto del ensamblado podr&#237;a estar da&#241;ado.
El sistema del proyecto no pudo leer un ensamblado al que hacía referencia el proyecto y, por tanto, no puede determinar a qué archivos hacía referencia el ensamblado.  
  
 **Para corregir este error**  
  
-   Reinstale Visual Studio \(si el ensamblado venía incluido con [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] o [!INCLUDE[dnprdnshort](../error-messages/tool-errors/includes/dnprdnshort_md.md)]\).  
  
     o bien  
  
-   Reinstale el control de terceros adecuado.  
  
     Este error no impedirá que se compile el proyecto. Sin embargo, podría producirse un error en tiempo de ejecución.
