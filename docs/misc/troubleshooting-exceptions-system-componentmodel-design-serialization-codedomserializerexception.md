# Soluci&#243;n de problemas de excepciones: System.ComponentModel.Design.Serialization.CodeDomSerializerException
La excepción que se produce cuando el serializador de CodeDOM encuentra un error.  
  
## Comentarios  
 Esta excepción permite al serializador devolver información sobre la ubicación y el tipo del error.
