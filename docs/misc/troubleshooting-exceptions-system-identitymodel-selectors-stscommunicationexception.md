# Soluci&#243;n de problemas de excepciones: System.IdentityModel.Selectors.StsCommunicationException
Se produce una excepción <xref:System.IdentityModel.Selectors.StsCommunicationException> cuando existe un problema de comunicación con el servicio de token de seguridad.  
  
 El problema puede deberse a las siguientes condiciones:  
  
-   Error de red  
  
-   No se puede alcanzar el extremo MEX  
  
-   El servicio de token de seguridad no tiene una identidad de extremo válida
