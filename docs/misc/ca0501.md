# CA0501
No se puede leer el informe de resultados  
  
 Se produce cuando el informe de resultados no se puede escribir debido a una de las razones siguientes:  
  
-   Disco lleno  
  
-   No hay ningún permiso para escribir en el directorio  
  
-   El directorio no existe
