# Error irrecuperable del compilador de recursos RC1049
argumento numérico 'string' no válido  
  
 El compilador de recursos esperaba un argumento numérico, pero recibió la cadena dada.  
  
 Este error puede deberse a un número hexadecimal sin el prefijo necesario \\x o un número de punto flotante con formato incorrecto.
