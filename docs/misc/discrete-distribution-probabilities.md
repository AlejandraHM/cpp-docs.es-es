# discrete_distribution::probabilities
Devuelve las probabilidades almacenadas para la distribución.  
  
## Sintaxis  
  
```  
std::vector<double> probabilities() const;  
```  
  
## Comentarios  
 Esta función miembro devuelve los valores de probabilidad almacenados.  
  
## Requisitos  
 **Encabezado:** \<random\>  
  
 **Espacio de nombres:** std
