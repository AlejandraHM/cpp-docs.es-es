# Path or drive is not valid.
Este error se produce normalmente cuando el cuadro Examinar del cuadro de diálogo Buscar en está vacío y presiona ENTRAR.  
  
### Para corregir este error  
  
1.  Escriba una ruta de acceso o un directorio y presione ENTRAR.
