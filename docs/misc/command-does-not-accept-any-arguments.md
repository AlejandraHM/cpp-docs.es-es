# Command does not accept any arguments.
Este error ocurre normalmente cuando se ha proporcionado un argumento para un comando que sólo reconoce modificadores.  
  
### Para corregir este error  
  
1.  Compruebe cuál es la sintaxis correcta en la documentación del comando e inténtelo de nuevo.
