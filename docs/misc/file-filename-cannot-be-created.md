# File &lt;filename&gt; cannot be created.
Este error se produce normalmente cuando se ha intentado crear un informe Web de comentario de código pero el usuario actual no tiene acceso de escritura.  
  
### Para corregir este error  
  
1.  Compruebe que dispone de acceso de escritura.  
  
2.  Compruebe que existe espacio suficiente para guardar las páginas.
