# Soluci&#243;n de problemas de excepciones: System.Resources.MissingManifestResourceException
La excepción que se produce si el ensamblado principal no contiene los recursos de referencia cultural neutra, cuando son necesarios dichos recursos porque falta el ensamblado satélite adecuado.
