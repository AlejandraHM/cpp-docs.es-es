# Soluci&#243;n de problemas de excepciones: System.Xml.Schema.XmlSchemaValidationException
La excepción que se produce cuando se encuentran errores de validación de esquema del lenguaje de definición de esquemas \(XSD\) de XML o existen advertencias en un documento XML que se está validando.
