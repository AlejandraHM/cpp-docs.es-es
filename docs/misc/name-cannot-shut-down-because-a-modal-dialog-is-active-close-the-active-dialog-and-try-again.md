# &lt;name&gt; cannot shut down because a modal dialog is active. Close the active dialog and try again.
Este error ocurre normalmente cuando el sistema operativo intenta cerrarse y hay un cuadro de diálogo modal abierto en alguna aplicación, lo que impide cerrarla.  
  
### Para corregir este error  
  
1.  Cierre el cuadro de diálogo modal y después cierre la aplicación, o cierre el sistema operativo.
