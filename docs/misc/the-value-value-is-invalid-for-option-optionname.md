# El valor &#39;&lt;valor&gt;&#39; no es v&#225;lido para la opci&#243;n &#39;&lt;nombreDeOpci&#243;n&gt;&#39;

 **Identificador de error:** BC2014  
  
### Para corregir este error  
  
1.  Compruebe los valores para encontrar el que no es válido.
