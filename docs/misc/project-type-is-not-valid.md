# Project type is not valid.
Este error se produce normalmente cuando se escribe un valor incorrecto para el argumento *templatename* del comando **Agregar nuevo proyecto**.  
  
### Para corregir este error  
  
1.  Compruebe en la documentación cuál es la sintaxis correcta del comando **Agregar nuevo proyecto** e inténtelo de nuevo.
