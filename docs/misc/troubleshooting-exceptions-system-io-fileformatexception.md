# Soluci&#243;n de problemas de excepciones: System.IO.FileFormatException
Se produce una excepción <xref:System.IO.FileFormatException> cuando un archivo de entrada o un flujo de datos que deben cumplir una especificación de formato de archivo determinada son incorrectos.
