# Error PRJ0011 al compilar el proyecto
Debido a que los recursos del sistema son muy escasos, no se puede crear un evento necesario para iniciar una compilación.  
  
 Si los recursos del sistema son escasos:  
  
-   Cierre algunas de las aplicaciones en ejecución.  
  
-   Reinicie Visual Studio para liberar recursos.  
  
-   Reinicie el equipo.
