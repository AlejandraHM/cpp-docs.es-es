# The following components could not be browsed: &lt;name&gt;
Este error ocurre normalmente cuando se intenta agregar un tipo de archivo que no admite examinar objetos, como los archivos .htm y .txt, en el cuadro de diálogo Selector de componentes.  El Examinador de objetos admite tipos de archivo como .bsc, .olb, .tlb, .dll, .exe y .ocx.  
  
### Para corregir este error  
  
1.  Quite el tipo de archivo no compatible de la lista Componentes seleccionados y seleccione un archivo diferente.
