# Demasiados clientes de aplicaci&#243;n DLL
La biblioteca de vínculos dinámicos \(DLL\) de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] solo puede admitir el acceso de un número limitado de aplicaciones host. La aplicación y otras aplicaciones que son hosts de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] \(es posible que su aplicación acceda a algunos de estos\) están intentando acceder a la DLL de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] al mismo tiempo.  
  
### Para corregir este error  
  
-   Reduzca el número de aplicaciones abiertas que acceden a [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)].
