# La herramienta requiere un nombre. Si no quiere guardar esta herramienta, haga clic en Eliminar en el cuadro de di&#225;logo Herramientas externas.
Este error se produce normalmente cuando se ha agregado una nueva herramienta al cuadro de diálogo Herramientas externas pero no se ha especificado ningún título.  
  
### Para corregir este error  
  
1.  Si desea guardar la herramienta, haga clic en **Aceptar** y especifique un título para la herramienta en el cuadro de diálogo **Herramientas externas**.  
  
2.  Si no tiene la intención de guardar la herramienta, haga clic en **Aceptar** y, a continuación, haga clic en **Eliminar** en el cuadro de diálogo **Herramientas externas**.
