# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Dispatcher.XPathNavigatorException
Se produce una excepción <xref:System.ServiceModel.Dispatcher.XPathNavigatorException> cuando se supera la cuota de nodos que se permite inspeccionar a un <xref:System.Xml.XPath.XPathNavigator>.
