# Error irrecuperable del compilador de recursos RC1106
modificador no válido: opción  
  
 La opción no es una opción válida del compilador de recursos.  
  
 Las opciones válidas son:  
  
 \/r  
  
 Se emite un archivo .res  
  
 \/v  
  
 Modo detallado \(se muestran mensajes de progreso\)  
  
 \/d  
  
 Se define un símbolo  
  
 \/fo  
  
 Se cambia el nombre del archivo .res  
  
 \/l  
  
 Identificador de lenguaje predeterminado en hexadecimal  
  
 \/I  
  
 Se agrega una ruta de acceso para las búsquedas **INCLUDE**  
  
 \/x  
  
 Se omite la variable de entorno **INCLUDE**  
  
 \/c  
  
 Se define una página de códigos que utiliza la conversión de NLS
