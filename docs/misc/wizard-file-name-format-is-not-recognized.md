# Wizard file &lt;name&gt; format is not recognized.
No se puede abrir el asistente.  Este error suele producirse porque un archivo de asistente está dañado o porque el número de versión de un asistente no coincide con el de Visual Studio.  
  
### Para corregir este error  
  
1.  Compruebe que el número de versión del archivo de asistente coincide con la versión de Visual Studio, que se muestra como el valor de VSWIZARD, por ejemplo `VSWIZARD 7.0`.  
  
     \-O bien\-  
  
     Vuelva a instalar Visual Studio para reparar el archivo dañado.
