# Error del compilador CS0148
El delegado 'delegate' no tiene un constructor válido  
  
 Ha importado y usado un programa administrado \(uno que usa .NET Framework Common Language Runtime\) que se creó con otro compilador. Ese compilador permitía un constructor [delegate](../Topic/delegate%20\(C%23%20Reference\).md) formado incorrectamente. Para obtener más información, consulta [Delegados](../Topic/Delegates%20\(C%23%20Programming%20Guide\).md).
