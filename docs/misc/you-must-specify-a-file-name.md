# Debe especificar un nombre de archivo.
Intentó una operación, como mover o copiar un archivo, lo que requiere proporcionar un nombre de archivo.  
  
### Para corregir este error  
  
-   Proporcione el nombre de archivo necesario.  
  
-   Asegúrese de que la ruta de acceso no termina con un carácter separador de directorio \(\\\).
