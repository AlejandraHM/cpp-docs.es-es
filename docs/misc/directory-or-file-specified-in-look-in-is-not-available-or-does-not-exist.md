# Directory or file specified in Look In is not available or does not exist.
Este error ocurre normalmente cuando no se encuentra la ruta de acceso o el nombre de archivo especificados en la lista Examinar.  
  
### Para corregir este error  
  
1.  Vuelva a escribir la ruta de acceso y el nombre de archivo.  
  
     \-O bien\-  
  
2.  Navegue hasta el archivo o carpeta correctos mediante la lista Examinar y el cuadro **Elementos disponibles**.
