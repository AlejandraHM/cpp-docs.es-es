# Invalid assembly strong name or name not specified
La propiedad de proyecto del modelo de objeto del sistema de proyecto que le permite especificar un nombre de clave o contenedor de clave para los proyectos que compilan ensamblados con nombres seguros está en blanco.  No se compilará el proyecto si se muestra este mensaje.  
  
### Para corregir este error  
  
1.  Asegúrese de que se encuentra especificado un nombre de clave o de contenedor de clave en el código.
