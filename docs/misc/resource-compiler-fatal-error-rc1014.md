# Error irrecuperable del compilador de recursos RC1014
hay demasiados archivos de inclusión: nivel \= 'nivel'  
  
 El número de niveles de anidamiento de las directivas `#include` era demasiado grande.  
  
 Use directivas anidadas para incluir archivos abiertos. El archivo de origen que contiene la directiva se cuenta como uno de los archivos.
