# Error de MSBuild MSB4136
**MSB4136: Error al leer la información del conjunto de herramientas del archivo de configuración.**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] recibió un error al intentar leer la información del conjunto de herramientas en el archivo de configuración de [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] \(msbuild.exe.config\).  
  
### Para corregir este error  
  
-   Asegúrese de que el archivo de configuración es correcto y está bien formado.  Por ejemplo, si ha personalizado el archivo .config agregando un conjunto de herramientas, compruebe la definición del conjunto de herramientas.
