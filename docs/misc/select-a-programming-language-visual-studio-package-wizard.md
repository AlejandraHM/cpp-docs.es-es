# Seleccione un lenguaje de programaci&#243;n, Asistente para paquetes de Visual Studio
Este paso le permite:  
  
-   Seleccionar el lenguaje de programación que desea usar para crear el paquete de VS.  
  
-   Generar o usar un archivo de clave existente para firmar el ensamblado del paquete de VS.  
  
 **Visual C\#**  
 Crea el paquete de VS en C\#.  
  
 **Visual Basic**  
 Crea el paquete de VS en Visual Basic.  
  
 **Visual C\+\+**  
 Crea el paquete de VS en Visual C\+\+.  
  
## Obtener claves para firmar el ensamblado  
 **Generar un nuevo archivo de clave para firmar el ensamblado**  
 Hace que el asistente genere un archivo de clave por usted. El archivo de clave generado se denomina key.snk.  
  
 **Usar el siguiente archivo de clave para firmar el ensamblado**  
 Permite escribir la ruta de acceso a un archivo de clave existente, o bien puede hacer clic en **Examinar** para obtener el archivo. El asistente realiza una copia del archivo y lo denomina key.snk.  
  
 **Cancelar**  
 Cancela la creación del paquete de VS.  
  
 **Anterior**  
 Se dirige a la página anterior del asistente.  
  
 **Siguiente**  
 Se dirige a la página siguiente del asistente.  
  
 **Finalizar**  
 Crea el paquete de VS.
