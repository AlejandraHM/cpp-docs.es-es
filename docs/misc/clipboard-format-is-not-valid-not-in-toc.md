# Clipboard format is not valid.
El Portapapeles sólo acepta datos en ciertos formatos especificados.  Este error ocurre normalmente cuando se intentan colocar datos de un componente en el Portapapeles, pero el formato de los datos es incompatible con éste.  
  
### Para corregir este error  
  
1.  Consulte la documentación del componente para determinar si se puede usar el Portapapeles para transferir datos desde el componente.  
  
     \-O bien\-  
  
2.  Seleccione otro elemento e inténtelo de nuevo.
