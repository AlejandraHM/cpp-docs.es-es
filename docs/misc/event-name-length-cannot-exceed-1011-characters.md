# La longitud del nombre del evento no puede superar los 1011 caracteres.
El nombre o identificador de cada evento declarado con una instrucción `Event` está limitado a 1011 caracteres.  
  
 **Identificador de error:** BC32204  
  
### Para corregir este error  
  
-   Reduzca la longitud del identificador.
