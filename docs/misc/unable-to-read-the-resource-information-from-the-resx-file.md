# Unable to read the resource information from the resx file
El sistema del proyecto no puede leer un archivo .resx.  
  
 Al final de este mensaje se adjunta una causa específica del sistema operativo \(si existe\).  El nombre del archivo que no se ha podido leer también se adjuntará.  
  
 Entre las causas de este error se incluyen la falta de permiso para la lectura de los archivos o la pérdida de comunicación con el servidor Web para proyectos web.  
  
 Esto provocará un error en el proceso de compilación.
