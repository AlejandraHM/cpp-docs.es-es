# Wizard file &lt;name&gt; is corrupted.
El archivo .vsz del asistente no tiene el formato correcto.  Este error se produce cuando un archivo .vsz está dañado o cuando el archivo proviene de un producto ajeno a Visual Studio.  
  
### Para corregir este error  
  
1.  Compruebe que el asistente sea un asistente para Visual Studio.  
  
     O bien  
  
     Vuelva a instalar Visual Studio para reemplazar el archivo dañado.
