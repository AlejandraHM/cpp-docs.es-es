# No se puede cambiar por un nombre con una unidad diferente
La instrucción `Rename` no puede especificar una letra de unidad en la ruta de acceso de archivo.  
  
### Para corregir este error  
  
-   Quite la letra de unidad de la ruta de acceso de archivo.
