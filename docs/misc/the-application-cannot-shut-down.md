# The application cannot shut down.
Este mensaje de error se produce normalmente cuando un programa externo, como el Administrador de tareas, intenta cerrar Visual Studio y todavía hay un cuadro de diálogo modal abierto.  
  
### Para corregir este error  
  
1.  Cierre el cuadro de diálogo modal e inténtelo de nuevo.
