# Error de MSBuild MSB3159
**MSB3159: Error de validación XML en el archivo '\<archivo\>': '\< error\>'**  
  
 Se genera este error cuando el producto o el archivo package.xml no es conforme al esquema XML.  La última parte de este error es un mensaje de error concreto.  
  
### Para corregir este error  
  
-   Actualice el archivo XML para resolver el error de validación.
