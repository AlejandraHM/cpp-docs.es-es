# Soluci&#243;n de problemas de excepciones: System.Windows.Controls.PrintDialogException
Se produce la excepción <xref:System.Windows.Controls.PrintDialogException> cuando se presenta una condición de error durante la apertura, el acceso o el uso de <xref:System.Windows.Controls.PrintDialog>.
