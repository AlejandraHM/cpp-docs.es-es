# Soluci&#243;n de problemas de excepciones: System.Workflow.ComponentModel.Serialization.WorkflowMarkupSerializationException
Los métodos que realizan la serialización y deserialización producen una excepción <xref:System.Workflow.ComponentModel.Serialization.WorkflowMarkupSerializationException>. La excepción incluye información sobre las ubicaciones de línea y columna que hicieron que se produjera la excepción.
