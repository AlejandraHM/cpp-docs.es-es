# Soluci&#243;n de problemas de excepciones: System.Web.Caching.TableNotEnabledForNotificationException
Excepción que se produce cuando una clase <xref:System.Web.Caching.SqlCacheDependency> se utiliza en una tabla de base de datos no habilitada para las notificaciones de cambios.
