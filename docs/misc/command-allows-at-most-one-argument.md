# Command allows at most one argument.
Este error ocurre normalmente cuando se han especificado demasiados argumentos para un comando que no acepta argumentos o sólo acepta un argumento.  
  
### Para corregir este error  
  
1.  Compruebe cuál es la sintaxis correcta en la documentación del comando e inténtelo de nuevo.
