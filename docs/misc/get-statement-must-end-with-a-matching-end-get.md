# La instrucci&#243;n &#39;Get&#39; debe terminar con la instrucci&#243;n &#39;End Get&#39; correspondiente.
Los procedimientos de propiedad `Get` deben terminar con una instrucción `End Get`.  
  
 **Identificador de error:** BC30631  
  
### Para corregir este error  
  
-   Asegúrese de que el procedimiento de la propiedad `Get` finalice con una construcción `End Get`.
