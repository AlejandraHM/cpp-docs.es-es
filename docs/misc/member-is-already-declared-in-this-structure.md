# &#39;&lt;miembro&gt;&#39; ya est&#225; declarado en esta estructura.
Un miembro de la enumeración se declaró más de una vez en esta estructura.  
  
 **Identificador de error:** BC31421  
  
### Para corregir este error  
  
-   Elimine la declaración superflua.
