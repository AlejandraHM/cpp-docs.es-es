# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
El complemento seleccionado no se encuentra.  Suele ocurrir cuando al complemento seleccionado le falta un archivo .dll u otro componente necesario.  
  
### Para corregir este error  
  
1.  Vuelva a instalar el complemento.
