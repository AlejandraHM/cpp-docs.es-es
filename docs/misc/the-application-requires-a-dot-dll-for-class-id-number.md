# The application requires a .dll for class ID &lt;number&gt;.
Visual Studio no pudo encontrar el archivo DLL para la clase que especificó.  Este mensaje es un error del sistema operativo.  
  
### Para corregir este error  
  
1.  Vuelva a instalar Visual Studio o la aplicación afectada e inténtelo de nuevo.
