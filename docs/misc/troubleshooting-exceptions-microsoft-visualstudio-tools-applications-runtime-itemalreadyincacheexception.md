# Soluci&#243;n de problemas de excepciones: Microsoft.VisualStudio.Tools.Applications.Runtime.ItemAlreadyInCacheException
La excepción que se produce cuando se realiza un intento de agregar un elemento a la caché del documento, pero el elemento ya existe en la caché.
