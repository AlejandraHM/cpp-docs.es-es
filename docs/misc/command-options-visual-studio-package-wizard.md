# Opciones de comandos, Asistente para paquetes de Visual Studio
Este paso le permite definir el nombre y el identificador de comando de un comando que aparecerá en el menú **Herramientas** de Visual Studio.  
  
 **Nombre de comando**  
 El texto de este cuadro se mostrará como:  
  
-   Nombre del comando.  
  
-   La información sobre herramientas del botón de comando en la barra de herramientas.  
  
 **Identificador de comando**  
 -   El texto de este cuadro define el identificador de comando del comando. Este identificador es una constante que se utiliza en el código generado para hacer referencia a este elemento de menú.  
  
 **Cancelar**  
 -   Cancela y detiene el asistente.  
  
 **Anterior**  
 -   Va a la página anterior del asistente.  
  
 **Finalizar**  
 -   Crea el VSPackage.  
  
 **Siguiente**  
 -   Va a la página siguiente del asistente.
