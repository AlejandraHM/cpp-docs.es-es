# Error de inicializaci&#243;n
Se produjo un error al inicializar el programa. Puede deberse a la cancelación de una instalación mientras estaba en curso.  
  
 **Identificador de error:** BC2000  
  
### Para corregir este error  
  
-   Vuelva a intentar la instalación.
