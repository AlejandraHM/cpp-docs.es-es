# No se declar&#243; &#39;&lt;funci&#243;n&gt;&#39;
No se declaró '\<función\>'. La función se movió al espacio de nombres Microsoft.VisualBasic.  
  
 **Identificador de error:** BC30818  
  
### Para corregir este error  
  
-   Use la función adecuada del espacio de nombres `Microsoft.VisualBasic`.
