# Error de MSBuild MSB3183
**MSB3183: La referencia '\<ensamblado\>' es un ensamblado de interoperabilidad que requiere plena confianza.**  
  
 Se genera esta advertencia cuando se encuentra un ensamblado de interoperabilidad entre las referencias al generar el manifiesto para una aplicación de confianza parcial.
