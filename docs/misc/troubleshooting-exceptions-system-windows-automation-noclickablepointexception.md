# Soluci&#243;n de problemas de excepciones: System.Windows.Automation.NoClickablePointException
Una excepción <xref:System.Windows.Automation.NoClickablePointException> contiene información sobre la excepción que se produce cuando se llama a <xref:System.Windows.Automation.AutomationElement.GetClickablePoint%2A> en un elemento <xref:System.Windows.Automation.AutomationElement> de interfaz de usuario que no tiene ningún punto donde hacer clic.  
  
## Comentarios  
 Esta excepción podría producirse cuando se minimiza la ventana de la aplicación o el elemento está fuera de la pantalla.
