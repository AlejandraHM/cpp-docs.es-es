# No se puede abrir el archivo de m&#243;dulo &#39;&lt;nombreDeArchivo&gt;&#39;: &lt;error&gt;
No se pudo abrir un archivo que contiene un módulo de este proyecto.  
  
 **Identificador de error:** BC31007  
  
### Para corregir este error  
  
1.  Compruebe que el archivo esté en la ubicación especificada y no esté bloqueado por otra aplicación.
