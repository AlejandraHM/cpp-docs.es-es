# Error de MSBuild MSB8020
Las herramientas de compilación de *\< plataforma \>* \(conjunto de herramientas de la plataforma \= '*\<versión\>*'\) no se encuentra. Para compilar mediante las herramientas de compilación *\<versión\>*, instale las herramientas de compilación de *\<plataforma\>*.  Como alternativa, puede actualizar a las herramientas de Visual Studio actuales si selecciona el menú Proyecto o hace clic con el botón derecho en la solución y luego selecciona "Redestinar solución".
