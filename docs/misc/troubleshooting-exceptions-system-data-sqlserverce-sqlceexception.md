# Soluci&#243;n de problemas de excepciones: System.Data.SqlServerCe.SqlCeException
Se produce una excepción `System.Data.SqlServerCe.SqlCeException` cuando el proveedor subyacente devuelve una advertencia o error de un origen de datos de SQL Server Compact. Para obtener más información sobre la causa de este error, vea [Errores de SQL Server Compact Edition](http://msdn2.microsoft.com/library/ms171849.aspx) en el sitio web de MSDN.  
  
## Comentarios  
 Un objeto `System.Data.SqlServerCe.SqlCeException` siempre contiene al menos una instancia de `System.Data.SqlServerCe.SqlCeError`. La clase no puede heredarse.  
  
 Para obtener más información, vea [Biblioteca de clases de SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
