# Error de carga del proyecto (Cuadro de di&#225;logo)
Este cuadro de diálogo aparece si no se puede cargar un archivo de proyecto.  
  
 **Corrija el error especificado y vuelva a cargar el proyecto.**  
 Proporciona información sobre el error.  Utilice la información de este cuadro de diálogo para buscar y resolver el error en el archivo de proyecto.  Guarde el proyecto y, a continuación, recárguelo en el IDE.  
  
 **No volver a mostrar este cuadro de diálogo en esta sesión**  
 Cuando se selecciona esta opción, el cuadro de diálogo **Error de carga del proyecto** no aparecerá hasta la próxima vez que se abra el IDE.
