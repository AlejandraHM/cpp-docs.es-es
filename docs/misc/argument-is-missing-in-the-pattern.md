# Argument is missing in the pattern.
Este error se produce normalmente cuando se ha especificado una sintaxis incompleta para una expresión regular en la cadena de búsqueda.  
  
### Para corregir este error  
  
1.  Vuelva a escribir la cadena.  Para obtener más información acerca de la sintaxis, vea [Expresiones regulares](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
