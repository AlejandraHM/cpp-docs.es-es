# Soluci&#243;n de problemas de excepciones: System.MulticastNotSupportedException
La excepción que se produce cuando se intenta combinar dos instancias de un tipo de delegado no combinable. No se produce una excepción si uno de los operandos es una referencia nula \(`Nothing` en Visual Basic\).
