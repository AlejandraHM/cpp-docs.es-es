# Soluci&#243;n de problemas de excepciones: System.Workflow.Activities.WorkflowAuthorizationException
Se produce una excepción <xref:System.Workflow.Activities.WorkflowAuthorizationException> cuando no se puede realizar la validación de roles debido a una identidad especificada no contenida en <xref:System.Workflow.Activities.WorkflowRoleCollection>.
