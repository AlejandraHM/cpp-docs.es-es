# No se admite Get (propiedad de solo escritura)
Una propiedad no se puede leer porque es de solo escritura.  
  
### Para corregir este error  
  
1.  Compruebe la propiedad y determine en qué condiciones se puede establecer.  
  
2.  Quite la referencia a la propiedad.
