# Seleccione m&#225;s de una configuraci&#243;n. Si quiere realizar cambios en una sola configuraci&#243;n, cancele el cuadro de di&#225;logo Varias configuraciones y seleccione la que quiera en la lista desplegable de configuraciones.
Este error normalmente se produce cuando se ha seleccionado una sola configuración de solución en el cuadro de diálogo **Varias configuraciones**.  
  
### Para corregir este error  
  
1.  Seleccione más configuraciones de solución.  
  
     \-O bien\-  
  
2.  Si quiere actualizar una única configuración, elija Cancelar y seleccione la configuración en cuestión en la lista de configuraciones del cuadro de diálogo **Propiedades de la solución**.
