# Soluci&#243;n de problemas de excepciones: Microsoft.Office.Tools.ControlNameAlreadyExistsException
Se produce una excepción <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> cuando se intenta crear mediante programación un control con el mismo nombre que otro control del proyecto.  
  
## Sugerencias asociadas  
 **Utilice otro nombre para uno de los controles.**  
 Los nombres de los controles deben ser únicos.
