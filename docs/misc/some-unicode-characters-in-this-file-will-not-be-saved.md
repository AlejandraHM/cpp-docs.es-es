# Some Unicode characters in this file will not be saved.
Este error ocurre generalmente cuando un archivo contiene caracteres que sólo pueden guardarse mediante la aplicación de una página de códigos Unicode.  
  
### Para guardar el archivo con caracteres Unicode  
  
1.  Haga clic en **Cancelar**.  
  
2.  En el menú **Archivo**, elija **Opciones avanzadas para guardar**.  
  
3.  En el cuadro de diálogo **Opciones avanzadas para guardar**, elija la codificación adecuada para el archivo.
