# Miscellaneous Files projects can contain &lt;number&gt; items.
El proyecto de archivos varios puede guardar hasta 256 elementos.  Este error ocurre normalmente cuando se ha escrito un número menor que 1 o mayor que 256 en el cuadro **Los proyectos de archivos varios guardan los últimos \<número\> elementos**, en el panel Documentos de las opciones de Entorno, del cuadro de diálogo Opciones.  
  
### Para corregir este error  
  
1.  Escriba un número entre uno y 256 en el cuadro.
