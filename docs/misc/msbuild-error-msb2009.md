# Error de MSBuild MSB2009
**El atributo '\< nombre de atributo\>' del elemento '\< nombre de elemento\>' no es válido.**  
  
 El nombre de atributo no se ha escrito correctamente o [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] no lo reconoce.  
  
### Para corregir este error  
  
-   Compruebe si se ha escrito correctamente el nombre de atributo.  
  
-   Compruebe si el archivo de proyecto ha sido modificado o dañado.  Si se ha modificado o dañado, abra el proyecto en la versión de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] en que se creó, guárdelo y, a continuación, intente convertirlo de nuevo.
