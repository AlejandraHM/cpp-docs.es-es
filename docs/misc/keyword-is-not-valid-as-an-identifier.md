# No es v&#225;lido el uso de la palabra clave como identificador
Una palabra clave aparece donde se requiere un nombre de elemento. El contexto llama a un elemento, como una variable o un procedimiento.  
  
 **Id. de error:** BC30183  
  
### Para corregir este error  
  
-   Reemplace la palabra clave por un nombre de elemento válido.
