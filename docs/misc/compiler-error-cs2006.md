# Error del compilador CS2006
Error de sintaxis de línea de comandos: falta 'texto' para la opción 'opción'  
  
 La sintaxis de *option* requiere texto adicional. Para obtener información, consulte [Opciones del compilador](../Topic/C%23%20Compiler%20Options.md).
