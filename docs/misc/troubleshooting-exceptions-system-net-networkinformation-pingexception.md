# Soluci&#243;n de problemas de excepciones: System.Net.NetworkInformation.PingException
Excepción que se produce cuando un método <xref:System.Net.NetworkInformation.Ping.Send%2A> o <xref:System.Net.NetworkInformation.Ping.SendAsync%2A> llama a un método que produce una excepción. Por ejemplo, un servidor puede no admitir el ping, o un firewall puede estar bloqueando la operación.
