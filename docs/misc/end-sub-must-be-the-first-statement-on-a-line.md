# &#39;End Sub&#39; debe ser la primera instrucci&#243;n de una l&#237;nea
Una instrucción `End Sub` sigue a un separador de instrucciones de dos puntos \(:\).`End Sub` debe ser la única instrucción en su línea de código fuente.  
  
 **Identificador de error:** BC32031  
  
### Para corregir este error  
  
-   Separe las distintas instrucciones en líneas diferentes.
