# Soluci&#243;n de problemas de excepciones: System.ServiceModel.PoisonMessageException
Se produce una excepción <xref:System.ServiceModel.PoisonMessageException> cuando se considera que un mensaje es dudoso. Un mensaje se considera dudoso si los repetidos intentos de enviarlo producen errores.
