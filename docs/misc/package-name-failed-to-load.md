# Package &lt;name&gt; failed to load.
No se pudo abrir el paquete.  Este error ocurre normalmente cuando no se ha registrado correctamente el paquete, a éste le falta una DLL, la DLL está dañada o falta una DLL que el paquete requiere de otro componente.  
  
### Para corregir este error  
  
1.  Vuelva a instalar el producto que proporciona el paquete.
