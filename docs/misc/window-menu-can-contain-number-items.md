# Window menu can contain &lt;number&gt; items.
La lista de ventanas del menú Ventana sólo puede mostrar hasta 24 nombres de ventana.  Este error se produce normalmente cuando se ha escrito un número menor que 1 o mayor que 24 en el cuadro **\<número\> elementos mostrados en el menú Ventana** en el panel General de las opciones de Entorno, en el cuadro de diálogo Opciones.  
  
### Para corregir este error  
  
1.  Escriba un número entre 1 y 24 en el cuadro.
