# No se puede eliminar el archivo &#39;&lt;filename&gt;&#39; porque est&#225; abierto.
Se ha intentado una eliminación en un archivo abierto.  
  
### Para corregir este error  
  
-   Cierre el archivo antes de eliminarlo.
