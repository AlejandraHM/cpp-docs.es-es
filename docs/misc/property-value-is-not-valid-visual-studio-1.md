# Property value is not valid. (Visual Studio) (1)
Este error ocurre normalmente cuando se escribe un valor no válido para la propiedad.  Por ejemplo, si la propiedad es Booleana con valores válidos de TRUE y FALSE, el valor MAYBE sería no válido para esta propiedad.  
  
### Para corregir este error  
  
1.  Escriba un valor que sea válido para dicha propiedad.  
  
    > [!TIP]
    >  Busque en la documentación en pantalla el nombre de la propiedad para comprobar qué valores son válidos.
