# Soluci&#243;n de problemas de excepciones: System.Net.NetworkInformation.NetworkInformationException
Excepción que se produce cuando se genera un error mientras se recupera la información de red.  
  
## Comentarios  
 Los tipos del espacio de nombres <xref:System.Net.NetworkInformation> producen esta excepción cuando falla una llamada a una función de Win32. La propiedad <xref:System.Net.NetworkInformation.NetworkInformationException.ErrorCode%2A> contiene el resultado devuelto por la función.
