# Error de MSBuild MSB1022
**El archivo de respuesta no existe.**  
  
 El archivo de respuesta no existe en la ubicación especificada.  
  
### Para corregir este error  
  
1.  Si ha especificado la ubicación absoluta o relativa del archivo de respuesta, compruebe que dicho archivo se encuentra en el directorio especificado.
