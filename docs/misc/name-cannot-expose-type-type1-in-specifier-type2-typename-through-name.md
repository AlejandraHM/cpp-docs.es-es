# &#39;&lt;name&gt;&#39; no puede exponer el tipo &lt;type1&gt; en &lt;specifier&gt; &lt;type2&gt; &#39;&lt;typename&gt;&#39; a trav&#233;s de &lt;name&gt;
Ha intentado exponer un tipo privado fuera de la clase pública `MyClass`.  
  
 **Id. de error:** BC30508  
  
### Para corregir este error  
  
-   Declare el tipo `Public`.
