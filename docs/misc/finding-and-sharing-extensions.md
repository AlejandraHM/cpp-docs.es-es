# Buscar y compartir extensiones
Puede descargar, instalar y administrar extensiones de Visual Studio mediante **Extensiones y actualizaciones**, que está disponible en el menú **Herramientas**. También puede obtener extensiones mediante la visita a sitios web, como la [Galería de Visual Studio](http://go.microsoft.com/fwlink/?LinkID=123847), o bien obtenerlas directamente de otros desarrolladores. Para más información acerca de cómo buscar e instalar las extensiones, consulte [Buscar y usar extensiones de Visual Studio](../Topic/Finding%20and%20Using%20Visual%20Studio%20Extensions.md).  
  
 Para crear extensiones, ajustarlas en paquetes de implementación y compartirlas con otros desarrolladores, también puede publicarlas en Internet o en un recurso compartido de red, o bien enviarlas por correo electrónico. Si ajusta una extensión en un paquete de implementación VSIX y la carga en el sitio web de la Galería de Visual Studio, otros desarrolladores podrán usar **Extensiones y actualizaciones** para instalarla. Para más información sobre la implementación VSIX, vea [Envío de extensiones de Visual Studio](../Topic/Shipping%20Visual%20Studio%20Extensions.md).
