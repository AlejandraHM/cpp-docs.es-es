# Error del compilador CS0014
No se pudo encontrar el archivo necesario 'archivo'.  
  
 Un archivo que es necesario para el compilador no se encuentra en el sistema. Asegúrese de que la ruta de acceso es correcta. Si el archivo es un archivo del sistema [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], puede que necesite reparar la instalación de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] o volver a instalarlo completamente.
