# piecewise_constant_distribution::densities
Devuelve las densidades almacenadas para la distribución.  
  
## Sintaxis  
  
```  
std::vector<double> densities() const;  
```  
  
## Comentarios  
 La función miembro devuelve las densidades almacenadas, calculadas a partir de los parámetros de peso.  
  
## Requisitos  
 **Encabezado:** \<random\>  
  
 **Espacio de nombres:** std
