# Soluci&#243;n de problemas de excepciones: System.ServiceModel.ActionNotSupportedException
Normalmente, se produce una excepción <xref:System.ServiceModel.ActionNotSupportedException> en el cliente cuando la acción relacionada con la operación invocada no coincide con ninguna acción de operaciones en el servidor.
