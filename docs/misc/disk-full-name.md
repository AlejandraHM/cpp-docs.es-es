# Disk full: &lt;name&gt;
La unidad que contiene el programa no contiene espacio suficiente para guardar los archivos de programa, como toolbox.tbd cuando se cierra éste.  
  
### Para corregir este error  
  
1.  Quite los archivos de la unidad y después intente cerrar de nuevo el programa.
