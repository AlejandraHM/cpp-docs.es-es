# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Dispatcher.MessageFilterException
Una excepción <xref:System.ServiceModel.Dispatcher.MessageFilterException> es el error de clase base de las excepciones que se producen cuando se supera la cuota de nodos inspeccionados por un filtro.
