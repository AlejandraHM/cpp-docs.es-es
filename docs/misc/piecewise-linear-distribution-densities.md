# piecewise_linear_distribution::densities
Devuelve las densidades almacenadas para la distribución.  
  
## Sintaxis  
  
```  
std::vector<double> densities() const;  
```  
  
## Comentarios  
 La función miembro devuelve los valores de densidad almacenados.  
  
## Requisitos  
 **Encabezado:** \<random\>  
  
 **Espacio de nombres:** std
