# Error de MSBuild MSB3041
**No se puede crear un nombre de recurso de manifiesto para "'\<archivo\>'".**  
  
 La tarea `CreateManifestResourceName` no puede crear un nombre de recurso de manifiesto para el archivo especificado.  
  
### Para corregir este error  
  
-   Para obtener más información sobre la causa de un error de MSBuild y las posibles soluciones, vea [Recursos adicionales](../Topic/Additional%20MSBuild%20Resources.md).
