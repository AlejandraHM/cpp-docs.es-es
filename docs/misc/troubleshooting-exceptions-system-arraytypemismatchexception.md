# Soluci&#243;n de problemas de excepciones: System.ArrayTypeMismatchException
Si el sistema no puede convertir el elemento al tipo declarado en la matriz, se produce una excepción <xref:System.ArrayTypeMismatchException>.  
  
## Sugerencias asociadas  
 **Asegúrese de que se puede convertir el tipo de objeto al tipo de matriz.**  
 Entre los dos tipos debe ser posible la conversión.  
  
 Para obtener más información, los usuarios de Visual Basic pueden consultar [Matrices](../Topic/Arrays%20in%20Visual%20Basic.md).
