# Error de MSBuild MSB3160
**MSB3160: Advertencia de validación XML en el archivo '\<archivo\>': '\< error\>'**  
  
 Se genera esta advertencia cuando el producto o el archivo package.xml no es conforme al esquema XML.  La última parte de esta advertencia es un mensaje de advertencia concreto.  
  
### Para corregir este error  
  
-   Actualice el archivo XML para resolver el error de validación.
