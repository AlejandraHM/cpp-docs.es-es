# Errores y advertencias de Editar y continuar (Visual Basic)
La característica Editar y continuar de [!INCLUDE[vb_current_short](../misc/includes/vb_current_short_md.md)] permite detener la ejecución del programa en el modo de interrupción, realizar cambios en el código en ejecución y, a continuación, reanudar la ejecución del programa con estos nuevos cambios incorporados.  
  
 Las modificaciones del código declarativo que afectan a la estructura pública de una clase suelen estar prohibidas, y no se permiten algunas modificaciones que se podrían hacer en un método, cuerpo de propiedad o declaraciones privadas en una clase. Siempre que es posible, Editar y continuar marca el código que no se puede editar en gris claro y muestra un mensaje de error. Para obtener más información acerca de las ediciones no compatibles en Editar y continuar para [!INCLUDE[vb_current_short](../misc/includes/vb_current_short_md.md)], consulte [Ediciones no compatibles en Editar y continuar de Visual Basic](../Topic/Unsupported%20Edits%20in%20Visual%20Basic%20Edit%20and%20Continue.md).  
  
 **Para corregir un error de Editar y continuar**  
  
1.  En el menú **Depurar**, elija **Deshacer** para deshacer el cambio.  
  
     O bien  
  
2.  Detenga la sesión de depuración, realice las tareas de edición e inicie una nueva sesión de depuración.  
  
 Si necesita más información acerca de un error o advertencia, puede buscar o publicar en MSDN [foro de IDE de Visual Basic](http://go.microsoft.com/fwlink/?LinkId=214679).
