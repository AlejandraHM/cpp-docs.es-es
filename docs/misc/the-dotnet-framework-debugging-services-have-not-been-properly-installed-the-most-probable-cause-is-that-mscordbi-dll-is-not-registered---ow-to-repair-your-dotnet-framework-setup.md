# Los servicios de depuraci&#243;n de .NET Framework no est&#225;n instalados correctamente. La causa m&#225;s probable es que el archivo mscordbi.dll no est&#233; registrado correctamente. Haga clic en Ayuda para obtener m&#225;s informaci&#243;n acerca de c&#243;mo reparar la instalaci&#243;n de .NET Framework.
Este error suele producirse cuando Visual Studio no puede encontrar el archivo mscordbi.dll, que se instala con los servicios de depuración de Common Language Runtime.  
  
### Para corregir este error  
  
-   Repare la instalación de .NET Framework en la máquina.
