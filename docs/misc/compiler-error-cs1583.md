# Error del compilador CS1583
'archivo' no es un archivo de recursos Win32 válido.  
  
 Este error se produce cuando se especifica un nombre de archivo con la opción de compilador [\/win32res](../Topic/-win32res%20\(C%23%20Compiler%20Options\).md) que no es un archivo de recursos Win32 válido o con el formato correcto. En Visual Studio, el nombre de archivo se especifica en el panel Aplicación del Diseñador de proyectos.
