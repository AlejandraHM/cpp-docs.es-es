# Soluci&#243;n de problemas de excepciones: System.ServiceModel.ChannelTerminatedException
Una excepción <xref:System.ServiceModel.ChannelTerminatedException> se produce normalmente en el cliente cuando se termina un canal debido a que el servidor cerró la conexión asociada.
