# Puntos suspensivos (...)
El operador de puntos suspensivos \(`...`\) tiene muchos usos en C y C\+\+, en función del contexto.  Se utilizaba originalmente en C como declarador abstracto para listas de parámetros de función variables y también se utiliza en programación de macros de preprocesador en forma de macros variádicas.  En C\+\+, `...` se utiliza en los bloques `catch` de control de excepciones y, en C\+\+11, se utiliza para las plantillas variádicas.
