# File &lt;name&gt; cannot be registered as an COM component.
El archivo no se puede agregar a las referencias del proyecto.  Este error ocurre normalmente cuando un archivo está dañado, no contiene bibliotecas de tipos, usa un tipo de archivo incompatible o el sistema no posee recursos suficientes para completar la operación.  
  
### Para corregir este error  
  
1.  Compruebe que el archivo contiene una biblioteca de tipos válida e inténtelo de nuevo.  
  
     \-O bien\-  
  
     Vuelva a instalar el producto que instaló el archivo.  
  
    > [!NOTE]
    >  Es posible que el producto que instaló el archivo sea ajeno a Visual Studio.
