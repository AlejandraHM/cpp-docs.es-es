# &#39;&lt;nombreDeTipo&gt;&#39; es un tipo de clase y no se puede usar como una expresi&#243;n.
Aparece un nombre de tipo cuando se necesita una expresión. Las expresiones deben constar de una combinación de variables, constantes, literales, propiedades y llamadas al procedimiento `Function`.  
  
 **Identificador de error:** BC30109  
  
### Para corregir este error  
  
-   Quite el nombre de tipo y construya la expresión con elementos válidos.
