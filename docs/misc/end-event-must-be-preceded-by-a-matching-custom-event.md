# La instrucci&#243;n &#39;End Event&#39; debe ir precedida de la instrucci&#243;n &#39;Custom Event&#39; correspondiente.
Una instrucción `End Event` aparece sin una instrucción `Custom Event` correspondiente.`End Event` debe ir precedida de una instrucción `Custom Event` correspondiente.  
  
 **Identificador de error:** BC31123  
  
### Para corregir este error  
  
-   Asegúrese de que una instrucción `Custom Event` anterior es válida y está escrita correctamente.
