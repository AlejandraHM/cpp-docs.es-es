# El argumento &#39;Access&#39; no es v&#225;lido. Los valores v&#225;lidos para el modo Output son &#39;OpenAccess.Write&#39; y &#39;OpenAccess.Default&#39;
Un valor de acceso no es válido en el modo Output.  
  
### Para corregir este error  
  
1.  Cambie el valor a `OpenAccess.Write` u `OpenAccess.Default`.
