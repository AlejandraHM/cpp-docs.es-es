# Switch &lt;name&gt; specified with conflicting values.
Este error aparece cuando escribe dos veces un modificador en la ventana Comandos y el valor especificado para el modificador es diferente en cada caso.  
  
### Para corregir este error  
  
-   Vuelva a especificar el comando, pero utilice sólo un modificador cada vez.
