# Soluci&#243;n de problemas de excepciones: System.ServiceModel.CommunicationObjectAbortedException
Se produce una excepción <xref:System.ServiceModel.CommunicationObjectAbortedException> cuando hay una llamada a un objeto <xref:System.ServiceModel.ICommunicationObject> que se ha anulado.
