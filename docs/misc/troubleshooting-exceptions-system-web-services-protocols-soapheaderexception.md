# Soluci&#243;n de problemas de excepciones: System.Web.Services.Protocols.SoapHeaderException
La excepción que se produce cuando se llama a un método de servicios Web XML sobre SOAP y se produce una excepción durante el proceso del encabezado de SOAP.
