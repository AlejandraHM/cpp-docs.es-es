# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException
Se produce una excepción <xref:System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException> cuando un filtro o tabla de filtros intenta obtener acceso al cuerpo de un mensaje no almacenado en búfer.
