# Caption cannot be empty.
Este error ocurre normalmente cuando no se ha escrito ningún nombre para la ficha del Cuadro de herramientas.  
  
### Para corregir este error  
  
1.  Escriba un nombre para la ficha del Cuadro de herramientas.
