# Soluci&#243;n de problemas de excepciones: System.Data.InvalidExpressionException
Se produce una excepción <xref:System.Data.InvalidExpressionException> cuando se intenta agregar una <xref:System.Data.DataColumn> que contiene una <xref:System.Data.DataColumn.Expression%2A> no válida en una <xref:System.Data.DataColumnCollection>.  
  
## Sugerencias asociadas  
 **Compruebe la sintaxis de la expresión.**  
 La propiedad <xref:System.Data.DataColumn.Expression%2A> se utiliza para calcular el valor de una columna o crear una columna agregada.
