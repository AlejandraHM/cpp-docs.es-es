# Error irrecuperable del compilador de recursos RC1207
La página de códigos predeterminada no es válida  
  
 La página de códigos especificada por la variable de entorno **RCCODEPAGE** no es válida. Para más información, vea [Páginas de códigos](../c-runtime-library/code-pages.md) en la *Referencia de la biblioteca en tiempo de ejecución*.
