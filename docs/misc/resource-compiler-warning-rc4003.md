# Advertencia del compilador de recursos RC4003
no hay suficientes parámetros reales para la macro 'identificador'  
  
 El número de parámetros reales especificado con el identificador dado era menor que el número de parámetros formales dado en la definición de macro del identificador.  
  
 Cuando se hace referencia a un parámetro formal en la definición y no se ha proporcionado el correspondiente parámetro real, el texto vacío se sustituye en la expansión de macro.
