# Soluci&#243;n de problemas de excepciones: System.Runtime.InteropServices.InvalidOleVariantTypeException
Es la excepción que produce el contador de referencias cuando encuentra un argumento de tipo variante cuyas referencias no se pueden calcular para un código administrado.
