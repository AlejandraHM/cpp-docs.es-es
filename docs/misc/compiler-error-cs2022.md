# Error del compilador CS2022
Las opciones '\/out' y '\/target' deben aparecer antes de los nombres de archivo de código fuente  
  
 Las opciones del compilador [\/out \(establecer el nombre de archivo de salida\)](../Topic/-out%20\(C%23%20Compiler%20Options\).md) y [\/target \(especificar el formato del archivo de salida\)](../Topic/-target%20\(C%23%20Compiler%20Options\).md), cuando se especifican en la línea de comandos, deben preceder a los archivos de código fuente.
