# Error de MSBuild MSB1001
**Modificador desconocido.**  
  
 Se ha especificado un modificador desconocido en la línea de comandos.  
  
### Para corregir este error  
  
1.  Compruebe si se ha escrito correctamente el modificador.  Para mostrar la lista de modificadores válidos, escriba `msbuild /help`.
