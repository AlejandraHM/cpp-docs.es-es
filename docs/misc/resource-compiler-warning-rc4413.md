# Advertencia del compilador de recursos RC4413
No se puede determinar el tipo de archivo para 'filename': se supone que es ASCII de 8 bits  
  
 El compilador de recursos no conoce el tipo de archivo especificado. Se ha compilado como un archivo ASCII de 8 bits.
