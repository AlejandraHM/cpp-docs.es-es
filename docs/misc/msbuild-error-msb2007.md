# Error de MSBuild MSB2007
**Visual Studio no puede encontrar el archivo de proyecto "{0}".**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] no encuentra el archivo de proyecto especificado.  
  
### Para corregir este error  
  
1.  Compruebe si el nombre de archivo del proyecto está correctamente escrito.
