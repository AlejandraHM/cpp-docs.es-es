# File &lt;name&gt; size is incorrect.
El archivo contiene menos bytes de lo esperado, posiblemente porque está truncado o dañado.  
  
### Para corregir este error  
  
1.  Cierre y reinicie Visual Studio.  
  
2.  Si sigue produciéndose el error, vuelva a instalar Visual Studio para reparar el archivo.
