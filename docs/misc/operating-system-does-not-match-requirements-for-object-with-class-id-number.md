# Operating system does not match requirements for object with class ID &lt;number&gt;.
Este error ocurre normalmente cuando se detecta una versión incompatible del sistema operativo o éste es incompatible.  
  
### Para corregir este error  
  
1.  Compruebe los requisitos del sistema para la aplicación.  
  
2.  Instale uno de los sistemas operativos compatibles e inténtelo de nuevo.
