# Error del compilador CS1021
La constante integral es demasiado extensa  
  
 Un valor asignado a un [tipo entero](../Topic/Integral%20Types%20Table%20\(C%23%20Reference\).md) es mayor que el mayor valor posible de entero sin signo.  
  
 El ejemplo siguiente genera la advertencia CS1021:  
  
```  
// CS1021.cs enum F : int { a=(int)2590000000000000000000   // CS1021 } public class clx { public static void Main() { } }  
```
