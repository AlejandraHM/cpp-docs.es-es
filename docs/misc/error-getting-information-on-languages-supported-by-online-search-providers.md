# Error getting information on languages supported by online search providers.
Este error suele producirse cuando se intenta seleccionar un idioma en la lista **Idioma del tema en línea** del [Help, Environment, Options Dialog Box](http://msdn.microsoft.com/es-es/a2a5dff5-70a5-4f14-bf4c-66793894de6a), pero no se puede tener acceso a los servicios en línea.  
  
### Para corregir este error  
  
1.  Compruebe que tiene una conexión a Internet activa.  
  
2.  Si utiliza un servidor proxy, compruebe que esté configurado correctamente.  
  
3.  Seleccione de nuevo la lista desplegable **Idioma del tema en línea**.
