# Soluci&#243;n de problemas de excepciones: System.ComponentModel.Win32Exception
La excepción que se produce para un código de error de Win32.  
  
## Comentarios  
 Los códigos de error de Win32 se traducen desde sus representaciones numéricas en un mensaje del sistema cuando se muestran. Utilice la propiedad <xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> para tener acceso a la representación numérica del código de error.
