# Error del compilador de recursos RC2176
**DIB antiguo en el nombre de archivo. Páselo a través de SDKPAINT.**  
  
 Se encontró un formato antiguo de mapa de bits independiente del dispositivo en el archivo especificado. El archivo se debería convertir al formato actual. Para ello, se puede usar la aplicación SDKPAINT proporcionada en el SDK de Windows 3.0, o una aplicación equivalente.
