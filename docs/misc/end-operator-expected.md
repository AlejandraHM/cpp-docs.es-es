# Se esperaba &#39;End Operator&#39;
Una instrucción `End` debe coincidir con el bloque que termina. Una definición de procedimiento `Operator` no se terminó correctamente.  
  
 **Identificador de error:** BC33005  
  
### Para corregir este error  
  
1.  Agregue una instrucción `End Operator` al final del procedimiento.
