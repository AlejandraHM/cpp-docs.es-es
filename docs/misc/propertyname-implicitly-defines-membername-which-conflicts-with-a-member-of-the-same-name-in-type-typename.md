# &#39;&lt;propertyname&gt;&#39; define impl&#237;citamente &#39;&lt;membername&gt;&#39;, que est&#225; en conflicto con un miembro con el mismo nombre en &lt;type&gt; &#39;&lt;typename&gt;&#39;
El nombre de un miembro de tipo está en conflicto con el nombre de un miembro creado implícitamente para una propiedad. Las propiedades crean implícitamente distintas variables implícitas que pueden estar en conflicto con los miembros definidos.  
  
 **Id. de error:** BC31060  
  
### Para corregir este error  
  
-   Cambie el nombre del miembro declarado explícitamente para quitar los conflictos de nombres.
