# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
No se pudo realizar una referencia entre proyectos a un proyecto determinado.  
  
 La referencia a un proyecto que compila un archivo .exe es la causa más común de este error.  Este error también indica que un proyecto hace referencia a un proyecto cuyos resultados no son un ensamblado \(como, por ejemplo, una aplicación C\+\+ nativa\).
