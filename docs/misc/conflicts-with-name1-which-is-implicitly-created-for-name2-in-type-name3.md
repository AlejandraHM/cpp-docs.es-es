# Conflictos con &#39;&lt;nombre1&gt;&#39;, que se crea impl&#237;citamente para &#39;&lt;nombre2&gt;&#39; en &lt;tipo&gt; &#39;&lt;nombre3&gt;&#39;.
El nombre de un miembro de tipo está en conflicto con otro miembro creado implícitamente.  
  
 **Identificador de error:** BC31058  
  
### Para corregir este error  
  
-   Cambie el nombre del miembro declarado explícitamente para eliminar los conflictos de nombres.
