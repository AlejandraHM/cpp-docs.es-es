# Error de MSBuild MSB3186
**MSB3186: No se puede generar la identidad de un ensamblado para el manifiesto generado desde los parámetros de entrada de la tarea.**  
  
 Este error se produce cuando el proceso de compilación no puede generar un nombre de ensamblado para el manifiesto de aplicación o de implementación.  El nombre de ensamblado no se proporciona explícitamente, no hay identidad en el manifiesto base y tampoco se especifica la identidad del punto de entrada.
