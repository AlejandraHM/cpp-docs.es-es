# Most recently used list can contain &lt;number&gt; items.
Las listas MRU pueden contener hasta veinticuatro elementos.  Este error ocurre normalmente cuando se ha escrito un número menor que 1 o mayor que 24 en el cuadro **Las listas de los usados más recientemente contienen \<número\> elementos** en el panel General de las opciones de Entorno, en el cuadro de diálogo Opciones.  
  
### Para corregir este error  
  
1.  Escriba un número entre 1 y 24 en el cuadro.
