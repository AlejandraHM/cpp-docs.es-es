# Error de MSBuild MSB4137
**MSB4137: Se ha especificado un valor no válido en el archivo de configuración en "\<x\>".  El nombre de la propiedad o el nombre de la versión de herramientas es una cadena vacía.**  
  
 Una definición de conjunto de herramientas contiene nombres o valores de propiedad no válidos.  
  
### Para corregir este error  
  
-   Asegúrese de que su definición de conjunto de herramientas no contiene nombres o valores de propiedad no válidos o vacíos.
