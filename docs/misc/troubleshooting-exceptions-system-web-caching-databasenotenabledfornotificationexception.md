# Soluci&#243;n de problemas de excepciones: System.Web.Caching.DatabaseNotEnabledForNotificationException
La excepción que se produce cuando una base de datos de Microsoft SQL Server no puede admitir dependencias asociadas a la clase <xref:System.Web.Caching.SqlCacheDependency>.
