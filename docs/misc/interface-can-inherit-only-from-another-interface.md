# Una interfaz solo puede heredar de otra interfaz.
Ha intentado heredar de un elemento que no es una interfaz.  
  
 **Identificador de error:** BC30354  
  
### Para corregir este error  
  
-   Asegúrese de que la interfaz base o las interfaces desde las que intenta heredar son realmente interfaces.
