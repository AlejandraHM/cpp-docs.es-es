# La secuencia pasada a TextFieldParser no se puede leer.
El [TextFieldParser \(Objeto\)](../Topic/TextFieldParser%20Object.md) no puede leer la secuencia que se le ha pasado. Puede ser como consecuencia de intentar leer un archivo que no es un archivo de texto.  
  
### Para corregir este error  
  
-   Compruebe que el archivo es un archivo de texto.
