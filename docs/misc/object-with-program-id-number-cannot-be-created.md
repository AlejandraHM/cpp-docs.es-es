# Object with program ID &lt;number&gt; cannot be created.
La cadena de clase que intenta usar no es válida.  Este mensaje es un error del sistema operativo.  
  
### Para corregir este error  
  
1.  Vuelva a instalar Visual Studio o la aplicación afectada e inténtelo de nuevo.
