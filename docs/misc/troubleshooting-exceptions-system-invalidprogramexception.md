# Soluci&#243;n de problemas de excepciones: System.InvalidProgramException
Excepción que se produce cuando un programa contiene lenguaje intermedio de Microsoft \(MSIL\) no válido o metadatos. Esto normalmente indica un error en el compilador que generó el programa.
