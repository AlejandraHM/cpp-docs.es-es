# Command was not specified.
Este error ocurre normalmente cuando se ha escrito una cadena vacía en la **Ventana de comandos** o en el cuadro **Buscar\/Comando**.  
  
### Para corregir este error  
  
1.  Especifique un comando válido.  Para obtener más información, vea [Comandos de Visual Studio](../Topic/Visual%20Studio%20Commands.md).
