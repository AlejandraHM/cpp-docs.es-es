# Error del compilador CS0584
Error interno del compilador: fase 'stage' símbolo 'symbol'  
  
 Intente determinar si el compilador está fallando porque no puede analizar una sintaxis inesperada. Si no es el caso, pruebe con [Hable con nosotros](../Topic/Talk%20to%20Us.md).
