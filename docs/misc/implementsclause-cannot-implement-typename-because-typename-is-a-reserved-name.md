# &#39;&lt;implementsclause&gt;&#39; no puede implementar &#39;&lt;typename&gt;&#39; porque &#39;&lt;typename&gt;&#39; es un nombre reservado
Una instrucción `Implements` especificó un nombre reservado.  
  
 **Id. de error:** BC31415  
  
### Para corregir este error  
  
-   Cambie el nombre del miembro de interfaz que no se puede implementar.
