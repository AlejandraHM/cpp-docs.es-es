# Error del compilador de recursos RC2014
El comando de preprocesador debe empezar con un primer espacio que no esté en blanco  
  
 Aparecen caracteres que no son de espacio en blanco antes del signo de número \(**\#**\) de una directiva de preprocesador en la misma línea.
