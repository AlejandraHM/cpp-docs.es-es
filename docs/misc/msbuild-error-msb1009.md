# Error de MSBuild MSB1009
**El archivo de proyecto no existe.**  
  
 Este archivo de proyecto no existe en el directorio especificado, o bien, si no se ha especificado ningún directorio, el archivo de proyecto no existe en el actual directorio de trabajo.  
  
### Para corregir este error  
  
1.  Compruebe si están bien escritos el nombre y la ruta de acceso del archivo.
