# Error al intentar abrir o crear el archivo de definici&#243;n de aplicaci&#243;n para este proyecto. &#39;&lt;error&gt;&#39;
Este error se produce al intentar abrir el archivo de definición de aplicación, pero éste no está disponible.  Es posible que se haya movido o eliminado.  
  
### Para corregir este error  
  
-   Vuelva a crear el archivo de definición de aplicación.
