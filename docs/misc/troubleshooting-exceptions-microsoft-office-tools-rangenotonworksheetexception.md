# Soluci&#243;n de problemas de excepciones: Microsoft.Office.Tools.RangeNotOnWorksheetException
Se produce una excepción <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException> cuando se crea mediante programación un control de vista con un intervalo que no esté en la hoja de cálculo correspondiente a la colección de controles.  
  
## Sugerencias asociadas  
 Asegúrese de que el intervalo especificado corresponde a la hoja de cálculo correcta.
