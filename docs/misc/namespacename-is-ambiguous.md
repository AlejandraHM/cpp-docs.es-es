# &#39;&lt;espacioDeNombres&gt;&#39; es ambiguo
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] no tiene reglas de resolución de conflictos. Es necesario eliminar la ambigüedad en los nombres de forma manual.  
  
 **Identificador de error:** BC30554  
  
### Para corregir este error  
  
-   Use el nombre completo.
