# &lt;name&gt; is not a valid switch for this command.
Este error ocurre normalmente cuando se ha especificado un modificador o combinación de modificadores de carácter individuales incorrectos.  
  
### Para corregir este error  
  
1.  Compruebe cuál es la sintaxis correcta en la documentación del comando e inténtelo de nuevo.
