# Cannot find main menu commandbar in resources.
Este error aparece cuando el entorno de desarrollo integrado no puede encontrar la barra de comandos del menú principal en el inicio.  Se ha eliminado o movido un archivo .dll, o está dañado.  
  
### Para corregir este error  
  
-   Utilice **Agregar o quitar programas** para reparar o reinstalar el producto.
