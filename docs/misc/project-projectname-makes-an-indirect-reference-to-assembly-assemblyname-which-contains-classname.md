# El proyecto &#39;&lt;nombreDeProyecto&gt;&#39; hace una referencia indirecta al ensamblado &#39;&lt;nombreDeEnsamblado&gt;&#39; que contiene &#39;&lt;nombreDeClase&gt;&#39;
El proyecto '\<nombreDeProyecto\>' hace una referencia indirecta al ensamblado '\<nombreDeEnsamblado\>' que contiene '\<nombreDeClase\>'. Agregue una referencia a '\<nombreDeEnsamblado\>' en el proyecto.  
  
 Este proyecto contiene miembros que dependen de un ensamblado que no está incluido en las referencias del proyecto.  
  
 **Identificador de error:** BC31515  
  
### Para corregir este error  
  
-   Haga clic con el botón derecho en **Referencias** en el **Explorador de soluciones** y, después, haga clic en **Agregar referencia** para agregar una referencia al ensamblado.
