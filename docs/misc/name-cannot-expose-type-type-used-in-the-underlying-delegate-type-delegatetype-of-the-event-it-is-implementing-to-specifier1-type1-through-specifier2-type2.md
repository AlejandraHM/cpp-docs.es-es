# &#39;&lt;name&gt;&#39; no puede exponer el tipo &#39;&lt;type&gt;&#39; utilizado en el tipo de delegado subyacente &#39;&lt;delegatetype&gt;&#39; del evento que implementa en &lt;specifier1&gt; &#39;&lt;type1&gt;&#39; mediante &lt;specifier2&gt; &#39;&lt;type2&gt;&#39;
El código expone un tipo privado fuera de la clase pública.  
  
 **Id. de error:** BC30923  
  
### Para corregir este error  
  
-   Declare el tipo `Public` o use otro tipo.
