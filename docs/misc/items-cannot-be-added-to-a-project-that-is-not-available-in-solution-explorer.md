# No se pueden agregar elementos a un proyecto que no est&#233; disponible en el Explorador de soluciones
Este error suele producirse cuando un proyecto especificado no está abierto en el Explorador de soluciones. Visual Studio no puede agregar elementos a un proyecto que no está disponible actualmente en el Explorador de soluciones.  
  
### Para corregir este error  
  
1.  Agregue el proyecto especificado a la solución actual o abra la solución que contiene el proyecto especificado.
