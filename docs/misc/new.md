# new
Describe brevemente los usos de la palabra clave `new` y proporciona enlaces a más información.  
  
|Título|Descripción|  
|------------|-----------------|  
|[operator new](../c-runtime-library/operator-new-crt.md)|Asigna un bloque de memoria del montón.|  
|[operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md)|La función llamada por una expresión nueva para asignar el almacenamiento para objetos individuales.|  
|[new \(Operador\)](../cpp/new-operator-cpp.md)|Asigna memoria para un objeto o una matriz de objetos del almacén libre y devuelve un puntero con tipo adecuado distinto de cero al objeto.|  
|[new \(nueva ranura en vtable\)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)|En un proyecto de C\+\+\/CLI, indica que un miembro virtual obtiene una nueva ranura en vtable.  Es decir, la función no reemplaza un método de clase base.|  
|[ref new, gcnew](../windows/ref-new-gcnew-cpp-component-extensions.md)|En un proyecto de [!INCLUDE[cppwrt_short](../build/reference/includes/cppwrt_short_md.md)] \(`ref new`\) o de C\+\+\/CLI \(`gcnew`\), esta palabra clave compuesta crea un objeto que se cuenta por referencia.|
