# Los caracteres de tipo no est&#225;n permitidos en los identificadores de etiqueta
Una etiqueta de instrucción contiene al menos un carácter de tipo de identificador.  
  
 Una etiqueta de instrucción debe ser un nombre de Visual Basic válido. Los caracteres permitidos no incluyen ninguno de los caracteres de tipo de identificador \(`%`, `&`, `@`, `!`, `#` y `$`\). Vea [Nombres de elementos declarados](../Topic/Declared%20Element%20Names%20\(Visual%20Basic\).md).  
  
 **Identificador de error:** BC31395  
  
### Para corregir este error  
  
-   Quite el carácter o los caracteres de tipo identificador de la etiqueta de la instrucción.
