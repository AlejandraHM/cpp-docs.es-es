# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
La subsección \(subsection\) es Include y la sección \(section\) es Files.  
  
 Es muy probable que el error esté causado por la edición manual del archivo de proyecto.  
  
 **Para corregir este error**  
  
-   Trabaje en modo **Mostrar todos los archivos** \(haga clic en ese botón en la barra de herramientas del Explorador de soluciones\) e incluya los archivos relevantes.  
  
     Si aparece este error, el proyecto se abrirá pero no se agregarán archivos.
