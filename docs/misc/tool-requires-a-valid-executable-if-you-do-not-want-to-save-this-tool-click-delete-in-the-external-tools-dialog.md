# La herramienta requiere un ejecutable v&#225;lido. Si no quiere guardar esta herramienta, haga clic en Eliminar en el cuadro de di&#225;logo Herramientas externas.
Este error se produce normalmente cuando se ha agregado una herramienta al cuadro de diálogo Herramientas externas pero no se ha especificado ningún ejecutable.  
  
### Para corregir este error  
  
1.  Si tiene la intención de guardar la herramienta, haga clic en **Aceptar** y especifique un nombre de archivo en el cuadro **Comando** de la herramienta o busque un archivo mediante la elección del botón de puntos suspensivos.  
  
2.  Si no tiene la intención de guardar la herramienta, haga clic en **Aceptar** y, a continuación, haga clic en **Eliminar** en el cuadro de diálogo **Herramientas externas**.
