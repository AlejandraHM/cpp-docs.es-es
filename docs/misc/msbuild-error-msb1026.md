# Error de MSBuild MSB1026
**El archivo de esquema no existe.**  
  
 El archivo de esquema no existe en la ubicación especificada.  
  
### Para corregir este error  
  
1.  Si ha especificado la ubicación absoluta o relativa del archivo de esquema, compruebe que dicho archivo se encuentra en el directorio especificado.
