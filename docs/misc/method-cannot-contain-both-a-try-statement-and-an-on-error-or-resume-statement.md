# El m&#233;todo no puede contener una instrucci&#243;n &#39;Try&#39; y una instrucci&#243;n &#39;On Error&#39; o &#39;Resume&#39;
Combinó una instrucción `Try` con `On Error` o `Resume`.  
  
 **Identificador de error:** BC30544  
  
### Para corregir este error  
  
1.  Quite `On Error` o `Resume`.
