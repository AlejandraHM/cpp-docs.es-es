# Failed to generate resource files
Se ha producido un error general del sistema del proyecto al leer un archivo .RESX como parte del proceso de creación de archivos binarios .RESOURCES comprensibles para el Administrador de recursos .NET.  
  
 Normalmente, este error está provocado por un archivo .RESX erróneo.  
  
 Esta advertencia no impedirá que se lleve a cabo la compilación.  No obstante, el archivo .RESX por el que se muestra este error no se incluirá en el proyecto como un recurso de manifiesto del ensamblado.
