# Error irrecuperable del compilador de recursos RC1023
no se puede abrir el archivo de origen 'filename'  
  
 El archivo especificado no existe, no se pudo abrir o no se encontró.  
  
 Asegúrese de que la configuración del entorno es válida y de que se especifica la ruta de acceso correcta para el archivo.  
  
 Si este error aparece sin un mensaje de error, el compilador de recursos se ha quedado sin identificadores de archivo.
