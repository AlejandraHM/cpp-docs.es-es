# Component is not correctly registered.
Este error ocurre normalmente cuando el Registro ha quedado dañado, el sistema no puede tener acceso normal al Registro o se escribió una entrada en éste con formato incorrecto.  
  
### Para corregir este error  
  
1.  Vuelva a instalar el componente.
