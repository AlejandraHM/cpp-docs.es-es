# Soluci&#243;n de problemas de excepciones: System.Windows.Media.Animation.AnimationException
Se produce una excepción <xref:System.Windows.Media.Animation.AnimationException> si se produce un error al animar una propiedad.  
  
## Comentarios  
 Esta clase no se utiliza para errores relacionados con el control del tiempo, como crear una animación con un valor <xref:System.Windows.Media.Animation.Timeline.Duration%2A> negativo o una configuración de <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> y <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> no válida.
