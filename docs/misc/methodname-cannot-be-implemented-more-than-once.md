# &#39;nombreM&#233;todo&#39; no se puede implementar m&#225;s de una vez
Intentó implementar un método que ya está implementado.  
  
 **Identificador de error:** BC30583  
  
### Para corregir este error  
  
1.  Compruebe el nombre del método para asegurarse de que está escrito correctamente.  
  
2.  Si el método es redundante, quite la declaración adicional.
