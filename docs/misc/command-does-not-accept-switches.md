# Command does not accept switches.
Este error ocurre normalmente cuando se ha proporcionado un modificador para un comando que sólo reconoce argumentos.  
  
### Para corregir este error  
  
1.  Revise los requisitos de sintaxis del comando.  Para obtener más información, vea [Comandos de Visual Studio con argumentos](../Topic/Visual%20Studio%20Commands.md).
