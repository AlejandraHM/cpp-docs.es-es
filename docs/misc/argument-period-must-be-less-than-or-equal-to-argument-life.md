# El argumento &#39;Period&#39; debe ser menor o igual que el argumento &#39;Life&#39;.
El valor del argumento `Period`, que especifica el período para el que se calcula la amortización del recurso, es superior al valor del argumento `Life`.  
  
### Para corregir este error  
  
-   Asegúrese de que los argumentos `Life` y `Period` se expresan en las mismas unidades. Por ejemplo, si `Life` se mide en meses, `Period` también debe medirse así.
