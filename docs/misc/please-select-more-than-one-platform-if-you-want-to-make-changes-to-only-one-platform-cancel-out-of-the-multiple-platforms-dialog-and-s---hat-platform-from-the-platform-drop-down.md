# Seleccione m&#225;s de una plataforma. Si quiere realizar cambios en una sola plataforma, cancele el cuadro de di&#225;logo Varias plataformas y seleccione la que quiere en la lista desplegable de plataformas.
Este error normalmente se produce cuando se ha seleccionado una sola plataforma en el cuadro de diálogo **Varias plataformas**.  
  
### Para corregir este error  
  
1.  Seleccione plataformas adicionales.  
  
     \-O bien\-  
  
2.  Si quiere actualizar una sola plataforma, seleccione Cancelar y luego seleccione la plataforma correcta en la lista de plataformas.
