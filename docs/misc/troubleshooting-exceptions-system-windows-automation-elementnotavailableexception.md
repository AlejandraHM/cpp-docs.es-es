# Soluci&#243;n de problemas de excepciones: System.Windows.Automation.ElementNotAvailableException
Una excepción <xref:System.Windows.Automation.ElementNotAvailableException> contiene información sobre la excepción que se produce al intentar obtener acceso a un elemento <xref:System.Windows.Automation.AutomationElement> de interfaz de usuario que corresponde a una parte de la interfaz de usuario que ya no está disponible.  
  
## Comentarios  
 Se puede producir esta excepción si el elemento está en un cuadro de diálogo que se ha cerrado o en una aplicación que se ha cerrado o ha finalizado.
