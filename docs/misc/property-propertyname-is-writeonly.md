# La propiedad &#39;&lt;nombreDePropiedad&#39; es &#39;WriteOnly&#39;
Intentó recuperar un valor de una propiedad declarada `WriteOnly`.  
  
 **Identificador de error:** BC30524  
  
### Para corregir este error  
  
-   Quite el especificador `WriteOnly` de declaración de propiedad.
