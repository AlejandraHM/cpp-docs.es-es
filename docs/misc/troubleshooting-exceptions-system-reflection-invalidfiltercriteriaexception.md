# Soluci&#243;n de problemas de excepciones: System.Reflection.InvalidFilterCriteriaException
Excepción que se produce en <xref:System.Type.FindMembers%2A> cuando el criterio de filtro no es válido para el tipo de filtro que se está utilizando.
