# Soluci&#243;n de problemas de excepciones: System.Printing.PrintSystemException
Se produce una excepción <xref:System.Printing.PrintSystemException> cuando una condición de error evita el acceso o la creación de <xref:System.Printing.PrintSystemObject>.
