# &#39;&lt;name&gt;&#39; ya se declar&#243; como par&#225;metro de este m&#233;todo.
El nombre de parámetro especificado ya se ha declarado para este método.  
  
 **Identificador de error:** BC30734  
  
### Para corregir este error  
  
-   Cambie el nombre del parámetro para que sea único.
