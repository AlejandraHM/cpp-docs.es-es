# &#39;&lt;typename&gt;&#39; es un tipo de &#39;&lt;containername&gt;&#39; y no se puede usar como una expresi&#243;n.
Uno de los nombres que se va a usar ya se ha declarado como el nombre de un tipo y no se puede usar en una expresión.  
  
 **Identificador de error:** BC30691  
  
### Para corregir este error  
  
-   Nombre único del miembro.  
  
-   Usar corchetes alrededor del nombre del miembro.
