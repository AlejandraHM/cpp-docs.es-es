# CA0503
La propiedad está desusada.  Utilice la propiedad que la sustituye.  
  
 La advertencia CA0503 se genera cuando una propiedad de análisis de código desusada contiene un valor en el archivo de configuración de compilación o proyecto.  Para solucionar esta advertencia, utilice la propiedad especificada que la sustituye.
