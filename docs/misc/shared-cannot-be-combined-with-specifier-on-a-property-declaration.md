# &#39;Shared&#39; no se puede combinar con &#39;&lt;especificador&gt;&#39; en una declaraci&#243;n de propiedad.
Ha intentado combinar el modificador `Shared` con un especificador que no es válido en una combinación de este tipo en una declaración de propiedad.  
  
 **Identificador de error:** BC30502  
  
### Para corregir este error  
  
-   Quite el especificador.
