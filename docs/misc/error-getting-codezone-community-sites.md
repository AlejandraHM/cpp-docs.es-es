# Error getting Codezone Community sites.
Este error aparece cuando no se puede encontrar una conexión al servicio Web en línea.  Los servicios Web llenan información en la página **En línea**, bajo **Ayuda**, **Entorno**, del cuadro de diálogo **Opciones**.  
  
### Para corregir este error  
  
1.  Compruebe que tiene una conexión a Internet activa.  
  
     \-O bien\-  
  
     Si se conecta a Internet a través de un servidor proxy, compruebe la configuración del servidor proxy.  
  
2.  Intente de nuevo tener acceso a la página En línea.
