# The application cannot run.
No hay memoria suficiente para que se ejecute Visual Studio.  
  
### Para corregir este error  
  
1.  Compruebe que el equipo cumple los requisitos mínimos para ejecutar Visual Studio.  
  
2.  Cierre los demás programas en ejecución en el equipo para liberar memoria.  
  
     \-O bien\-  
  
     Vuelva a instalar Visual Studio.
