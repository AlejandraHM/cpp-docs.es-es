# Soluci&#243;n de problemas de excepciones: System.IdentityModel.Selectors.CardSpaceException
Se produce una excepción <xref:System.IdentityModel.Selectors.CardSpaceException> cuando se han producido una o más excepciones en el nivel del servicio CardSpace. La causa del error está en el registro de eventos.  
  
## Comentarios  
 Entre las condiciones que causan esta excepción se incluye la aparición de un error al cargar un almacén o libro mayor.
