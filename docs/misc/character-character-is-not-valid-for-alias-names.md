# Character &lt;character&gt; is not valid for alias names.
Este error ocurre normalmente cuando se escribe un carácter incorrecto como parte de un nombre de alias.  Los nombres de alias no pueden contener espacios en blanco, \=, \[, \], \/, \- o ".  
  
### Para corregir este error  
  
1.  Vuelva a escribir el nombre de alias, omitiendo el carácter incorrecto.
