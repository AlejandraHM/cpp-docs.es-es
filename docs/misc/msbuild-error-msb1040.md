# Error de MSBuild MSB1040
**MSB1040: El valor de ToolsVersion no es válido.**  
  
 El valor de `ToolsVersion` especificado con el modificador **\/ToolsVersion** no es válido.  
  
### Para corregir este error  
  
-   Especifique un valor de `ToolsVersion` válido al compilar utilizando el modificador **\/ToolsVersion** en la línea de comandos.
