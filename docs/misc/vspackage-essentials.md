# Elementos fundamentales de VSPackage
Los VSPackages son módulos de software que conforman y amplían Visual Studio con elementos de interfaz de usuario, servicios, proyectos, editores y diseñadores. Los VSPackages son la unidad de la arquitectura central de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] en sí mismo está escrito principalmente como una colección de VSPackages.  
  
 Para crear una aplicación de extensibilidad [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], cree primero un VSPackage. A continuación, puede agregar comandos de menú, ventanas de herramientas, ventanas de documentos u otros código para ampliar el IDE de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
 Para usar VSPackages, debe instalar el SDK de Visual Studio. Para seguir este tutorial, debe instalar el SDK de Visual Studio. Para obtener más información, consulta [Visual Studio SDK](../Topic/Visual%20Studio%20SDK.md).
