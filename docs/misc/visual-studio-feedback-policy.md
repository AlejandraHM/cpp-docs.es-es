# Directiva de comentarios de Visual Studio
Gracias por sus comentarios.  Los equipos de desarrolladores usan sus comentarios para mejorar la documentación.  Mientras revisamos sus comentarios, es posible que le enviemos un mensaje de correo electrónico para solicitar alguna aclaración o comentario sobre una solución.  No usamos su dirección de correo electrónico para ningún otro fin y la eliminamos una vez acabada la revisión.  Para obtener más información sobre las directivas de privacidad de Microsoft, vea [http:\/\/privacy.microsoft.com\/default.aspx](http://privacy.microsoft.com/default.aspx).
