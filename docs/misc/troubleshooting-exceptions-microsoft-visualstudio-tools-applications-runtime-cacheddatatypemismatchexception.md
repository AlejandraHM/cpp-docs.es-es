# Soluci&#243;n de problemas de excepciones: Microsoft.VisualStudio.Tools.Applications.Runtime.CachedDataTypeMismatchException
La excepción que se produce cuando el estado serializado de XML del objeto de la caché no coincide con la información de tipo que figura en el manifiesto de datos almacenado en la memoria caché.
