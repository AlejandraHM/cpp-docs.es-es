# Please specify valid search string.
Este error ocurre normalmente cuando se intenta realizar una búsqueda sin especificar una cadena en el cuadro **Buscar** del cuadro de diálogo **Buscar símbolo**.  
  
### Para corregir este error  
  
1.  Escriba el texto correcto en el cuadro **Buscar** y elija de nuevo **Buscar**.
