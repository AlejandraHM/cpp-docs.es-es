# String contains too many tagged expressions.
Este error ocurre normalmente cuando se especifican más de nueve expresiones etiquetadas en la cadena de búsqueda.  
  
### Para corregir este error  
  
1.  Reduzca el número de expresiones etiquetadas especificadas.
