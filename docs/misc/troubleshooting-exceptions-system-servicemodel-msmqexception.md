# Soluci&#243;n de problemas de excepciones: System.ServiceModel.MsmqException
Una excepción <xref:System.ServiceModel.MsmqException> encapsula los errores devueltos por Microsoft Message Queuing. Esta excepción la producen el transporte y el canal de integración de Microsoft Message Queuing.
