# El nombre de archivo no existe, no es v&#225;lido o contiene par&#225;metros que no se pueden validar. &#191;Quiere cambiar la entrada?
Este error suele producirse cuando se escriben una ruta de acceso o un nombre de archivo incorrectos en el cuadro de diálogo **Agregar programa**.  
  
### Para corregir este error  
  
1.  Seleccione **Examinar** para ir al archivo correcto.  
  
     \-O bien\-  
  
2.  Vuelva a escribir la ruta de acceso y el nombre de archivo.
