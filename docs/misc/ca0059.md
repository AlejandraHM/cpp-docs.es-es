# CA0059
Error de modificador de la línea de comandos.  
  
 Este error se produce si falta un modificador necesario de la línea de comandos, por ejemplo,  
  
 si los modificadores \/out o \/console no se han especificado.  Este error también se produce si hay un modificador no válido, como un valor de tiempo de espera incorrecto.
