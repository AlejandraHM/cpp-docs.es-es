# Se produjo una divisi&#243;n por cero al evaluar esta expresi&#243;n
Al evaluar esta expresión, intentó dividir un número por cero.  
  
 **Identificador de error:** BC30542  
  
### Para corregir este error  
  
1.  Compruebe los valores de constantes y variables para determinar cómo se devuelve cero en el denominador de la expresión.
