# Command requires at least &lt;number&gt; arguments.
Este error ocurre normalmente cuando se han especificado demasiados pocos argumentos para un comando.  
  
### Para corregir este error  
  
1.  Compruebe cuál es la sintaxis correcta en la documentación del comando e inténtelo de nuevo.
