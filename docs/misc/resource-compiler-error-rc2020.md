# Error del compilador de recursos RC2020
número de dígitos no válido para la base  
  
 El `digit` especificado no es un dígito válido para la base especificada por `radix`. El dígito o la base pueden ser incorrectos.  
  
 Los dígitos octales deben ser números del 0 al 7 y los dígitos hexadecimales deben ser números del 0 al 9 o letras de la A a la E.
