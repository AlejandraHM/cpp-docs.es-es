# Clipboard cannot be opened.
Este error ocurre normalmente cuando se intenta cortar, copiar o pegar un elemento pero no se tiene acceso al Portapapeles.  
  
### Para corregir este error  
  
1.  Cierre los demás programas que puedan estar usando el Portapapeles e inténtelo de nuevo.
