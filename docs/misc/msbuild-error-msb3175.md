# Error de MSBuild MSB3175
**MSB3175: Valor no válido para '\<file\>' del elemento '\<tarea de compilación\>'.**  
  
 Esta advertencia se genera cuando el proceso de compilación no reconoce el valor de varias propiedades de elementos de tarea basados en la enumeración, como `AssemblyType`, `DependencyType`, `FileType` o `TargetZone`.  Se aplica tanto a los manifiestos de aplicación como a los manifiestos de implementación.
