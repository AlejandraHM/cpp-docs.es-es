# Toolbox item could not be copied to another tab.
Este error se produce normalmente cuando el elemento está dañado o no hay memoria suficiente en el equipo.  
  
### Para corregir este error  
  
1.  Cierre las aplicaciones abiertas y, a continuación, intente mover el elemento de nuevo.  
  
     \-O bien\-  
  
2.  Seleccione otro elemento para moverlo.
