# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Persistence.InstanceLockException
La excepción <xref:System.ServiceModel.Persistence.InstanceLockException> está diseñada para ser utilizada por clases que heredan de <xref:System.ServiceModel.Persistence.LockingPersistenceProvider> cuando la operación no se puede realizar debido al estado de bloqueo de la instancia.
