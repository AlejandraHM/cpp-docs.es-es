# Error de MSBuild MSB3114
**MSB3114: No se pudo encontrar el archivo '\<archivo\>' al que hace referencia el ensamblado '\<ensamblado\>'.**  
  
 Este error se genera cuando se encuentra una referencia irresoluble en un manifiesto existente.  
  
### Para corregir este error  
  
-   Compruebe en los manifiestos si hay conflictos entre las referencias de archivo.
