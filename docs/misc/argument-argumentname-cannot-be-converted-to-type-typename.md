# El argumento &#39;&lt;nombreDeArgumento&gt;&#39; no se puede convertir en el tipo &#39;&lt;nombreDeTipo&gt;&#39;
Una conversión intenta cambiar el argumento a un tipo que no es válido.  
  
### Para corregir este error  
  
-   Cambie el tipo a uno que sea válido.
