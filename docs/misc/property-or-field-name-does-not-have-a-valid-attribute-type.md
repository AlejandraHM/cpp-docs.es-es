# La propiedad o el campo &#39;&lt;name&gt;&#39; no tiene un tipo de atributo v&#225;lido.
Puede que esté utilizando un constructor de atributos que no acepta un tipo de atributo válido, aun cuando el atributo pueda tener otros constructores adecuados.  
  
 **Identificador de error:** BC30659  
  
### Para corregir este error  
  
1.  Compruebe el uso de este atributo.
