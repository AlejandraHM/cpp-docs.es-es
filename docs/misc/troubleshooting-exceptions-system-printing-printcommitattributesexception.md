# Soluci&#243;n de problemas de excepciones: System.Printing.PrintCommitAttributesException
Se produce una excepción <xref:System.Printing.PrintCommitAttributesException> cuando una condición de error evita que <xref:System.Printing.PrintSystemObject> confirme atributos con el equipo, impresora o dispositivo real que representa el objeto.
