# Custom Tool Error
Una herramienta personalizada ha generado un error o advertencia.  Las herramientas personalizadas son componentes que se pueden utilizar para transformar archivos de un tipo a otro en tiempo de diseño.  Por ejemplo, una herramienta personalizada puede ser un generador de código de conjunto de datos que lee un archivo de esquema XML \(.xsd\) y genera clases en un archivo de código que expone sus tablas y columnas mediante programación.  
  
 **Para corregir este error**  
  
-   Consulte la documentación proporcionada por el proveedor de la herramienta personalizada.
