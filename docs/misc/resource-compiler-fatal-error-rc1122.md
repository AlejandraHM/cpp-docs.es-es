# Error irrecuperable del compilador de recursos RC1122
Error de E\/S al escribir en el archivo  
  
 El compilador de recursos no pudo escribir en un archivo.  
  
### Posibles causas del error:  
  
1.  Espacio en disco insuficiente. El espacio libre debe ser igual al doble del tamaño del archivo ejecutable que se está creando.  
  
2.  El volumen es de solo lectura.  
  
3.  Sector defectuoso.  
  
4.  Infracción de uso compartido.
