# Soluci&#243;n de problemas de excepciones: System.Reflection.CustomAttributeFormatException
Excepción que se produce cuando el formato binario del atributo personalizado no es válido.  
  
## Comentarios  
 Puede ser el resultado de un error en una herramienta utilizada para crear el tipo de datos que produce el error.
