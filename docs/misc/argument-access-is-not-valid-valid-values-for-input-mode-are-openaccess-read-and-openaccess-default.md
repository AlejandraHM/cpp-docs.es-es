# El argumento &#39;Access&#39; no es v&#225;lido. Los valores v&#225;lidos para el modo Input son &#39;OpenAccess.Read&#39; y &#39;OpenAccess.Default&#39;
Un valor de acceso no es válido en el modo `Input`.  
  
### Para corregir este error  
  
-   Cambie el valor a `OpenAccess.Read` u `OpenAccess.Default`.
