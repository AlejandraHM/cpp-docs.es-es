# Error de compilaci&#243;n de proyecto PRJ0043
La propiedad 'Outputs' de la regla de compilación personalizada 'rule' asignada al archivo 'file' no es válida. La propiedad contenía una cadena 'string' que se evalúa en 'value'.  
  
 La propiedad Outputs se evaluó en una cadena vacía. Las reglas de compilación personalizadas necesitan la salida de la acción de compilación.
