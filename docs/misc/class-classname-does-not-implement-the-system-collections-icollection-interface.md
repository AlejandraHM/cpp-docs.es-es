# La clase &#39;&lt;classname&gt;&#39; no implementa la interfaz System.Collections.ICollection
La clase no implementa la interfaz `System.Collections.ICollection`, que define los métodos de tamaño, enumeradores y sincronización de todas las colecciones. Puede que sea necesario volver a instalar [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
### Para corregir este error  
  
1.  Vuelva a instalar [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].
