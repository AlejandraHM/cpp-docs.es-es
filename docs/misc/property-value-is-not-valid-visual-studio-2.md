# Property value is not valid. (Visual Studio) (2)
Este error ocurre normalmente cuando se escribe un valor no válido para la propiedad.  Por ejemplo, al especificar el formato incorrecto para un valor entero o un valor demasiado grande para un valor short.  
  
### Para corregir este error  
  
1.  Escriba un valor que sea válido para dicha propiedad.  
  
    > [!TIP]
    >  Busque en la documentación en pantalla el nombre de la propiedad para comprobar qué valores son válidos.
