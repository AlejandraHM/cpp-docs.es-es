# Unable to Open Linked File
Se ha agregado un archivo vinculado al proyecto.  No obstante, al abrir el proyecto, el sistema de proyecto no encuentra el archivo.  
  
 **Para corregir este error**  
  
-   Elimine el archivo vinculado desde el Explorador de soluciones.  
  
     \-O bien\-  
  
-   Devuelva el archivo a la ubicación donde el sistema del proyecto espera encontrarlo.
