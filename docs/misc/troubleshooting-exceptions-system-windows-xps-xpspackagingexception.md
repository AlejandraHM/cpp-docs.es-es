# Soluci&#243;n de problemas de excepciones: System.Windows.Xps.XpsPackagingException
Se produce una excepción <xref:System.Windows.Xps.XpsPackagingException> al leer, escribir, registrar u obtener acceso de alguna otra forma a un objeto <xref:System.Windows.Xps.Packaging.XpsDocument>.
