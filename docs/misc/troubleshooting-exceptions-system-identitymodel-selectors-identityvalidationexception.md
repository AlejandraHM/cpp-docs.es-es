# Soluci&#243;n de problemas de excepciones: System.IdentityModel.Selectors.IdentityValidationException
Se produce una excepción <xref:System.IdentityModel.Selectors.IdentityValidationException> cuando el certificado del destinatario no es válido. Esto puede ocurrir si el certificado no se encadena correctamente al certificado raíz.
