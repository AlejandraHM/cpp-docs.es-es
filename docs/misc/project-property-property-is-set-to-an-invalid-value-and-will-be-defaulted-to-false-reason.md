# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
El valor almacenado para una propiedad `Boolean` no es uno de los siguientes:  
  
-   1 ó 0  
  
-   On u Off  
  
-   true o false  
  
 Es muy probable que el error esté causado por la edición manual del archivo de proyecto.  
  
 **Para corregir este error**  
  
-   Edite otra vez el archivo de proyecto y restáurelo en una condición satisfactoria.  
  
     El valor de la propiedad afectada se establecerá en false.
