# Error irrecuperable del compilador de recursos RC1107
uso no válido; ¿usar RC\/? para Ayuda  
  
 Se especificó una opción del compilador de recursos no válida. Las opciones válidas son \/r, \/v, \/d, \/fo, \/l, \/i, \/x y \/c.
