# Item cannot be added to a project when multiple or no items are selected in Project Explorer.
Este error se produce normalmente cuando no se ha seleccionado ningún elemento en el Explorador de soluciones, o se ha seleccionado más de un elemento.  
  
### Para corregir este error  
  
1.  Seleccione un solo elemento en el Explorador de soluciones.
