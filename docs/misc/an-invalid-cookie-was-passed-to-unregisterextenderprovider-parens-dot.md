# An invalid cookie was passed to UnregisterExtenderProvider().
Se ha especificado una cookie que ya no es válida.  
  
### Para corregir este error  
  
1.  Devuelva el mismo valor de cookie que obtuvo al llamar a RegisterExtenderProvider\(\).
