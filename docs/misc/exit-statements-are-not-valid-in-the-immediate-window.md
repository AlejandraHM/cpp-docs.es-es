# Las instrucciones &#39;Exit&#39; no son v&#225;lidas en la ventana Inmediato
Las instrucciones `Exit` solo pueden usarse en el código fuente.  
  
 **Id. de error:** BC30713  
  
### Para corregir este error  
  
-   Quite las instrucciones `Exit` del código de depuración.
