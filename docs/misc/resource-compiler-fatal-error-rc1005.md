# Error irrecuperable del compilador de recursos RC1005
archivo no encontrado: nombre de archivo  
  
 No se encontró el archivo especificado en la línea de comandos del compilador de recursos. Compruebe si el archivo se ha movido a otro directorio y si se ha escrito correctamente el nombre de archivo o ruta de acceso.  
  
 Los archivos se buscan usando la variable de entorno **INCLUDE** o la configuración de directorios de Visual C\+\+.
