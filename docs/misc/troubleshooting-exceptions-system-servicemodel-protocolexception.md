# Soluci&#243;n de problemas de excepciones: System.ServiceModel.ProtocolException
Se produce una excepción <xref:System.ServiceModel.ProtocolException> en el cliente cuando la comunicación con la parte remota es imposible debido a que los protocolos de transferencia de datos no coinciden.  
  
## Comentarios  
 La excepción <xref:System.ServiceModel.ProtocolException> puede estar causada por enlaces incompatibles entre los extremos, el uso de una dirección de extremo remoto incorrecta o una configuración similar incorrecta.
