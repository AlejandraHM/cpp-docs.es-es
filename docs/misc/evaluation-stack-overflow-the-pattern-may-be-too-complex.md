# Evaluation stack overflow. The pattern may be too complex.
Este error se produce normalmente cuando se ha utilizado una expresión regular compleja en una cadena de búsqueda que el entorno de desarrollo integrado \(IDE\) no puede procesar.  
  
### Para corregir este error  
  
1.  Reduzca el número de cierres y de cláusulas e inténtelo de nuevo.
