# No se puede obtener acceso a la definici&#243;n del m&#233;todo &lt;m&#233;todo&gt; en este contexto
Una función de agregado que no existe se incluyó con el operador `Into`.  
  
 **Identificador de error:** BC36594  
  
### Para corregir este error  
  
-   Compruebe la ortografía del nombre de función de agregado y realice las correcciones necesarias.  
  
-   Agregue el método como una función de agregado personalizada \(método de extensión\). Para obtener más información acerca de cómo crear una función de agregado personalizada, consulte [Aggregate \(Cláusula\)](../Topic/Aggregate%20Clause%20\(Visual%20Basic\).md).
