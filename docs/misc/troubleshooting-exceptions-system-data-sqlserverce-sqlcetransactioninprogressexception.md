# Soluci&#243;n de problemas de excepciones: System.Data.SqlServerCe.SqlCeTransactionInProgressException
Se produce una excepción `System.Data.SqlServerCe.SqlCeTransactionInProgressException` cuando una aplicación intenta modificar una base de datos mientras otra transacción está en curso. Para más información, vea [Biblioteca de clases de SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
