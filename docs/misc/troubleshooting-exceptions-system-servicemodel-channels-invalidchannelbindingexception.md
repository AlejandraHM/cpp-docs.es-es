# Soluci&#243;n de problemas de excepciones: System.ServiceModel.Channels.InvalidChannelBindingException
Se produce una excepción <xref:System.ServiceModel.Channels.InvalidChannelBindingException> cuando el enlace especificado no es coherente con los requisitos de contrato del servicio.
