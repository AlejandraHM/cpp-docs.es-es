# Soluci&#243;n de problemas de excepciones: Microsoft.Office.Tools.CannotRemoveControlException
Se produce una excepción <xref:Microsoft.Office.Tools.CannotRemoveControlException> cuando se produce un intento de quitar mediante programación un control que no se ha agregado mediante programación.  
  
## Sugerencias asociadas  
 Solamente se puede quitar mediante programación un control agregado mediante programación.  
 -   Solamente se puede quitar mediante programación un control agregado mediante programación. No se puede quitar si el control se agregó al documento en tiempo de diseño en lugar de en tiempo de ejecución.
