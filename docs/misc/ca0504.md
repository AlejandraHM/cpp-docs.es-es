# CA0504
Se omitió el directorio de la regla porque no existe  
  
 La advertencia CA504 se genera cuando no se encuentra un directorio especificado en la propiedad **CodeAnalysisRuleDirectories** del archivo de configuración de compilación o de proyecto.  Para resolver este problema, escriba una ruta de acceso al directorio que sea correcta.
