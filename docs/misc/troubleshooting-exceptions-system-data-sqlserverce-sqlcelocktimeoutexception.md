# Soluci&#243;n de problemas de excepciones: System.Data.SqlServerCe.SqlCeLockTimeoutException
Se produce una excepción `System.Data.SqlServerCe.SqlCeLockTimeoutException` si se alcanza el tiempo de espera de bloqueo y SQL Server Compact agota el tiempo de espera mientras espera un bloqueo. El tiempo de espera de bloqueo predeterminado es 2000 milisegundos. Para más información, vea [Biblioteca de clases de SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
