# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
El complemento ha provocado un error y puede que esté causando otros problemas en el entorno de desarrollo integrado.  
  
### Para corregir este error  
  
1.  Elija Sí para quitar el complemento de la lista de complementos disponibles.  
  
     O bien  
  
     Elija No.  
  
    > [!NOTE]
    >  Póngase en contacto con el autor del complemento para informarle del problema.
