# La referencia de ensamblado predeterminado &#39;&lt;referencia&gt;&#39; ya se agreg&#243; autom&#225;ticamente; ignorada
Se realizó un intento de agregar la referencia de ensamblado predeterminada, pero el ensamblado ya se agregó, por lo que se omitió el intento.  
  
 **Identificador de error:** BC2024  
  
### Para corregir este error  
  
-   Quite el código superfluo.
