# No se admite Get en tiempo de ejecuci&#243;n.
Se intentó leer una propiedad en tiempo de ejecución que solo es accesible en tiempo de diseño.  
  
### Para corregir este error  
  
1.  Compruebe la propiedad y determine bajo qué condiciones se puede establecer.  
  
2.  Quite la referencia a la propiedad.
