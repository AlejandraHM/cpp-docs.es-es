# Command &lt;name&gt; does not accept arguments or switches.
Este error ocurre normalmente cuando se ha escrito un modificador o argumento con un comando o alias que no usa argumentos ni modificadores.  Si se ha escrito un alias, aparece el nombre de comando completo en lugar del nombre de alias en el mensaje de error.  Para obtener una lista de comandos válidos que utilizan modificadores y argumentos, vea [Comandos de Visual Studio](../Topic/Visual%20Studio%20Commands.md).  
  
### Para corregir este error  
  
1.  Vuelva a escribir el comando y los argumentos, o escriba un nuevo comando.
