# CA0053
No se pudo cargar el ensamblado de regla.  
  
 Esto indica que no se pudo cargar un ensamblado de regla por reflexión.  Puede suceder si el usuario tiene reglas personalizadas que no se han compilado correctamente, o si la herramienta de análisis no puede cargar el ensamblado por otros motivos \(por ejemplo, si se le deniega el acceso al sistema de archivos\).
