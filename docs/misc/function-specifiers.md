# Especificadores de funci&#243;n
Puede usar las palabras clave [inline](../misc/inline-inline-forceinline.md) y [virtual](../cpp/virtual-specifier.md) como especificadores en declaraciones de función. Este uso de **virtual** difiere del uso del especificador de clase base de una definición de clase.
