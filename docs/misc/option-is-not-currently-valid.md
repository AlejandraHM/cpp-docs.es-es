# Option is not currently valid.
Este error se produce normalmente cuando se ha especificado un argumento o un modificador para un comando que no está disponible en la actualidad.  Por ejemplo, si especificó `>Edit.Find test /sel` y no se ha realizado ninguna selección en el archivo, el modificador \/sel no estará disponible.  
  
### Para corregir este error  
  
1.  Compruebe que los argumentos y los modificadores especificados están disponibles en la actualidad.  Para obtener más información, vea [Comandos de Visual Studio con argumentos](../Topic/Visual%20Studio%20Commands.md).
