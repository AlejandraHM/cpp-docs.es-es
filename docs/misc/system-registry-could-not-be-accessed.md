# System registry could not be accessed.
Este error ocurre normalmente cuando el sistema operativo no encuentra el archivo del Registro, el archivo está dañado o falló un intento de leer o escribir en el archivo.  
  
### Para corregir este error  
  
1.  Vuelva a instalar Visual Studio.
