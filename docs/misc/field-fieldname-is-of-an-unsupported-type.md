# El campo &#39;&lt;fieldname&gt;&#39; es de un tipo no compatible.
Un campo está declarado como un tipo que no se puede representar por [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
 **Identificador de error:** BC30656  
  
### Para corregir este error  
  
1.  Defina el campo como un tipo compatible.  
  
2.  Desarrolle un nuevo tipo que describa los datos en el campo.
