# &#39;&lt;nombre&gt;&#39; no puede exponer el tipo &#39;&lt;tipo1&gt;&#39; que se usa en el tipo delegado subyacente &#39;&lt;tipoDelegado&gt;&#39; del evento que implementa fuera del proyecto a trav&#233;s de &lt;especificador&gt; &#39;&lt;tipo2&gt;&#39;
El código expone un tipo privado fuera de la clase pública.  
  
 **Identificador de error:** BC30924  
  
### Para corregir este error  
  
-   Declare el tipo `Public` o use otro tipo.
