# No se puede obtener la informaci&#243;n de &#39;&lt;nombreDeTipo&gt;&#39;
Se hizo referencia a un tipo que no ha sido cargado por Common Language Runtime.  
  
 **Identificador de error:** BC30751  
  
### Para corregir este error  
  
-   Reestructure el código para que se defina el tipo dentro del ámbito actual.
