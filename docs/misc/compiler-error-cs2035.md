# Error del compilador CS2035
Error de sintaxis de la línea de comandos: falta ':\<number\>' para la opción 'compiler\_option'  
  
 Algunas opciones del compilador requieren un valor.  
  
## Ejemplo  
 El ejemplo siguiente genera la advertencia CS2035.  
  
```  
// CS2035.cs // compile with: /baseaddress // CS2035 expected  
```
