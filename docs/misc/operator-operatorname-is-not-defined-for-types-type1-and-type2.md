# El operador &#39;&lt;operatorname&gt;&#39; no est&#225; definido para los tipos &#39;&lt;type1&gt;&#39; y &#39;&lt;type2&gt;&#39;
Ha intentado utilizar un operador binario para realizar una operación que no es válida para los dos elementos de código que devuelven valores implicados.  
  
 **Id. de error:** BC30452  
  
### Para corregir este error  
  
-   Compruebe los dos elementos y asegúrese de que sean compatibles.
