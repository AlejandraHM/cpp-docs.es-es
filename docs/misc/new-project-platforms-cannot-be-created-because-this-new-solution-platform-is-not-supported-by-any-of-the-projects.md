# New project platforms cannot be created because this new solution platform is not supported by any of the projects.
Este error aparece cuando selecciona **Crear nuevas plataformas de proyecto** pero ninguno de los proyectos admite la nueva plataforma de soluciones.  
  
### Para corregir este error  
  
-   Anule la selección de **Crear nuevas plataformas de proyecto** y haga clic en **Aceptar**.  
  
     \-O bien\-  
  
     Anule la selección de **Crear nuevas plataformas de proyecto**, seleccione un nombre de plataforma diferente en la lista desplegable y haga clic en **Aceptar**.
