# Error del compilador de recursos RC2132
Se esperaba la palabra clave VALUE, BLOCK o END  
  
 Un bloque de un recurso **VERSION** no termina correctamente o un bloque nuevo no comienza correctamente. El resultado es una instrucción **VERSION** que no es un bloque válido.
