# El &#237;ndice de la colecci&#243;n debe estar en el intervalo 1 del tama&#241;o de la colecci&#243;n.
El valor de índice proporcionado está fuera del intervalo válido, que es 1 para el tamaño de la colección.  
  
### Para corregir este error  
  
1.  Cambie el valor proporcionado a uno válido.
