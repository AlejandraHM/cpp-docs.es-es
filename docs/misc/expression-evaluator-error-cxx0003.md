# Error del evaluador de expresiones CXX0003
error interno en el evaluador de expresiones  
  
 Se encontró un error interno en el evaluador de expresiones.  
  
 Anote las circunstancias del error y notifíquelo a Microsoft Corporation. Puede informar del error en [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) o puede llamar a los servicios de soporte técnico de Microsoft. Para obtener más información, haga clic en **Soporte técnico** del menú **Ayuda** de Visual C\+\+.  
  
 Este error es idéntico a CAN0003.
