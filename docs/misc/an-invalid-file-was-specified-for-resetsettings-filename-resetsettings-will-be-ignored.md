# An invalid file was specified for /resetsettings: &lt;filename&gt;.  /resetsettings will be ignored.
Este error ocurre generalmente cuando el archivo de configuración no es legible, no existe o contiene XML incorrecto.  
  
### Para corregir este error  
  
1.  Compruebe que el nombre de archivo es correcto.  
  
2.  Compruebe que el archivo tiene el formato correcto.  
  
3.  Compruebe que el archivo contiene código XML formado correctamente.
