# Las etiquetas no son v&#225;lidas en la ventana Inmediato.
Las etiquetas de línea habilitan la bifurcación y no están permitidas en un contexto de depuración.  
  
 **Identificador de error:** BC30134  
  
### Para corregir este error  
  
-   No escriba una etiqueta de línea en la ventana **Inmediato**.
