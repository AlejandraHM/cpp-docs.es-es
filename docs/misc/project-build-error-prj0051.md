# Error PRJ0051 al compilar el proyecto
Error al copiar 'file' en el directorio de destino.  Asegúrese de que el archivo de destino no está en uso.  
  
 El sistema de compilación no pudo copiar un archivo.  
  
 Asegúrese de que el archivo no está abierto en la máquina o en otro programa.
