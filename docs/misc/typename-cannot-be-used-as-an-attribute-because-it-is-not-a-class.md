# &#39;&lt;nombreDeTipo&gt;&#39; no se puede usar como atributo porque no es una clase
Se intentó usar como atributo un tipo que no es una clase.  
  
 **Identificador de error:** BC31503  
  
### Para corregir este error  
  
1.  Defina atributos personalizados como clases que derivan de `System.Attribute`.
