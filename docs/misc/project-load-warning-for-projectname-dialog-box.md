# Cuadro de di&#225;logo Advertencia de carga de proyecto para &lt;nombreProyecto&gt;
Este cuadro de diálogo aparecerá si la configuración de los proyectos abiertos en el IDE no se carga correctamente.  
  
 **Corrija el error especificado y vuelva a cargar el proyecto**  
 Proporciona información relacionada con la compilación, como una ruta de acceso o un nombre de archivo no válidos.  Para resolver los errores, quite o reemplace cualquier referencia no válida de las páginas de propiedades del proyecto.  Guarde el proyecto y, a continuación, recárguelo en el IDE.  
  
 **No volver a mostrar este cuadro de diálogo en esta sesión**  
 Cuando se selecciona, el cuadro de diálogo de advertencia de carga del proyecto para \<nombreDeProyecto\> no volverá a aparecer hasta la próxima vez que se abra el IDE.
