# Advertencias de conformidad con CLS
Los temas de esta sección describen las advertencias que puede generar el vinculador al realizar comprobaciones de conformidad con Common Language Subset \(CLS\) en el ensamblado.  
  
 Para más información, consulte [CLS Compliant Assemblies](http://msdn.microsoft.com/es-es/3320b57e-ea55-4697-a17d-f509a36a3c93) \(Ensamblados conformes a CLS\).
