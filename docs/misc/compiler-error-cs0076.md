# Error del compilador CS0076
El nombre de enumerador 'value\_\_' está reservado y no se puede usar.  
  
 Un elemento de una [enumeración](../Topic/enum%20\(C%23%20Reference\).md) no puede tener un identificador denominado **value\_\_**.
