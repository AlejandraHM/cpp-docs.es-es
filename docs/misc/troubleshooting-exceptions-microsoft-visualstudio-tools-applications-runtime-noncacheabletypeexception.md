# Soluci&#243;n de problemas de excepciones: Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
La excepción que se produce cuando se intenta almacenar en caché un tipo, pero este no es <xref:System.Data.DataSet>, <xref:System.Data.DataTable> ni un tipo que cumpla los requisitos especificados en [Caching Data](http://msdn.microsoft.com/es-es/e60dc731-2cb0-4de6-a861-0f0d131d780a).
