# Command &lt;name&gt; is not valid.
Este error se produce normalmente cuando se ha escrito un comando o un alias de comando que Visual Studio no reconoce en el cuadro **Buscar\/Comando** o en la ventana **Comando**.  Si se ha escrito un alias, aparece el nombre de comando completo en lugar del nombre de alias en el mensaje de error.  
  
### Para corregir este error  
  
1.  Vuelva a escribir el nombre de comando o elija un nombre de comando válido.
