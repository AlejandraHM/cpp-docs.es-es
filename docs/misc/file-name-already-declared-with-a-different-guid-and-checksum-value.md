# El nombre de archivo ya est&#225; declarado con un GUID y un valor de suma de comprobaci&#243;n distintos
Se intentó declarar un archivo, pero el archivo ya existe con un GUID y un valor de suma de comprobación diferentes.  
  
 **Identificador de error:** BC42034  
  
### Para corregir este error  
  
-   Compruebe el nombre de archivo y la ruta de acceso para asegurarse de que ambos son correctos.  
  
-   Elimine el archivo redundante si ya no es necesario.
