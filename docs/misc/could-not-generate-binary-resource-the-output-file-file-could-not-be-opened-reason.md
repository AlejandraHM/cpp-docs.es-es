# No se pudo generar el recurso binario. No se pudo abrir el archivo de salida &#39;file&#39;. &lt;motivo&gt;
No se pudo eliminar una copia obsoleta del archivo .resources mediante el sistema del proyecto. Como parte de la preparación de recursos, el sistema del proyecto transformará los archivos XML con la extensión .resx en archivos .resources binarios para el Administrador de recursos. NET.  
  
 El motivo específico del sistema operativo que se indica al final del texto del mensaje de error proporcionará aquí información más detallada.  
  
 **Para corregir este error**  
  
-   Reinicie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     Este error impedirá que el proceso de compilación se lleve a cabo.
