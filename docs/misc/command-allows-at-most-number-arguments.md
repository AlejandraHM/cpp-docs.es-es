# Command allows at most &lt;number&gt; arguments.
Este error ocurre normalmente cuando se especifican demasiados argumentos para un comando o se utiliza una combinación incorrecta de argumentos.  
  
### Para corregir este error  
  
1.  Compruebe cuál es la sintaxis correcta en la documentación del comando e inténtelo de nuevo.
