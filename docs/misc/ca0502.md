# CA0502
Lenguaje no compatible  
  
 Este error aparece cuando el usuario trata de analizar un proyecto de ASP.NET que contiene una o más páginas o clases escritas en un lenguaje distinto de C\# o Visual Basic.  Para corregir este error, quite el archivo del proyecto o deshabilite el análisis de código.
