# Template &lt;name&gt; is not valid for the selected project.
Este error se produce normalmente cuando se ha especificado un nombre de plantilla desconocido para el comando de **Nuevo archivo** \(`File.NewFile`\) o para el comando **Agregar nuevo elemento de solución** \(`File.AddNewSolutionItem`\).  
  
### Para corregir este error  
  
1.  Vuelva a escribir el comando y sus argumentos.  
  
     \-O bien\-  
  
2.  Vea los temas "Nuevo archivo \(Comando\)" o "Agregar nuevo elemento de solución \(Comando\)" para conocer la sintaxis adecuada.
