# El operador &#39;If&#39; requiere dos o tres operandos
El operador `If` acepta solo dos o tres argumentos.  
  
 **Identificador de error:** BC33104  
  
### Para corregir este error  
  
-   Examine la expresión `If` para ver si el número de operadores se puede reducir o ampliar a dos o tres argumentos.  
  
-   Considere la posibilidad de usar otra construcción condicional, como una instrucción `If...Then...Else` o una instrucción `If...Then...Else` anidada para organizar los argumentos.
