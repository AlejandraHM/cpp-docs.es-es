# Error de MSBuild MSB3484
**MSB3484: No se encontró el destino de firma '\<archivo\>'.**  
  
 Se genera este error cuando no se encuentra el archivo que se intenta firmar.  
  
### Para corregir este error  
  
-   Asegúrese de que ha especificado un archivo válido para la firma.
