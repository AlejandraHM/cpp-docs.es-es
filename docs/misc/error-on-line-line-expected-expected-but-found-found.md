# Error on line &lt;line&gt;. Expected &#39;expected&#39; but found &#39;found&#39;
Éste es un error general de análisis XML al leer el archivo de proyecto.  Este error puede producirse en archivos de proyecto \(.VBPROJ\/.CSPROJ\) así como en archivos de configuración de cada usuario \(.vbproj.user\/.csproj.user\).  El sistema del proyecto proporcionará más información en los parámetros `expected` y `found`.  
  
 Es muy probable que el error esté causado por la edición manual del archivo de proyecto \(o de usuario\).  
  
 **Para corregir este error**  
  
-   Edite otra vez el archivo de proyecto y restáurelo en una condición satisfactoria.  
  
     En función de la gravedad del error, podrá o no podrá abrir el proyecto.
