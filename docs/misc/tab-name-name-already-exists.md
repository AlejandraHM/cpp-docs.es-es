# Tab name &lt;name&gt; already exists.
Todas las fichas del Cuadro de herramientas deben tener un nombre único.  Este error ocurre normalmente cuando el nombre especificado para la ficha del Cuadro de herramientas es el mismo que otra ficha existente del mismo.  
  
### Para corregir este error  
  
1.  Escriba un nombre único para la ficha.
