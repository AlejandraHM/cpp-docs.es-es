# Soluci&#243;n de problemas de excepciones: System.IO.Log.ReservationNotFoundException
Se produce una excepción <xref:System.IO.Log.ReservationNotFoundException> cuando no se encuentra una reserva de espacio específica en <xref:System.IO.Log.LogRecordSequence>.
