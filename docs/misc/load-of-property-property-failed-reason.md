# Load of property &#39;property&#39; failed. &lt;reason&gt;
El valor de una propiedad está fuera del intervalo.  
  
 Es muy probable que el error esté causado por la edición manual del archivo de proyecto.  
  
 **Para corregir este error**  
  
-   Vuelva a editar el archivo de proyecto y corrija manualmente el error.  
  
     Las propiedades afectadas se establecerán en el valor predeterminado, es decir, el valor que tenía la propiedad cuando se creó el proyecto.
