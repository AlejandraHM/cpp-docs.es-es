# Soluci&#243;n de problemas de excepciones: System.Drawing.Printing.InvalidPrinterException
Se produce una excepción <xref:System.Drawing.Printing.InvalidPrinterException> cuando se intenta tener acceso a una impresora mediante una configuración de impresora no válida.  
  
## Sugerencias asociadas  
 **Asegúrese de que existe la impresora.**  
 La causa más común de que se produzca una configuración de impresora no válida es debido a que se ha hecho referencia a una impresora no existente. Para obtener más información, consulta <xref:System.Drawing.Printing>.  
  
 **Asegúrese de que se ha instalado una impresora predeterminada.**  
 Si no se ha especificado ninguna impresora, asegúrese de que se ha instalado una impresora predeterminada. Para obtener más información, vea <xref:System.Drawing.Printing.PrintDocument.PrinterSettings%2A>
