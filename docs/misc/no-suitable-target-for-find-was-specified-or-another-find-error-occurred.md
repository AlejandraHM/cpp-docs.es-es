# No suitable target for find was specified or another find error occurred.
Este error se produce normalmente cuando se ha intentado realizar una búsqueda con el ámbito de búsqueda establecido en Documento actual pero no hay ningún archivo abierto.  
  
### Para corregir este error  
  
1.  Abra un archivo e intente realizar la búsqueda de nuevo o cambie el ámbito de la búsqueda.
