# Tagged expression number is out of range.
Este error ocurre normalmente cuando se ha usado una referencia a una expresión etiquetada en una cadena de búsqueda y reemplazo, pero el número especificado estaba fuera del intervalo 0\-9.  
  
### Para corregir este error  
  
1.  Escriba un número entre cero y nueve.
