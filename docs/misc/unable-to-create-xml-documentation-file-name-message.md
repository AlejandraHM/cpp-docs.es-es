# No se puede crear el archivo de documentaci&#243;n XML &#39;&lt;nombre&gt;&#39;: &lt;mensaje&gt;
No se puede crear el archivo de documentación XML debido a un error.  
  
 **Identificador de error:** BC42311  
  
 Los mensajes siguientes se pueden mostrar con este error:  
  
-   [Un bloque de comentario XML no se puede asociar con ningún elemento de lenguaje que admita la aplicación de comentarios de documentación XML](../misc/xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments.md).  
  
-   [No se permite la etiqueta de comentario XML 'returns' en una propiedad 'WriteOnly'](../misc/xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property.md).  
  
-   [El comentario XML no se puede aplicar más de una vez en un \<tipo\> parcial.](../misc/xml-comment-cannot-be-applied-more-than-once-on-a-partial-type.md).  
  
-   [No se permite la etiqueta de comentario XML 'returns' en un elemento de lenguaje 'declare sub'](../misc/xml-comment-tag-returns-is-not-permitted-on-a-declare-sub-language-element.md).  
  
-   [Error de análisis de documentación XML: la etiqueta inicial '\<etiqueta\>' no tiene la correspondiente etiqueta de cierre.](../misc/xml-documentation-parse-error-start-tag-tag-doesn-t-have-a-matching-end-tag.md).  
  
-   [El parámetro \<parámetro\> de tipo de comentario XML no coincide con un parámetro de tipo de la instrucción \<palabra clave\> correspondiente.](../misc/xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement.md).  
  
-   [El parámetro de tipo de comentario XML debe tener un atributo 'name'](../misc/xml-comment-type-parameter-must-have-a-name-attribute.md).  
  
-   [La excepción del comentario XML debe tener un atributo 'cref'](../Topic/XML%20comment%20exception%20must%20have%20a%20'cref'%20attribute.md).  
  
### Para corregir este error  
  
-   Consulte el mensaje proporcionado para obtener recomendaciones.
