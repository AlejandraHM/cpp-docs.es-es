# Advertencia PRJ0045 al compilar el proyecto
No se estableció la propiedad 'Outputs' de la regla de compilación personalizada 'regla' asignada al archivo 'archivo'. Se omitirá la ejecución de la regla.  
  
 La propiedad Outputs no está definida para una regla de compilación personalizada. Las reglas de compilación personalizadas necesitan la salida de la acción de compilación.
