# Top-level menu names cannot contain separators.
Este error ocurre normalmente cuando se intenta arrastrar texto que contiene un carácter separador, como el guión \(\-\), en un nombre de menú.  Los menús de nivel superior no pueden contener este carácter, que está reservado para separar conjuntos de elementos de menú.  
  
### Para corregir este error  
  
1.  Escriba el texto sin usar el carácter separador.
