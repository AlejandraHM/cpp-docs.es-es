# Todos los proyectos de una soluci&#243;n de Visual Basic deben tener como destino la misma plataforma, pero el proyecto que est&#225; intentando agregar tiene como destino una plataforma distinta de la especificada en la soluci&#243;n.
Todos los proyectos de una solución de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] deben tener como destino la misma plataforma, pero el proyecto que está agregando tiene como destino una plataforma distinta de la especificada en la solución.  
  
 **Identificador de error:** BC30765  
  
### Para corregir este error  
  
-   Agregue únicamente los proyectos que tienen como destino la misma plataforma especificada en la solución.
