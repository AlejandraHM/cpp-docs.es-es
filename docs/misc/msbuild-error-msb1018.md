# Error de MSBuild MSB1018
**Nivel de contenido no válido.**  
  
 El nivel de contenido especificado no se corresponde con ninguno de los niveles de contenido disponibles.  
  
### Para corregir este error  
  
1.  Compruebe si se ha escrito correctamente el nivel de contenido.  Los niveles de contenido disponibles son: q\[uiet\], m\[inimal\], n\[ormal\], d\[etailed\] y diag\[nostic\], por ejemplo, `/verbosity:quiet`,  `/verbosity:q` o  `/v:q`.
