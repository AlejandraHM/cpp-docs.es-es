# El valor de ConnectionTimeout debe ser mayor que 0
Al cargar y descargar archivos con el [My.Computer.Network \(Objeto\)](../Topic/My.Computer.Network%20Object.md), debe especificar un `connectionTimeout` mayor que `0`.  
  
### Para corregir este error  
  
-   Proporcione un valor de `connectionTimeout` que sea mayor que `0`.
