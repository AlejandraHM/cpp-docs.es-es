# Grouped expression is missing &#39;)&#39;.
Este error ocurre normalmente cuando se especifican paréntesis de apertura "`(`", pero se omiten los de cierre "`)`" en la cadena de búsqueda.  
  
### Para corregir este error  
  
1.  Para buscar el literal de cadena `(`, utilice `\(`.  
  
2.  Para agrupar una expresión, escriba el carácter `)` que falta.
