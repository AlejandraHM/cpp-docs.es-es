# Error del compilador CS1647
Una expresión es demasiado larga o compleja para compilar cerca de 'code'  
  
 Se produjo un desbordamiento de pila en el compilador que procesa el código. Para resolver este error, simplifique el código. Si el código es válido, póngase en contacto con [soporte técnico](http://msdn.microsoft.com/es-es/77e75b8b-817d-45bf-9c38-458930d873b4).
