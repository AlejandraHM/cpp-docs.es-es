# No se puede usar el tipo &#39;&lt;nombreDeTipo&gt;&#39; como un atributo porque no est&#225; declarado como &#39;Public&#39;.
Los atributos deben ser públicos.  
  
 **Identificador de error:** BC31516  
  
### Para corregir este error  
  
1.  Agregue el modificador `Public` a la clase de atributo.
