# This project cannot be added to the solution because it already exists in the solution in a hidden Solution Folder.
En una solución, un proyecto sólo puede estar en una ubicación.  
  
### Para corregir este error  
  
1.  Para mostrar las carpetas de soluciones ocultas, seleccione la solución o la carpeta de solución que contenga una carpeta de solución oculta y, a continuación, en el menú **Proyecto**, seleccione **Mostrar carpetas**.
