# Soluci&#243;n de problemas de excepciones: System.IO.PathTooLongException
Cuando un nombre de ruta o de archivo es más largo que la longitud máxima definida por el sistema, se produce una excepción <xref:System.IO.PathTooLongException>.  
  
## Sugerencias asociadas  
 **Asegúrese de que la ruta de acceso no supere el máximo definido por el sistema.**  
 En plataformas basadas en Windows, las rutas de acceso deben ser menos de 248 caracteres y los nombres de archivo deben tener menos de 260 caracteres.
