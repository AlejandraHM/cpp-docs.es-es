# This tool could not be executed. Please ensure that it points to a valid executable.
Este error ocurre normalmente cuando se intenta iniciar una herramienta externa y ésta provoca un error al ejecutarse.  Este error puede deberse a que el archivo especificado no es un ejecutable o archivo de lotes válido.  
  
### Para corregir este error  
  
1.  Compruebe que la herramienta externa sea un archivo ejecutable o un archivo por lotes válido utilizando el cuadro de diálogo **Herramientas externas**, disponible en el menú **Herramientas**.
