# Soluci&#243;n de problemas de excepciones: Microsoft.VisualStudio.Tools.Applications.Runtime.UpdateFailedException
La excepción que se produce cuando un manifiesto de implementación hace referencia a un manifiesto de aplicación cuya versión es diferente de la del manifiesto de aplicación del documento, pero éste último no se puede actualizar.
