# Soluci&#243;n de problemas de excepciones: System.IO.DirectoryNotFoundException
Si no se encuentra parte de una ruta de acceso a un archivo o un directorio, se produce una excepción <xref:System.IO.DirectoryNotFoundException>.  
  
## Sugerencias asociadas  
 **Compruebe que el directorio aparece en la ubicación especificada.**  
 Compruebe que el directorio existe en la ubicación especificada.  
  
 **Cuando utilice rutas de acceso relativas, asegúrese de que el directorio sea el correcto.**  
 La ruta de acceso puede ser incorrecta si se asume un directorio actual incorrecto.
