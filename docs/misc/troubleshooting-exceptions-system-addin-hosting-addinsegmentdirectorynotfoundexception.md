# Soluci&#243;n de problemas de excepciones: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
La excepción <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> se produce cuando falta un directorio de segmentos de la estructura de directorios de canalizaciones.  
  
## Comentarios  
 Cuando se activa un complemento, se construye su canalización al host. El sistema del complemento requiere que los segmentos de canalizaciones se coloquen en una estructura de directorios concreta con nombres de directorio especificados, como se describe en [Requisitos del desarrollo de canalizaciones](http://msdn.microsoft.com/es-es/ef9fa986-e80b-43e1-868b-247f4c1d9da5).
