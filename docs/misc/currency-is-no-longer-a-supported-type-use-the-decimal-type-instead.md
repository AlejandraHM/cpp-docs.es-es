# Ya no se admite el tipo &#39;Currency&#39;; use el tipo &#39;Decimal&#39; en su lugar
El tipo de datos `Currency` se reemplazó por el tipo de datos `Decimal`.  
  
 **Identificador de error:** BC30803  
  
### Para corregir este error  
  
-   Utilice el tipo de datos `Decimal` en lugar de `Currency`.
