# Look In list must contain valid files or folders, separated by semicolons.
Este error se produce normalmente cuando el directorio o los archivos mostrados en la lista **Buscar en** contienen caracteres que no son válidos.  
  
### Para corregir este error  
  
1.  Utilice el cuadro de diálogo **Buscar en** para buscar el nombre de directorio o archivo en los que se va buscar, en lugar de escribir la ruta de acceso o el nombre de archivo.  Para obtener acceso al cuadro de diálogo **Buscar en** elija el botón de puntos suspensivos situado junto a la lista **Buscar en**.
