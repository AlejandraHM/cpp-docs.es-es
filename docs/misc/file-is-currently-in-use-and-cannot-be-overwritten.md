# File is currently in use and cannot be overwritten.
No se puede guardar el archivo actual.  Este error ocurre normalmente cuando se intenta guardar un archivo con el nombre de un archivo abierto en ese momento.  
  
### Para corregir este error  
  
1.  Cierre el archivo que posee el mismo nombre e inténtelo de nuevo.  
  
     \-O bien\-  
  
     Cambie el nombre del archivo y guárdelo.
