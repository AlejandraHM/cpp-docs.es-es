# VbStrConv.Wide y VbStrConv.Narrow no son aplicables a la configuraci&#243;n regional especificada
La aplicación está intentando usar los miembros `Wide` o `Narrow` de la enumeración `VbStrConv`, que no son aplicables a la configuración regional especificada.  
  
### Para corregir este error  
  
1.  Quite `VbStrConv.Wide` o `VbStrConv.Narrow`.
