# Soluci&#243;n de problemas de excepciones: System.ServiceModel.MessageHeaderException
Se produce una excepción <xref:System.ServiceModel.MessageHeaderException> cuando no se cumplen las expectativas con respecto a los encabezados de un mensaje SOAP al procesar el mensaje.
