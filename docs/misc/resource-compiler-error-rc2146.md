# Error del compilador de recursos RC2146
Falta una coma en la instrucción LANGUAGE  
  
 Falta la coma que separa el idioma principal y el secundario.  
  
 La instrucción **LANGUAGE** debe usar la sintaxis siguiente:  
  
 **LANGUAGE** *Id\_idioma\_primario*,*Id\_idioma\_secundario*
