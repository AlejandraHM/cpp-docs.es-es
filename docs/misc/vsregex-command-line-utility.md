# Utilidad de l&#237;nea de comandos VsRegEx
La utilidad VsRegEx se ha reemplazado por la [Utilidad de CreateExpInstance](../Topic/CreateExpInstance%20Utility.md) en Visual Studio 2010 y versiones posteriores.  
  
## Comentarios  
 En Visual Studio 2008, la información de configuración sobre la instancia experimental del IDE se almacenaba en el registro del sistema. En Visual Studio 2010 y versiones posteriores, se almacena en el sistema de archivos.
