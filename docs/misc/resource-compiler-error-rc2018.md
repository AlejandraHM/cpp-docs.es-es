# Error del compilador de recursos RC2018
carácter 'hexnumber' desconocido  
  
 El carácter ASCII correspondiente al número hexadecimal dado aparece en el archivo de origen, pero no es un carácter válido.  
  
 Este error puede deberse a daños en el archivo de origen.
