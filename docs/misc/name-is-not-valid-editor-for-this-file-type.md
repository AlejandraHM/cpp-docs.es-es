# &lt;name&gt; is not valid editor for this file type.
Este error ocurre normalmente cuando se escribe un valor incorrecto para el argumento \/e:*editorname* en la **Ventana de comandos** o el cuadro **Buscar\/Comando**.  
  
### Para corregir este error  
  
1.  Compruebe la sintaxis correcta del argumento \/e:*editorname* en la documentación y vuelva a escribir el comando.
