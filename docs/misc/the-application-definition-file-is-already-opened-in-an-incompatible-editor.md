# El archivo de definici&#243;n de aplicaci&#243;n ya est&#225; abierto en un editor incompatible
**El archivo de definición de aplicación ya está abierto en un editor incompatible.  Cierre el otro editor y vuelva a cargar la página de propiedades del proyecto.**  
  
 Este error se produce al intentar abrir el archivo de definición de aplicación cuando ya está abierto en otro editor.  
  
### Para corregir este error  
  
1.  Cierre el otro editor.  
  
2.  Cierre el Diseñador de proyectos.  
  
3.  En el menú **Proyecto**, haga clic en **Propiedades** para volver a abrir el Diseñador de proyectos.
