# Soluci&#243;n de problemas de excepciones: System.ComponentModel.WarningException
Especifica una excepción que se controla como una advertencia en lugar de un error.  
  
## Comentarios  
 Si no se detecta esta excepción, se presenta al usuario en forma de mensaje de advertencia.
