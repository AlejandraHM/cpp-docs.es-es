# No se pudo analizar el c&#243;digo XAML: &lt;error&gt;
Existe un error en el archivo XML que se está usando.  
  
 **Id. de error:** BC31023  
  
### Para corregir este error  
  
-   Compruebe que el código XML esté bien formado, que sea válido y que el archivo no esté dañado.
