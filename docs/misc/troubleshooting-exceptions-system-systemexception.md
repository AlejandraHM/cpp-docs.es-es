# Soluci&#243;n de problemas de excepciones: System.SystemException
Define la clase base para las excepciones predefinidas en el espacio de nombres `System`.  
  
## Comentarios  
 Esta clase se proporciona como medio para diferenciar entre las excepciones que define el sistema y las excepciones que define las aplicaciones.
