# No se declar&#243; &#39;&lt;funci&#243;nMatem&#225;tica1&gt;&#39;
No se declaró '\<funciónMatemática1\>'. La función se movió a la clase System.Math y ahora se denomina 'funciónMatemática2'.  
  
 Varias funciones que eran intrínsecas de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] en versiones anteriores se movieron al espacio de nombres <xref:System.Math?displayProperty=fullName>. Esto hace que sus características estén disponibles de manera más general para todos los lenguajes de programación.  
  
 **Identificador de error:** BC30819  
  
### Para corregir este error  
  
-   Use los métodos declarados en <xref:System.Math?displayProperty=fullName>.
