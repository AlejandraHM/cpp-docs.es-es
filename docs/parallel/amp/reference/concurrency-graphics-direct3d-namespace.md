# Concurrency::graphics::direct3d (Espacio de nombres)
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Provee los métodos [get\_texture](../Topic/get_texture%20Function.md) y [make\_texture](../Topic/make_texture%20Function.md).  
  
## Sintaxis  
  
```  
namespace direct3d;  
```  
  
## Miembros  
  
### Funciones  
  
|Name|Descripción|  
|----------|-----------------|  
|[get\_sampler \(Función\)](../Topic/get_sampler%20Function.md)|Obtenga la interfaz de estado de muestra de Direct3D en la vista de acelerador proporcionada que representa el objeto de ejemplo especificado.|  
|[get\_texture \(Función\)](../Topic/get_texture%20Function.md)|Obtiene la interfaz de textura de Direct3D que subyace al objeto [textura](../../../parallel/amp/reference/texture-class.md) especificado.|  
|[make\_sampler \(Función\)](../Topic/make_sampler%20Function.md)|Cree una muestra de un puntero de interfaz de estado de muestra Direct3D.|  
|[make\_texture \(Función\)](../Topic/make_texture%20Function.md)|Crea un objeto [texture](../../../parallel/amp/reference/texture-class.md) utilizando los parámetros especificados.|  
|[msad4 \(Función\)](../Topic/msad4%20Function.md)|Compara un valor de referencia de 4 bytes y un valor de origen de 8 bytes y acumula un vector de 4 sumas.|  
  
## Requisitos  
 **Encabezado:** amp\_graphics.h  
  
 **Espacio de nombres:** Concurrency::graphics
