##  <a name="address_mode"></a>  address_mode (enumeración)  
 Tipo de enumeración que se use para indicar los modos de dirección admitidos para el muestreo de textura.  
  
```  
enum address_mode;  
```  
  
##  <a name="filter_mode"></a>  filter_mode (enumeración)  
 Tipo de enumeración se utiliza para denotar modos de filtro que se admite para el muestreo de textura.  
  
```  
enum filter_mode;  
```
