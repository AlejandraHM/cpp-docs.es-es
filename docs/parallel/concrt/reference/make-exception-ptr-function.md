# <a name="makeexceptionptr-function"></a>Función make_exception_ptr

## <a name="syntax"></a>Sintaxis

```
template<class _E>
exception_ptr make_exception_ptr(_E _Except);
```

#### <a name="parameters"></a>Parámetros

*_E*<br/>
Tipo de excepción.

*_Except*<br/>
Valor de la excepción.
