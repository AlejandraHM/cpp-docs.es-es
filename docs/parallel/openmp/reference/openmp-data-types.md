# OpenMP Data Types
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

proporciona vínculos a los tipos de datos utilizados en el OpenMP API.  
  
 Implementación de Visual C\+\+ del estándar de OpenMP incluye los siguientes tipos de datos.  
  
|Tipo de datos|Descripción|  
|-------------------|-----------------|  
|[omp\_lock\_t](../../../parallel/openmp/reference/omp-lock-t.md)|Un tipo que contiene el estado de un bloqueo, si el bloqueo está disponible o si un subproceso posee un bloqueo.|  
|[omp\_nest\_lock\_t](../../../parallel/openmp/reference/omp-nest-lock-t.md)|Un tipo que contiene uno de la siguiente información sobre un bloqueo: si el bloqueo está disponible, y la identidad del subproceso que posee el bloqueo y un recuento de anidamiento.|
