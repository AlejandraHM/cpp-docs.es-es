# <a name="openmp-data-types"></a>Tipos de datos de OpenMP

Proporciona vínculos a los tipos de datos utilizados en la API de OpenMP.

La implementación de Visual C++ del estándar OpenMP incluye los siguientes tipos de datos.

|Tipo de datos|Descripción|
|---------------|-----------------|
|[omp_lock_t](../../../parallel/openmp/reference/omp-lock-t.md)|Un tipo que contiene el estado de un bloqueo, si el bloqueo esté disponible o si un subproceso posee un bloqueo.|
|[omp_nest_lock_t](../../../parallel/openmp/reference/omp-nest-lock-t.md)|Un tipo que contiene uno de los siguientes fragmentos de información sobre un bloqueo: si el bloqueo esté disponible y la identidad del subproceso que posee el bloqueo y un recuento de anidamiento.|
