# <a name="ompgetthreadnum"></a>omp_get_thread_num
Devuelve el número de subprocesos de la ejecución de subprocesos dentro de su equipo de subproceso.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
int omp_get_thread_num( );  
```
