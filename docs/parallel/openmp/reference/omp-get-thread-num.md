# omp_get_thread_num
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Devuelve el número de subprocesos del subproceso que se ejecuta dentro del equipo de subproceso.  
  
## Sintaxis  
  
```  
int omp_get_thread_num( );  
```
