# OpenMP in Visual C++
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La interfaz de programación de aplicaciones de OpenMP c y C\+\+ permite escribir aplicaciones que utilizan eficazmente varios procesadores.  Visual C\+\+ admite el estándar de OpenMP 2,0.

 [Library Reference](../../parallel/openmp/reference/openmp-library-reference.md)
