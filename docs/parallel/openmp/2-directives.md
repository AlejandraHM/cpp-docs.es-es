# 2. Directives
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Las directivas se basan en las directivas de **\#pragma** definido en las normas de c y C\+\+.  Los compiladores que admiten el OpenMP C y C\+\+ API tendrán una opción de la línea de comandos que active y permite la interpretación de todas las directivas de compilador de OpenMP.
