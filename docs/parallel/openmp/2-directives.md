# <a name="2-directives"></a>2. Directivas
Las directivas se basan en **#pragma** directivas definidas en las normas de C y C++.  Los compiladores que admiten OpenMP C y C++ API incluirá una opción de línea de comandos que activa y permite la interpretación de todas las directivas de compilador de OpenMP.
