# C.1 Notation
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Reglas gramaticales consisten en el nombre de un no terminal, seguido de dos puntos, seguido de alternativas de reemplazo en líneas independientes.  
  
 El término sintáctico de expresiónopt indica que el término es opcional dentro de reemplazo.  
  
 *El término*sintáctico de laoptseq expresión es equivalente *a término\-seqopt*con reglas adicionales siguientes:  
  
 *término\-seq* :  
  
 *término*  
  
 *término término\-seq*  
  
 *término\-seq* , *término*
