# <a name="exitsuccess-exitfailure"></a>EXIT_SUCCESS, EXIT_FAILURE

## <a name="required-header"></a>Encabezado necesario

```c
#include <stdlib.h>
```

## <a name="remarks"></a>Comentarios

Estos son los argumentos para las funciones [exit](reference/exit-exit-exit.md) y [_exit](reference/exit-exit-exit.md) y los valores devueltos para las funciones [atexit](reference/atexit.md) y [_onexit](reference/onexit-onexit-m.md).

|Constante|Valor definido|
|-|-|
|EXIT_SUCCESS|0|
|EXIT_FAILURE|1|
