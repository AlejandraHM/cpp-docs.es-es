# <a name="exitsuccess-exitfailure"></a>EXIT_SUCCESS, EXIT_FAILURE
## <a name="syntax"></a>Sintaxis  
  
```  
  
#include <stdlib.h>  
```  
  
## <a name="remarks"></a>Comentarios  
 Estos son los argumentos para las funciones **exit** y `_exit` y los valores devueltos para las funciones `atexit` y `_onexit`.
