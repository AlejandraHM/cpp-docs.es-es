# Asignaciones de texto gen&#233;rico
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Para simplificar el código de escritura para los mercados internacionales, las asignaciones de genérico\- texto se definen en TCHAR.H para:  
  
-   [Tipos de datos](../c-runtime-library/data-type-mappings.md)  
  
-   [Constantes y variables globales](../c-runtime-library/constant-and-global-variable-mappings.md)  
  
-   [Asignaciones rutinarias](../c-runtime-library/routine-mappings.md)  
  
 Para obtener más información, vea [Mediante asignaciones de Genérico\- texto](../c-runtime-library/using-generic-text-mappings.md).  Las asignaciones de Genérico\- texto son extensiones de Microsoft que no son compatibles con ANSI.
