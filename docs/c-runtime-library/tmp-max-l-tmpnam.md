# TMP_MAX, L_tmpnam
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxis  
  
```  
#include <stdio.h>  
```  
  
## Comentarios  
 `TMP_MAX` es el número máximo de nombres de archivo únicos que la función de `tmpnam` puede generar.  `L_tmpnam` es la longitud de los nombres de archivo temporales generados por `tmpnam`.
