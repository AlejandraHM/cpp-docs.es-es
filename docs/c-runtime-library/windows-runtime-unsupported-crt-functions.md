# <a name="windows-runtime-unsupported-crt-functions"></a>Funciones de CRT no compatibles con Windows Runtime

Muchas API en tiempo de ejecución de C (CRT) no se pueden usar en aplicaciones de la Plataforma universal de Windows que se ejecutan en Windows Runtime. Estas aplicaciones se crean con la marca de compilador/ZW. Para obtener una lista de funciones de CRT no compatibles, vea [Funciones de CRT no admitidas en aplicaciones de la Plataforma universal de Windows](../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

Todas las API en CRT se describen en la sección [Referencia alfabética de funciones](../c-runtime-library/reference/crt-alphabetical-function-reference.md) de la documentación.
