# _CIatan
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Calcula el arco tangente del valor superior de la pila.  
  
## Sintaxis  
  
```  
void __cdecl _CIatan();  
```  
  
## Comentarios  
 Esta versión de la función de `atan` tiene una convención de llamada especializada que el compilador entienda.  Acelera la ejecución porque evita que las copias genera y contribuye a la asignación del registro.  
  
 El valor resultante se incrusta en la parte superior de la pila.  
  
## Requisitos  
 **Plataforma:** x86
