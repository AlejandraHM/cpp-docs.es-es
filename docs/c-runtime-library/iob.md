# <a name="iob"></a>_iob
Matriz de estructuras de control de stdio.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## <a name="remarks"></a>Comentarios  
 `IOB_ENTRIES` se define como 20 en stdio.h.
