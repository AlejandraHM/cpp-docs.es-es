# _FREEENTRY, _USEDENTRY
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxis  
  
```  
#include <malloc.h>  
```  
  
## Comentarios  
 Estas constantes representan los valores asignados por las rutinas de `_heapwalk` al elemento de **\_useflag** de la estructura de **\_HEAPINFO** .  Indican el estado de la entrada de la pila.
