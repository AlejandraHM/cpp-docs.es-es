# <a name="tznamemax"></a>TZNAME_MAX

**Obsoleto**. La longitud máxima permitida de la cadena de una variable de nombre de zona horaria. Esta macro estaba definida en \<limits.h> Visual Studio 2012 y versiones anteriores. No está definida en Visual Studio 2013 ni versiones posteriores. Para obtener la longitud requerida para contener el nombre de la zona horaria actual, use [_get_tzname](../c-runtime-library/reference/get-tzname.md).

## <a name="syntax"></a>Sintaxis

```
#include <limits.h>
```
