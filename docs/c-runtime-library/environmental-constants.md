# Constantes de entorno
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxis  
  
```  
#include <stdlib.h>  
```  
  
## Comentarios  
 Esta constante define la longitud del entorno para las cadenas.  
  
|Constante|Significado|  
|---------------|-----------------|  
|`_MAX_ENV`|Tamaño máximo de la cadena de una cadena ambiental.|
