# <a name="randmax"></a>RAND_MAX
## <a name="syntax"></a>Sintaxis  
  
```  
  
#include <stdlib.h>  
  
```  
  
## <a name="remarks"></a>Comentarios  
 La constante `RAND_MAX` es el valor máximo que la función `rand` puede devolver. `RAND_MAX` se define como el valor 0x7fff.
