# Manipulaci&#243;n de cadenas (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Estas rutinas funcionan en cadenas de caracteres de un solo byte terminadas en NULL, de caracteres anchos y de caracteres multibyte.  Use las rutinas de control del búfer, descritas en [Control del búfer](../c-runtime-library/buffer-manipulation.md), para trabajar con matrices de caracteres que no finalizan con un carácter nulo.
