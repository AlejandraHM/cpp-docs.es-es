# putenv
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta función POSIX está desusada.  Use en su lugar las funciones compatibles con ISO C\+\+ [\_putenv](../../c-runtime-library/reference/putenv-wputenv.md) o [\_putenv\_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md), de seguridad mejorada.  
  
> [!IMPORTANT]
>  Esta API no se puede usar en aplicaciones que se ejecutan en Windows en tiempo de ejecución.  Para obtener más información, vea [Funciones de CRT no admitidas con \/ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
