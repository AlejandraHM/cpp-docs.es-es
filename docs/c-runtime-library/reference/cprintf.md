# cprintf
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta función POSIX está desusada.  En su lugar, use la función [\_cprintf](../../c-runtime-library/reference/cprintf-cprintf-l-cwprintf-cwprintf-l.md) compatible con ISO C\+\+, o la función [\_cprintf\_s](../../c-runtime-library/reference/cprintf-s-cprintf-s-l-cwprintf-s-cwprintf-s-l.md) con mejoras de seguridad.  
  
> [!IMPORTANT]
>  Esta API no se puede usar en aplicaciones que se ejecutan en Windows en tiempo de ejecución.  Para obtener más información, vea [Funciones de CRT no admitidas con \/ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
