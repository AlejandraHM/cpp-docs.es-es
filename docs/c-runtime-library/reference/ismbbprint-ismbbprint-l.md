# <a name="ismbbprint-ismbbprintl"></a>_ismbbprint, _ismbbprint_l

Determina si el carácter multibyte especificado es un carácter imprimible.

## <a name="syntax"></a>Sintaxis

```C
int _ismbbprint(
   unsigned int c
);
int _ismbbprint_l(
   unsigned int c,
   _locale_t locale
);
```

### <a name="parameters"></a>Parámetros

*c*<br/>
Entero que se va a probar.

*locale*<br/>
Configuración regional que se va a usar.

## <a name="return-value"></a>Valor devuelto

**_ismbbprint** devuelve un valor distinto de cero si la expresión:

`isprint(c) || _ismbbkprint(c)`

es distinto de cero para *c*, o 0 si no lo está. **_ismbbprint** usa la configuración regional actual para cualquier comportamiento dependiente de la configuración regional. **_ismbbprint_l** es idéntica, salvo que usa la configuración regional que se pasa en su lugar. Para obtener más información, vea [Locale](../../c-runtime-library/locale.md).
