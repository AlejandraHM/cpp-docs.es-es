# <a name="visual-c-features-deprecated-in-visual-studio-2019-preview"></a>Características de Visual C++ en desuso en la versión preliminar de Visual Studio 2019

## <a name="support-for-windows-xp-development"></a>Compatibilidad con el desarrollo de Windows XP

La versión de Visual C++ en Visual Studio 2019 Preview 0 y posterior no incluye compatibilidad con la creación de código en Windows XP mediante el uso del conjunto de herramientas v142. La compatibilidad con el desarrollo de Windows XP mediante el uso del conjunto de herramientas v141 que se incluye en Visual Studio 2017 está disponible como componente opcional en el Instalador de Visual Studio. Para información sobre cómo instalar el conjunto de herramientas de la plataforma Windows XP v141, consulte [Configuración de programas para Windows XP](../build/configuring-programs-for-windows-xp.md).
