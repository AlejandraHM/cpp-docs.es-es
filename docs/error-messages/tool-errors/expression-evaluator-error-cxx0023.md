# <a name="expression-evaluator-error-cxx0023"></a>Error del evaluador de expresiones CXX0023
raíz errónea  
  
 El evaluador de expresiones de C no reconoce la base especificada. Solo son válidas las raíces octales, hexadecimales y decimales.  
  
 Este error es idéntico a CAN0023.
