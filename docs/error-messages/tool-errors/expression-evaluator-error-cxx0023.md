# Error del evaluador de expresiones CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

base errónea  
  
 El evaluador de la expresión C no reconoce la base especificada.  Sólo son válidas las bases decimales, hexadecimales y octales.  
  
 Este error es idéntico a CAN00023.
