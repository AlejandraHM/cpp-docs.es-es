# <a name="linker-tools-error-lnk1166"></a>Error de las herramientas del vinculador LNK1166

no se puede ajustar el código en offset = offset, va = value

VÍNCULO no pudo rellenar el código según sea necesario.

Algunas instrucciones no pueden cruzar los límites de página en algunos procesadores. VÍNCULO intenta agregar paneles para corregir esta situación. En este caso, el vínculo no funcionara solucionar el problema.
