# Error de las herramientas del vinculador LNK1166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede ajustar el código en offset\=offset, va\=value  
  
 LINK no pudo rellenar el código según lo requerido.  
  
 No se permite a algunas instrucciones cruzar límites de página en ciertos procesadores.  LINK intenta agregar bytes de relleno para corregir esta situación.  En este caso, LINK no pudo resolver el problema.
