# <a name="linker-tools-warning-lnk4199"></a>Advertencia de las herramientas del vinculador LNK4199

/DELAYLOAD:dllname omitida; no se encontró importación desde dllname

El vinculador omite `dllname` porque no necesita ninguna de las funciones que `dllname` exporta.
