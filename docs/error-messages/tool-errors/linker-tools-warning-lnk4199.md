# Advertencia de las herramientas del vinculador LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se ha omitido \/DELAYLOAD:nombrededll; no se encontró ninguna importación desde nombrededll  
  
 El vinculador omite `dllname` porque no necesita ninguna de las funciones que exporta `dllname`.
