# Error irrecuperable del compilador de recursos RC1019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#else' inesperada  
  
 La directiva `#else` no apareció en una construcción `#if`, **\#ifdef** o **\#ifndef**.  
  
 Asegúrese de que hay una instrucción `#if`, **\#ifdef** o **\#ifndef** en vigor antes de esta instrucción.
