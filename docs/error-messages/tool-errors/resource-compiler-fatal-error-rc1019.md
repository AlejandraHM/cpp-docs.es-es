# <a name="resource-compiler-fatal-error-rc1019"></a>Error irrecuperable del compilador de recursos RC1019
inesperado ' #else '  
  
 El `#else` directiva no apareció dentro de un `#if`, **#ifdef**, o **#ifndef** construir.  
  
 Asegúrese de que hay un `#if`, **#ifdef**, o **#ifndef** instrucción en vigor antes de esta instrucción.
