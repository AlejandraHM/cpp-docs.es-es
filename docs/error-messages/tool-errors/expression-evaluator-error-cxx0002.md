# Error del evaluador de expresiones CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de acceso a la memoria de usuario  
  
 La expresión intenta hacer referencia a memoria que no está asignada al programa en depuración.  
  
 Este error es idéntico a CAN0002.
