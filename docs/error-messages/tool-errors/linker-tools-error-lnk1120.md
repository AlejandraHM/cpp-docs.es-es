# Error de las herramientas del vinculador LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

número externos sin resolver  
  
 El error LNK1120 da un número \(`number`\) de externos sin resolver para este vínculo.  Las circunstancias que causan externos sin resolver se describen en el error [LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md), que precede a este mensaje de error \(uno por cada externo sin resolver\).  
  
 Para obtener más información, vea [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;815650](http://support.microsoft.com/default.aspx?scid=kb;en-us;815650).
