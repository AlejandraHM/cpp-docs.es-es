# Error grave de NMAKE U1034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de sintaxis : falta separador  
  
 Falta el signo de dos puntos \(**:**\) que separa destinos y dependientes.
