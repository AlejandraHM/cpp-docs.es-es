# Error del compilador de recursos RC2169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el nombre del archivo de recursos no tiene el formato de 2.03  
  
 El recurso especificado ha utilizado un formato anterior a la versión 2.03.  El archivo de recursos se debe convertir o volver a crear usando el formato para la versión 3.00 o posterior.
