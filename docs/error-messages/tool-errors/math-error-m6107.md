# Error matem&#225;tico M6107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no emulado  
  
 Se ha intentado ejecutar una instrucción del coprocesador no válida o no admitida por el emulador.  
  
 El programa termina con el código de salida 135.
