# Error irrecuperable del compilador de recursos RC1102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error interno : demasiados argumentos para RCPP  
  
 Se pasaron demasiados argumentos al preprocesador del compilador de recursos.  Reduzca el número de símbolos definidos con la opción \/d \(Definir símbolos\) definiéndolos en el código fuente.  Este error también puede deberse a que se especifican demasiadas rutas de búsqueda de archivos de inclusión utilizando la opción \/i \(Incluir la ruta de búsqueda\).
