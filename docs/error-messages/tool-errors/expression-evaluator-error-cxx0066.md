# Error del evaluador de expresiones CXX0066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el miembro estático no está presente  
  
 No se puede encontrar un miembro estático de una clase o no se ha definido.  Este error puede aparecer por un miembro estático de clase declarado pero no definido, o sólo definido y al que se hace referencia en módulos que no contienen información de depuración.  
  
 Este error es idéntico a CAN0066.
