# <a name="project-build-error-prj0023"></a>Error PRJ0023 al compilar el proyecto
'Herramienta', la propiedad desconocida contiene el nombre de archivo no válido 'archivo'.  
  
 El nombre de archivo ***archivo***, especificado para la herramienta, ***herramienta***, no era válido.  
  
 Es probable que haya usado una macro desconocida o no es válida.  
  
 Vea [establecer las propiedades de un proyecto de Visual C++](../../ide/working-with-project-properties.md) para obtener información sobre cómo obtener acceso a las propiedades del proyecto.
