# <a name="linker-tools-error-lnk1248"></a>Error de las herramientas del vinculador LNK1248

> tamaño de imagen ('*tamaño*') supera el tamaño máximo permitido (80000000)

El vinculador determinó que el tamaño del archivo de salida superará el tamaño más grande posible para una imagen de programa de 32 bits. Es posible que desee que su programa en varios archivos DLL.
