# Error grave de NMAKE U1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de sintaxis : falta '\)' en llamada de macro  
  
 Un paréntesis izquierdo, **\(**, ha aparecido sin su correspondiente paréntesis derecho, **\)**, en una llamada de macro.  El formato correcto es **$\(***nombre***\)**; `$`*n* está permitido para nombres de un carácter.
