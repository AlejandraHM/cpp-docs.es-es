# <a name="nmake-fatal-error-u1000"></a>Error grave de NMAKE U1000
error de sintaxis: ')' falta en la llamada de macro  
  
 Un paréntesis izquierdo, **(**, aparecido sin su correspondiente paréntesis derecho, **)**, en una llamada de macro. El formato correcto es **$(***nombre***)**; `$` *n* está permitido para nombres de un carácter.
