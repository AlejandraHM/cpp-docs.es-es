# <a name="linker-tools-error-lnk1332"></a>Error de las herramientas del vinculador LNK1332
detectado\<recuento > tipos de Windows Runtime importadas en un módulo y definido en otro módulo  
  
 Cuando genera el destino actual, el vinculador detectó <`count`> tipos de Windows en tiempo de ejecución, cada uno de los cuales se importa en un módulo y también está definido en otro módulo.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Corregir cada uno de los errores de LNK2039 en la compilación de acuerdo con la sugerencia en el mensaje de error.
