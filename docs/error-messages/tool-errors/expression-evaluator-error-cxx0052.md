# Error del evaluador de expresiones CXX0052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la función miembro no está presente  
  
 Se ha especificado una función miembro como punto de interrupción pero no se pudo encontrar.  Este error puede ocurrir por definir un punto de interrupción en una función que se ha dividido en líneas.  
  
 Vuelva a compilar el archivo dejando de forzar la división entre líneas \(\/Ob0\) para definir un punto de interrupción.  
  
 Una expresión llamó a una función que no estaba definida.  
  
 Este error es idéntico a CAN00052.
