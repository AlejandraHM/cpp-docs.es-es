# Error del evaluador de expresiones CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no es puntero struct  
  
 Se ha aplicado el operador de la miembro\- selección \(**\-\>**\) a una expresión que no es un puntero a una estructura.  
  
 Compruebe que toda la expresión tenga los paréntesis correctos, o convierta el tipo de expresión de la dirección al tipo de puntero de estructura correcto.  
  
 Este error es idéntico a CAN00029.
