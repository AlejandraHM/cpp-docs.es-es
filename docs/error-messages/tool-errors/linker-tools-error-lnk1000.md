# Error de las herramientas del vinculador LNK1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error desconocido; consulte la documentación para conocer las opciones de soporte técnico  
  
 Anote las circunstancias del error, intente aislar el problema y crear un caso de prueba reproducible y, a continuación, póngase en contacto con `Microsoft Product Support Services`.  Para obtener información sobre cómo investigar y crear informes sobre estos errores, [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;134650](http://support.microsoft.com/default.aspx?scid=kb;en-us;134650)vea.  
  
 Este error se puede producir si mezcla archivos de encabezado estándar \(por ejemplo, dos.h\) y sus propios archivos.  `#include` los archivos de encabezado estándar primero, seguidos de sus propios archivos de encabezado.
