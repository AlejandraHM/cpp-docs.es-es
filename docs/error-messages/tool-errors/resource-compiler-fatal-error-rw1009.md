# Error irrecuperable del compilador de recursos RW1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Error al crear el nombre de recurso  
  
 No se pudo crear el archivo .RES especificado.  Asegúrese de que no se esté creando en una unidad de sólo lectura.  Utilice la opción \/V para averiguar si el archivo se está creando.
