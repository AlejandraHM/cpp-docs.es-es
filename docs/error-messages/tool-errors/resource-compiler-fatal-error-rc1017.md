# Error irrecuperable del compilador de recursos RC1017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expresión constante de tipo entero no válida  
  
 La expresión en una directiva `#if` no existe o no se evalúa como una constante.
