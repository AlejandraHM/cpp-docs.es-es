# Error del evaluador de expresiones CXX0067
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

evaluación de función no compatible  
  
 La expresión contiene una llamada de función.  Algunos evaluadores de expresiones no admiten llamadas de función.  
  
 Este error es idéntico a CAN0067.
