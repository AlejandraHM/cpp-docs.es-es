# Errores y advertencias de compilaci&#243;n del proyecto (PRJxxxx)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta sección hace referencia a los errores generados por las herramientas de compilación.  Para obtener ayuda sobre un mensaje de error concreto, haga clic con el mouse en el número de error mostrado en la Ventana de salida y presione F1, o escriba el número de error en el cuadro **Buscar** en el Índice.
