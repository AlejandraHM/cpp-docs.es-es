# Error irrecuperable del compilador de recursos RW1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Error de E\/S al escribir en el archivo**  
  
 El compilador de recursos no pudo escribir en un archivo.  
  
### Posibles causas del error:  
  
1.  No hay suficiente espacio en el disco.  El espacio disponible debe ser al menos igual al doble del tamaño del archivo ejecutable que se crea.  
  
2.  El volumen es de sólo lectura.  
  
3.  Sector defectuoso.  
  
4.  Infracción de uso compartido.
