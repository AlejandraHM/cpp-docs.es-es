# Error del evaluador de expresiones CXX0020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipos de operando erróneos para esta operación  
  
 Se aplicó un operador a una expresión con un tipo no válido para ese operador.  
  
 Por ejemplo, no es valido tomar la dirección de un registro o subíndices de una matriz con una expresión de punto flotante.  
  
 Este error es idéntico a CAN00020.
