# <a name="expression-evaluator-error-cxx0020"></a>Error del evaluador de expresiones CXX0020

tipos de operando incorrectos para esta operación

Se aplicó un operador en una expresión con un tipo no válido para ese operador.

Por ejemplo, no es válido para tomar la dirección de un registro o subíndices de matriz con una expresión de punto flotante.

Este error es idéntico a CAN0020.
