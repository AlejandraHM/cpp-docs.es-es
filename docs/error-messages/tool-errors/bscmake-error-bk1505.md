# Error de BSCMAKE BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede leer el archivo 'nombredearchivo' \[: razón\]  
  
 BSCMAKE no puede leer en el archivo.  
  
### Posibles causas del error:  
  
1.  Archivo dañado.  
  
2.  Truncamiento de archivo en el compilador sin espacio en disco o interrumpido durante la creación del archivo .sbr.
