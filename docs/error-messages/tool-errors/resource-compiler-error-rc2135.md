# Error del compilador de recursos RC2135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**archivo no encontrado: '**   
 ***nombredearchivo* '**  
  
 No se encontró el archivo especificado en la línea de comandos del compilador de recursos.  Compruebe si se ha movido el archivo a otro directorio y si se ha escrito correctamente el nombre de archivo y la ruta de acceso.  
  
 Los archivos se buscan en función de si utilizan la variable de entorno **INCLUDE** o la configuración **INCLUDE** de Visual C\+\+.
