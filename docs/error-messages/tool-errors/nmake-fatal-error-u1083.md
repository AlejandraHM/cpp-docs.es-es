# Error grave de NMAKE U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la macro de destino 'target' se expande a nada  
  
 El destino dado es una llamada de una macro que no se ha definido o tiene un valor nulo.  NMAKE no puede procesar un destino nulo.
