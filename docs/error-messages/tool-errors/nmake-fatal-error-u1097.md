# <a name="nmake-fatal-error-u1097"></a>Error grave de NMAKE U1097
sintaxis de partes del nombre de archivo requiere dependientes  
  
 La dependencia actual no tiene un dependiente explícito o implícito. Sintaxis de partes del nombre de archivo, que utiliza el porcentaje (`%`) especificador, representa componentes del primer dependiente del destino actual.
