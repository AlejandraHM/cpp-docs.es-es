# Error grave de NMAKE U1097
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la sintaxis de partes del nombre de archivo requiere dependientes  
  
 La dependencia actual no tiene un dependiente explícito ni implícito.  La sintaxis de partes del nombre de archivo, que utiliza el especificador \(`%`\), representa componentes del primer dependiente del destino actual.
