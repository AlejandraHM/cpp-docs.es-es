# Error del evaluador de expresiones CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

secuencia de llamada no compatible  
  
 Una función especificada en la expresión usa una secuencia de llamada no compatible con el evaluador de la expresión de C.  Esta función no se puede llamar en una expresión de la ventana Inspección.  
  
 Este error es idéntico a CAN00048.
