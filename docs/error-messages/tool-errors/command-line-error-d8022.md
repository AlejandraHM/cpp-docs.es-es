# <a name="command-line-error-d8022"></a>Error de la línea de comandos D8022

no se puede abrir 'archivo ' de mensajes

El archivo indicado era no en el directorio actual o en un directorio especificado en la variable de entorno PATH. El archivo de mensaje contiene un breve resumen de sintaxis de línea de comandos del compilador y las opciones.

Mueva este archivo en el directorio actual o un directorio en la ruta de acceso actual. Si no se encuentra este archivo, ejecute el programa de instalación para copiarlo desde los discos de distribución.
