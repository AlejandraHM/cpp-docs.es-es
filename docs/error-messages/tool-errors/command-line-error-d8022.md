# Error de la l&#237;nea de comandos D8022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede abrir 'archivo de mensaje'  
  
 El archivo dado no se encuentra en el directorio actual o en un directorio especificado en la variable de entorno PATH.  El archivo de mensajes contiene un resumen de la sintaxis y las opciones de la línea de comandos del compilador.  
  
 Mueva este archivo al directorio actual o a un directorio de la ruta de acceso actual.  Si no se encuentra este archivo, ejecute el programa de instalación para copiarlo de los discos de distribución.
