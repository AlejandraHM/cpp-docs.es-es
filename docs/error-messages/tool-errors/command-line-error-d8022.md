# <a name="command-line-error-d8022"></a>Error de la línea de comandos D8022
no se puede abrir 'archivo ' de mensajes  
  
 El archivo especificado era no está en el directorio actual o en un directorio especificado en la variable de entorno PATH. El archivo de mensaje contiene un breve resumen de opciones y la sintaxis de línea de comandos del compilador.  
  
 Mueva este archivo en el directorio actual o un directorio en la ruta de acceso actual. Si no se encuentra este archivo, ejecute el programa de instalación para copiar desde los discos de distribución.
