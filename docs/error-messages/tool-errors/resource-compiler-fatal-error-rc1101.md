# <a name="resource-compiler-fatal-error-rc1101"></a>Error irrecuperable del compilador de recursos RC1101
ningún nombre de recurso binario archivo especificado  
  
 El resultado de cambiar el nombre (/ fo) no siguió la opción por un nombre de archivo.  
  
 Use la siguiente sintaxis para la opción/fo:  
  
 **/Fo RC**   
 ***archivo_salida archivo_entrada.*  rc**
