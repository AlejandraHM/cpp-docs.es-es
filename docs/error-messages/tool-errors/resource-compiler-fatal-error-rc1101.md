# Error irrecuperable del compilador de recursos RC1101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se ha especificado un nombre de archivo binario de recursos  
  
 La opción \/fo \(Cambiar el nombre de resultados\) no iba seguida de un nombre de archivo.  
  
 Utilice la siguiente sintaxis para la opción \/fo:  
  
 **RC \/fo**   
 ***archivo\_salida archivo\_entrada.* rc**
