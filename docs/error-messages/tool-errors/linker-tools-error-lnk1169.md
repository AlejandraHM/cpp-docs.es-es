# Error de las herramientas del vinculador LNK1169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se encontraron uno o más símbolos definidos simultáneamente  
  
 La compilación produjo un error debido a la existencia de varias definiciones para uno o varios símbolos.  Este error viene precedido del error [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 La opción [\/FORCE](../../build/reference/force-force-file-output.md) o **\/FORCE:MULTIPLE** reemplaza este error.
