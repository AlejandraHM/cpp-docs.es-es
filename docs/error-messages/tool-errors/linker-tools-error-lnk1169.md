# <a name="linker-tools-error-lnk1169"></a>Error de las herramientas del vinculador LNK1169
uno o varios multiplicación encontrado los símbolos definidos  
  
 Error en la compilación debido a varias definiciones de uno o más símbolos. Este error es precedido del error [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 El [/FORCE](../../build/reference/force-force-file-output.md) o **/Force: Multiple** opción la reemplaza este error.
