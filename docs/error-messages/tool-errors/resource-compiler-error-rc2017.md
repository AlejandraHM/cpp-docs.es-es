# <a name="resource-compiler-error-rc2017"></a>Error del compilador de recursos RC2017

secuencia de escape no válida

Una secuencia de escape apareció donde no se esperaba.

Una secuencia de escape, una barra diagonal inversa ( **\\** ) seguido por un número o una letra, sólo puede aparecer en un carácter o cadena constante.
