# Error del compilador de recursos RC2017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

secuencia de escape no válida  
  
 Aparece una secuencia de escape donde no se esperaba.  
  
 Una secuencia de escape, una barra diagonal inversa \( **\\** \) seguida de un número o de una letra, sólo puede aparecer en una constante de caracteres o de cadena.
