# Error del evaluador de expresiones CXX0011
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta corchete derecho  
  
 La expresión contiene corchetes no coincidentes.  Vuelva a escribir la expresión con corchetes coincidentes.  
  
 Este error es idéntico a CAN0011.
