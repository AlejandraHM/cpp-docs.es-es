# <a name="resource-compiler-fatal-error-rc1203"></a>Error irrecuperable del compilador de recursos RC1203
especificada el identificador de idioma predeterminado hexadecimal no válido.  
  
 Especificar idioma predeterminado (/ l) siguió la opción por un identificador de idioma hexadecimal no válido. Vea [cadenas de idioma y país o región](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) en el *referencia de la biblioteca de tiempo de ejecución* para obtener una lista de identificadores de idioma válidos.
