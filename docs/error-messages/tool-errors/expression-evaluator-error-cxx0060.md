# <a name="expression-evaluator-error-cxx0060"></a>Error del evaluador de expresiones CXX0060

registro no está disponible

Especificó un registro que no se puede usar en una expresión. Este error puede deberse a intentar obtener acceso a un registro que no existe en el equipo que ejecuta.

Este error es idéntico a CAN0060.
