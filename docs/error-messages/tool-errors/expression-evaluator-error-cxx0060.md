# Error del evaluador de expresiones CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

registro no disponible  
  
 Una expresión ha especificado un registro que no se puede usar.  La causa de este error puede ser el intento de acceso a un registro que no está en el equipo activo.  
  
 Este error es idéntico a CAN0060.
