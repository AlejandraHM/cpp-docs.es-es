# <a name="resource-compiler-fatal-error-rc1109"></a>Error irrecuperable del compilador de recursos RC1109
Error al crear el nombre del recurso  
  
 No se pudo crear el archivo .res especificado. Asegúrese de que no se está creando en una unidad de sólo lectura. Utilice la opción /V para averiguar si el archivo se está creando.
