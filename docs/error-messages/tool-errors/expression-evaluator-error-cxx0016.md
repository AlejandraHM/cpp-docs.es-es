# Error del evaluador de expresiones CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

constante demasiado grande  
  
 El evaluador de la expresión C no puede aceptar una constante entera no firmada mayor de 4.294.967.295 \(0FFFFFFFF hexadecimal\), o una constante de coma flotante cuya magnitud sea mayor de 1.8E\+308 aproximadamente.  
  
 Este error es idéntico a CAN0016.
