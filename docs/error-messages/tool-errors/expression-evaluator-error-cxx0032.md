# Error del evaluador de expresiones CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

división por cero  
  
 La expresión contiene un divisor de cero que es ilegal.  Este divisor puede ser incluso el número cero o puede ser una expresión equivalente.  
  
 Este error es idéntico a CAN00032.
