# <a name="expression-evaluator-error-cxx0032"></a>Error del evaluador de expresiones CXX0032

división por 0

La expresión contiene un divisor de cero, lo que no es válido. Este divisor puede ser incluso el número cero o una expresión que se evalúa como cero.

Este error es idéntico a CAN0032.
