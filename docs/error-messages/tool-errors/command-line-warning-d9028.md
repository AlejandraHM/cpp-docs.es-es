# Advertencia de la l&#237;nea de comandos D9028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de recompilación mínimo, revirtiendo a la compilación normal  
  
 El archivo .idb del proyecto está dañado.  Elimine el archivo y recompile.  
  
 Para obtener más información sobre recompilaciones mínimas y el archivo .idb, vea la [opción del compilador \/Gm](../../build/reference/gm-enable-minimal-rebuild.md).
