# Error del compilador de recursos RC2167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

campo VERSIONINFO no reconocido; se esperaba BEGIN o una coma  
  
 Se encontró un campo no reconocido en la parte **FIXED** de una declaración de estructura **VERSIONINFO**.  Un campo **VERSIONINFO** debe ser DWORDS separado por una coma.
