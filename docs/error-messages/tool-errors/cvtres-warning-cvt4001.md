# Advertencia de CVTRES CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**no se especificó el tipo de equipo; se supone**   
 ***tipo***  
  
 CVTRES no ha encontrado una especificación de equipo.  Se ha asumido el tipo de equipo dado.  Si el valor predeterminado no es correcto, ejecute CVTRES con la opción \/MACHINE.
