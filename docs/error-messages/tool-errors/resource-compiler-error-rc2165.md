# Error del compilador de recursos RC2165
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cadena no encontrada en la instrucción DLGINCLUDE  
  
 La instrucción no especificó un archivo de inclusión válido.  
  
 La instrucción **DLGINCLUDE** debe usar la siguiente sintaxis:  
  
 **DLGINCLUDE "**   
 ***nombredearchivo* .h"**
