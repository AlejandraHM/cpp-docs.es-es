# Error del evaluador de expresiones CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

llamadas al constructor no compatibles  
  
 Una expresión ha llamado a un constructor.  Las expresiones no pueden llamar explícitamente a constructores o realizar conversiones que requieran una llamada al constructor.  
  
 Este error es idéntico a CAN0062.
