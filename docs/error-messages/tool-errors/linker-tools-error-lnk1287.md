# Error de las herramientas del vinculador LNK1287
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

función de punto de entrada administrado no válido  
  
 El punto de entrada no es válido para una imagen administrada.  El tipo de valor devuelto de una función de punto de entrada administrado sólo puede ser `void` o `int`.  El tipo del parámetro de una función de punto de entrada administrado sólo puede ser `void` o `String []`.
