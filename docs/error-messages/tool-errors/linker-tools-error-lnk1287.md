# <a name="linker-tools-error-lnk1287"></a>Error de las herramientas del vinculador LNK1287

función de punto de entrada administrado no válido

El punto de entrada no es válido para una imagen administrada. Solo puede ser el tipo de valor devuelto de una función de punto de entrada administrado `void` o `int`. Solo puede ser el tipo del parámetro de una función de punto de entrada administrado `void` o `String []`.
