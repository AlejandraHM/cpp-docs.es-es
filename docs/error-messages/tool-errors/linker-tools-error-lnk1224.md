# Error de las herramientas del vinculador LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

dirección base de imagen no válida  
  
 Se especificó una dirección base de la imagen no válida.  Las direcciones base deben ser de 64KB alineados \(los últimos cuatro dígitos hexadecimales deben ser cero\) y la base de imagen debe ajustarse a un valor con signo o sin signo de 32 bits.
