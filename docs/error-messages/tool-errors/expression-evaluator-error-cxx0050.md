# Error del evaluador de expresiones CXX0050
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la parte izquierda de :: debe ser del tipo class\/struct\/union  
  
 El símbolo a la izquierda del operador de resolución de ámbito \(`::`\) no era una clase, una estructura ni una unión.  
  
 Este error es idéntico a CAN00050.
