# Error de las herramientas del vinculador LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

información de depuración dañada; compile de nuevo el módulo  
  
 Se puede producir este error porque la compilación finalizó antes de que se creara un archivo objeto válido.  
  
 Vuelva a compilar el archivo objeto dado.  Si eso no resuelve el problema, anote las circunstancias del error, intente aislar el problema y cree un caso de prueba reproducible.  A continuación, consulte el archivo de ayuda de soporte técnico o la sección correspondiente de alguno de los manuales.
