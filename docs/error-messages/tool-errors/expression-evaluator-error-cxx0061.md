# Error del evaluador de expresiones CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se superó el número de niveles de anidamiento de la función  
  
 El número de niveles de anidamiento de la función de la expresión supera el límite.  Modifique la expresión para reducir los niveles de anidamiento.  
  
 Este error es idéntico a CAN0061.
