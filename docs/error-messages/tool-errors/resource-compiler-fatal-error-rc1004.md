# <a name="resource-compiler-fatal-error-rc1004"></a>Error irrecuperable del compilador de recursos RC1004

final inesperado de archivo encontrado

Este error puede deberse a falta de avance de línea y caracteres de retorno de carro en la última línea de un archivo de texto.
