# <a name="linker-tools-warning-lnk4071"></a>Advertencia de las herramientas del vinculador LNK4071

no puede estar vinculado incrementalmente en vínculos subsiguientes

VÍNCULO puede encontrar varias definiciones para uno o más símbolos, pero [/FORCE](../../build/reference/force-force-file-output.md) o **/Force: Multiple** se usó para crear un archivo de salida independientemente de los errores. VÍNCULO elimina el archivo de estado incremental (.ilk).
