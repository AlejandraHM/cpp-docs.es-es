# <a name="expression-evaluator-error-cxx0018"></a>Error del evaluador de expresiones CXX0018
nombre de registro incorrecto  
  
 Falta un registro especificado no existe o no se puede mostrar.  
  
 La ventana Inspección puede mostrar los registros siguientes:  
  
 AX SP DS IP  
  
 BX BP ES FL  
  
 CX SI SS GS  
  
 DX DI CS SS  
  
 EAX ESP DS EIP  
  
 EBX EBP ES EFL  
  
 ECX ESI FS  
  
 EDX EDI CS  
  
 Este error es idéntico a CAN0018.
