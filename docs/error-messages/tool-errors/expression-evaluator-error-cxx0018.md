# Error del evaluador de expresiones CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nombre de registro erróneo  
  
 Falta un registro especificado o no se puede mostrar.  
  
 La ventana Inspección puede mostrar los siguientes registros:  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 Este error es idéntico a CAN0018.
