# Error del evaluador de expresiones CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Error: falta la información de tipos o es desconocida  
  
 El archivo .pch no se pudo vincular, o el código hace referencia a un tipo que está en un módulo no compilado con \/Zi.  
  
 Este error es idéntico a CAN0067.
