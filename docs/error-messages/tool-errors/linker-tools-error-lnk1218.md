# <a name="linker-tools-error-lnk1218"></a>Error de las herramientas del vinculador LNK1218
Advertencia tratada como error; no se generó ningún archivo de salida  
  
 Cuando se vincula con **/WX**, se tratará las advertencias del vinculador como errores y no se creará ningún archivo de salida.  
  
 Vea [/WX (tratar advertencias del vinculador como errores)](../../build/reference/wx-treat-linker-warnings-as-errors.md) para obtener más información.
