# Error de las herramientas del vinculador LNK1218
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

advertencia tratada como error; no se generó ningún archivo de salida  
  
 Cuando vincula con **\/WX**, todas las advertencias del vinculador se tratarán como errores y no se creará ningún archivo de salida.  
  
 Para obtener más información, vea [\/WX \(Tratar advertencias del vinculador como errores\)](../../build/reference/wx-treat-linker-warnings-as-errors.md).
