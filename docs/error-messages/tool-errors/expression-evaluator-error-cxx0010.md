# Error del evaluador de expresiones CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta corchete izquierdo  
  
 La expresión contiene corchetes no coincidentes.  Vuelva a escribir la expresión con corchetes coincidentes.  
  
 Este error es idéntico a CAN0010.
