# Error de la l&#237;nea de comandos D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\/opción' no permitida con varios archivos de código fuente  
  
 Estas opciones de compilador no se pueden utilizar con varios archivos de código fuente:  
  
-   Asignar un nombre al listado de archivos del ensamblado \(\/Fa\)  
  
-   Cambiar el nombre del archivo objeto \(\/Fo\)  
  
-   Crear la información del explorador de código fuente sin variables locales \(\/Fr\)  
  
-   Crear la información del explorador de código fuente que incluya variables locales \(\/FR\)
