# <a name="resource-compiler-error-rc2175"></a>Error del compilador de recursos RC2175

nombre de archivo de archivo de recurso no está en formato 3.00

El recurso especificado ha utilizado un formato anterior a la versión 3.00. El archivo de recursos se debe convertir o volver a crearse con el formato para la versión 3.00 o posterior.
