# Error del compilador de recursos RC2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el nombre del archivo de recursos no tiene el formato de 3.00  
  
 El recurso especificado ha utilizado un formato anterior a la versión 3.00.  El archivo de recursos se debe convertir o volver a crear usando el formato para la versión 3.00 o posterior.
