# <a name="project-build-error-prj0019"></a>Error PRJ0019 al compilar el proyecto
Una herramienta devolvió un código de error de  
  
 Un nivel de error era distinto de cero para un paso de compilación personalizada o un evento de compilación.  
  
 También verá PRJ0019 cuando una herramienta devolvió un código de error, pero ningún mensaje de error. Esto puede ocurrir, por ejemplo, si se redirige la salida de MIDL a NUL.  
  
 Vea [solución de problemas de los pasos de compilación personalizada y eventos de compilación](../../ide/troubleshooting-build-customizations.md) para obtener más información.  
  
 Este error también puede producirse cuando se ejecuta como un miembro del grupo de usuarios y se necesita acceso administrativo. Para obtener más información, consulte [ejecutar como miembro del grupo de usuarios](../../security/running-as-a-member-of-the-users-group.md).
