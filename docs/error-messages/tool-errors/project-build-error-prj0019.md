# Error PRJ0019 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Una herramienta devolvió un código de error de  
  
 Había un nivel de error distinto de cero para un paso de generación personalizada o un evento de compilación.  
  
 También verá el error PRJ0019 cuando una herramienta devuelve código de error sin mensaje de error.  Esto puede suceder, por ejemplo, si redirige el resultado de MIDL a NUL.  
  
 Vea [Solución de problemas de pasos de generación personalizada y eventos de compilación](../../ide/troubleshooting-build-customizations.md) para obtener más información.  
  
 Este error también puede producirse cuando se actúa como miembro del grupo de usuarios y se necesita acceso administrativo.  Para obtener más información, vea [Ejecutar como miembro del grupo de usuarios](../../top/running-as-a-member-of-the-users-group.md).
