# Error del compilador de recursos RC2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se esperaba un nombre de la clase del control  
  
 El campo `class` de una instrucción CONTROL en una instrucción **DIALOG** debe ser de uno de los siguientes tipos: BUTTON, COMBOBOX, EDIT, LISTBOX, SCROLLBAR, STATIC o uno definido por el usuario.  Asegúrese de que la clase se haya escrito correctamente.
