# <a name="resource-compiler-error-rc2114"></a>Error del compilador de recursos RC2114
nombre de la clase de control esperado  
  
 El `class` campo de una instrucción de CONTROL en el **diálogo** instrucción debe ser uno de los siguientes tipos: BUTTON, COMBOBOX, EDIT, LISTBOX, barra de desplazamiento, estático, o definido por el usuario. Asegúrese de que la clase se ha escrito correctamente.
