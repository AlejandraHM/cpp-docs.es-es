# <a name="bscmake-error-bk1504"></a>Error de BSCMAKE BK1504
no se puede ubicar en el archivo 'nombredearchivo' [: motivo]  
  
 BSCMAKE no puede mover a una ubicación en el archivo.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Posibles causas del error:  
  
1.  Un disco lleno.  
  
2.  Un error de hardware.  
  
3.  Truncamiento de archivo debido a espacio en disco insuficiente o interrupción del compilador al crear el archivo .sbr.
