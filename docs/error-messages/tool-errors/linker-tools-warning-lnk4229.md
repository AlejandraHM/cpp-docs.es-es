# <a name="linker-tools-warning-lnk4229"></a>Advertencia de las herramientas del vinculador LNK4229

/ Directiva de la directiva no válido que se encuentra; pasa por alto

Una directiva que se pasa a través de la [comentario](../../preprocessor/comment-c-cpp.md) pragma no era válida. El vinculador omite `/directive`.
