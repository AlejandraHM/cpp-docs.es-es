# Advertencia de las herramientas del vinculador LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se encontró la directiva \/directive no válida; se omitió  
  
 Una directiva pasada a través de la directiva pragma [comment](../../preprocessor/comment-c-cpp.md) no era válida.  El vinculador pasa por alto `/directive`.
