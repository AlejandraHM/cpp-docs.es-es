# Error del evaluador de expresiones CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la lista de argumentos no coincide con una función  
  
 Una expresión ha llamado a una función con una lista de parámetros actuales que no coincide con la lista de parámetros formales de ninguna función con el mismo nombre definido en el programa.  
  
 Sólo se puede llamar a las funciones sobrecargadas si hay una coincidencia de parámetros exacta o una coincidencia que no requiere la construcción de un objeto.  
  
 Este error es idéntico a CAN00047.
