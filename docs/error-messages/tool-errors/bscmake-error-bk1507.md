# Error de BSCMAKE BK1507
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede abrir el archivo temporal 'nombredearchivo' \[: razón\]  
  
 BSCMAKE no puede abrir un archivo temporal.  
  
### Posibles causas del error:  
  
1.  La variable de entorno **TMP** no se ha establecido en una unidad o un directorio válidos.  
  
2.  Disco lleno.
