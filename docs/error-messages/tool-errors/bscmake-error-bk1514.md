# <a name="bscmake-error-bk1514"></a>Error de BSCMAKE BK1514

todos los. SBR están truncados, no se encuentra en el nombre de archivo ninguno

Ninguno de los archivos .sbr especificados para una actualización formaban parte del archivo original de examen (.bsc) de la información. Para obtener los nombres de los archivos .sbr que causó este error, lea la [BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) advertencias que lo preceden.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Posibles causas del error:

1. Nombre de archivo incorrecto especificado para .sbr o BSC.

1. Archivo .bsc dañado requiere BSCMAKE volver a crearla.
