# <a name="bscmake-error-bk1514"></a>Error de BSCMAKE BK1514
todo. SBR están truncados, no se encuentra en nombre de archivo ninguno  
  
 Ninguno de los archivos .sbr especificados para una actualización formaban parte del archivo de información (.bsc) Examinar original. Para buscar los nombres de los archivos .sbr que causó este error, lea la [BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) advertencias que lo preceden.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Posibles causas del error:  
  
1.  Nombre de archivo incorrecto especificado para .sbr o BSC.  
  
2.  Archivo .bsc dañado requiere que BSCMAKE vuelva a generarlo.
