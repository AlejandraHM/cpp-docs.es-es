# <a name="cvtres-fatal-error-cvt1105"></a>Error irrecuperable de CVTRES CVT1105
no se puede buscar en archivos  
  
 CVTRES no se pudo ir a una ubicación en el archivo.  
  
 Este error puede deberse a daños en el archivo.
