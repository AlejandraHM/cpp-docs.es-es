# Error irrecuperable de CVTRES CVT1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede buscar en el archivo  
  
 CVTRES no pudo desplazarse a una ubicación en el archivo.  
  
 Este error puede estar motivado por los daños del archivo.
