# Error de BSCMAKE BK1515
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'archivo\_bsc': versión incompatible; no se puede actualizar incrementalmente  
  
 El archivo .bsc se creó con otra versión de BSCMAKE.  Vuelva a crear el archivo .bsc.
