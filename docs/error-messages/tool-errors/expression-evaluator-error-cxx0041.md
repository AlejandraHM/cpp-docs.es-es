# Error del evaluador de expresiones CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el elemento de la clase debe ser un miembro estático o una función miembro.  
  
 Se utiliza un miembro no estático de una clase \(o estructura o unión\) sin especificar qué instancia de la clase se utiliza.  
  
 Sólo los miembros de datos estáticos o las funciones miembro se pueden utilizar sin especificar una instancia.  
  
 Este error es idéntico a CAN00041.
