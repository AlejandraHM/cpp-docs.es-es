# <a name="expression-evaluator-error-cxx0049"></a>Error del evaluador de expresiones CXX0049
OMF obsoleto - volver a vincular programa  
  
 El programa utiliza una antigua OMF (Object Module Format).  
  
 Volver a vincular el programa usando la versión actual del vinculador.  
  
 Este error es idéntico a CAN0049.
