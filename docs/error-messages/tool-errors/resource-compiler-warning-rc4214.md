# Advertencia del compilador de recursos RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Página de códigos no válida : se omite  
  
 El archivo .RC contiene un argumento de página de códigos y la página de códigos especificada no es válida.  Para obtener más información, vea [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674).
