# <a name="expression-evaluator-error-cxx0009"></a>Error del evaluador de expresiones CXX0009
Falta ' después de la constante de caracteres  
  
 Faltaban las comillas simples esperadas al final de la constante de caracteres. Vuelva a escribir la expresión, incluyendo la constante de caracteres entre comillas simples.  
  
 Este error es idéntico a CAN0009.
