# Error del evaluador de expresiones CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta ' después de la constante del carácter  
  
 Faltan las comillas simples esperadas al final de la constante del carácter.  Vuelva a escribir la expresión encerrando la constante del carácter entre comillas simples.  
  
 Este error es idéntico a CAN0009.
