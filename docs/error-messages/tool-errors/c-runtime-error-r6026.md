# Error en tiempo de ejecuci&#243;n de C R6026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no hay espacio suficiente para la inicialización de stdio  
  
 Este error aparece cuando no hay bastante memoria libre disponible para inicializar la compatibilidad de E\/S estándar.
