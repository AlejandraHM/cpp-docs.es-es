# Error en tiempo de ejecuci&#243;n de C R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

espacio insuficiente para la tabla \_onexit\/atexit  
  
 No hay memoria disponible para la función `_onexit` o `atexit`.  Este error está provocado por una condición de poca memoria.
