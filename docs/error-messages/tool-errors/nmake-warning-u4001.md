# Advertencia de NMAKE U4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sólo se puede llamar al archivo de comandos desde la línea de comandos  
  
 Un archivo de comandos, que se llama mediante el especificador del signo arroba \(**@**\), no puede contener una especificación para otro archivo de comandos.  No se permite tal anidamiento.  Se ha omitido la especificación.
