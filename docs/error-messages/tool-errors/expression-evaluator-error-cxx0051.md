# <a name="expression-evaluator-error-cxx0051"></a>Error del evaluador de expresiones CXX0051
más de un símbolo sobrecargado especificado en el punto de interrupción  
  
 El evaluador de expresiones no pudo determinar cuál de más de un símbolo sobrecargado para usarlo como un punto de interrupción.  
  
 Este error es idéntico a CAN0051.
