# Error del evaluador de expresiones CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se especificó más de un símbolo sobrecargado en el punto de interrupción  
  
 El evaluador de expresión no pudo determinar cuál de los símbolos sobrecargados distintos utilizar como punto de interrupción.  
  
 Este error es idéntico a CAN00051.
