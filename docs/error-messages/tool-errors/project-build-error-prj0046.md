# Error PRJ0046 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

No se pudo generar una línea de comandos porque la que se especificó estaba vacía.  
  
 Se ha especificado una línea de comandos vacía para la configuración de un archivo MAKE y la línea de comandos es imprescindible.
