# <a name="command-line-warning-d9040"></a>Advertencia de la línea de comandos D9040
la opción '/analyze' se omitirá; las advertencias de análisis de código no están disponibles en esta edición del compilador  
  
 El **/ analyze** opción de línea de comandos no está disponible en todas las ediciones de Visual Studio. La solución es pasarse a una edición compatible de Visual Studio Team System o quitar la opción de línea de comandos.
