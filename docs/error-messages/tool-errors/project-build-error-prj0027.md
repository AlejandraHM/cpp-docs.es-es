# Error PRJ0027 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

El mensaje de registro 'contenido' de Unicode incluye contenido que no se puede traducir a la página de códigos ANSI del usuario.  
  
 Normalmente sólo verá esta advertencia junto con los errores al crear archivos de proceso por lotes o archivos de respuesta.  
  
 La solución de este error es actualizar el contenido del registro de compilación para que utilice ANSI o instalar la página de códigos en el equipo y establecerla como la predeterminada del sistema.
