# Error matem&#225;tico M6110
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

desbordamiento de pila  
  
 Una expresión de punto flotante ha provocado un desbordamiento de pila de punto flotante.  
  
 Las excepciones de punto flotante de desbordamiento de pila se interceptan hasta un límite de siete niveles además de los ocho niveles habitualmente admitidos por el coprocesador 8087\/287\/387.  
  
 El programa termina con el código de salida 138.
