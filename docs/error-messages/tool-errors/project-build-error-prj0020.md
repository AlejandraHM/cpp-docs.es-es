# <a name="project-build-error-prj0020"></a>Error PRJ0020 al compilar el proyecto
'Herramienta', propiedad 'property' contiene el nombre de archivo no válido 'nombre_archivo'.  
  
 El nombre de archivo ***archivo***, especificado en la propiedad ***propiedad***, para la herramienta, ***herramienta***, no era válido.  
  
 Es probable que haya usado una macro desconocida o no es válida.  
  
 Vea [establecer las propiedades de un proyecto de Visual C++](../../ide/working-with-project-properties.md) para obtener información sobre cómo obtener acceso a las propiedades del proyecto.
