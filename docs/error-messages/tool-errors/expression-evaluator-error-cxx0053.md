# Error del evaluador de expresiones CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el símbolo de no función coincide al enlazar puntos de interrupción  
  
 Un símbolo usado como punto de interrupción no era una función.  La especificación de un miembro de datos como punto de interrupción puede provocar este error.  
  
 Este error es idéntico a CAN00053.
