# <a name="expression-evaluator-error-cxx0053"></a>Error del evaluador de expresiones CXX0053
el símbolo coincide al enlazar puntos de interrupción  
  
 Un símbolo usado como un punto de interrupción no era una función. Especifica a un miembro de datos como un punto de interrupción, puede producir este error.  
  
 Este error es idéntico a CAN0053.
