# Error matem&#225;tico M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

denormal  
  
 Una operación ha generado un número muy pequeño de punto flotante que puede no ser válido por carecer de significado.  Las excepciones de punto flotante denormal se suelen enmascarar, provocando su captura y tratamiento posterior.  
  
 El programa termina con el código de salida 130.
