# Error del evaluador de expresiones CXX0054
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Registro no válido en la expresión de puntos de interrupción  
  
 No se puede utilizar un registro en una expresión de punto de interrupción.  
  
 Este error es idéntico a CAN00054.
