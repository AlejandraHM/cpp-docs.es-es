# Error de las herramientas del vinculador LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tabla de símbolos COFF dañada o no válida \(símbolo estático o de etiqueta sin definir\)  
  
 El contenido generado por el compilador para una sección del objeto dado parece estar dañado.  
  
 Puede tratarse de un error del compilador.
