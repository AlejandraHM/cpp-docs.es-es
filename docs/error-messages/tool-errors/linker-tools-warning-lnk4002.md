# <a name="linker-tools-warning-lnk4002"></a>Advertencia de las herramientas del vinculador LNK4002
símbolo definido en objeto  
  
 El símbolo, mostrado en su forma representativa, se especificó en su formato no representativo en `object`, pero no se pudo encontrar una coincidencia única con un símbolo representativo. Esta advertencia siempre precedida de advertencia [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) y seguido del error irrecuperable [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
