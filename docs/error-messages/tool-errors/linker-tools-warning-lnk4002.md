# Advertencia de las herramientas del vinculador LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

definición de símbolo en objeto  
  
 El símbolo, que se muestra en forma representativa, se especificó en su forma no representativa en `object`, pero no se encontró ninguna coincidencia única con un símbolo representativo.  Esta advertencia viene precedida de la advertencia [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) y seguida del error irrecuperable [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
