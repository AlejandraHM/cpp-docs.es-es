# Error del evaluador de expresiones CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede encontrar el operador sobrecargado  
  
 Se ha especificado un tipo de clase como el operando izquierdo de una expresión, pero no se definió un operador sobrecargado para la clase.  
  
 Este error es idéntico a CAN00058.
