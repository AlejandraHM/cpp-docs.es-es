# Error matem&#225;tico M6205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : error \_TLOSS  
  
 Se ha producido una pérdida total de significado \(precisión\).  
  
 La causa de este error puede ser un número muy grande como el operando de sin, cos o tan, ya que el operando debe pertenecer al intervalo 0 \- 2\*pi.
