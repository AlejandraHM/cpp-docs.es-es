# <a name="expression-evaluator-error-cxx0038"></a>Error del evaluador de expresiones CXX0038

discrepancia de cuenta o el tipo de argumento de función

La llamada de función según lo especificado no coincide con el prototipo de la función.

Vuelva a escribir la llamada con el número correcto de argumentos. Conversión de tipo cada argumento para que coincida con el prototipo, según sea necesario.

Este error es idéntico a CAN0038.
