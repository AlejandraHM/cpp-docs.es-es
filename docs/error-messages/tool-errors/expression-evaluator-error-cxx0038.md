# Error del evaluador de expresiones CXX0038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el número de argumentos de la función o los tipos no coinciden  
  
 La llamada de función especificada no coincide con el prototipo para la función.  
  
 Vuelva a escribir la llamada con el número de argumentos correcto.  Convierta el tipo de cada argumento para que coincida con el prototipo según sea necesario.  
  
 Este error es idéntico a CAN00038.
