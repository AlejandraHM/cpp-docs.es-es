# <a name="profile-guided-optimization-warning-pg1087"></a>Advertencia de la optimización guiada por perfiles PG1087

El archivo PGC 'archivo' no es válido.

Un archivo .pgc está dañado, posiblemente debido a una finalización anormal en una sesión de generación de perfiles.  Vuelva a ejecutar el escenario de generación de perfiles y finalizar con normalidad.
