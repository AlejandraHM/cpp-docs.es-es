# Advertencia de la optimizaci&#243;n guiada por perfiles PG1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Archivo PGC 'archivo' no válido.  
  
 Un archivo .pgc está dañado, posiblemente debido a una finalización anormal de una sesión de generación de perfiles.  Vuelva a ejecutar el escenario de generación de perfiles y finalice la sesión con normalidad.
