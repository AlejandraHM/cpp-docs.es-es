# Error grave de NMAKE U1077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'programa' : código devuelto 'valor'  
  
 El comando o programa dado llamado desde NMAKE ha dado un error y ha devuelto el código de salida dado.  
  
 Para suprimir este error y continuar la sesión de NMAKE, utilice la opción \/I, la directiva **.IGNORE** o el modificador de comandos guión \(**\-**\).  Para continuar la sesión de NMAKE para partes no relacionadas del árbol de dependencias, utilice la opción \/K.
