# Error del evaluador de expresiones CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

símbolo ambiguo en el operador de contexto  
  
 Un símbolo en el operador de contexto \(**{}**\) hacía referencia a más de un símbolo del programa.  
  
 El operador de resolución de ámbito \(`::`\) podría resolver la ambigüedad.  
  
 Este error es idéntico a CAN00055.
