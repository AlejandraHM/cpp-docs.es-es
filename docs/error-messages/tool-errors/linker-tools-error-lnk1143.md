# <a name="linker-tools-error-lnk1143"></a>Error de las herramientas del vinculador LNK1143

archivo no válido o dañado: no hay símbolo para el número de la sección COMDAT

Este error puede deberse a si el archivo de objeto está dañado.

Vuelva a generar el archivo.
