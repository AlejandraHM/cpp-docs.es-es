# <a name="resource-compiler-error-rc2171"></a>Error del compilador de recursos RC2171
formato de encabezado DIB desconocido  
  
 El encabezado de mapa de bits no es un **estructura BITMAPCOREHEADER** o **BITMAPINFOHEADER** estructura.
