# Advertencia PRJ0018 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

No se encontraron las siguientes variables de entorno:  
  
 Una variable de entorno no está definida.  Este error muestra las variables de entorno que no están definidas.  
  
 Vea [Macros para comandos de compilación personalizada](../../ide/common-macros-for-build-commands-and-properties.md) para obtener información acerca de las macros del proceso de compilación.
