# Error irrecuperable del compilador de recursos RC1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

página de códigos no válida  
  
 La opción \/c \(Especificar la página de códigos\) iba seguida de una página de códigos no válida.  Vea [Páginas de códigos](../../c-runtime-library/code-pages.md) en la *Referencia de la biblioteca en tiempo de ejecución* para obtener más información.
