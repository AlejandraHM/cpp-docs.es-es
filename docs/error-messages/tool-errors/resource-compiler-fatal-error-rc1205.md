página de códigos no válida  
  
 Especificar la página de código (/ c) opción iba seguida de una página de códigos no válida. Vea [páginas de códigos](../../c-runtime-library/code-pages.md) en el *referencia de la biblioteca de tiempo de ejecución* para obtener más información.
