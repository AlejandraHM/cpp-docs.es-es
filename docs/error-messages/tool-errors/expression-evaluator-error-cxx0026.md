# Error del evaluador de expresiones CXX0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cadena con formato erróneo  
  
 Se ha especificado de forma incorrecta una cadena de formato.  Compruebe la sintaxis de la expresión.  
  
 Este error es idéntico a CAN00026.  
  
 Para obtener más información, vea [Especificadores de formato en C\+\+](../Topic/Format%20Specifiers%20in%20C++.md).
