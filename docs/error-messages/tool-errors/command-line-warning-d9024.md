# Advertencia de la l&#237;nea de comandos D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo de archivo de código fuente 'nombre de archivo' no reconocido, se supone el archivo objeto  
  
 No se reconoce la extensión del archivo especificado.  Se supone que el archivo es un archivo objeto y se ha pasado al vinculador.  
  
 Se reconocen las siguientes extensiones:  
  
-   .c \(Archivo de código fuente de C\)  
  
-   .cxx \(Archivo de código fuente de C\+\+\)  
  
-   .cpp \(Archivo de código fuente de C\+\+\)  
  
-   .obj \(Archivo objeto\)  
  
-   .lib \(Archivo de bibliotecas\)  
  
-   .def \(Archivo de definiciones de módulo\)  
  
-   .exp \(Archivo de exportaciones del vinculador, creado por LINK \/LIB\)
