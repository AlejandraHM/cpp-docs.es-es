# Advertencia del compilador de recursos RW4003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Se ha utilizado SHIFT o CONTROL sin VIRTKEY  
  
 Dentro de un recurso de tabla de aceleradores, SHIFT o CONTROL requieren VIRTKEY.  Debido a que SHIFT y CONTROL se indican como bits de marcadores en un acelerador de tipo VIRTKEY, no pueden existir independientemente de VIRTKEY.
