# Error del evaluador de expresiones CXX0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no es un elemento struct\/union  
  
 Una expresión de formato “Struct.Member” o “pStruct\- miembro\>” se ha especificado, pero \<el miembro\> no es un elemento de la estructura.  
  
 La expresión puede tener mal escritos los paréntesis.  
  
 Este error es idéntico a CAN00028.
