# Error irrecuperable del compilador de recursos RC1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**modificador no válido, '**   
 ***opción* ': hay demasiados modificadores \/d**  
  
 Se definieron demasiados símbolos utilizando la opción \/d \(Definir símbolos\).  Defina algunos símbolos en el código fuente y vuelva a compilar.
