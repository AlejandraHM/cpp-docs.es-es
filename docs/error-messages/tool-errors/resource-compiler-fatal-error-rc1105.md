# <a name="resource-compiler-fatal-error-rc1105"></a>Error irrecuperable del compilador de recursos RC1105

modificador no válido, opción: hay demasiados modificadores /d

Demasiados símbolos definidos con definir símbolos (/ d) opción. Definir algunos símbolos en el origen y vuelva a compilar.
