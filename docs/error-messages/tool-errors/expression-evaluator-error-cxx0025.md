# <a name="expression-evaluator-error-cxx0025"></a>Error del evaluador de expresiones CXX0025
el operador necesita struct/union  
  
 Un operador que toma una expresión de `struct` o **union** tipo se aplicó a una expresión que no es un `struct` o **union**.  
  
 Componentes de clase, estructura o unión variables deben tener un nombre completo. Componentes no pueden especificarse sin especificación completa.  
  
 Este error es idéntico a CAN0025.
