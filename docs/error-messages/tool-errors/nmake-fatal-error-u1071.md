# Error grave de NMAKE U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ciclo en árbol de dependencia para destino 'nombrededestino'  
  
 Existe una dependencia circular en el árbol de dependencias para el destino dado.  El destino dado es un dependiente de uno de los dependientes del destino dado.  Las dependencias circulares no son válidas.
