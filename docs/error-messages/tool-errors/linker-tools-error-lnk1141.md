# Error de las herramientas del vinculador LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error al compilar el archivo de exportaciones  
  
 [LINK](../../build/reference/linker-command-line-syntax.md) no pudo compilar el archivo de exportaciones \(.exp\).  
  
 Las causas de este error incluyen problemas del sistema, como la falta de memoria, y los errores de sintaxis en las opciones o en las instrucciones de definición de módulos.  
  
 Este error es precedido por otro error que proporciona información adicional.
