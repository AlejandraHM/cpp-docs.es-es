# Error PRJ0015 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

El dispositivo NULL no se encuentra en el sistema.y no se puede iniciar la compilación.  
  
 Este error puede deberse a que el sistema tiene pocos recursos.  Cierre algunas aplicaciones o reinicie el equipo.  
  
 También puede deberse a que no tenga privilegios suficientes para el dispositivo NULL en el equipo.
