# <a name="expression-evaluator-error-cxx0013"></a>Error del evaluador de expresiones CXX0013
Falta el operador  
  
 Un operador se esperaba en la expresión pero no se encontró. Cambie la sintaxis de la expresión.  
  
 Este error es idéntico a CAN0013.
