# Error del evaluador de expresiones CXX0013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta un operador  
  
 Se esperaba un operador en la expresión pero no se encontró.  Compruebe la sintaxis de la expresión.  
  
 Este error es idéntico a CAN0013.
