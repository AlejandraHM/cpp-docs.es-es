# <a name="resource-compiler-error-rc2111"></a>Error del compilador de recursos RC2111
tipo de control no válido.  
  
 Cada instrucción de CONTROL en un **diálogo** instrucción debe ser uno de los siguientes: 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, icono, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, SCROLLBAR, USERBUTTON.  
  
 Asegúrese de que estas instrucciones de CONTROL estén escritas correctamente.
