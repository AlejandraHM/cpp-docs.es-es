# Error del compilador de recursos RC2111
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo de control no válido  
  
 Cada instrucción CONTROL dentro de una instrucción **DIALOG** debe ser una de las siguientes: 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ICON, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, SCROLLBAR, USERBUTTON.  
  
 Asegúrese de que las instrucciones CONTROL están escritas correctamente.
