# <a name="linker-tools-warning-lnk4010"></a>Advertencia de las herramientas del vinculador LNK4010
número de números de versión de subsistema no válido; versión predeterminada del subsistema supone  
  
 Puede especificar una versión para el subsistema de la imagen ([/Subsystem](../../build/reference/subsystem-specify-subsystem.md)). Cada subsistema tiene un requisito de versión mínima. Si la versión especificada es inferior al mínimo, se producirá esta advertencia y el vinculador utilizará el subsistema predeterminado.
