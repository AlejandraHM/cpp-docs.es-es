# Error del compilador de recursos RC2127
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se esperaba una versión WORD separada por comas  
  
 Los números de versión en un recurso de versión deben ser del tipo **WORD y estar separados por comas.**
