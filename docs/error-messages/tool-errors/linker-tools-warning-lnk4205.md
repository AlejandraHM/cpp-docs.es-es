# Advertencia de las herramientas del vinculador LNK4205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta información de depuración actual ‘nombredearchivo' para hacer referencia al módulo; se vinculará el objeto sin tener en cuenta información de depuración  
  
 El archivo .pdb tiene información no actualizada.  El vinculador continuará vinculando el objeto sin información de depuración.  Puede resultar conveniente volver a compilar el archivo objeto con la opción [Zi](../../build/reference/z7-zi-zi-debug-information-format.md).
