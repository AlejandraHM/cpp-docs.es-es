# Error PRJ0044 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La propiedad 'Dependencias adicionales' correspondiente a la regla de compilación personalizada 'regla' asignada al archivo 'archivo' no es válida.La propiedad contiene 'cadena' que se evalúa como 'valor'.  
  
 La propiedad **Dependencias adicionales** se ha evaluado como cadena vacía o cadena con caracteres no válidos, es decir, caracteres que no pueden estar en ningún nombre de archivo ni de directorio.  Las reglas de compilación personalizadas necesitan el resultado de la acción de compilación.  
  
 Para obtener más información, vea [Especificar las herramientas de compilación personalizadas](../../ide/specifying-custom-build-tools.md).
