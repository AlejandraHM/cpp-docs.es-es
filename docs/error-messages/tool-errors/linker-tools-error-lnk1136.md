# Error de las herramientas del vinculador LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

archivo no válido o dañado  
  
 El archivo de entrada tenía un encabezado dañado, o su tamaño es cero o anormalmente pequeño.  
  
 También puede producirse el error LNK1136 si el archivo se ha especificado incorrectamente como una dependencia adicional.
