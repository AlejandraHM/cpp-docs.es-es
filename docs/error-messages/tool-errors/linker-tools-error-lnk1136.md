# <a name="linker-tools-error-lnk1136"></a>Error de las herramientas del vinculador LNK1136

archivo no válido o dañado

El archivo de entrada tiene un encabezado dañado o es cero, tamaño o pequeño de forma anómala.

También puede deberse LNK1136 si el archivo se ha especificado incorrectamente como una dependencia adicional.
