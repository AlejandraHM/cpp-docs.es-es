# Error del evaluador de expresiones CXX0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de sintaxis  
  
 La sintaxis de la expresión es incorrecta.  
  
 Vuelva a escribir la expresión con la sintaxis correcta.  
  
 Este error es idéntico a CAN0004.
