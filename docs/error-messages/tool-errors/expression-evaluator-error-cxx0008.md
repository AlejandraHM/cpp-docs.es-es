# Error del evaluador de expresiones CXX0008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**falta " al final de la cadena**  
  
 Faltan las comillas dobles esperadas al final de la cadena literal.  Vuelva a escribir la expresión encerrando la cadena literal entre comillas dobles.  
  
 Este error es idéntico a CAN0008.
