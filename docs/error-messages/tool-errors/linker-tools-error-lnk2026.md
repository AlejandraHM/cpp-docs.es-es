# Error de las herramientas del vinculador LNK2026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

módulo no seguro de una imagen SAFESEH  
  
 [Se ha especificado \/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md), pero un módulo no es compatible con la característica de control de excepciones seguro.  Si desea utilizar este módulo con **\/SAFESEH**, tendrá que volver a compilar dicho módulo utilizando el compilador de Visual C\+\+ .NET 2003 o posterior.
