# Error del evaluador de expresiones CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expresión no ampliable  
  
 El evaluador de expresiones encontró un error interno.  
  
 Quizá pueda escribir una expresión equivalente que se evalúe correctamente.  
  
 Este error es idéntico a CAN00031.
