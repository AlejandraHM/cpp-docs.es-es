# Error irrecuperable del compilador de recursos RC1018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#elif' inesperada  
  
 La directiva `#elif` no apareció en una construcción `#if`, **\#ifdef** o **\#ifndef**.  
  
 Asegúrese de que hay una instrucción `#if`, **\#ifdef** o **\#ifndef** en vigor antes de esta instrucción.
