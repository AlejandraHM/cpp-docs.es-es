# <a name="resource-compiler-fatal-error-rc1018"></a>Error irrecuperable del compilador de recursos RC1018
'#elif' inesperada  
  
 El `#elif` directiva no apareció dentro de un `#if`, **#ifdef**, o **#ifndef** construir.  
  
 Asegúrese de que hay un `#if`, **#ifdef**, o **#ifndef** instrucción en vigor antes de esta instrucción.
