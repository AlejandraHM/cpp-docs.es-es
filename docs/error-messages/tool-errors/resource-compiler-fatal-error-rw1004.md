# Error irrecuperable del compilador de recursos RW1004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Final de archivo inesperado  
  
 Este error puede deberse a la falta de caracteres de avance de línea o retorno de carro en la última línea de un archivo de texto.
