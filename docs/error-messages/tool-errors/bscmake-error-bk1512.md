# Error de BSCMAKE BK1512
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nombre de archivo: se superó la capacidad  
  
 BSCMAKE no puede compilar un archivo de información de examen debido a que el número de definiciones, referencias, módulos u otra información supera el límite.  
  
### Se corrige mediante algunas de las siguientes posibles soluciones  
  
1.  Excluya cierta información mediante \/Em, \/Es o \/Ei.  
  
2.  Omita la opción \/Iu.
