# Error matem&#225;tico M6101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no válido  
  
 Operación no válida.  
  
 Este error puede aparecer cuando un operando es NaN \(no un número\) o infinito.  
  
 El programa termina con el código de salida 129.
