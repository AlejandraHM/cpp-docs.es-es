# Advertencia de la optimizaci&#243;n guiada por perfiles PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

No se pudo esperar el tiempo de inactividad para rastrear.  
  
 La aplicación no permitió que se ejecutara [pgosweep](../../build/reference/pgosweep.md).  Si es posible, ejecute pgosweep cuando la aplicación se encuentre en estado inactivo.
