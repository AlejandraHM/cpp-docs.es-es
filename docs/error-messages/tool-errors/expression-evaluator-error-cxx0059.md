# Error del evaluador de expresiones CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el operando izquierdo es una clase, no un nombre de función  
  
 El operando izquierdo de una llamada de función es un nombre de clase y no se puede resolver para una llamada de función.  La omisión del nombre de una función miembro en una expresión puede provocar este error.  
  
 Este error es idéntico a CAN00059.
