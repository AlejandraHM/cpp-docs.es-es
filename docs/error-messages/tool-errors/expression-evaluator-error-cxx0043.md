# <a name="expression-evaluator-error-cxx0043"></a>Error del evaluador de expresiones CXX0043
Este puntero usado fuera de la función miembro  
  
 El **esto** puntero solo puede usarse para funciones miembro no estáticas.  
  
 Este error es idéntico a CAN0043.
