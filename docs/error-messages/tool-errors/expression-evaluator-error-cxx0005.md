# Error del evaluador de expresiones CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operador no compatible  
  
 Se especificó un operador C no compatible en una expresión.  Escriba una expresión equivalente usando los operadores C compatibles.  
  
 Este error es idéntico a CAN0005.
