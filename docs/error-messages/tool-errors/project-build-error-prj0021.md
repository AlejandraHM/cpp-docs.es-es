# <a name="project-build-error-prj0021"></a>Error PRJ0021 al compilar el proyecto
'Herramienta', propiedad 'property' contiene el nombre de archivo no válido.  
  
 El nombre de archivo especificado en la propiedad ***propiedad***, para la herramienta, ***herramienta***, no era válido.  
  
 Es probable que haya usado una macro desconocida o no es válida.  
  
 Vea [establecer las propiedades de un proyecto de Visual C++](../../ide/working-with-project-properties.md) para obtener información sobre cómo obtener acceso a las propiedades del proyecto.
