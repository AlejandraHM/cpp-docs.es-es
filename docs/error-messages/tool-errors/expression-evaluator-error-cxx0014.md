# Error del evaluador de expresiones CXX0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta un operando  
  
 Se especificó un operador sin el operando requerido.  Compruebe la sintaxis de la expresión.  
  
 Este error es idéntico a CAN0014.
