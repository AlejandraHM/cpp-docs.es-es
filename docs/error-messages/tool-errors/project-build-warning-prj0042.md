# Advertencia PRJ0042 al compilar el proyecto
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**La propiedad 'Outputs' del paso de compilación personalizado correspondiente al archivo '**   
 ***archivo* ' no está establecida. El paso de compilación personalizada se omitirá.**  
  
 No se ejecutó un paso de compilación personalizada porque no se especificó ningún resultado.  
  
 Para solucionar este error, realice una de las acciones siguientes:  
  
-   Excluya el paso de compilación personalizada de la compilación.  
  
-   Agregue un resultado.  
  
-   Elimine el contenido del comando del paso de compilación personalizada.
