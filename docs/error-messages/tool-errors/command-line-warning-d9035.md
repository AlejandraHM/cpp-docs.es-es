# <a name="command-line-warning-d9035"></a>Advertencia de la línea de comandos D9035
opción 'opción' ha quedado desusada y se quitará en una versión futura  
  
 Se especificó una opción del compilador que se quitará en una versión futura del compilador de Visual C++. Esta advertencia va seguida de advertencia [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) si no hay un reemplazo sugerido para *opción*.  
  
 La opción especificada todavía funciona, pero debe actualizar ahora la configuración de compilación. Como resultado, el proyecto es más probable que seguirán compilándose al actualizar el compilador.
