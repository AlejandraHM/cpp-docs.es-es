# <a name="command-line-warning-d9035"></a>Advertencia de la línea de comandos D9035

opción 'opción' está desusada y se quitará en una versión futura

Ha especificado una opción del compilador que se quitará en una versión futura del compilador de Visual C++. Esta advertencia va seguida de advertencia [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) si hay un reemplazo sugerido para *opción*.

La opción especificada sigue funcionando, pero debe actualizar la configuración de compilación ahora. Como resultado, el proyecto es más probable que seguirán compilándose al actualizar el compilador.
