# Advertencia de la l&#237;nea de comandos D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la opción 'opción' ha quedado desusada y se quitará en próximas versiones  
  
 Especificó una opción del compilador que se quitará en una versión futura del compilador de Visual C\+\+.  Esta advertencia va seguida advirtiendo [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) si es un reemplazo sugerido para *option*.  
  
 La opción especificada todavía funciona, pero debería actualizar ahora su configuración de compilación.  Como resultado, es más probable que su proyecto se siga compilando al actualizar el compilador.
