# <a name="project-build-error-prj0002"></a>Error PRJ0002 al compilar el proyecto
resultado del error devuelto desde 'línea de comandos'.  
  
 Un comando, ***línea de comandos***, que se formó desde proporcionados por el usuario en el **páginas de propiedades** se abre el cuadro de diálogo, devuelve un código de error, pero no hay información en la ventana de salida.  
  
 La solución a este error depende de la herramienta que generó el error. Para MIDL, obtendrá una idea de qué ha salido mal si se ha definido /o (Redirigir resultados).  
  
 Un archivo por lotes, como un paso de compilación personalizada o un evento de compilación que no es proporcionar información útil acerca de las condiciones de error también podría ser la causa de este error.
