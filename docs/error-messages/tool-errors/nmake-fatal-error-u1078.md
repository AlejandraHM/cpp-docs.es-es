# Error grave de NMAKE U1078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

desbordamiento constante en 'expresión'  
  
 La expresión especificada contenía una constante que superaba el intervalo comprendido entre \-2.147.483.648 y 2.147.483.647.  La constante aparecía en una de las situaciones siguientes:  
  
-   Una expresión especificada con una directiva de preprocesamiento.  
  
-   Un nivel de error especificado con el modificador de comandos guión \(**\-**\).
