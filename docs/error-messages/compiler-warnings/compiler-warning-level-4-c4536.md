# Advertencia del compilador (nivel 4) C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nombre de tipo' : el nombre de tipo supera el límite de metadatos de 'limite' caracteres  
  
 Un nombre de tipo quedaría truncado en los metadatos si fuera un tipo administrado.  Para obtener más información, vea [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md).  
  
 De forma predeterminada, esta advertencia está desactivada.  Para obtener más información, vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
