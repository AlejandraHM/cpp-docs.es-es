# <a name="compiler-warning-level-4-c4536"></a>Advertencia del compilador (nivel 4) C4536
'type name': nombre de tipo supera el límite de metadatos de 'limite' caracteres  
  
 Un nombre de tipo quedaría truncado en los metadatos si fuera un tipo administrado. Vea [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) para obtener más información.  
  
 De forma predeterminada, esta advertencia está desactivada. Vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para más información.
