# Advertencia del compilador (nivel 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se supone que la biblioteca de tipos se compiló para punteros de \(número\) bits  
  
 No se proporcionó información explícita con el atributo **ptrsize** de la directiva [\#import](../../preprocessor/hash-import-directive-cpp.md); el compilador interpretó que el tamaño de puntero de la biblioteca de tipos es ***número***.  
  
 De forma predeterminada, esta advertencia está desactivada.  Para obtener más información, vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
