# <a name="compiler-warning-level-4-c4931"></a>Advertencia del compilador (nivel 4) C4931
se supone que la biblioteca de tipos se compiló para punteros de (número) bits  
  
 No se proporcionó información explícita con el **ptrsize** atributo de la [#import](../../preprocessor/hash-import-directive-cpp.md) directiva; el compilador concluyó que el tamaño del puntero de la biblioteca de tipos es ***número***.  
  
 De forma predeterminada, esta advertencia está desactivada. Vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para más información.
