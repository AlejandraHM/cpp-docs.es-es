# <a name="compiler-warning-level-3-c4306"></a>Advertencia del compilador (nivel 3) C4306
**'**   
 ***identificador de* ': conversión de '**   
 ***tipo1* 'to'**   
 ***type2* ' de mayor tamaño**  
  
 El identificador es de tipo en un puntero mayor. Los bits superiores no rellenados del nuevo tipo será rellenan con ceros.  
  
 Esta advertencia puede indicar una conversión no deseada. El puntero resultante puede no ser válido.
