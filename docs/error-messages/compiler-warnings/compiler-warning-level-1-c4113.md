# Advertencia del compilador (nivel 1) C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador1' es distinto en las listas de parámetros de 'identificador2'  
  
 Un puntero a función está asignado a otro puntero a función, pero las listas de parámetros formales de las funciones no coinciden.  Se compiló la asignación sin modificaciones.
