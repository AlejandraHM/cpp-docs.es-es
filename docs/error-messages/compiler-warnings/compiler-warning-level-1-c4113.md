# <a name="compiler-warning-level-1-c4113"></a>Compilador advertencia (nivel 1) C4113
'identificador1' se diferencia en listas de parámetros de 'identificador2'  
  
 Un puntero de función se asigna a otro puntero de función, pero las listas de parámetros formales de las funciones no coinciden. La asignación se compila sin ninguna modificación.
