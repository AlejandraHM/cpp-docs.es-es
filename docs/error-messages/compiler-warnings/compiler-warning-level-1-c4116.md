# Advertencia del compilador (nivel 1) C4116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

definición de tipo sin nombre en paréntesis  
  
 Una estructura, unión o tipo enumerado sin nombre se ha definido dentro de una expresión entre paréntesis.  La definición de tipo no tiene sentido.  
  
 En una llamada de función de C, la definición tiene ámbito global.  En una llamada de función de C\+\+, la definición tiene el mismo ámbito que la función a la que se llama.
