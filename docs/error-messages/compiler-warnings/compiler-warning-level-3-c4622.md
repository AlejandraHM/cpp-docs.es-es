# <a name="compiler-warning-level-3-c4622"></a>Advertencia del compilador (nivel 3) C4622
Reemplazando la información de depuración formada durante la creación del encabezado precompilado en el archivo objeto: 'file'  
  
 Se perdió la información de CodeView del archivo especificado cuando se compila con la [/Yu](../../build/reference/yu-use-precompiled-header-file.md) opción (utilizar encabezado precompilado).  
  
 Cambie el nombre del archivo de objeto (mediante [/Fo](../../build/reference/fo-object-file-name.md)) al crear o utilizar encabezado precompilado de archivos y vincúlelo con el nuevo archivo de objeto.
