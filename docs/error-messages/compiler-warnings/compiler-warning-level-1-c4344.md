# Advertencia del compilador (nivel 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cambio de comportamiento: el uso de argumentos de plantilla explícitos da como resultado la llamada a 'función'  
  
 Una llamada a función mediante argumentos explícitos de plantilla ha llamado a una función diferente a la que llamaría si no se hubieran especificado argumentos explícitos.
