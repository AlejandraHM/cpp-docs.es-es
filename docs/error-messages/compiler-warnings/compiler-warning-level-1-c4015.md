# <a name="compiler-warning-level-1-c4015"></a>Compilador advertencia (nivel 1) C4015
'identificador': tipo de campo de bits debe ser integral  
  
 El campo de bits no se declara como un tipo entero. El compilador supone que el tipo base del campo de bits es unsigned. Campos de bits se deben declarar como tipos de entero sin signo.
