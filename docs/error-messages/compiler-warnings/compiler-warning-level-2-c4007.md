# Advertencia del compilador (nivel 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : debe ser 'atributo'  
  
 Un atributo requerido para una función no está expresado explícitamente.  Por ejemplo, la función **main** debe tener el atributo `__cdecl`.  El compilador fuerza el atributo.
