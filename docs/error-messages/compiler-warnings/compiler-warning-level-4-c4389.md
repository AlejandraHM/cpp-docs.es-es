# Advertencia del compilador (nivel 4) C4389
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operador' : no coinciden signed\/unsigned  
  
 Una operación incluía variables signed y unsigned.  Puede haberse producido pérdida de datos.  
  
 El código siguiente genera el error C4389:  
  
```  
// C4389.cpp  
// compile with: /W4  
#pragma warning(default: 4389)  
  
int main()  
{  
   int a = 9;  
   unsigned int b = 10;  
   if (a == b)   // C4389  
      return 0;  
   else  
      return 0;  
};  
```
