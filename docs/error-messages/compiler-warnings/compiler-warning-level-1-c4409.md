# Advertencia del compilador (nivel 1) C4409
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tamaño de instrucción máquina no válido  
  
 La instrucción de código máquina no tenía el tamaño especificado.  Se utilizó el tamaño permitido más pequeño.
