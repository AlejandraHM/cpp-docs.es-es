# <a name="compiler-warning-level-2-c4948"></a>Advertencia del compilador (nivel 2) C4948
tipo de valor devuelto de 'descriptor de acceso' no coincide con el tipo último parámetro del método establecedor correspondiente  
  
El compilador encontró una discrepancia entre el tipo de datos es la que se va a obtener y establece para una propiedad indizada.  
  
Solo es accesible mediante la opción del compilador obsoleta C4948 **/CLR: oldSyntax**.
