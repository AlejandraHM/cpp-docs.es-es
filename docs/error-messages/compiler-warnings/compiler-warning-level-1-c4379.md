# <a name="compiler-warning-level-1-c4379"></a>Advertencia del compilador (nivel 1) C4379
Este compilador no admite la versión de common language runtime. Con esta versión puede provocar resultados inesperados.  
  
 Tiene una versión anterior de common language runtime en su equipo, pero no la versión actual. Para resolver el error C4379, instale la versión de common language runtime que se incluye con el compilador.
