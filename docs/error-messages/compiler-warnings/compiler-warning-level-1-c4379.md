# Advertencia del compilador (nivel 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Este compilador no admite la versión versión de Common Language Runtime.El uso de esta versión puede causar resultados inesperados.  
  
 Tiene una versión anterior de Common Language Runtime en su equipo, pero no la versión actual.  Para resolver el error C4379, instale la versión de Common Language Runtime que se distribuye con su compilador.
