# Advertencia del compilador (nivel 1) C4628
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

los digramas no son compatibles con \-Ze.La secuencia de caracteres 'digrama' no interpretado como token alternativo de 'char'  
  
 No se admiten los digramas con la opción [\/Ze](../../build/reference/za-ze-disable-language-extensions.md).  Esta advertencia viene seguida de un error.  
  
 De forma predeterminada, esta advertencia está desactivada.  Para obtener más información, vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md).  
  
 El código siguiente genera el error C4628:  
  
```  
// C4628.cpp  
// compile with: /WX  
#pragma warning(default : 4628)  
int main()  
<%   // C4628 <% digraph for {  
}  
```
