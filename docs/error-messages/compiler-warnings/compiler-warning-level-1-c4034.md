# Advertencia del compilador (nivel 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof devuelve 0  
  
 Se ha aplicado el operador `sizeof` a un operando de tamaño cero \(una estructura, unión, clase o tipo enumerado vacíos, o de tipo `void`\).
