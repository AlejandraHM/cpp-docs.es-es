# <a name="compiler-warning-level-1-c4034"></a>Compilador advertencia (nivel 1) C4034

sizeof devuelve 0

El `sizeof` operador se aplica a un operando de tamaño cero (una estructura vacía, unión, clase o tipo enumerado, tipo o `void`).
