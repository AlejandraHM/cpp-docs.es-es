# Advertencia del compilador (nivel 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : truncamiento de 'tipo1' a 'tipo2'  
  
 El identificador se convierte a un tipo más pequeño, lo que da como resultado la pérdida de información.  
  
 Si se utilizó el Asistente para puntos de conexión ATL, véase el artículo de Knowledge Base:  
  
-   Q250847 : ATL Connection Point Wizard Generated Code for Event with VARIANT Argument Gives C4305 Warning
