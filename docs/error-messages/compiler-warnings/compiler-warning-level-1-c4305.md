# <a name="compiler-warning-level-1-c4305"></a>Advertencia del compilador (nivel 1) C4305
'identificador': truncamiento de 'tipo1' a 'tipo2'  
  
 El identificador se convierte en un tipo más pequeño, lo que provoca la pérdida de información.  
  
 Si utiliza al Asistente para puntos de conexión ATL, vea el artículo de Knowledge Base:  
  
-   Q250847: Asistente para puntos de conexión ATL genera código para eventos con el argumento de tipo VARIANT emite la advertencia de C4305
