# Advertencia del compilador (nivel 1) C4264
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función\_virtual' : no hay un reemplazo disponible para la función miembro virtual desde 'clase' base; la función está oculta  
  
 Siempre se genera la advertencia C4264 después de [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).  
  
 De forma predeterminada, esta advertencia está desactivada.  Para obtener más información, vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
