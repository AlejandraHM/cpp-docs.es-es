# <a name="compiler-warning-level-1-c4024"></a>Compilador advertencia (nivel 1) C4024
'función': tipos diferentes de parámetros formales y reales 'número'  
  
 Los parámetros formales y reales correspondientes tienen tipos diferentes. El compilador pasa el parámetro real sin cambios. La función receptora convierte el tipo de parámetro para el tipo esperado.
