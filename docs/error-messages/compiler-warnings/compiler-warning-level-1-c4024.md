# Advertencia del compilador (nivel 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : tipos distintos para el parámetro formal y el parámetro real 'número'  
  
 Los parámetros formal y real correspondientes tienen tipos diferentes.  El compilador pasa el parámetro real sin cambios.  La función receptora convierte el tipo de parámetro al tipo que se esperaba.
