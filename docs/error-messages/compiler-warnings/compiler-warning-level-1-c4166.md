# <a name="compiler-warning-level-1-c4166"></a>Advertencia del compilador (nivel 1) C4166
**convención de llamada no válida para el constructor o destructor**  
  
 Los constructores y destructores no pueden tener convenciones de llamada distintas del valor predeterminado de la plataforma (excepto cuando se especifica explícitamente **__clrcall**).
