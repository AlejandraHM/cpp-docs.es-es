# Advertencia del compilador (nivel 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**convención de llamada no válida para el constructor o destructor**  
  
 Los constructores y destructores no pueden tener convenciones de llamada distintas del valor predeterminado de la plataforma \(excepto cuando se especifica explícitamente **\_\_clrcall**\).
