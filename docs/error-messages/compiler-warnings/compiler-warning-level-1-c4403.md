# <a name="compiler-warning-level-1-c4403"></a>Advertencia del compilador (nivel 1) C4403
operador PTR no válido  
  
 Operador PTR se utiliza incorrectamente en código ensamblador en línea.  
  
 El compilador omitió el operador PTR.
