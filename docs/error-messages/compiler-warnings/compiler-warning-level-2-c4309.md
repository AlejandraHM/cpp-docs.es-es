# Advertencia del compilador (nivel 2) C4309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'conversión' : truncamiento de valor constante  
  
 La conversión de tipos provoca que una constante supere el espacio que tiene asignado.  Puede que sea necesario usar un tipo mayor para la constante.  
  
 El código siguiente genera el error C4309:  
  
```  
// C4309.cpp  
// compile with: /W2  
int main()  
{  
   char c = 128;   // C4309  
}  
```
