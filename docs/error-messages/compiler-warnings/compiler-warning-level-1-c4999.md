# <a name="compiler-warning-level-1-c4999"></a>Advertencia del compilador (nivel 1) C4999

ADVERTENCIA DESCONOCIDA En el menú Ayuda, elija el comando Soporte técnico o abra el archivo de ayuda de soporte técnico para obtener más información.

Anote las circunstancias del error, intente aislar el problema y crear un caso de prueba reproducible, luego póngase en contacto con los [servicios de soporte técnico de Microsoft](/visualstudio/ide/talk-to-us).
