# <a name="compiler-warning-level-1-c4541"></a>Advertencia del compilador (nivel 1) C4541
'identificador' utilizado en el tipo polimórfico 'tipo' con/GR; puede dar lugar a un comportamiento impredecible  
  
 Se intentó usar una característica que requiere información de tipo en tiempo de ejecución sin habilitar la información de tipo en tiempo de ejecución. Vuelva a compilar con [/GR](../../build/reference/gr-enable-run-time-type-information.md).
