# Advertencia del compilador (nivel 1) C4026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

función declarada con lista de parámetros formales  
  
 La declaración de función tiene parámetros formales, pero su definición no. Cualquier llamada posterior a esta función supone que la función no toma parámetros.
