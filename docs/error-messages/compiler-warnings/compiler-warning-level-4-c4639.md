# Advertencia del compilador (nivel 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

El error MSXML, los comentarios del documento XML no se procesará. motivo  
  
 Esta advertencia puede aparecer por distintas razones.  
  
 Para solucionarlo:  
  
-   Vuelva a compilar.  
  
-   Reinstale MSXML reinstalando Common Language Runtime.  
  
-   Modifique o quite el comentario de documentación que causa la advertencia y vuelva a compilar.  
  
 Cuando aparece la advertencia C4639, se deshabilita el procesamiento posterior de comentarios XML y no se genera el archivo .xdc.
