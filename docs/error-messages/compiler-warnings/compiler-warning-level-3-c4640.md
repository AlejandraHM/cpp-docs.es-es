# <a name="compiler-warning-level-3-c4640"></a>Advertencia del compilador (nivel 3) C4640
'instancia': la construcción del objeto estático local no es segura para subprocesos  
  
 Una instancia estática de un objeto no es segura para subprocesos.  
  
 De forma predeterminada, esta advertencia está desactivada. Vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para más información.  
  
 El ejemplo siguiente genera C4640:  
  
```  
// C4640.cpp  
// compile with: /W3  
#pragma warning(default:4640)  
  
class X {  
public:  
   X() {  
   }  
};  
  
void f() {  
   static X aX;   // C4640  
}  
  
int main() {  
   f();  
}  
```
