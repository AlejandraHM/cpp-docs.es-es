# <a name="compiler-warning-level-1-c4411"></a>Advertencia del compilador (nivel 1) C4411
'identificador': símbolo se resuelve como un registro de desplazamiento  
  
 El identificador es un símbolo local que se resuelve en un registro de desplazamiento y, por tanto, puede usarse en un operando con otro símbolo.
