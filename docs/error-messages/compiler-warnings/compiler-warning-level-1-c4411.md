# Advertencia del compilador (nivel 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : el símbolo se resuelve como un registro de desplazamiento  
  
 El identificador es un símbolo local que se resuelve como un registro de desplazamiento y por lo tanto puede usarse en un operando con otro símbolo.
