# <a name="compiler-warning-level-1-c4022"></a>Compilador advertencia (nivel 1) C4022
'función': puntero no coincide con el parámetro real 'número'  
  
 El tipo de puntero del parámetro real difiere del tipo de puntero del parámetro formal correspondiente. El parámetro real se pasa sin cambios.
