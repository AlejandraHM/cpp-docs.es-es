# Advertencia del compilador (nivel 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : tipos distintos en parámetro real 'número', parámetro formal 'número'  
  
 Los parámetros formal y real correspondientes tienen tipos diferentes.  El parámetro real se pasa sin cambios.  La función convierte el parámetro real al tipo especificado en la definición de función.
