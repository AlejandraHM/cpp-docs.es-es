# <a name="compiler-warning-level-1-c4381"></a>Advertencia del compilador (nivel 1) C4381

'function1': el método no público 'function2' no implementará el método de interfaz

Una clase debe implementar la función all en una interfaz. Una clase puede satisfacer esta condición si uno de sus clases base implementa la función. Sin embargo, la función debe implementarse como una función pública.
