# <a name="compiler-warning-level-1-c4381"></a>Advertencia del compilador (nivel 1) C4381
'función1': el método no público 'función2' no implementará el método de interfaz  
  
 Una clase debe implementar todas las funciones en una interfaz. Una clase puede satisfacer esta condición si una de sus clases base implementa la función. Sin embargo, la función se debe implementar como una función pública.
