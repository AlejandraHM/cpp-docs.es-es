# <a name="compiler-warning-level-1-c4381"></a>Advertencia del compilador (nivel 1) C4381
'función1': el método no público 'función2' no implementará el método de interfaz  
  
 Una clase debe implementar la función all en una interfaz. Una clase satisface esta condición si una de sus clases bases implementa la función. Sin embargo, la función debe implementarse como una función pública.
