# <a name="compiler-warning-level-1-c4631"></a>Advertencia del compilador (nivel 1) C4631

MSXML o XPath no disponible; no se procesarán los comentarios del documento XML. reason

La instalación de Common Language Runtime no tenía los archivos necesarios para admitir el procesamiento de comentario de documento. Reinstale Common Language Runtime.
