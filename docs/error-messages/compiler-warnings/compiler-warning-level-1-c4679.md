# <a name="compiler-warning-level-1-c4679"></a>Advertencia del compilador (nivel 1) C4679

> '*miembro*': no se pudo importar el miembro

El compilador encontró una construcción compatibles, que no se puede importar desde los metadatos.

No intente utilizar la construcción.
