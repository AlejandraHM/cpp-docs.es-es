# Error del compilador C2773
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import y \#using sólo están disponibles en el compilador de C\+\+  
  
 El compilador de C no reconoce la directiva de preprocesador `#import`.  Compile el código fuente como C\+\+.  Utilice [\/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) si es necesario.
