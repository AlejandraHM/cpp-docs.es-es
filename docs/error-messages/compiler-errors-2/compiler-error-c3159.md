# <a name="compiler-error-c3159"></a>Error del compilador C3159

'pointer': no se puede declarar la matriz de punteros a tipo de valor

No se puede declarar una matriz de punteros a un tipo de valor.

Solo es accesible a través de la opción del compilador obsoleto C3159 **/CLR: oldSyntax**.
