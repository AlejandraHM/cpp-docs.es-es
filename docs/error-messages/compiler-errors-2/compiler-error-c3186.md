# Error del compilador C3186
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se permiten matrices nativas multidimensionales  
  
 Se declaró una matriz multidimensional nativa incorrectamente.  
  
 El código siguiente genera el error C3186:  
  
```  
// C3186.cpp  
// compile with: /clr  
#using <mscorlib.dll>  
int main()  
{  
   int a[,];   // C3186  
   int b[2][3];   // native multidimension array  
   array<int,2> ^c = new array<int, 2>(2,5);   // managed multidimension array  
}  
```
