# Error del compilador C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador': el objeto de clase no tiene constructor para la inicialización  
  
 No hay un constructor con una lista de parámetros que coincida con la inicialización.  
  
 Parámetros incorrectos de inicialización pueden producir este error.
