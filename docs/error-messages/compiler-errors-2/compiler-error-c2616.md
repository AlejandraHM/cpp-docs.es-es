# Error del compilador C2616
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'conversión': no se puede convertir implícitamente un tipo 'tipo1' que no sea un valor L a un tipo 'tipo2' que no sea const  
  
 No se puede inicializar una referencia a partir de un valor que no sea lvalue.  
  
 Se trata de un error cuando está habilitada la compatibilidad con ANSI \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) y una advertencia en las extensiones de Microsoft \(**\/Ze**\).
