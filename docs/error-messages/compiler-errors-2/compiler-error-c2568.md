# <a name="compiler-error-c2568"></a>Error del compilador C2568

'identificador1': no se puede resolver la sobrecarga de función

El compilador no puede determinar qué función para llamar a sobrecargada. Los parámetros reales que se pasa a la función se deben convertir para que coincida con los parámetros formales para uno de la función sobrecargada, pero ninguna es inequívocamente mejor que todos los demás.
