# Error del compilador C2873
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'símbolo' : el símbolo no se puede utilizar en una declaración using  
  
 Falta una palabra clave [namespace](../../misc/namespace-declaration.md) en una directiva `using`.  Esto hace que el compilador interprete el código incorrectamente como una [declaración using](../../cpp/using-declaration.md) en lugar de una [directiva using](../../misc/using-directive-cpp.md).
