# Error del compilador C3126
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede definir una unión 'unión' dentro del tipo administrado 'tipo'  
  
 No se puede definir una unión dentro de un tipo administrado.  
  
 El código siguiente genera el error C3126:  
  
```  
// C3126_2.cpp  
// compile with: /clr /c  
ref class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```  
  
 El código siguiente genera el error C3126:  
  
```  
// C3126.cpp  
// compile with: /clr:oldSyntax /c  
#using <mscorlib.dll>  
  
__gc class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```
