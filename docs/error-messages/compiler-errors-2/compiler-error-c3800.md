# <a name="compiler-error-c3800"></a>Error del compilador C3800

'declaration': no se pueden mezclar propiedades y eventos

No se puede declarar una construcción para que sea una propiedad y un evento.

Solo es accesible a través de la opción del compilador obsoleto C3800 **/CLR: oldSyntax**.
