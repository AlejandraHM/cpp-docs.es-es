# <a name="compiler-error-c2700"></a>Error del compilador C2700
'identifier': no se puede iniciar (use /W4 para más información)  
  
 El objeto no se puede iniciar. Compile con [/W4](../../build/reference/compiler-option-warning-level.md) para obtener más información de diagnóstico.
