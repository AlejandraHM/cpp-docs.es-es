# Error del compilador C2794
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : no es un miembro ninguna clase base directa ni indirecta de 'clase'  
  
 Se intentó usar [super](../../cpp/super.md) para llamar a una función miembro inexistente.  
  
 El ejemplo siguiente genera C2794:  
  
```  
// C2794.cpp  
struct B {  
   void mf();  
};  
  
struct D : B {  
   void mf() {  
      __super::f();  // C2794  
   }  
};  
```
