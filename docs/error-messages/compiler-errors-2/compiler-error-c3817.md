# <a name="compiler-error-c3817"></a>Error del compilador C3817

'declaration': sólo puede aplicarse a una función de propiedad

El `property` palabra clave solo se pueden aplicar a una definición de función.

Solo es accesible a través de la opción del compilador obsoleto C3817 **/CLR: oldSyntax**.
