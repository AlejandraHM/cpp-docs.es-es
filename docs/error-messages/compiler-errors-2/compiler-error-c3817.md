# Error del compilador C3817
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'declaración' : property sólo se puede aplicar a una función  
  
 La palabra clave [property](../../misc/property.md) sólo se puede aplicar a una definición de función.  
  
 Sólo se puede reproducir el error C3817 utilizando **\/clr:oldSyntax**.  
  
 El código siguiente genera el error C3817:  
  
```  
// C3817.cpp  
// compile with: /clr:oldSyntax  
#using <mscorlib.dll>  
  
__gc class G {  
      __property int x;   // C3817  
   };  
  
// the following class defines a property  
__gc class X {  
public:  
   __property int get_N( int i ) {  
      Console::WriteLine( L"int" );  
      return m_val[i];  
   }  
  
   __property void set_N( int i, int val ) {  
      m_val[i] = val;  
   }  
  
private:  
   int m_val[10];  
};  
  
int main() {  
}  
```
