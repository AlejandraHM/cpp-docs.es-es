# Error del compilador C2869
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nombre' : ya se ha definido como espacio de nombres  
  
 No se puede volver a utilizar ningún nombre ya usado como espacio de nombres.  
  
 El código siguiente genera el error C2869:  
  
```  
// C2869.cpp  
// compile with: /c  
namespace A { int i; };  
  
class A {};   // C2869, A is already used  
```
