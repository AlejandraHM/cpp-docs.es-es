# Error del compilador C3858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'tipo': no se puede declarar de nuevo en el ámbito actual  
  
 No se puede declarar dos veces el tipo en el mismo ámbito.  
  
 El código siguiente genera el error C3858:  
  
```  
// C3858.cpp  
// compile with: /LD  
template <class T>  
struct Outer  
{  
   struct Inner;  
};  
  
template <class T>  
struct Outer<T>::Inner;   // C3858  
// try the following line instead  
// struct Outer<T>::Inner{};  
```
