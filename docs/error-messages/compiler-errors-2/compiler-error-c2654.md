# Error del compilador C2654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : intento de acceso a un miembro fuera de una función miembro  
  
 Se obtiene acceso a un miembro en una declaración.  Sólo se puede obtener acceso a los datos de miembros en funciones miembro.  
  
 Se puede producir este error al intentar inicializar las variables en una declaración.  Para este propósito utilice un constructor.
