# <a name="compiler-error-c3370"></a>Error del compilador C3370
'idl_module name': idl_module no se ha definido a√∫n  
  
 Para poder usar [idl_module](../../windows/idl-module.md) para especificar un punto de entrada en un archivo DLL, primero debe usar `idl_module` para especificar el nombre de archivo DLL.  
  
 El ejemplo siguiente genera la advertencia C3370:  
  
```  
// C3370.cpp  
[module(name=MyLibrary)];  
// uncomment the following line to resolve the error  
// [idl_module(name="name1", dllname=x.dll)];  
[idl_module(name="name1"), entry(100)] // C3370  
int f1();  
  
int main()  
{  
}  
```
