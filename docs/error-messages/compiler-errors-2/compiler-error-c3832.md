# Error del compilador C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'biblioteca de tipos': parece que la biblioteca de tipos se ha compilado para punteros de 32 bits; cambie el calificador 'ptrsize'  
  
 La información explícita proporcionada con el atributo `ptrsize` de la directiva [\#import](../../preprocessor/hash-import-directive-cpp.md) no coincide con lo detectado por el compilador en la biblioteca de tipos.
