# <a name="compiler-error-c2721"></a>C2721 de Error del compilador
'especificador': especificador de clase de almacenamiento no es válido entre la palabra clave del operador y el tipo  
  
 Conversiones de tipos definidos por el usuario se aplican a todas las clases de almacenamiento, por lo que no puede especificar una clase de almacenamiento en una conversión de tipos.
