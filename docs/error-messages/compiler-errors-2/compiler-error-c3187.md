# <a name="compiler-error-c3187"></a>Error del compilador C3187
' __func\_\_': solamente está disponible en el cuerpo de una función  
  
 La macro predefinida no está disponible fuera del cuerpo de una función.  
  
 Para corregir este error, introduzca la macro en el cuerpo de una función.
