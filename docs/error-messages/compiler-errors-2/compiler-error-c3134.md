# <a name="compiler-error-c3134"></a>Error de C3134 de Error de compilador
'valor': valor del argumento de atributo 'atributo' no tiene tipo válido de 'type'  
  
 Se detectó un error de sintaxis cuando se asigna un valor a un atributo.
