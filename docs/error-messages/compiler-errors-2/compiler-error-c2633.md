# Error del compilador C2633
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : 'inline' es la única clase de almacenamiento válida para constructores  
  
 Un constructor se ha declarado como una clase de almacenamiento distinta de inline.  
  
 El código siguiente genera el error C2633:  
  
```  
// C2633.cpp  
// compile with: /c  
class C {  
   extern C();   // C2633, not inline  
   inline C();   // OK  
};  
```
