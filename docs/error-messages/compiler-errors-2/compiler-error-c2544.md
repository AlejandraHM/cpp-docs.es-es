# Error del compilador C2544
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se esperaba '\)' para el operador '\(\)'  
  
 Al operador de llamada de función le falta un paréntesis izquierdo.  
  
 Este error puede estar motivado por la expansión de macro.
