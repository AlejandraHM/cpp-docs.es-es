# Error del compilador C2996
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': definición de plantilla de función recursiva  
  
 Una definición de función intenta crear instancias de su clase basada en modelo raíz. No se permite la creación de instancias de plantilla recursiva.
