# Error del compilador C2976
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : no hay suficientes argumentos de tipo  
  
 Faltan uno o varios argumentos en una plantilla o genérico.  Compruebe la declaración de plantilla o genérico para encontrar el número correcto de parámetros.  
  
 Este error puede deberse a la falta de argumentos de plantilla en componentes STL.  
  
 El código siguiente genera el error C2976:  
  
```  
// C2976.cpp  
template <class T>   
struct TC {  
   T t;  
};  
int main() {  
   TC<>* t;   // C2976  
   TC<int>* t2;   // OK  
}  
```  
  
 El error C2976 también puede producirse cuando se utilizan genéricos:  
  
```  
// C2976b.cpp  
// compile with: /clr  
generic <class T>  
ref struct GC {  
   T t;  
};  
  
int main() {  
   GC<>^ g;   // C2976  
   GC<int>^ g2;   // OK  
}  
```
