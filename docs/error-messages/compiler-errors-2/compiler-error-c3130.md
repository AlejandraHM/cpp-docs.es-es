# Error del compilador C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Error interno del compilador: error al escribir bloque de código insertado en PDB  
  
 Este error se produce cuando el compilador no logra escribir un bloque de código insertado en el archivo .pdb  La causa más común del error es la falta de espacio en disco.
