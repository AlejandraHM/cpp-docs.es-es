# <a name="compiler-error-c3812"></a>Error del compilador C3812

'property' debe ser el primer símbolo (token) en una declaración de propiedad

Al declarar una propiedad, el `__property` palabra clave debe ser el primer símbolo de la línea.

Solo es accesible a través de la opción del compilador obsoleto C3812 **/CLR: oldSyntax**.
