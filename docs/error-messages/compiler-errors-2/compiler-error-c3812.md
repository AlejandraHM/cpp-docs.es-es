# <a name="compiler-error-c3812"></a>Error del compilador C3812
'propiedad' debe ser el primer símbolo en una declaración de propiedad  
  
 Al declarar una propiedad, el `__property` palabra clave debe ser el primer símbolo de la línea.  
  
 Solo es accesible mediante la opción del compilador obsoleta C3812 **/CLR: oldSyntax**.
