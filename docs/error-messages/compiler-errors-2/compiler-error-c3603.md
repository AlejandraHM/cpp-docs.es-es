# Error del compilador C3603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Símbolo' : el tipo 'Tipo' no se admite aún  
  
 Se intentó utilizar un tipo de datos que aún no se admite en el tiempo de ejecución de .NET en un objeto administrado.
