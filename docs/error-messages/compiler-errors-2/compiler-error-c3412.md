# Error del compilador C3412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'plantilla' : no se puede especializar la plantilla en el ámbito actual  
  
 No se ha podido especializar una plantilla en el ámbito de clase, sólo en el ámbito global o del espacio de nombres.  
  
## Ejemplo  
 El ejemplo siguiente genera el error C3412.  
  
```  
// C3412.cpp  
template <class T>  
struct S {  
   template <>  
   struct S<int> {};   // C3412 in a class  
};  
```  
  
## Ejemplo  
 En el ejemplo siguiente se muestra una posible solución:  
  
```  
// C3412b.cpp  
// compile with: /c  
template <class T>  
struct S {};  
  
template <>  
struct S<int> {};  
```
