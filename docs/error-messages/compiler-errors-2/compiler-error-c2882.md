# Error del compilador C2882
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nombre': uso no válido del identificador de espacio de nombres en la expresión  
  
 Se intentó usar el nombre de un espacio de nombres en una expresión.  
  
 El código siguiente genera el error C2882:  
  
```  
// C2882.cpp  
// compile with: /c  
namespace A {  
   int k;  
}  
  
int i = A;   // C2882, can't assign A to i  
```
