# Error del compilador C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la opción de la línea de comandos 'opción' no es coherente con el encabezado precompilado  
  
 La opción especificada en la línea de comandos es distinta de la utilizada para crear el encabezado precompilado.  Vuelva a generar el encabezado precompilado con la opción dada de la línea de comandos.
