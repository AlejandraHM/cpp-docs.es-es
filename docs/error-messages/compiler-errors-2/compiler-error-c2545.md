# Error del compilador C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operador' : no se puede encontrar el operador sobrecargado  
  
 El operador no se puede utilizar con los operandos proporcionados.  Es necesario proporcionar un operador sobrecargado con los operandos requeridos.  
  
 Puede producirse este error si el tipo de los operandos es incorrecto.  
  
 Este error puede corregirse definiendo un operador de conversión o un constructor que tome un único parámetro.
