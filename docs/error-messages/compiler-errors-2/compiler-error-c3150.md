# <a name="compiler-error-c3150"></a>Error del compilador C3150

'element': 'attribute' solo puede aplicarse a una clase, interfaz, matriz o puntero

`__gc` solo puede usarse en una clase, interfaz o matriz.

Solo es accesible a través de la opción del compilador obsoleto C3150 **/CLR: oldSyntax**.
