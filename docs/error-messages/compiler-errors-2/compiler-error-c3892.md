# Error del compilador C3892
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'var' : no se puede asignar a una variable constante  
  
 No se puede cambiar una variable const después de que se declara y se inicializa.  
  
 El código siguiente genera el error C3892:  
  
```  
// C3892.cpp  
// compile with: /clr  
ref struct Y1 {  
   static const int staticConst = 9;  
};  
  
int main() {  
   Y1::staticConst = 0;   // C3892  
}  
```
