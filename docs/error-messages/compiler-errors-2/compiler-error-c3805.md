# Error del compilador C3805
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'símbolo \(token\)' : símbolo \(token\) inesperado, se esperaba '}' o un identificador  
  
 Al definir una propiedad, se encontró un símbolo \(token\) inesperado.  Quite el símbolo \(token\) no válido.
