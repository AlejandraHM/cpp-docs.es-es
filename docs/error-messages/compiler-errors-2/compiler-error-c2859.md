# Error del compilador C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nombredearchivo no es el archivo tipo que se utilizó al crear este encabezado precompilado, vuelva a crear el encabezado precompilado.  
  
 La base de datos del proyecto y los archivos de encabezado precompilado deben crearse juntos para garantizar una información coherente.  Recompile el proyecto para crear de nuevo el encabezado precompilado.  
  
 Para obtener información acerca de encabezados precompilados, vea [\/Y \(Encabezados precompilados\)](../../build/reference/y-precompiled-headers.md):
