# <a name="compiler-error-c3631"></a>Error del compilador C3631

'function': no puede sobrecargar eventos administrados o de WinRT

Un evento administrado o de WinRT no se puede sobrecargar.

Solo es accesible a través de la opción del compilador obsoleto C3631 **/CLR: oldSyntax**.
