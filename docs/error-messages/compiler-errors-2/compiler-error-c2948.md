# Error del compilador C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

creación de instancias explícita; el especificador de clase de almacenamiento 'especificador' no está permitido en la especialización  
  
 No se pueden usar especificadores de clase de almacenamiento \(como `extern`\) en la especialización de una clase de plantilla de la que ya se han creado instancias de forma explícita.
