# <a name="compiler-error-c2948"></a>Error del compilador C2948
creación de instancias explícita; especificador de clase de almacenamiento 'especificador' no está permitido en la especialización  
  
 No se puede usar especificadores de clase de almacenamiento (como `extern`) en una especialización de una clase de plantilla que ya se han creado instancias de forma explícita.
