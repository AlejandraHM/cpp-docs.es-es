# <a name="compiler-error-c2958"></a>Error del compilador C2958

el delimitador izquierdo encontrado en 'ubicación' no tiene correspondencia

Un delimitador no se hace coincidir correctamente.

Este error puede deberse a paréntesis no coincidentes en una declaración de plantilla de función.
