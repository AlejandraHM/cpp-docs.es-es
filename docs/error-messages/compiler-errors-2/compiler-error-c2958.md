# Error del compilador C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el delimitador izquierdo encontrado en 'ubicación' no tiene correspondencia  
  
 Falta el delimitador correspondiente a otro delimitador.  
  
 La causa de este error puede ser que falte uno de los paréntesis en la declaración de una plantilla de función.
