# Error del compilador C3276
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'palabra clave' : al saltar desde el bloque \_\_finally se produce un comportamiento no definido durante el control de finalización  
  
 Este error es el mismo que la advertencia [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md).  Sin embargo, cuando se utiliza \/clr, esta condición no se puede deshabilitar con pragma [warning](../../preprocessor/warning.md).
