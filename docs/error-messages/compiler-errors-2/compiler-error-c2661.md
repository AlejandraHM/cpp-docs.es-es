# <a name="compiler-error-c2661"></a>Error del compilador C2661

'function': ninguna función sobrecargada acepta número parámetros

Causas posibles:

1. Parámetros reales incorrectos en la llamada de función.

1. Declaración de función que falta.

El ejemplo siguiente genera C2661:

```
// C2661.cpp
void func( int ){}
void func( int, int ){}
int main() {
   func( );   // C2661 func( void ) was not declared
   func( 1 );   // OK func( int ) was declared
}
```
