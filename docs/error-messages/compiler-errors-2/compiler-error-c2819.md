# Error del compilador C2819
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el tipo “tipo” no tiene un miembro sobrecargado “operador \-\>”  
  
 Es necesario definir `operator->()` para utilizar esta operación de puntero.  
  
 El código siguiente genera el error C2819:  
  
```  
// C2819.cpp  
// compile with: /c  
class A {  
   public:  
      int i;  
};  
  
class B {};  
  
void C(B j) {  
   j->i;   // C2819  
}  
  
class D {  
   A* pA;  
  
   public:  
      A* operator->() {  
         return pA;  
      }  
};  
  
void F(D j) {  
   j->i;  
}  
```  
  
 El error C2819 también puede producirse cuando se usa [Semántica de pila de C\+\+ para los tipos de referencia](../../dotnet/cpp-stack-semantics-for-reference-types.md).  El código siguiente genera el error C2819:  
  
```  
// C2819_b.cpp  
// compile with: /clr  
ref struct R {  
   void Test() {}  
};  
  
int main() {  
   R r;  
   r->Test();   // C2819  
   r.Test();   // OK  
}  
```
