# Error del compilador C2517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador': el operando derecho de '::' no está definido  
  
 El identificador situado a la derecha del operador de resolución de ámbito \(`::`\) debe ser un miembro definido de la clase, estructura o unión situada a la izquierda.  Si no aparece ningún nombre de clase, estructura o unión, debe declararse el identificador de la derecha con ámbito global.
