# <a name="compiler-error-c3322"></a>Error del compilador C3322
'property': no es una propiedad válida para el atributo 'attribute'  
  
 Se pasó una propiedad no válida o desconocida (o parámetro) a un atributo. Compruebe la documentación del atributo.
