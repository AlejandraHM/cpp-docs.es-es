# Error del compilador C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : modificador para referencia no válido  
  
 Se cambió la referencia.  No se pueden modificar las referencias para que hagan referencia a otro objeto.  En su lugar, utilice un puntero.
