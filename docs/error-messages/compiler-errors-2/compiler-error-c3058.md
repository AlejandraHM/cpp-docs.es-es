# <a name="compiler-error-c3058"></a>Error del compilador C3058
'symbol': símbolo no declarado como 'threadprivate' antes de utilizarse en la cláusula 'copyin'  
  
 En primer lugar se debe declarar un símbolo [threadprivate](../../parallel/openmp/reference/threadprivate.md) antes de que se puede utilizar en una [copyin](../../parallel/openmp/reference/copyin.md) cláusula.  
  
 El ejemplo siguiente genera la advertencia C3058:  
  
```  
// C3058.cpp  
// compile with: /openmp  
int x, y, z;  
#pragma omp threadprivate(x, z)  
  
void test() {  
   #pragma omp parallel copyin(x, y)   // C3058  
   {  
   }  
}  
```  
  
 Posible solución:  
  
```  
// C3058b.cpp  
// compile with: /openmp /LD  
int x, y, z;  
#pragma omp threadprivate(x, y)  
  
void test() {  
   #pragma omp parallel copyin(x, y)  
   {  
   }  
}  
```
