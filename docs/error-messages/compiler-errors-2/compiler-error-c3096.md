# Error del compilador C3096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'atributo': el atributo solamente se permite en miembros de datos de clases de atributos.  
  
 Se aplicó incorrectamente un atributo.  
  
 Para obtener más información, consulta [Atributos definidos por el usuario](../../windows/user-defined-attributes-cpp-component-extensions.md).
