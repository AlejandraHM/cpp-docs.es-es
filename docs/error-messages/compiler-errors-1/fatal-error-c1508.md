# Error irrecuperable C1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Límite del compilador: 'function': más de 65535 bytes de argumentos  
  
 Los parámetros formales para la función superan el límite de 65535 bytes.
