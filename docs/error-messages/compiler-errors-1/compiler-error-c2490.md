# Error del compilador C2490
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'palabra clave' no se permite en la función con atributo 'naked'  
  
 Una función definida como [naked](../../cpp/naked-cpp.md) no puede utilizar el control de excepciones estructurado.  
  
 El código siguiente genera el error C2490:  
  
```  
// C2490.cpp  
// processor: x86  
__declspec( naked ) int func() {  
   __try{}   // C2490, structured exception handling  
}  
```
