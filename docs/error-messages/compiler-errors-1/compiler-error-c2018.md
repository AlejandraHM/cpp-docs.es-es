# <a name="compiler-error-c2018"></a>C2018 de Error del compilador
carácter 'hexnumber' desconocido  
  
 El archivo de origen contiene un carácter ASCII, que se identifica mediante su número hexadecimal. Para resolver el error, quite el carácter.
