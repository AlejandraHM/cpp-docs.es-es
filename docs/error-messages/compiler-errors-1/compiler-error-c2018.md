# Error del compilador C2018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

carácter 'número hexadecimal' desconocido  
  
 El archivo de código fuente contiene un carácter ASCII, que se identifica mediante su número hexadecimal.  Para resolver el error, quite el carácter en cuestión.
