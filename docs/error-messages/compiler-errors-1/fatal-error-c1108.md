# Error irrecuperable C1108
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede encontrar la DLL: 'dll'  
  
 La DLL especificada \(***dll***\) no se encontró en la ruta de acceso.  
  
 Para resolver este error, vuelva a instalar Visual C\+\+ o copie el archivo .dll adecuado del CD de instalación en el equipo.
