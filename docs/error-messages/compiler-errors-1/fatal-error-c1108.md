# <a name="fatal-error-c1108"></a>Error irrecuperable C1108

> no se puede encontrar la DLL: '*nombre del archivo dll*'

La DLL especificada (*nombre del archivo dll*) no se encontró en la ruta de acceso.

Para resolver este error, vuelva a instalar Visual C++ o copie el archivo .dll adecuado de la instalación en el equipo.
