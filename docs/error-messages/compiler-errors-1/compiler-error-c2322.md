# Error del compilador C2322
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': la dirección de dllimport 'dllimport' no es estática  
  
 Se proporciona un valor no estático como dirección de una función declarada con `dllimport`.
