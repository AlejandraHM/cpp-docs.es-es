# Error del compilador C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?' : operando izquierdo no válido, se esperaba un tipo aritmético o de puntero  
  
 Existe una expresión a la izquierda de un operador `?` que no puede compararse con cero.  Deberá utilizar una expresión aritmética o de puntero que sí se pueda comparar con cero.
