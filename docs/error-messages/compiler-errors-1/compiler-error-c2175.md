# <a name="compiler-error-c2175"></a>Error del compilador C2175
'configuración regional': configuración regional no válido  
  
 La configuración regional especificada no es válida. Vea [cadenas de idioma y país o región](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) en el *referencia de la biblioteca de tiempo de ejecución* para compatibilidad con configuraciones regionales.
