# Error irrecuperable C1046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

límite del compilador : la estructura está demasiado anidada  
  
 La estructura, unión o clase supera el límite de anidamiento de 15 niveles.  Vuelva a escribir la expresión y reduzca el nivel de anidamiento.  Divida la estructura, unión o clase en varias partes utilizando `typedef`  para definir las estructuras anidadas.
