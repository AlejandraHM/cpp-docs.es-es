# Error irrecuperable C1192
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de \#using en 'archivo'  
  
 Sólo puede pasarse un archivo en formato MSIL \(lenguaje intermedio de Microsoft\) a una directiva [\#using](../../preprocessor/hash-using-directive-cpp.md).  La opción [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) del compilador permite crear un archivo de salida MSIL.  Otros lenguajes de Visual Studio también generan archivos MSIL.
