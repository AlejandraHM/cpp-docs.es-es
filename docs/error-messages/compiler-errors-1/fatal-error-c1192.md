# <a name="fatal-error-c1192"></a>Error irrecuperable C1192
\#Error en 'archivo'  
  
 Sólo un archivo en el formato de lenguaje intermedio de Microsoft (MSIL) puede pasarse a una [#using](../../preprocessor/hash-using-directive-cpp.md) directiva. El [/CLR](../../build/reference/clr-common-language-runtime-compilation.md) opción del compilador permite crear un archivo de salida MSIL. Otros lenguajes de Visual Studio también generan archivos MSIL.
