# Error del compilador C2315
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type1': 'type2' ha capturado una referencia en el número de línea  
  
 El tipo de excepción está controlado por un controlador anterior. La referencia del segundo elemento catch tiene el tipo de la primera.
