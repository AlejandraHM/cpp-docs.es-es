# Error del compilador C2412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'etiqueta' : etiqueta redefinida sin distinción de mayúsculas y minúsculas  
  
 La etiqueta se ha definido más de una vez en la función actual.  Cambie la ortografía de la etiqueta y de sus referencias.
