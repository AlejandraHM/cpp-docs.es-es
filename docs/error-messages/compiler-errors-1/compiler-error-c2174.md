# Error del compilador C2174
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : el parámetro real es de tipo 'void' : parámetro number1, lista de parámetros number2  
  
 El parámetro `number1` pasado a la lista de parámetros `number2` es un parámetro de tipo `void`.  Los parámetros no pueden ser de tipo `void`.  Utilice `void*` en su lugar.
