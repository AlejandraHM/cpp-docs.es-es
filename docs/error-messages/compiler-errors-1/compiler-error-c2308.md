# Error del compilador C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

concatenación de cadenas no coincidentes  
  
 Se especificaron cadenas de caracteres anchos y no anchos para la concatenación.  No se puede concatenar una cadena de caracteres anchos con una de caracteres no anchos.
