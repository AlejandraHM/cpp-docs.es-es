# <a name="fatal-error-c1075"></a>Error irrecuperable C1075

el token de la izquierda estaba desemparejado al final del archivo

El compilador esperaba un *token* coincidente antes de alcanzar el final del archivo.

Este error puede deberse a un paréntesis, una llave u otro carácter sin su pareja correspondiente.
