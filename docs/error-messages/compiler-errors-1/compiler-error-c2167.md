# Error del compilador C2167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': hay demasiados parámetros reales para la función intrínseca  
  
 Una referencia a una función `intrinsic` tiene demasiados parámetros.
