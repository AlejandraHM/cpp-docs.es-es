# Error del compilador C2380
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipos que preceden a 'identificador' \(¿constructor con tipo de valor devuelto o nueva definición no válida de nombre de clase actual?\)  
  
 Un constructor devuelve un valor o realiza una nueva definición del nombre de clase.  
  
 El código siguiente genera el error C2326:  
  
```  
// C2380.cpp  
// compile with: /c  
class C {  
public:  
   int C();   // C2380, specifies an int return  
   int C;   // C2380, redefinition of i  
   C();   // OK  
};  
```
