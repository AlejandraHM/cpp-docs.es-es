# Error del compilador C2403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador': el registro debe ser de base\/índice en 'contexto'.  
  
 El registro usado en un operando de memoria indirecta debe ser un registro de base o índice en este contexto.
