# Error del compilador C2419
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

división \(módulo\) por cero en 'contexto'  
  
 El parámetro de la derecha del operador `MOD` es cero en este contexto.
