# Error irrecuperable C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La llamada API 'nombre de función' produjo un error 'ubicación de llamada' : 'texto en tiempo de ejecución'  
  
 Error en la llamada a una función .NET.  El runtime de COM puede suministrar o no el ***texto en tiempo de ejecución***.  
  
 Para obtener más información sobre los mensajes del error del sistema, vea el archivo de sistema winerror.h y [FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351).
