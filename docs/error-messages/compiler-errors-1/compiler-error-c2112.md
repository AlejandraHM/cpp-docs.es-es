# Error del compilador C2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-' : la resta de punteros requiere un operando de tipo puntero o integral  
  
 Se ha intentado restar punteros que apuntan a tipos diferentes.
