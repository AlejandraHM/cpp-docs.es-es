# Error irrecuperable C1904
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

interacción con el proveedor incorrecta: 'file'  
  
 Este error indica un error de un proveedor de atributos.  
  
 Anote las circunstancias del error, intente aislar el problema y cree un caso de prueba reproducible, y luego póngase en contacto con [Hable con nosotros](../Topic/Talk%20to%20Us.md).
