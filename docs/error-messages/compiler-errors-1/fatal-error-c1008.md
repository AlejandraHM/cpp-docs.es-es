# Error irrecuperable C1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se ha especificado un archivo de entrada  
  
 No se proporcionó al compilador un archivo de código fuente de C o C\+\+ para compilar. Compruebe la línea de comandos de **CL** y la variable de entorno para las especificaciones de nombre de archivo.
