# Error del compilador C2338
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Proveedor de atributos ATL 'identifier'  
  
 El atributo hace referencia a un identificador no declarado. Es posible que vea este error asociado al [Error del proveedor ATL ATL2010](http://msdn.microsoft.com/es-es/27a25924-b52b-4971-80d3-c3b3e9ffbe41).
