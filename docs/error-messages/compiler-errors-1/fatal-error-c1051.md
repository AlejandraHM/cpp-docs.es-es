# Error irrecuperable C1051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el archivo de base de datos de programa, 'archivopdb', tiene un formato obsoleto; elimínelo y vuelva a compilar  
  
 El compilador no puede actualizar el archivo de base de datos de programa, cuyo número de versión es más antiguo.  Elimine el archivo y vuelva a compilar el programa con **\/Zi** o **\/ZI**.  Para obtener más información, vea [\/Z7, \/Zi, \/ZI \(Formato de la información de depuración\)](../../build/reference/z7-zi-zi-debug-information-format.md)
