# Error del compilador C2426
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'símbolo \(token\)' : operador no válido en 'contexto'  
  
 El token no se puede utilizar como operador en este contexto.  Los operadores de índice, por ejemplo, no se pueden anidar.
