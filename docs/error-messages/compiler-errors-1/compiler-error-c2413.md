# Error del compilador C2413
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token' : tamaño de alineación no válido  
  
 Falta el tamaño utilizado con la directiva `ALIGN` o se encuentra fuera del intervalo válido.
