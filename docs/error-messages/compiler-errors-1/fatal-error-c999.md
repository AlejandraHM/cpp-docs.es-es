# <a name="fatal-error-c999"></a>Error irrecuperable C999
MENSAJE desconocido elija el comando soporte técnico en el menú Ayuda de Visual C++, o abra el archivo de Ayuda de soporte técnico para obtener más información  
  
 Este error normalmente significa que se han combinado varios archivos de versiones diferentes del compilador o bien que la instalación del compilador está dañada. Use la **programas y características** applet del Panel de Control para reparar o reinstalar el producto.
