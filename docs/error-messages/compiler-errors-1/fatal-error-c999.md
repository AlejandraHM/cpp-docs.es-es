# Fatal Error C999
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MENSAJE DESCONOCIDO    Elija el comando Soporte técnico en el menú Ayuda de Visual C\+\+, o abra el archivo de ayuda Soporte técnico para obtener más información  
  
 Este error normalmente significa que se han combinado varios archivos de versiones diferentes del compilador o bien que la instalación del compilador está dañada.  Use el applet **Programas y características** del Panel de control para reparar o reinstalar el producto.
