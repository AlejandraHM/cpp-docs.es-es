# Error del compilador C2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator': puntero a la izquierda; se requiere un valor de tipo integral a la derecha  
  
 El operando izquierdo de `operator` era un puntero, por lo que el operando derecho debe ser un valor entero.
