# Error del compilador C2095
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : el parámetro real es de tipo 'void' : parámetro 'number'  
  
 El parámetro pasado a la función es de tipo `void`, lo cual no está permitido.  Utilice un puntero a void \(`void *`\)en su lugar.  
  
 El parámetro `number` indica qué parámetro es `void`.
