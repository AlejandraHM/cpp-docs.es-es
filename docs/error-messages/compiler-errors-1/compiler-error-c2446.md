# <a name="compiler-error-c2446"></a>Error del compilador C2446

'operador': no hay conversión de 'tipo1' a 'tipo2'

No se puede convertir el compilador `type1` a `type2`. La conversión no puede tener sentido porque infringe la semántica de C o C++.
