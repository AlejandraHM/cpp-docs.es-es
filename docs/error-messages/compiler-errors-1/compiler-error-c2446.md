# Error del compilador C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operador' : no hay conversión de 'type1' en 'type2'  
  
 El compilador no puede convertir `type1` en `type2`.  Puede que la conversión no tenga sentido porque infringe la semántica de C\/C\+\+.
