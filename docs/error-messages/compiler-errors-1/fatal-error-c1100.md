# <a name="fatal-error-c1100"></a>Error irrecuperable C1100

no se puede inicializar OLE: mensaje de error del sistema

El compilador no puede inicializar la biblioteca de modelo de objetos componentes (COM). Consulte [CoInitialize](/windows/desktop/api/objbase/nf-objbase-coinitialize).
