# Error irrecuperable C1201
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede continuar después del error de sintaxis en la definición de plantilla de clase  
  
 Se produjo un error inesperado mientras se analizaba una definición de plantilla de clase.  
  
 Corrija cualquier otro error y vuelva a compilar. Si esto no funciona, anote las circunstancias del error, intente aislar el problema y crear un caso de prueba reproducible, luego póngase en contacto con los [servicios de soporte técnico de Microsoft](../Topic/Talk%20to%20Us.md).
