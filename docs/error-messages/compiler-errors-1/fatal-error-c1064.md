# Error irrecuperable C1064
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

límite del compilador: desbordamiento de símbolo \(token\) de búfer interno  
  
 Un identificador supera la longitud del búfer interno utilizado para los identificadores.  Para solucionarlo, reduzca el nombre.
