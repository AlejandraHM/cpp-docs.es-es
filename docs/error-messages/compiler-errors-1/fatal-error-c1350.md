# Error irrecuperable C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error al cargar la DLL 'DLL': no se encontró la DLL  
  
 No se ha encontrado la DLL que admite la operación deseada.  
  
 Existe algún problema con la instalación: debería volver a instalar la DLL desde el CD del producto.
