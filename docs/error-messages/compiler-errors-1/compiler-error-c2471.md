# Error del compilador C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se puede actualizar la base de datos del programa 'archivo'  
  
 El compilador no puede escribir en el archivo de base de datos.  
  
### Para corregir el problema  
  
-   Asegúrese de que el disco tenga suficiente espacio disponible.  
  
-   Asegúrese de que el archivo no es de solo lectura.  
  
-   Asegúrese de que no haya infracciones de uso compartido.
