# Error del compilador C2486
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\_\_LOCAL\_SIZE' sólo se permite en la función con el atributo 'naked'  
  
 En funciones con ensamblado en línea, el nombre `__LOCAL_SIZE` se reserva para funciones declaradas con el atributo [naked](../../cpp/naked-cpp.md).  
  
 El código siguiente genera el error C2486:  
  
```  
// C2486.cpp  
// processor: x86  
void __declspec(naked) f1() {  
   __asm {  
      mov   eax,   __LOCAL_SIZE  
   }  
}  
void f2() {  
   __asm {  
      mov   eax,   __LOCAL_SIZE   // C2486  
   }  
}  
```
