# Error irrecuperable C1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error de \#using en 'archivo'  
  
 Sólo puede pasarse un archivo en formato MSIL \(lenguaje intermedio de Microsoft\) a una directiva [\#using](../../preprocessor/hash-using-directive-cpp.md).  La opción [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) del compilador permite crear un archivo de salida MSIL.  Otros lenguajes de [!INCLUDE[vsprvs](../../assembler/masm/includes/vsprvs_md.md)] también generan archivos MSIL.
