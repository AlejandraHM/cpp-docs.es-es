# <a name="compiler-error-c2036"></a>Error del compilador C2036

'identifier': tamaño desconocido

Una operación en `identifier` requiere que el tamaño del objeto de datos, que no se puede determinar.

## <a name="example"></a>Ejemplo

El ejemplo siguiente genera C2036.

```
// C2036.c
// a C program
struct A* pA;
struct B { int i; } *pB;
int main() {
   pA++;   // C2036, size of A not known
   ((char*)pA)++;   // OK

   pB++;   // OK
}
```

## <a name="example"></a>Ejemplo

El ejemplo siguiente genera C2036.

```
// C2036_2.cpp
// a C++ program
struct A* pA;
int main() {
   pA++;   // C2036, size of A not known
   ((char*&)pA)++;   // OK, if sizeof(A) == sizeof(char)
}
```
