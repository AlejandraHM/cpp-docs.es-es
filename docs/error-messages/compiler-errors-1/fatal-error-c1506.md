# Error irrecuperable C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error irrecuperable de ámbito de bloque  
  
 El bloque es demasiado grande para ser compilado.  
  
### Posibles causas del error:  
  
1.  Correspondencia incorrecta en las llaves  
  
2.  Función o clase demasiado grande
