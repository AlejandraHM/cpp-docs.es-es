# <a name="compiler-error-c2145"></a>Error del compilador C2145
error de sintaxis: falta 'token' delante del identificador  
  
 El compilador esperaba `token` y encontró un identificador en su lugar.  
  
 Este error puede deberse a la falta de un punto y coma después de la última declaración de un bloque.
