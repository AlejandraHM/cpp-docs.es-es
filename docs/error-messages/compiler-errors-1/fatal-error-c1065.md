# Error irrecuperable C1065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

límite del compilador : no hay etiquetas  
  
 El archivo de código fuente contiene más de 65.523 clases, structs, uniones, espacios de nombres o enumeraciones.  
  
 Divida el archivo en archivos más pequeños.
