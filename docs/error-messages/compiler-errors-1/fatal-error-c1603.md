# <a name="fatal-error-c1603"></a>Error irrecuperable C1603

destino de bifurcación de ensamblado alineado fuera de intervalo en bytes de 'número'

La distancia calculada entre un JCXZ o JECXZ y su etiqueta de destino especificado era superior a 128 bytes. Actualice el código para que la etiqueta está más cerca de la instrucción.
