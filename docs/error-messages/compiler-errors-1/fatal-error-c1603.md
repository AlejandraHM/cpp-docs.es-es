# Error irrecuperable C1603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

destino de bifurcación de ensamblado en línea fuera de intervalo en 'número' bytes  
  
 La distancia calculada entre una instrucción JCXZ o JECXZ y la etiqueta de destino especificada para ella era mayor de 128 bytes.  Actualice el código de modo que la etiqueta esté más cerca de la instrucción.
