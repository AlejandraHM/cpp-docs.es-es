# Error del compilador C2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : punteros a funciones con distintos atributos  
  
 Un puntero a una función con una convención de llamada `__cdecl`, `__stdcall` o `__fastcall` está asignado a un puntero a una función con otra convención de llamada.
