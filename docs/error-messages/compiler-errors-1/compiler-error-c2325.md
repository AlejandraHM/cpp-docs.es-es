# <a name="compiler-error-c2325"></a>Error del compilador C2325
'type': tipo no esperado a la derecha de 'nombre'  
  
 Se realiza una llamada a un destructor de tipo incorrecto.  
  
 El ejemplo siguiente genera C2325:  
  
```  
// C2325.cpp  
// compile with: /c  
class A {};  
typedef A* pA_t;  
void f() {  
    A** ppa = new A *;  
    ppa->~A*;   // C2325  
  
   pA_t *ppa2 = new pA_t;  
   ppa2->~pA_t();   // OK  
}  
```
