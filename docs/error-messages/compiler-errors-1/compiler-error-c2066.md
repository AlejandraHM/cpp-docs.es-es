# Error del compilador C2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la conversión a tipo de función no es válida  
  
 En ANSI C, no se permite la conversión entre punteros a función y punteros a datos.
