# Error irrecuperable C1053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\<identificador\>': la función es demasiado grande  
  
 La función es demasiado grande para compilarla.  
  
### Use las soluciones posibles siguientes para corregirlo  
  
1.  Intente compilarla sin optimizaciones.  
  
2.  Divida la función en funciones más pequeñas.  
  
3.  Reduzca las llamadas a funciones insertadas.
