# Error irrecuperable C1313
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

límite del compilador: los bloques tipo no se pueden anidar más de número niveles  
  
 Los bloques de control de excepciones \(o el control de excepciones estructurado\) se anidaron demasiado profundamente.  Simplifique el código.
