# <a name="fatal-error-c1602"></a>Error irrecuperable C1602

intrínseco no admitido

Este error indica una incoherencia en los archivos .exe del compilador, que se haya producido debido a una instalación incompleta. Por ejemplo, es posible que ha instalado un service pack, pero no en el módulo de procesador. Instale todos los productos necesarios.
