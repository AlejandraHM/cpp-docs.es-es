# Error irrecuperable C1602
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

intrínseco no admitido  
  
 Este error indica que los archivos .exe del compilador no coinciden, debido posiblemente a una instalación incompleta.  Por ejemplo, puede que se haya instalado un Service Pack pero no el Processor Pack correspondiente.  En dicho caso, instale todos los productos necesarios.
