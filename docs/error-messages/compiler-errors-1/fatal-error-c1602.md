# <a name="fatal-error-c1602"></a>Error irrecuperable C1602
intrínseco no admitido  
  
 Este error indica un error de coincidencia en los archivos .exe de compilador, que se haya producido debido a una instalación incompleta. Por ejemplo, quizás ha instalado un service pack, pero no en el módulo de procesador. Instale todos los productos necesarios.
