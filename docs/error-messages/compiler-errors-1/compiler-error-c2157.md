# <a name="compiler-error-c2157"></a>Error del compilador C2157
'function': se debe declarar antes de usarse en la lista pragma  
  
 El nombre de funci√≥n no se declara antes de que se hace referencia en la lista de funciones para una [alloc_text](../../preprocessor/alloc-text.md) pragma.  
  
 El ejemplo siguiente genera la advertencia C2157:  
  
```  
// C2157.cpp  
// compile with: /c  
#pragma alloc_text( "func", func)   // C2157  
  
// OK  
extern "C" void func();  
#pragma alloc_text( "func", func)  
```
