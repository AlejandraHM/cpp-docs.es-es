# Error del compilador C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función' : no está disponible como función intrínseca  
  
 Un pragma `intrinsic` o `function` muestra una función no disponible de forma intrínseca.  Por ejemplo, ciertas funciones intrínsecas no están disponibles cuando se compila un programa que utiliza programación \/clr.
