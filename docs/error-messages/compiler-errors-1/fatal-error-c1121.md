# <a name="fatal-error-c1121"></a>Error irrecuperable C1121

no se pudo llamar a CryptoAPI

El compilador realiza una llamada a CryptoAPI y el error en la llamada. Vuelva a instalar Visual Studio y, posiblemente, volver a instalar el sistema operativo.
