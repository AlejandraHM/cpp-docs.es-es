# Error irrecuperable C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

no se pudo llamar a CryptoAPI  
  
 El compilador realizó una llamada a CryptoAPI sin éxito.   Reinstale Visual Studio y es posible que tenga que reinstalar el sistema operativo.
