# <a name="compiler-error-c2100"></a>Error del compilador C2100
direccionamiento indirecto no v√°lido  
  
 Operador de direccionamiento indirecto ( `*` ) se aplica a un valor no puntero.  
  
 El ejemplo siguiente genera C2100:  
  
```  
// C2100.cpp  
int main() {  
   int r = 0, *s = 0;  
   s = &r;  
   *r = 200;   // C2100  
   *s = 200;   // OK  
}  
```
