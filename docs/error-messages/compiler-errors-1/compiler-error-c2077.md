# Error del compilador C2077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

inicializador de campo no escalar 'identificador'  
  
 Se ha intentado inicializar un campo de bits con un inicializador de campo no escalar \(struct, unión, matriz o clase\).  En su lugar, deberá utilizarse un valor entero o de punto flotante.
