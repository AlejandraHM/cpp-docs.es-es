# <a name="compiler-error-c2077"></a>C2077 de Error del compilador
inicializador de campo no escalar 'identificador'  
  
 Se intentó inicializar un campo de bits con un inicializador (estructura, unión, matriz o clase). Utilice un entero o un valor de punto flotante.
