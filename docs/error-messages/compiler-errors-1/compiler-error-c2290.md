# Error del compilador C2290
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Se omite la sintaxis de asm de C\+\+. Use \_\_asm.  
  
 La sintaxis de `asm` se reserva para uso futuro.
