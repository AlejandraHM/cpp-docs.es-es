# regex_traits&lt;wchar_t&gt; (Clase)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Especialización de regex\_traits para wchar\_t.  
  
## Sintaxis  
  
```  
template <>  
    class regex_traits<wchar_t>  
```  
  
## Comentarios  
 La clase es una especialización explícita de la clase de plantilla [regex\_traits \(Clase\)](../standard-library/regex-traits-class.md) para los elementos del tipo `wchar_t` \(de modo que pueda aprovechar las funciones de biblioteca que manipulan objetos de este tipo\).  
  
## Requisitos  
 **Encabezado:** \<regex\>  
  
 **Espacio de nombres:** std
