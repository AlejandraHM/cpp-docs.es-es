# defer_lock_t (Estructura)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxis  
  
```cpp  
struct defer_lock_t;  
```  
  
## Requisitos  
 **Encabezado:** mutex  
  
 **Espacio de nombres:** std
