# <a name="container-classiterator"></a>Clase de contenedor::iterator
> [!NOTE]
>  Este tema se incluye en la documentación de Visual C++ como un ejemplo no funcional de los contenedores usados en la biblioteca estándar de C++. Para obtener más información, vea [Contenedores de la biblioteca estándar de C++](../standard-library/stl-containers.md).  
  
 Describe un objeto que puede actuar como un iterador de la secuencia controlada.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
 
typedef T5 iterator;  
```  
  
## <a name="remarks"></a>Comentarios  
 Aquí se describe como sinónimo del tipo sin especificar **T5**. Un objeto de tipo **iterator** se puede convertir en un objeto de tipo [const_iterator](../standard-library/container-class-const-iterator.md).
