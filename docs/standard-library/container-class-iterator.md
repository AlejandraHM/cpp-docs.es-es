# Clase de contenedor::iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Este tema está en la documentación de Visual C\+\+ como ejemplo no funcional de contenedores utilizados en la biblioteca estándar de C\+\+.  Para obtener más información, vea [Contenedores STL](../standard-library/stl-containers.md).  
  
 Describe un objeto que puede actuar como iterador para la secuencia controlada.  
  
## Sintaxis  
  
```  
  
typedef T5 iterator;  
  
```  
  
## Comentarios  
 Se describe aquí como sinónimo del tipo sin especificar **T5**.  Un objeto de **iterator** tipo se pueda convertir en un objeto de [const\_iterator](../standard-library/container-class-const-iterator.md)escrito.
