## <a name="swap"></a>  swap

Intercambia los elementos de dos deques.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Parámetros

`left` Un objeto de tipo `deque`.

`right` Un objeto de tipo `deque`.
