##  <a name="swap"></a>  swap  
 Intercambia los elementos de dos deques.  
  
```
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>Parámetros  
 `left`  
 Objeto de tipo `deque`.  
  
 `right`  
 Objeto de tipo `deque`.
