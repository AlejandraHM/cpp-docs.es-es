# &lt; lista &gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Define la lista de clases de plantilla de contenedores y varias plantillas auxiliares.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
#include <list>  
```  
  
### <a name="operators"></a>Operadores  
  
|||  
|-|-|  
|[operador! =](../Topic/%3Clist%3E%20operators.md#operator_neq)|Comprueba si el objeto de lista del lado izquierdo del operador no es igual que el objeto de lista del lado derecho.|  
|[operador <](../Topic/%3Clist%3E%20operators.md#operator_lt_)|Comprueba si el objeto de lista del lado izquierdo del operador es menor que el objeto de lista del lado derecho.|  
|[operador \< =](../Topic/%3Clist%3E%20operators.md#operator_lt__eq)|Comprueba si el objeto de lista del lado izquierdo del operador es menor o igual que el objeto de lista del lado derecho.|  
|[operador ==](../Topic/%3Clist%3E%20operators.md#operator_eq_eq)|Comprueba si el objeto de lista del lado izquierdo del operador es igual que el objeto de lista del lado derecho.|  
|[operador >](../Topic/%3Clist%3E%20operators.md#operator_gt_)|Comprueba si el objeto de lista del lado izquierdo del operador es mayor que el objeto de lista del lado derecho.|  
|[operador > =](../Topic/%3Clist%3E%20operators.md#operator_gt__eq)|Comprueba si el objeto de lista del lado izquierdo del operador es mayor o igual que el objeto de lista del lado derecho.|  
  
### <a name="classes"></a>Clases  
  
|||  
|-|-|  
|[List (clase)](../standard-library/list-class.md)|Una clase de plantilla de contenedores de secuencias que mantienen sus elementos en disposición lineal y permiten realizar inserciones y eliminaciones de manera eficiente en cualquier ubicación de la secuencia.|
