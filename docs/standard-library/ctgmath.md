# &lt;ctgmath&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Incluye los encabezados \<ccomplex\> y \<cmath\> de la Biblioteca estándar de C\+\+, que proporcionan las macros math genéricas de tipo equivalentes a \<tgmath.h\>.  
  
## Sintaxis  
  
```  
  
#include <ctgmath>  
  
```  
  
## Comentarios  
 La funcionalidad del encabezado \<tgmath.h\> de la Biblioteca estándar de C la proporcionan las sobrecargas en \<ccomplex\> y \<cmath\>.  
  
 Incluir este encabezado también garantiza que los nombres declarados mediante vinculación externa en el encabezado de la biblioteca estándar de C se declaran en el espacio de nombres `std`.
