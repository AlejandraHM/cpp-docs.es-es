# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;
El encabezado `<cvt/wstring>` en versiones anteriores de Visual Studio definía la clase de plantilla [wstring_convert (Clase)](../standard-library/wstring-convert-class.md) en el espacio de nombres stdext::cvt. El encabezado se conserva por razones de compatibilidad con las versiones anteriores. El código nuevo debe usar la versión de la clase que se define en [\<locale>](../standard-library/locale.md), en el espacio de nombres `std`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
#include <cvt/wstring>  
  
```
