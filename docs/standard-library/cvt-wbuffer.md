# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer&gt;
El encabezado `<cvt/wstring>` en versiones anteriores de Visual Studio definía la clase de plantilla [wbuffer_convert (Clase)](../standard-library/wbuffer-convert-class.md) en el espacio de nombres stdext::cvt. El encabezado se conserva por razones de compatibilidad con las versiones anteriores. El código nuevo debe usar la versión de la clase que se define en [\<locale>](../standard-library/locale.md), en el espacio de nombres `std`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
#include <cvt/wbuffer>  
  
```
