# Clase de contenedor::difference_type
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Este tema está en la documentación de Visual C\+\+ como ejemplo no funcional de contenedores utilizados en la biblioteca estándar de C\+\+.  Para obtener más información, vea [Contenedores STL](../standard-library/stl-containers.md).  
  
 Describe un objeto que puede representar la diferencia entre las direcciones de los dos elementos de la secuencia controlada.  
  
## Sintaxis  
  
```  
  
typedef T1 difference_type;  
  
```  
  
## Comentarios  
 Se describe aquí como sinónimo del tipo sin especificar **T1** \(normalmente **Alloc::difference\_type**\).
