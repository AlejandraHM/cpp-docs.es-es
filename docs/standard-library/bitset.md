# &lt;bitset&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Define la clase de plantilla bitset y dos funciones auxiliares de plantilla para representar y manipular secuencias de bits de tamaño fijo.  
  
## Sintaxis  
  
```  
  
#include <bitset>  
  
```  
  
### Operadores  
  
|||  
|-|-|  
|[operator&](../Topic/operator&%20\(%3Cbitset%3E\).md)|Efectúa una operación AND bit a bit entre dos conjuntos de bits.|  
|[operador \<\<](../Topic/operator%3C%3C%20\(%3Cbitset%3E\).md)|Inserta una representación de texto de la secuencia de bits en el flujo de salida estándar.|  
|[operador \>\>](../Topic/operator%3E%3E%20\(%3Cbitset%3E\).md)|Inserta una representación de texto de la secuencia de bits en el flujo de entrada estándar.|  
|[operator^](../Topic/operator%5E%20\(%3Cbitset%3E\).md)|Efectúa una operación EXCLUSIVE\-OR bit a bit entre dos conjuntos de bits.|  
|[operator&#124;](../Topic/operator%7C%20\(%3Cbitset%3E\).md)|Efectúa una operación OR bit a bit entre dos conjuntos de bits.|  
  
### Clases  
  
|||  
|-|-|  
|[bitset \(Clase\)](../standard-library/bitset-class.md)|La clase de plantilla describe un tipo de objeto que almacena una secuencia que consta de un número fijo de bits que proporcionan una manera compacta de mantener indicadores para un conjunto de elementos o condiciones.|
