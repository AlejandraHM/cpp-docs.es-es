# C&#243;mo: Crear un proyecto de C++ a partir del c&#243;digo existente
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En [!INCLUDE[vsPro](../ide/includes/vspro_md.md)] y [!INCLUDE[vsUltShort](../ide/includes/vsultshort_md.md)], puede trasladar los archivos de código existentes a un proyecto de Visual C\+\+ mediante el asistente **Crear nuevo proyecto de archivos de código fuente existentes**.  Este asistente no está disponible en las ediciones Express.  
  
 Trasladar los archivos de código existentes a un proyecto de Visual C\+\+ permite utilizar características del IDE como [IntelliSense](../Topic/Using%20IntelliSense.md), el **Explorador de llamadas** y el [Administrador de propiedades](../misc/property-manager.md).  
  
### Para crear un proyecto de C\+\+ a partir de código existente  
  
1.  En el menú **Archivo**, elija **Nuevo** y, a continuación, haga clic en **Proyecto del código existente**.  
  
2.  En la primera página del asistente **Crear nuevo proyecto de archivos de código fuente existentes**, seleccione **Visual C\+\+** en la lista **¿Qué tipo de proyecto desea crear?**.  
  
3.  En las páginas subsiguientes del asistente, especifique la configuración del nuevo proyecto que desee crear.  
  
4.  Haga clic en **Finalizar** para generar el nuevo proyecto.
