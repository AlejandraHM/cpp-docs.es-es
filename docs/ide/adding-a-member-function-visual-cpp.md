# <a name="adding-a-member-function-visual-c"></a>Agregar una función miembro (Visual C++)
En la **Vista de clases**, puede agregar una función miembro a cualquier clase. Cuando lo hace, se agrega una declaración al archivo de encabezado, así como el cuerpo de una función miembro auxiliar al archivo de implementación de la clase, que después puede modificarse.  
  
### <a name="to-add-a-member-function-to-a-class"></a>Para agregar una función miembro a una clase  
  
1.  En la **Vista de clases**, expanda el nodo del proyecto para mostrar las clases del proyecto. (Para abrir la **Vista de clases**, en la barra de menús, seleccione **Ver**, **Vista de clases**).  
  
2.  Abra el menú contextual de la clase a la que quiera agregar una función miembro y, después, seleccione **Agregar**, **Agregar función**.  
  
3.  Proporcione los detalles deseados sobre la función miembro. Para obtener más información, vea [Asistente para agregar funciones miembro](../ide/add-member-function-wizard.md).  
  
4.  Haga clic en el botón **Finalizar** para generar el código de la función miembro.
