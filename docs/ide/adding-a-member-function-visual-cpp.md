# Agregar una funci&#243;n miembro (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En **Vista de clases**, puede agregar una función miembro a cualquier clase.  Cuando lo hace, se agrega una declaración al archivo de encabezado, así como el cuerpo de una función miembro auxiliar al archivo de implementación de la clase, que después puede modificarse.  
  
### Para agregar una función miembro a una clase  
  
1.  En la **Vista de clases**, expanda el nodo del proyecto para mostrar las clases del proyecto. \(Para abrir **Vista de clases**, en la barra de menús, elija **Ver**, **Vista de clases**\).  
  
2.  Abra el menú contextual de la clase a la que desea agregar una función miembro y, a continuación, elija **Agregar**, **Agregar función**.  
  
3.  Proporcione los detalles deseados sobre la función miembro.  Para obtener más información, vea [Asistente para agregar funciones miembro](../ide/add-member-function-wizard.md).  
  
4.  Elija el botón **Finalizar** para generar el código de la función miembro.
