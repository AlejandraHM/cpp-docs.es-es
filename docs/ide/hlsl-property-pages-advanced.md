# P&#225;ginas de propiedades HLSL: Avanzadas
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Para configurar las propiedades siguientes del compilador de HLSL \(fxc.exe\), utilice la página de propiedades **Avanzadas** .  Para obtener información sobre cómo tener acceso a la página de propiedades **Avanzadas** en la carpeta de HLSL, vea [Cómo: Especificar propiedades de proyecto con páginas de propiedades](../misc/how-to-specify-project-properties-with-property-pages.md).  
  
## Lista de UIElement  
 **Suprimir la pancarta de inicio**  
 **Sí \(\/nologo\)** para suprimir la presentación de la pancarta y el mensaje de información de versión; si no, **No**.  De forma predeterminada, el valor es **Sí \(\/nologo\)**.  
  
 **Tratar advertencias como errores**  
 **Sí \(\/WX\)** para tratar todas las advertencias del compilador como errores; si no, **No**.
