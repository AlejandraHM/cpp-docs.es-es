# asistente personalizado
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El Asistente personalizado de Visual C\+\+ es un tipo específico de asistente diseñado para ayudarle a crear sus propios asistentes.  Una vez configurado, el asistente genera los archivos de inicio necesarios para los proyectos.  
  
## Información general  
 En esta página del asistente, se describe la configuración actual de la aplicación para el proyecto del Asistente personalizado que se está creando.  De forma predeterminada, el asistente crea un proyecto de la manera siguiente:  
  
-   Se asigna como nombre descriptivo del asistente el nombre del proyecto indicado en el **cuadro de diálogo Nuevo proyecto**.  
  
-   El asistente tiene una interfaz de usuario.  
  
-   La interfaz de usuario del asistente tiene una página.  
  
 En la página [Configuración de la aplicación](../ide/application-settings-custom-wizard.md) del asistente se puede cambiar la configuración predeterminada.
