# Agregar un evento (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En la Vista de clases puede agregar un evento, mediante el [Asistente para agregar eventos](../ide/add-event-wizard.md), solamente a la clase de control de su proyecto de [control ActiveX de MFC](../mfc/reference/creating-an-mfc-activex-control.md).  Si desea agregar un evento a un proyecto de otro tipo, use el botón **Eventos** de la [ventana Propiedades](../Topic/Properties%20Window.md).  
  
### Para agregar un evento a un proyecto de control ActiveX de MFC  
  
1.  En la Vista de clases, expanda el nodo del proyecto hasta mostrar las clases del proyecto.  
  
2.  Haga clic con el botón secundario del mouse en la clase de control del proyecto.  
  
3.  En el menú contextual, haga clic en **Agregar** y, después, en **Agregar evento** para ver el Asistente para agregar eventos.  
  
4.  Suministre la información del evento en los cuadros apropiados del asistente.  
  
5.  Haga clic en **Finalizar** para agregar el evento al proyecto.
