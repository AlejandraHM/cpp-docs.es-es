# <a name="navigating-the-class-structure-visual-c"></a>Navegar por la estructura de clases (Visual C++)
Puede navegar por una estructura de clase de proyecto, incluyendo las funciones, herencia y las interfaces mediante [vista de clases](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925) o [Examinador de objetos](http://msdn.microsoft.com/en-us/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Puede ir a una declaración de función haciendo doble clic en el nombre de función en la vista de clases. Puede ir a una definición de clase haciendo doble clic en un nombre de clase en la vista de clases.
