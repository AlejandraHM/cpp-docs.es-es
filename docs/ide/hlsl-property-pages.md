# P&#225;ginas de propiedades HLSL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Puede utilizar las páginas de propiedades del compilador de HLSL \(fxc.exe\) para configurar cómo se compilan los archivos individuales del sombreador de HLSL.  Las propiedades se dividen en tres categorías:  
  
-   [Páginas de propiedades HLSL: General](../ide/hlsl-property-pages-general.md)  
  
     Proporciona propiedades para especificar las opciones de uso general para el compilador de HLSL.  
  
-   [Páginas de propiedades HLSL: Avanzadas](../ide/hlsl-property-pages-advanced.md)  
  
     Proporciona propiedades para especificar opciones adicionales para el compilador de HLSL.  
  
-   [Páginas de propiedades HLSL: archivos de salida](../ide/hlsl-property-pages-output-files.md)  
  
     Proporciona propiedades para especificar cómo los archivos son generados por el compilador de HLSL.  
  
 Puede especificar argumentos de línea de comandos del compilador de HLSL mediante la propiedad **Opciones adicionales** de la página de propiedades **Línea de comandos** ; esto incluye los argumentos que no se pueden configurar utilizando otras propiedades de páginas de propiedades de HLSL.  Para obtener información sobre el compilador de HLSL, vea [Herramienta de Compilador de efectos](http://go.microsoft.com/fwlink/p/?LinkID=258285&clcid=0x409)
