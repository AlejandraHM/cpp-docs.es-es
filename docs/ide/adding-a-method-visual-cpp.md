# Agregar un m&#233;todo (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Puede utilizar el [Asistente para agregar métodos](../ide/add-method-wizard.md) con el fin de agregar una propiedad a una interfaz del proyecto.  Si el proyecto contiene una clase asociada a la interfaz, el asistente también modifica la clase.  
  
### Para agregar un método al objeto  
  
1.  En la Vista de clases, expanda el nodo del proyecto de modo que aparezca la interfaz a la que desee agregar el método.  
  
    > [!NOTE]
    >  También puede agregar métodos a interfaces dispinterfaces que, a menos que el proyecto tenga atributos, se encontrarán bajo el nodo de biblioteca.  
  
2.  Haga clic con el botón secundario en el nombre de la interfaz.  
  
3.  En el menú contextual, haga clic en **Agregar** y, a continuación, en **Agregar método**.  
  
4.  En el Asistente para agregar métodos, especifique la información necesaria para crear el método.  
  
5.  Especifique la configuración de lenguaje de definición de interfaz para este método en la página [atributos IDL](../ide/idl-attributes-add-method-wizard.md) del asistente.  
  
6.  Haga clic en **Finalizar** para agregar el método.
