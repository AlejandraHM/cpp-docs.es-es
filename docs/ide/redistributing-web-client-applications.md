# Redistribuir aplicaciones cliente web
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Si su aplicación utiliza las clases MFC que implementan el control WebBrowser \(por ejemplo, `CHtmlView` o `CHtmlEditView`\), debe haber instalada la versión 4.0 o posterior de Microsoft Internet Explorer en el equipo de destino.  
  
 Instalar la última versión de Internet Explorer también servirá para garantizar que el equipo de destino posea los archivos de control comunes más recientes.  
  
 En el siguiente artículo de Knowledge Base encontrará información sobre cómo instalar los componentes esenciales de Internet Explorer:  
  
-   Q185375, HOWTO: Create a Single EXE Install of Internet Explorer \([http:\/\/support.microsoft.com\/support\/kb\/articles\/q185\/3\/75.asp](http://support.microsoft.com/support/kb/articles/q185/3/75.asp)\)  
  
 Encontrará artículos de Knowledge Base en MSDN Library o en la dirección [http:\/\/support.microsoft.com](http://support.microsoft.com).
