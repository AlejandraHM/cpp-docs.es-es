# Herramienta Generador de datos HTML (P&#225;gina de propiedades)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La página de propiedades **Herramienta generadora de datos XML** aparece disponible cuando se agrega un conjunto de datos a un proyecto.  
  
 La página de propiedades **Herramienta generadora de datos XML** contiene las siguientes propiedades:  
  
 **Archivo de resultados**  
 Especifica el nombre del archivo de salida que se va a utilizar.  
  
 **Suprimir la pancarta de inicio**  
 Suprime la presentación de la pancarta de inicio y los mensajes de información.  
  
 **Lenguaje del proxy generado**  
 Determina si se debe o no emitir código administrado.  
  
 Para obtener información sobre cómo obtener acceso a la página de propiedades **Herramienta Generador de datos XML**, vea [Cómo: Especificar propiedades de proyecto con páginas de propiedades](../misc/how-to-specify-project-properties-with-property-pages.md).  
  
 Para obtener información sobre cómo obtener acceso mediante programación a los miembros de este objeto, vea <xref:Microsoft.VisualStudio.VCProjectEngine.VCXMLDataGeneratorTool>.
