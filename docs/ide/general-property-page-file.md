# P&#225;gina de propiedades General (Archivo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cuando se selecciona un archivo en el **Explorador de soluciones**, la página de propiedades **General** bajo el nodo **Propiedades de configuración** contiene las propiedades siguientes:  
  
 **Excluir de la compilación**  
 Especifica si el archivo debe estar en la compilación de la configuración actual.  
  
 Para obtener acceso a esta propiedad mediante programación, vea <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild%2A>.  
  
 **Herramienta**  
 La herramienta que se utilizará para compilar este archivo.  Para obtener más información, consulte [Especificar las herramientas de compilación personalizadas](../ide/specifying-custom-build-tools.md).  
  
 Para obtener acceso a esta propiedad mediante programación, vea <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool%2A>.  
  
 Para obtener información sobre cómo tener acceso a la página de propiedades **General** bajo el nodo **Propiedades de configuración**, vea [Cómo: Especificar las propiedades de un proyecto con las páginas de propiedades](../misc/how-to-specify-project-properties-with-property-pages.md).
