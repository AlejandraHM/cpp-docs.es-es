# P&#225;ginas de propiedades L&#237;nea de comandos
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La mayoría de las carpetas de la página de propiedades contienen una página de propiedades **Línea de comandos**.  Esta página muestra las propiedades establecidas en la carpeta.  La página de propiedades **Línea de comandos** también contiene un cuadro **Opciones adicionales**, en el que se pueden especificar propiedades válidas para la herramienta, pero para las que no hay ninguna propiedad en la carpeta.  
  
 Cualquier comando que escriba en el cuadro de edición se pasará a través de la herramienta para la carpeta.  No se realizarán comprobaciones en la entrada, ni se analizarán las dependencias.  
  
 Para obtener información sobre cómo tener acceso a la página de propiedades **Línea de comandos**, vea [Cómo: Especificar las propiedades de un proyecto con las páginas de propiedades](../misc/how-to-specify-project-properties-with-property-pages.md).
