# <a name="adding-a-generic-c-class"></a>Agregar una clase genérica de C++
Puede agregar una clase genérica de C++ mediante **vista de clases**. Una clase genérica de C++ es una clase que defina o que se deriva de una clase que defina.  
  
### <a name="to-add-a-generic-c-class-to-a-project"></a>Para agregar una clase genérica de C++ a un proyecto  
  
1.  En **vista de clases**, haga clic en el proyecto al que desea agregar la nueva clase, haga clic en **agregar**y, a continuación, haga clic en **clase**.  
  
2.  En el [Agregar clase](../ide/add-class-dialog-box.md) cuadro de diálogo, en el panel Plantillas, haga clic en **clases de C++**. Haga clic en **agregar** para mostrar la [Asistente de clases genéricas de C++](../ide/generic-cpp-class-wizard.md).  
  
3.  En el asistente, proporcione un nombre de clase y, a continuación, definir la configuración o acepte los valores predeterminados.  
  
4.  Para cerrar el asistente y ver la nueva clase genérica de C++ en el proyecto, haga clic en **finalizar**.
