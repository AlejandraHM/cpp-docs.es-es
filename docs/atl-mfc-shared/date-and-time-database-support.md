# Date and Time: Database Support
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A partir de la versión 4,0, la programación de base de datos MFC utiliza clases de [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) y de [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) para representar datos de fecha y hora.  Estas clases, también utilizadas en la automatización, se derivan de la clase [COleVariant](../mfc/reference/colevariant-class.md).  Proporciona mejor compatibilidad para administrar datos de fecha y hora que [CTime](../atl-mfc-shared/reference/ctime-class.md) y [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md).
