# ATL/MFC Shared Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A partir de Visual C\+\+ .NET 2002, varias clases de utilidad existentes de MFC se reescritas o revisadas para reducir las dependencias en otras clases MFC.  Estas clases de utilidad ahora se puede utilizar en cualquier proyecto de C\+\+ nativo.  Esta sección incluye únicamente las clases que eran disponibles para los proyectos de MFC y ahora se han compartido, más algunas nuevas clases relacionadas con los cambios en `CString`.  No incluye las clases ATL, que se pueden utilizar en cualquier tipo de proyecto de C\+\+ nativo mediante la inclusión del encabezado adecuado.
