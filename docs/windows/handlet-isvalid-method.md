# HandleT::IsValid (M&#233;todo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica si el objeto actual de HandleT representa un identificador.  
  
## Sintaxis  
  
```  
bool IsValid() const;  
```  
  
## Valor devuelto  
 `true` si el HandleT representa un identificador; si no, `false`.  
  
## Requisitos  
 **Encabezado:** corewrappers.h  
  
 **Espacio de nombres:** Microsoft::WRL::Wrappers
