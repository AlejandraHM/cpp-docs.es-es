# <a name="vararg"></a>vararg
Especifica que la función toma un número variable de argumentos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[vararg]  
```  
  
## <a name="remarks"></a>Comentarios  
 El **vararg** atributo de C++ tiene la misma funcionalidad que el [vararg](http://msdn.microsoft.com/library/windows/desktop/aa367304) atributo MIDL.  
  
## <a name="example"></a>Ejemplo  
 El código siguiente muestra un uso de **vararg**:  
  
```cpp  
// cpp_attr_ref_vararg.cpp  
// compile with: /LD  
#include "unknwn.h"  
#include "oaidl.h"  
[module(name="MyLibrary")];  
  
[object, uuid("00000000-0000-0000-0000-000000000001")]  
__interface X : public IUnknown   
{  
   [vararg] HRESULT Button([in, satype(VARIANT)]SAFEARRAY *psa);  
};  
```
