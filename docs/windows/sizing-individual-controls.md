# Sizing Individual Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilice los cuadros de tamaño para cambiar el tamaño de un control.  Cuando se sitúa el puntero sobre un cuadro de tamaño, su forma cambia para indicar las direcciones en que se puede variar el tamaño del control.  Los cuadros de tamaño activos son sólidos; si un cuadro de tamaño está hueco, el tamaño del control no podrá variarse siguiendo ese eje.  
  
 También puede cambiarse el tamaño de un control ajustándolo a las guías o los márgenes o desplazando un control ajustado y alejándolo de otro.  
  
### Para cambiar el tamaño de un control  
  
1.  Seleccione el control.  
  
2.  Arrastre los cuadros de tamaño para cambiar el tamaño del control:  
  
    -   Los cuadros superior y laterales cambian o el tamaño horizontal o el vertical.  
  
    -   Los cuadros de tamaño de las esquinas cambian el tamaño horizontal y el vertical.  
  
    > [!TIP]
    >  Puede cambiar el tamaño del control en una unidad de cuadro de diálogo \(DLU\) cada vez manteniendo presionada la tecla MAYÚS y usando las teclas de dirección derecha y abajo.  
  
### Para cambiar automáticamente el tamaño de un control en función del texto que contiene  
  
1.  Elija **Ajustar al contenido** en el menú **Formato**.  
  
 \-O bien\-  
  
-   Haga clic con el botón secundario del mouse y elija **Ajustar al contenido** en el menú contextual.
