# Compiler Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los atributos del compilador proporcionan una variedad de funcionalidades.  
  
|Atributo|Descripción|  
|--------------|-----------------|  
|[emitidl](../windows/emitidl.md)|Determina si todos los atributos siguientes IDL se procesarán y colocados en el archivo generado .idl.|  
|[event\_receiver](../windows/event-receiver.md)|Crea un receptor de eventos.|  
|[event\_source](../windows/event-source.md)|crea un origen de eventos.|  
|[export](../windows/export.md)|Genera una estructura de datos que se almacena en el archivo .idl.|  
|[implements](../Topic/implements%20\(C++\).md)|Especifica las interfaces de distribución que se convierten para ser miembros coclass IDL.|  
|[importidl](../windows/importidl.md)|Inserta el archivo especificado .idl en el archivo generado .idl.|  
|[importlib](../windows/importlib.md)|Muestra tipos que han sido ya compilados en otra biblioteca de tipos disponible a la biblioteca de tipos que se creó.|  
|[includelib](../windows/includelib-cpp.md)|Produce un archivo .idl o .h que se incluirá en el archivo generado .idl.|  
|[library\_block](../windows/library-block.md)|Coloca una construcción dentro del bloque de la biblioteca del archivo .idl.|  
|[no\_injected\_text](../windows/no-injected-text.md)|Evita que el compilador inserte código como resultado de uso del atributo.|  
|[satype](../windows/satype.md)|especifica el tipo de datos de **SAFEARRAY**.|  
|[version](../windows/version-cpp.md)|Identifica una determinada versión entre varias versiones de una interfaz o una clase.|
