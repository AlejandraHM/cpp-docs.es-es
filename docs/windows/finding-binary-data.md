# Finding Binary Data
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Las búsquedas pueden ser de cadenas ASCII o de bytes hexadecimales.  Por ejemplo, para encontrar "Hello", puede buscar la cadena "Hello" o su equivalente hexadecimal "48 65 6C 6C 6F".  
  
### Para buscar datos binarios  
  
1.  En el menú **Edición**, haga clic en [Buscar](../Topic/Find%20Command.md).  
  
2.  En el cuadro **Buscar**, seleccione una cadena de búsqueda que se haya utilizado previamente en la lista desplegable o escriba el dato que desea encontrar.  
  
3.  Seleccione cualquiera de las opciones de **Buscar**.  
  
4.  Haga clic en **Buscar siguiente**.

Para obtener información sobre cómo agregar manualmente archivos de recursos a proyectos administrados, cómo obtener acceso a recursos, cómo mostrar recursos estáticos y cómo asignar cadenas de recursos a propiedades, vea [Tutorial: Adaptar formularios Windows Forms](http://msdn.microsoft.com/es-es/9a96220d-a19b-4de0-9f48-01e5d82679e5) y [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
