# Filling a Bounded Area of an Image with a Color (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La herramienta **Rellenar** del Editor de imágenes permite rellenar el área de la imagen delimitada con el color del dibujo actual o con el color de fondo actual.  
  
> [!TIP]
>  Aparece información sobre herramientas cuando se desplaza el cursor sobre un botón de la barra de herramientas.  Estas sugerencias ayudan a identificar la función de cada botón.  
  
### Para usar la herramienta Rellenar  
  
1.  En la barra de herramientas del **Editor de imágenes** \(o en el menú **Imagen**, comando **Herramientas**\), haga clic en la herramienta **Rellenar**.  
  
2.  Si es necesario, elija los colores de dibujo: en la [paleta Colores](../Topic/Colors%20Window%20\(Image%20Editor%20for%20Icons\).md), haga clic con el botón primario del mouse para seleccionar un color de primer plano, o con el botón secundario para seleccionar un color de fondo.  
  
3.  Desplace la herramienta **Rellenar** al área que desee rellenar.  
  
4.  Haga clic con el botón primario o secundario del mouse para rellenar con el color de primer plano o de fondo, respectivamente.
