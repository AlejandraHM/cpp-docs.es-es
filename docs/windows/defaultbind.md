# defaultbind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica la propiedad única, enlazable que mejor representa el objeto.  
  
## Sintaxis  
  
```  
  
[defaultbind]  
  
```  
  
## Comentarios  
 el atributo de **defaultbind** C\+\+ tiene la misma funcionalidad que el atributo de [defaultbind](http://msdn.microsoft.com/library/windows/desktop/aa366790) MIDL.  
  
## Ejemplo  
 Vea el ejemplo para [enlazable](../windows/bindable.md) para obtener un ejemplo de cómo utilizar **defaultbind**.
