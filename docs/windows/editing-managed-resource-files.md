# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los archivos de recursos administrados \(.resx\) son archivos XML.  Cuando se agrega un archivo de recursos administrado al proyecto desde el cuadro de diálogo **Agregar nuevo elemento**, el **Editor de recursos administrados** se abre de forma predeterminada.  
  
 Para obtener más información, vea [Recursos en formato de archivos .Resx](http://msdn.microsoft.com/es-es/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5).
