# Module::Terminate (M&#233;todo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Hace que todos los generadores de crear una instancia por el módulo de apagar.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
void Terminate();  
```  
  
## <a name="remarks"></a>Comentarios  
 Libera los generadores en la memoria caché.  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** module.h  
  
 **Espacio de nombres:** Microsoft::WRL
