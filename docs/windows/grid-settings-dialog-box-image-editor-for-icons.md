# Configuraci&#243;n de la cuadr&#237;cula (Cuadro de di&#225;logo) (Editor de im&#225;genes para iconos)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Permite especificar la configuración de la cuadrícula de una imagen y muestra las líneas de la cuadrícula encima de la imagen editada.  Las líneas son útiles para editar la imagen, pero no se guardan como una parte de la imagen.  
  
 **Cuadrícula de píxeles**  
 Si está activada, presenta una cuadrícula alrededor de cada píxel en el Editor de imágenes.  La cuadrícula sólo aparece a resoluciones de 4× y superiores.  
  
 **Cuadrícula de mosaico**  
 Si está seleccionada, muestra una cuadrícula alrededor de los bloques de píxeles en el Editor de imágenes, que se especifica por los valores de espaciado de cuadrícula.  
  
 **Ancho**  
 Especifica el ancho de cada bloque del mosaico.  Es útil para dibujar mapas de bits que contengan varias imágenes dispuestas a intervalos regulares.  
  
 **Alto**  
 Especifica el alto de cada bloque del mosaico.  Es útil para dibujar mapas de bits que contengan varias imágenes dispuestas a intervalos regulares.
