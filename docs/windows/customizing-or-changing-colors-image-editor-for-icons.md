# Customizing or Changing Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El editor de imágenes [Paleta de colores](../Topic/Colors%20Window%20\(Image%20Editor%20for%20Icons\).md) muestra inicialmente los 16 colores estándar.  Además de los colores mostrados, puede crear sus propios colores personalizados.  A continuación, puede [guardar y cargar una paleta de colores personalizada](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
### Para cambiar los colores de la paleta de colores  
  
1.  Desde el menú **Imagen**, elija **Ajustar colores**.  
  
2.  En el [cuadro de diálogo Selector de colores personalizados](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md), defina el color escribiendo los valores RGB o HSL en el cuadro de texto correspondiente o elija un color en el cuadro **Muestra de color degradado**.  
  
3.  Establezca la luminosidad moviendo el control deslizante de la barra **Luminosidad**.  
  
4.  Muchos colores personalizados están interpolados.  Si desea el color sólido más próximo al color interpolado, haga doble clic en el cuadro **Color**.  
  
     Si más adelante prefiere el color interpolado, mueva el control deslizante de la barra **Luminosidad** o mueva la cruz del cuadro **Muestra de color degradado** de nuevo para restaurar la interpolación.  
  
5.  Haga clic en **Aceptar** para agregar el color nuevo.  
  
 Para obtener información sobre cómo agregar recursos a proyectos administrados, vea [Recursos de aplicaciones](../Topic/Resources%20in%20Desktop%20Apps.md) en la *Guía del desarrollador de .NET Framework*. Para obtener información sobre cómo agregar manualmente archivos de recursos a proyectos administrados, acceder a recursos, mostrar recursos estáticos y asignar cadenas de recursos a propiedades, vea este [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
