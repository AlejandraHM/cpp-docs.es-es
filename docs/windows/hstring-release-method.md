# HString::Release (M&#233;todo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Elimina el valor de cadena subyacente e inicializa el objeto HString actual con un valor vacío.  
  
## Sintaxis  
  
```  
void Release() throw()  
```  
  
## Requisitos  
 **Encabezado:** corewrappers.h  
  
 **Espacio de nombres:** Microsoft::WRL::Wrappers
