# <a name="adding-a-control-to-a-dialog-box"></a>Agregar un control a un cuadro de diálogo
### <a name="to-add-a-control-to-a-dialog-box"></a>Para agregar un control a un cuadro de diálogo  
  
1.  Asegúrese de que la ventana con pestañas de cuadro de diálogo sea el documento actual en el marco del editor. Si un cuadro de diálogo no es el documento actual, no verá la **pestaña Editor de cuadros de diálogo** en el **Cuadro de herramientas**.  
  
2.  En la [pestaña Editor de cuadros de diálogo](../windows/dialog-editor-tab-toolbox.md) de la [ventana Cuadro de herramientas](/visualstudio/ide/reference/toolbox), seleccione el control que desee y luego:  
  
    -   Haga clic en el cuadro de diálogo en la ubicación donde quiere colocar el control. El control aparecerá donde haga clic. Para obtener información, vea [Agregar varios controles](../windows/adding-multiple-controls.md).  
  
         \- o -  
  
    -   Arrastre y coloque el control desde el **cuadro de herramientas** ventana hasta la ubicación en el cuadro de diálogo. Para obtener más información, vea [Cambiar el tamaño de un control al agregarlo](../windows/sizing-a-control-while-you-add-it.md).  
  
         \- o -  
  
    -   Haga doble clic en el control en el **cuadro de herramientas** ventana (aparece en el cuadro de diálogo), a continuación, cambiar de posición el control a la ubicación que prefiera.  
  
 Para obtener información sobre los tipos de controles disponibles en el **cuadro de herramientas** ventana, consulte [pestaña Editor de cuadro de diálogo, cuadro de herramientas](../windows/dialog-editor-tab-toolbox.md).  
  
 Para obtener información sobre cómo agregar recursos a proyectos administrados, vea [Resources in Desktop Apps](/dotnet/framework/resources/index) en el *Guía del desarrollador de .NET Framework*. Para obtener información sobre cómo agregar manualmente archivos de recursos a proyectos administrados, acceder a los recursos, mostrar recursos estáticos y asignar cadenas de recursos a propiedades, vea [crear archivos de recursos para las aplicaciones de escritorio](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Para obtener información sobre la globalización y localización de recursos en aplicaciones administradas, vea [Globalizar y localizar aplicaciones de .NET Framework](/dotnet/standard/globalization-localization/index).
