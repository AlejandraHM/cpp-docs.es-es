# Crear una aplicaci&#243;n de consola
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La forma más fácil de crear una aplicación de consola Win32 es con el Asistente para aplicaciones Win32.  
  
### Para crear una aplicación de consola Win32  
  
1.  Siga las instrucciones descritas en el tema de Ayuda [Crear un proyecto con el Asistente para aplicaciones de Visual C\+\+](../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  En el cuadro de diálogo **Nuevo proyecto**, seleccione **Proyecto de consola Win32** en el panel Plantillas para abrir el asistente.  
  
3.  Defina la [configuración de la aplicación](../Topic/Application%20Settings,%20Win%2032%20Project%20Wizard.md) mediante el [Asistente para aplicaciones Win32](../windows/win32-application-wizard.md).  
  
    > [!NOTE]
    >  Omita este paso para mantener la configuración predeterminada del asistente.  
  
4.  Haga clic en **Finalizar** para cerrar el asistente y abrir el proyecto recién creado en el **Explorador de soluciones**.
