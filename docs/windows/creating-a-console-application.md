# <a name="creating-a-console-application"></a>Crear una aplicación de consola
La forma más fácil de crear una aplicación de consola Win32 es con el Asistente para aplicaciones Win32.  
  
### <a name="to-create-a-win32-console-application"></a>Para crear una aplicación de consola Win32  
  
1.  Siga las instrucciones del tema de Ayuda [crear un proyecto con un Asistente para aplicaciones de Visual C++](../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  En el **nuevo proyecto** cuadro de diálogo, seleccione **proyecto de consola Win32** en el panel Plantillas para abrir el asistente.  
  
3.  Definir la [configuración de la aplicación](../windows/application-settings-win-32-project-wizard.md) mediante la [Asistente para aplicaciones Win32](../windows/win32-application-wizard.md).  
  
    > [!NOTE]
    >  Omita este paso para mantener la configuración predeterminada del asistente.  
  
4.  Haga clic en **finalizar** cerrar el asistente y el proyecto recién creado se abre en **el Explorador de soluciones**.
