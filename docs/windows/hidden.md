# <a name="hidden"></a>hidden
Indica que el elemento existe pero no se debe mostrar en un explorador orientado al usuario.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
[hidden]  
  
```  
  
## <a name="remarks"></a>Comentarios  
 El **oculto** atributo C++ tiene la misma funcionalidad que la [oculto](http://msdn.microsoft.com/library/windows/desktop/aa366861) atributo MIDL.  
  
## <a name="example"></a>Ejemplo  
 Vea el ejemplo de [enlazables](../windows/bindable.md) para obtener un ejemplo de cómo usar **oculto**.
