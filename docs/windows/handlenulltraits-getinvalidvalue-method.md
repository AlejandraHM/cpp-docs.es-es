# HANDLENullTraits::GetInvalidValue (M&#233;todo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxis  
  
```  
inline static Type GetInvalidValue();  
```  
  
## Valor devuelto  
 Siempre devuelve `nullptr`.
