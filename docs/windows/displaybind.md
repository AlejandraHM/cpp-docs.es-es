# displaybind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica una propiedad que se debe mostrar al usuario como enlazable.  
  
## Sintaxis  
  
```  
  
[displaybind]  
  
```  
  
## Comentarios  
 el atributo de **displaybind** C\+\+ tiene la misma funcionalidad que el atributo de [displaybind](http://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL.  
  
## Ejemplo  
 Vea el ejemplo para [enlazable](../windows/bindable.md) para obtener un ejemplo de cómo utilizar **displaybind**.
