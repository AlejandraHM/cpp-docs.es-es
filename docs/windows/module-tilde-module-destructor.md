# <a name="modulemodule-destructor"></a>Module::~Module (Destructor)

Desinicializa la instancia actual de la **módulo** clase.

## <a name="syntax"></a>Sintaxis

```cpp
virtual ~Module();
```

## <a name="requirements"></a>Requisitos

**Encabezado:** module.h

**Espacio de nombres:** Microsoft::WRL
