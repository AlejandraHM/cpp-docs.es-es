# <a name="open-ltdevicegt-image-dialog-box-image-editor-for-icons"></a>Abra &lt;dispositivo&gt; imagen, cuadro de diálogo (Editor de imágenes para iconos)
Utilice este cuadro de diálogo para abrir las imágenes de dispositivo. Enumera las imágenes de dispositivo existentes en el recurso actual (imágenes que forman parte del recurso actual).  
  
### <a name="current-images"></a>Imágenes actuales  
 Enumera las imágenes incluidas en el recurso. Seleccione el tipo de imagen que desea abrir.  
  
## <a name="requirements"></a>Requisitos  
 Ninguna
