# Attributes by Usage
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Este tema enumeran los atributos de acuerdo a los elementos del lenguaje C\+\+ a los que se aplican.  
  
 Si un atributo precede a un elemento de Visual C\+\+ que no está en el ámbito del atributo, el bloque de atributos se trata como un comentario.  
  
|Atributo|Descripción|  
|--------------|-----------------|  
|[Atributos de módulo](../windows/module-attributes.md)|Se aplica a la  [módulo de](../windows/module-cpp.md) atributo.|  
|[Atributos de interfaz](../windows/interface-attributes.md)|Se aplica a la  [\_\_interface](../cpp/interface.md) palabra clave de C\+\+.|  
|[Atributos de clase](../windows/class-attributes.md)|Se aplica a la palabra clave de C\+\+.|  
|[Atributos de método](../windows/method-attributes.md)|Se aplica a los métodos en una clase, una coclase o una interfaz.|  
|[Atributos de parámetro](../windows/parameter-attributes.md)|Se aplica a los parámetros de un método en una clase o interfaz.|  
|[Atributos de miembro de datos](../windows/data-member-attributes.md)|Se aplica a los miembros de datos en una clase, una coclase o una interfaz.|  
|[TypeDef, enumeración, unión y atributos de estructura](../windows/typedef-enum-union-and-struct-attributes.md)|Se aplica a las palabras clave de C\+\+.|  
|[Matriz de atributos](../windows/array-attributes.md)|Se aplica a arreglos de discos o  **SAFEARRAY**s.|  
|[Atributos independientes](../Topic/Stand-Alone%20Attributes.md)|Funciona como una línea de código, pero no funciona en una palabra clave de C\+\+.  Las instrucciones de atributo independiente requieren un punto y coma al final de la línea.|  
|[Atributos personalizados](../windows/custom-attributes-cpp.md)|Permite al usuario extender metadatos.|
