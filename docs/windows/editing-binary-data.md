# Editing Binary Data
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Para modificar un recurso en el Editor binario  
  
1.  Seleccione el byte que desee modificar.  
  
     La tecla TABULADOR desplaza el foco entre las secciones hexadecimal y ASCII del Editor binario.  Puede utilizar las teclas RE PÁG y AV PÁG para desplazarse por el recurso de pantalla en pantalla.  
  
2.  Escriba el nuevo valor.  
  
     El valor cambia inmediatamente en ambas secciones, hexadecimal y ASCII, y el foco se desplaza hasta el siguiente valor en la línea.  
  
    > [!NOTE]
    >  El Editor binario acepta los cambios automáticamente al cerrar su ventana.
