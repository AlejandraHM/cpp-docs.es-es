# immediatebind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica que la base de datos se notifique inmediatamente de todos los cambios de una propiedad de un objeto enlazado a datos.  
  
## Sintaxis  
  
```  
  
[immediatebind]  
  
```  
  
## Comentarios  
 el atributo de **immediatebind** C\+\+ tiene la misma funcionalidad que el atributo de [immediatebind](http://msdn.microsoft.com/library/windows/desktop/aa367045) MIDL.  
  
## Ejemplo  
 Vea [enlazable](../windows/bindable.md) para obtener un ejemplo de cómo utilizar **immediatebind**.
