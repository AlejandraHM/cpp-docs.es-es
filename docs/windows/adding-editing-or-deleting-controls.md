# <a name="adding-editing-or-deleting-controls"></a>Agregar, editar o eliminar controles
Con el editor de cuadro de diálogo, puede agregar, cambiar el tamaño, editar y eliminar los controles de cuadros de diálogo. También puede editar las propiedades de un control, como su identificador, o si está visible inicialmente en tiempo de ejecución.  
  
 Puede usar el menú contextual en el editor de cuadro de diálogo para agregar rápidamente controles ActiveX registrados en un cuadro de diálogo, y puede agregar controles ActiveX al cuadro de herramientas para un acceso rápido (para obtener más información, consulte [Personalizar cuadro de diálogo](http://msdn.microsoft.com/bd07835f-18a8-433e-bccc-7141f65263bb)).  
  
-   [Agregar un control a un cuadro de diálogo](adding-a-control-to-a-dialog-box.md)  
  
-   [Tamaño de un control al agregarlo](sizing-a-control-while-you-add-it.md)  
  
-   [Agregar varios controles](adding-multiple-controls.md)  
  
-   [Agregar un control ActiveX a un cuadro de diálogo](viewing-and-adding-activex-controls-to-a-dialog-box.md)  
  
-   [Editar las propiedades de un control o controles](editing-control-properties.md)  
  
-   [Eliminar un control](deleting-a-control.md)  
  
-   [Mostrar u ocultar la ventana de cuadro de herramientas](showing-or-hiding-the-dialog-editor-toolbar.md)  
  
 Para obtener información sobre cómo agregar recursos a proyectos administrados, vea [Resources in Desktop Apps](/dotnet/framework/resources/index) en el *Guía del desarrollador de .NET Framework.* Para obtener información sobre cómo agregar manualmente archivos de recursos a proyectos administrados, acceder a los recursos, mostrar recursos estáticos y asignar cadenas de recursos a propiedades, vea [crear archivos de recursos para las aplicaciones de escritorio](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Para obtener información sobre la globalización y localización de recursos en aplicaciones administradas, vea [Globalizar y localizar aplicaciones de .NET Framework](/dotnet/standard/globalization-localization/index).
