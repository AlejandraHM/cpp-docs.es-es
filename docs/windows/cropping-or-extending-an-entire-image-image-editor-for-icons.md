# <a name="cropping-or-extending-an-entire-image-image-editor-for-icons"></a>Recortar o ampliar una imagen completa (Editor de imágenes para iconos)
Los procedimientos siguientes se muestra cómo recortar o ampliar una imagen completa.  
  
### <a name="to-crop-or-extend-an-entire-image"></a>Recortar o ampliar una imagen completa  
  
1.  Seleccione toda la imagen.  
  
     Si parte de la imagen está seleccionado actualmente y desea seleccionar toda la imagen, haga clic en cualquier parte en la imagen de fuera del borde de selección actual.  
  
2.  Arrastre un controlador de tamaño hasta que la imagen tenga el tamaño deseado.  
  
 Normalmente, el editor de imágenes se recorta o aumenta el tamaño de una imagen cuando cambia el tamaño al mover un controlador de tamaño. Si mantiene presionada la tecla MAYÚS al mover un controlador de tamaño, el editor de imágenes [se reduce o ajusta](../windows/shrinking-or-stretching-an-entire-image-image-editor-for-icons.md) la imagen.  
  
 Para obtener información sobre cómo agregar recursos a proyectos administrados, vea [recursos en aplicaciones de escritorio](/dotnet/framework/resources/index) en el *Guía del desarrollador de .NET Framework.* Para obtener información sobre cómo agregar manualmente archivos de recursos a proyectos administrados, tener acceso a recursos, mostrar recursos estáticos y asignar cadenas de recursos a propiedades, vea [crear archivos de recursos para las aplicaciones de escritorio](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Para obtener información sobre la globalización y localización de recursos en aplicaciones administradas, vea [Globalizar y localizar aplicaciones de .NET Framework](/dotnet/standard/globalization-localization/index).  
  
 Requisitos  
  
 Ninguna
