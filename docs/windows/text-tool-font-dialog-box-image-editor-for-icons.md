# Fuente de herramienta de texto (Cuadro de di&#225;logo) (Editor de im&#225;genes para iconos)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Especifica la configuración de fuente cuando se utiliza el [cuadro de diálogo Herramienta de texto](../mfc/text-tool-dialog-box-image-editor-for-icons.md) del [Editor de imágenes](../mfc/image-editor-for-icons.md).  
  
 Este cuadro de diálogo se utiliza para:  
  
-   [Cambiar la fuente o el texto de una imagen](../mfc/changing-the-font-of-text-on-an-image-image-editor-for-icons.md)  
  
 Para tener acceso a este cuadro de diálogo, haga clic en el botón **Fuente** en el cuadro de diálogo Herramienta de texto.  
  
 **Fuente**  
 Muestra las fuentes disponibles.  
  
 **Estilo de fuente**  
 Muestra los estilos disponibles para la fuente especificada.  
  
 **Size**  
 Muestra los tamaños de punto disponibles para la fuente especificada.  
  
 **Ejemplo**  
 Muestra un ejemplo del aspecto que tendrá el texto con la configuración de fuente especificada.  
  
 **Script**  
 Muestra los scripts de comandos de idioma disponibles para la fuente especificada.  Si selecciona un script de otro idioma, el juego de caracteres del idioma queda disponible para crear documentos multilingües.  
  
## Requisitos  
 None
