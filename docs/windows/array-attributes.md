# <a name="array-attributes"></a>Atributos de matriz
Los siguientes atributos se aplican a las matrices o `SAFEARRAY`s.  
  
|Atributo|Descripción|  
|---------------|-----------------|  
|[library_block](../windows/library-block.md)|Coloca una construcción dentro del bloque de biblioteca del archivo. idl.|  
|[satype](../windows/satype.md)|Especifica el tipo de datos de la estructura SAFEARRAY.|  
|[string](../windows/string-cpp.md)|Indica que unidimensional **char**, **wchar_t**, **bytes** (o equivalentes) el puntero a este tipo de matriz o matriz debe tratarse como una cadena.|
