# Array Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los atributos siguientes se aplican a las matrices o en s de **SAFEARRAY**.  
  
|Atributo|Descripción|  
|--------------|-----------------|  
|[library\_block](../windows/library-block.md)|Coloca una construcción dentro del bloque de la biblioteca del archivo .idl.|  
|[satype](../windows/satype.md)|especifica el tipo de datos de la estructura de SAFEARRAY.|  
|[string](../windows/string-cpp.md)|Indica que `char`unidimensional, `wchar_t`, la matriz de **byte** \(o equivalente\) o un puntero a este tipo de matriz se deben tratar como una cadena.|
