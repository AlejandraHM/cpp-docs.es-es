# Specifying the Dominant Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El control seleccionado en primer lugar será el control dominante.  
  
### Para especificar el control dominante  
  
1.  Mantenga presionada la tecla **CTRL** y haga clic *primero* en el control que desee usar para definir el tamaño o la ubicación de los otros controles.  
  
     **Nota** Los cuadros de tamaño del control dominante serán sólidos, mientras que los de los controles subordinados serán huecos.  Todas las operaciones posteriores de cambio de tamaño o alineación se basarán en el control dominante.  
  
### Para cambiar el control dominante  
  
1.  Desactive la selección actual, haciendo clic fuera de todos los controles que estén seleccionados.  
  
2.  Repita el procedimiento anterior, pero seleccionando primero otro control.
