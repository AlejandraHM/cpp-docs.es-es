# OLE DB Consumer Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los atributos de consumidor OLE DB insertan código, basándose en [Plantillas de consumidor OLE DB](../data/oledb/ole-db-consumer-templates-reference.md), para crear un consumidor OLE DB que funciona realizar tareas como tablas de apertura, ejecutando datos de los comandos, y de acceso.  
  
|Atributo|Descripción|  
|--------------|-----------------|  
|[db\_accessor](../windows/db-accessor.md)|Enlaza las columnas de un conjunto de filas y los enlaza a los mapas correspondientes de descriptor de acceso.|  
|[db\_column](../windows/db-column.md)|Enlaza una columna especificada al conjunto de filas.|  
|[db\_command](../windows/db-command.md)|Ejecuta un comando OLE DB.|  
|[db\_param](../windows/db-param.md)|Asociado a la variable miembro especificada a una entrada o parámetro de salida.|  
|[db\_source](../windows/db-source.md)|Crea y encapsula una conexión, a través de un proveedor, un origen de datos.|  
|[db\_table](../windows/db-table.md)|Abra una tabla de OLE DB.|
