# <a name="creating-moving-and-editing-toolbar-buttons"></a>Crear, mover y editar botones de la barra de herramientas
Puede crear fácilmente, mover, copiar y editar botones de barra de herramientas.  
  
 De forma predeterminada, se muestra un botón nuevo o en blanco en el extremo derecho de la barra de herramientas. Puede mover este botón antes de editarlo. Cuando se crea un nuevo botón, otro botón en blanco situado a la derecha del botón editado. Cuando se guarda una barra de herramientas, el botón en blanco no se guarda.  
  
-   [Crear un nuevo botón de barra de herramientas](creating-a-new-toolbar-button.md)  
  
-   [Mover un botón de barra de herramientas](moving-a-toolbar-button.md)  
  
-   [Copiar botones desde un recurso de barra de herramientas](../windows/copying-buttons-from-a-toolbar.md)  
  
-   [Eliminar un botón de barra de herramientas](deleting-a-toolbar-button.md)  
  
-   [Inserte un espacio entre los botones de un recurso de barra de herramientas](../windows/inserting-a-space-between-buttons-on-a-toolbar.md)  
  
-   [Primer plano de un espacio entre los botones de una barra de herramientas](../windows/removing-space-between-buttons-on-a-toolbar.md)  
  
-   [Editar la página de propiedades de una barra de herramientas](../windows/changing-the-properties-of-a-toolbar-button.md)
