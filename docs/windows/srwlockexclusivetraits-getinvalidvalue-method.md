# SRWLockExclusiveTraits::GetInvalidValue (M&#233;todo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Recupera un objeto de SRWLockExclusiveTraits que siempre no válido.  
  
## Sintaxis  
  
```  
inline static Type GetInvalidValue();  
```  
  
## Valor devuelto  
 Un objeto vacío de SRWLockExclusiveTraits.  
  
## Requisitos  
 **Encabezado:** corewrappers.h  
  
 **Espacio de nombres:** Microsoft::WRL::Wrappers::HandleTraits
