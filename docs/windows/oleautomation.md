# oleautomation
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica que una interfaz es compatible con automatización.  
  
## Sintaxis  
  
```  
  
[oleautomation]  
  
```  
  
## Comentarios  
 el atributo de **oleautomation** C\+\+ tiene la misma funcionalidad que el atributo de [oleautomation](http://msdn.microsoft.com/library/windows/desktop/aa367129) MIDL.  
  
## Ejemplo  
 Vea los ejemplos de [defaultvalue](../Topic/defaultvalue.md) y [nonextensible](../Topic/nonextensible.md) para un ejemplo de uso de **oleautomation**.
