# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Especifica una función de descriptor de acceso de la propiedad.  
  
## Sintaxis  
  
```  
  
[propget]  
  
```  
  
## Comentarios  
 el atributo de **propget** C\+\+ tiene la misma funcionalidad que el atributo de [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL.
