# Editing Accelerator Tables
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En Visual C\+\+ se puede editar una tabla de aceleradores directamente mediante la edición en contexto del Editor de aceleradores.  
  
 Los procedimientos que siguen se refieren al uso de las páginas de propiedades estándar, aunque los métodos de edición en contexto y de la página de propiedades tienen el mismo resultado.  Los cambios efectuados, tanto mediante las páginas de propiedades como mediante la edición en contexto, se aplicarán inmediatamente a la tabla de aceleradores.  
  
-   [Editar una tabla de aceleradores](../windows/editing-in-an-accelerator-table.md)  
  
-   [Buscar una entrada en una tabla de aceleradores](../windows/finding-an-entry-in-an-accelerator-table.md)  
  
-   [Agregar una entrada a una tabla de aceleradores](../windows/adding-an-entry-to-an-accelerator-table.md)  
  
-   [Eliminar una entrada de una tabla de aceleradores](../Topic/Deleting%20an%20Entry%20From%20an%20Accelerator%20Table.md)  
  
-   [Mover o copiar una entrada de una tabla de aceleradores](../windows/moving-or-copying-an-accelerator-table-entry-to-another-resource-script-file.md)  
  
-   [Cambiar las propiedades de varias teclas de aceleración](../windows/changing-the-properties-of-multiple-accelerator-keys.md)  
  
 Para obtener información sobre cómo agregar recursos a proyectos administrados, vea [Recursos de aplicaciones](../Topic/Resources%20in%20Desktop%20Apps.md) en la *Guía del desarrollador de .NET Framework*. Para obtener información sobre cómo agregar manualmente archivos de recursos a proyectos administrados, cómo obtener acceso a recursos, cómo mostrar recursos estáticos y cómo asignar cadenas de recursos a propiedades, vea [Tutorial: Adaptar formularios Windows Forms](http://msdn.microsoft.com/es-es/9a96220d-a19b-4de0-9f48-01e5d82679e5) y [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
