# <a name="makeallocatormakeallocator-constructor"></a>MakeAllocator::MakeAllocator (Constructor)

## <a name="syntax"></a>Sintaxis

```cpp
MakeAllocator();
```

## <a name="remarks"></a>Comentarios

Inicializa una nueva instancia de la **MakeAllocator** clase.
