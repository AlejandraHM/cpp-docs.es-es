# EventSource::GetSize (M&#233;todo)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Recupera el número de controladores de eventos asociados con el objeto de origen de eventos actual  
  
## <a name="syntax"></a>Sintaxis  
  
```  
size_t GetSize() const;  
```  
  
## <a name="return-value"></a>Valor devuelto  
 El número de controladores de eventos de [targets_](../Topic/EventSource::targets_%20Data%20Member.md).  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** event.h  
  
 **Espacio de nombres:** Microsoft::WRL
