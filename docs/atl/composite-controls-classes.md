# Composite Controls Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La siguiente clase proporciona compatibilidad para crear controles compuestos  
  
-   Los controles de[CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md) ActiveX derivados de `CComCompositeControl` se hospedan en un cuadro de diálogo estándar.  Se llama a estos tipos de controles controles compuestos porque pueden hospedar otros controles \(controles nativos de Windows y controles ActiveX\).
