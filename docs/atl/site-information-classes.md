# Site Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Las clases siguientes permiten que un objeto se comunique con el sitio:  
  
-   Recupera y conjuntos de[IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) un puntero al sitio de un objeto.  Utilizado para los objetos que no son controles.  
  
-   Recupera y conjuntos de[IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) un puntero al sitio de un objeto.  Se usa para controles.
