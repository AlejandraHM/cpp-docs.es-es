# <a name="mmc-snap-in-classes"></a>Clases de complemento de MMC
Las clases siguientes proporcionan compatibilidad para desarrollar componentes de complemento de Microsoft Management Console (MMC):  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) implementa un objeto de nodo del complemento, como la adición de elementos de menú y las barras de herramientas y comandos para el nodo de complemento a la función de controlador adecuado de reenvío.  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) implementa un objeto de página de propiedades del complemento.
