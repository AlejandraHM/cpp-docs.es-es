# Componente de registro de ATL (Registrador)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El registro ATL proporciona acceso optimizado en el registro del sistema a través de una interfaz personalizada.  Libre\- subproceso y permite al registro en lugar de código para los clientes de C\+\+.  
  
> [!NOTE]
>  El código fuente del registro ATL se puede encontrar en el atlmfc \\ inclusión \\ atliface.h.

 Una guía de crear scripts del registro.  Incluye temas en sintaxis de BNF, árboles de análisis sintácticos, ejemplos de script de registro, utilizando parámetros reemplazables, e invocar scripts.  
  
 [Colocando un vínculo Static en el registro Code \(sólo C\+\+\)](../atl/setting-up-a-static-link-to-the-registrar-code-cpp-only.md)  
 Muestra los pasos para configurar la vinculación estática al registro.
