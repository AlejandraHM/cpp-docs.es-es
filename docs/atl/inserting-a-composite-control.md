# Inserting a Composite Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El cuadro de diálogo de **Agregar clase** permite insertar un objeto ATL en un proyecto.  Tiene acceso a este cuadro de diálogo haciendo clic con el botón secundario en el nombre del proyecto en el explorador de soluciones, elija **Agregar**, haga clic en **Agregar clase**.  
  
 en el cuadro de diálogo de **Agregar clase** , elija **Control ATL**.  Esto iniciará [Asistente para controles ATL](../atl/reference/atl-control-wizard.md).  Para crear un control compuesto, seleccione la ficha de **Opciones** , y haga clic en la casilla de **control compuesto** .  
  
 Una página HTML predeterminada se creará para ver el control.
