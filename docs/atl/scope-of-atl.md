# Scope of ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL le permite crear fácilmente los controles de los objetos COM, servidores de automatización, y ActiveX.  ATL proporciona compatibilidad integrada para muchas de las interfaces COM fundamentales.  
  
 ATL se envía como código fuente que se incluya en la aplicación.  ATL también crea DLL disponibles \(atl90.dll\), que contiene el código que se puede compartir entre los componentes.  Sin embargo, esta DLL no es necesario.
