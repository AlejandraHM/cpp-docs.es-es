# <a name="error-information-classes"></a>Clases de información de error

La clase siguiente indica cómo se controla la información de error:

- [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) determina si el objeto admite el [IErrorInfo](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-ierrorinfo) interfaz. `IErrorInfo` permite que la información de error que se propaguen al cliente.
