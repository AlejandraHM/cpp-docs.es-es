# <a name="modifying-the-atl-project"></a>Modificar el proyecto ATL
En este momento, el proyecto de control compuesto implementará los objetos necesarios para el control compuesto. El siguiente paso es agregar los controles que contenga el control compuesto y controlar los eventos necesarios.  
  
 Para agregar controles ActiveX o Windows adicionales, agregue un nuevo script de recursos y, a continuación, utilice el editor de cuadro de diálogo. Para obtener más información sobre cómo agregar controles (y las tareas relacionadas), vea [Editor de cuadro de diálogo](../windows/dialog-editor.md).  
  
 Para controlar los eventos necesarios desde los controles ActiveX, vea [agregar funcionalidad al Control compuesto](../atl/adding-functionality-to-the-composite-control.md).
