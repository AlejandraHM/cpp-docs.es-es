# Modifying the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En este punto, el proyecto de control compuesto implementa los objetos necesarios para el control compuesto.  El paso siguiente es agregar cualquier control que el control compuesto contiene y controle los eventos necesarios.  
  
 Para agregar controles adicionales de ActiveX o de Windows, agregue un nuevo script de recursos y utilice el editor de cuadros de diálogo.  Para obtener más información sobre los controles de suma \(y las tareas relacionadas\), vea [editor de cuadros de diálogo](../mfc/dialog-editor.md).  
  
 Para administrar cualquier evento necesario de controles ActiveX, vea [Agregar funcionalidad al Control compuesto](../atl/adding-functionality-to-the-composite-control.md).
