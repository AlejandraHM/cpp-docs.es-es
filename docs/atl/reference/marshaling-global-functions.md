# Marshaling Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas funciones proporcionan compatibilidad para formar y convertir datos de cálculo de referencias de punteros de interfaz.  
  
> [!IMPORTANT]
>  Las funciones enumeradas en la tabla siguiente no se pueden utilizar en las aplicaciones que se ejecutan en [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlFreeMarshalStream](../Topic/AtlFreeMarshalStream.md)|Libera los datos de cálculo de referencias y el puntero de `IStream` .|  
|[AtlMarshalPtrInProc](../Topic/AtlMarshalPtrInProc.md)|Crea un nuevo objeto de secuencia y forma el puntero de interfaz especificado.|  
|[AtlUnmarshalPtr](../Topic/AtlUnmarshalPtr.md)|Convierte los datos de cálculo de referencias de una secuencia en un puntero de interfaz.|
