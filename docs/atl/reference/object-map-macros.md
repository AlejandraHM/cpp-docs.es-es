# Object Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros definen mapas y entradas del objeto.  
  
|||  
|-|-|  
|[DECLARE\_OBJECT\_DESCRIPTION](../Topic/DECLARE_OBJECT_DESCRIPTION.md)|Permite especificar la descripción de texto de un objeto de clase, que se insertan en el mapa del objeto.|  
|[OBJECT\_ENTRY\_AUTO](../Topic/OBJECT_ENTRY_AUTO.md)|Escribe un objeto ATL en el mapa del objeto, actualice el registro, y crea una instancia de objeto.|  
|[OBJECT\_ENTRY\_NON\_CREATEABLE\_EX\_AUTO](../Topic/OBJECT_ENTRY_NON_CREATEABLE_EX_AUTO.md)|Permite especificar que inicializan el objeto está registrado y, pero no se debe externamente pueden crearse mediante `CoCreateInstance`.|
