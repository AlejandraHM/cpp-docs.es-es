# CAtlDebugInterfacesModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta clase proporciona compatibilidad para la depuración interfaces.  
  
## Sintaxis  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## Comentarios  
 `CAtlDebugInterfacesModule` proporciona la compatibilidad requerida para depurar interfaces.  Se incluye en cualquier proyecto que define el símbolo [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md).  
  
## Requisitos  
 **encabezado:** atlbase.h
