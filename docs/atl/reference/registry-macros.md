# Registry Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros definen las funciones útiles de la biblioteca de tipos y del registro.  
  
|||  
|-|-|  
|[\_ATL\_STATIC\_REGISTRY](../Topic/_ATL_STATIC_REGISTRY.md)|Indica que desea que el código de registro para que el objeto está en el objeto para evitar dependencias en ATL.DLL.|  
|[DECLARE\_LIBID](../Topic/DECLARE_LIBID.md)|Proporciona una manera de que ATL obtenga *el libid* de la biblioteca de tipos.|  
|[DECLARE\_NO\_REGISTRY](../Topic/DECLARE_NO_REGISTRY.md)|Evita el registro predeterminado ATL.|  
|[DECLARE\_REGISTRY](../Topic/DECLARE_REGISTRY.md)|Las ENTRAR o quitan la entrada del objeto principal del sistema.|  
|[DECLARE\_REGISTRY\_APPID\_RESOURCEID](../Topic/DECLARE_REGISTRY_APPID_RESOURCEID.md)|Especifica la información necesaria automáticamente para registrar *el appid*.|  
|[DECLARE\_REGISTRY\_RESOURCE](../Topic/DECLARE_REGISTRY_RESOURCE.md)|Encuentra el recurso denominado y ejecuta el script de registro dentro de.|  
|[DECLARE\_REGISTRY\_RESOURCEID](../Topic/DECLARE_REGISTRY_RESOURCEID.md)|Encuentra el recurso identificado por un identificador y ejecuta el script de registro dentro de.|
