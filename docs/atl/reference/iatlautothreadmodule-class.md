# IAtlAutoThreadModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

esta clase representa una interfaz a un método de `CreateInstance` .  
  
> [!IMPORTANT]
>  Esta clase y sus miembros no se pueden utilizar en las aplicaciones que se ejecutan en [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
## Sintaxis  
  
```  
  
__interface IAtlAutoThreadModule  
  
```  
  
## Comentarios  
 La clase [CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md) deriva de `IAtlAutoThreadModule`, su uso para proporcionar código para crear un objeto y recuperar un puntero de interfaz.  
  
## Requisitos  
 **encabezado:** atlbase.h
