# Composite Control Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros definen mapas y entradas de receptor de eventos.  
  
|||  
|-|-|  
|[BEGIN\_SINK\_MAP](../Topic/BEGIN_SINK_MAP.md)|Marca el principio del mapa de receptor de eventos para el control compuesto.|  
|[END\_SINK\_MAP](../Topic/END_SINK_MAP.md)|Marca el final del mapa de receptor de eventos para el control compuesto.|  
|[SINK\_ENTRY](../Topic/SINK_ENTRY.md)|Entrada a la asignación de receptor de eventos.|  
|[SINK\_ENTRY\_EX](../Topic/SINK_ENTRY_EX.md)|Entrada al receptor de eventos asignado con un parámetro adicional.|  
|[SINK\_ENTRY\_INFORMATION](../Topic/SINK_ENTRY_INFO.md)|Entrada a la asignación de receptor de eventos con información de tipos manualmente proporcionada para el uso con [IDispEventSimpleImpl](../../atl/reference/idispeventsimpleimpl-class.md).|
