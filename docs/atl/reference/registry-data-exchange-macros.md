# Registry Data Exchange Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros realizan las operaciones de Data Exchange del registro.  
  
|||  
|-|-|  
|[BEGIN\_RDX\_MAP](../Topic/BEGIN_RDX_MAP.md)|Marca el principio del mapa de Data Exchange del registro.|  
|[END\_RDX\_MAP](../Topic/END_RDX_MAP.md)|Marca el final del mapa de Data Exchange del registro.|  
|[RDX\_BINARY](../Topic/RDX_BINARY.md)|Asocia la entrada de Registro especificada una variable miembro especificada de BYTE escrito.|  
|[RDX\_CSTRING\_TEXT](../Topic/RDX_CSTRING_TEXT.md)|Asocia la entrada de Registro especificada una variable miembro especificada de tipo CString.|  
|[RDX\_DWORD](../Topic/RDX_DWORD.md)|Asocia la entrada de Registro especificada una variable miembro especificada del tipo DWORD.|  
|[RDX\_TEXT](../Topic/RDX_TEXT.md)|Asocia la entrada de Registro especificada una variable miembro especificada de TCHAR escrito.|
