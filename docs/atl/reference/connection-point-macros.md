# Connection Point Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros definen mapas y entradas de punto de conexión.  
  
|||  
|-|-|  
|[BEGIN\_CONNECTION\_POINT\_MAP](../Topic/BEGIN_CONNECTION_POINT_MAP.md)|Marca el principio de las entradas del mapa de puntos de conexión.|  
|[CONNECTION\_POINT\_ENTRY](../Topic/CONNECTION_POINT_ENTRY.md)|Escribe los puntos de conexión en el mapa.|  
|[END\_CONNECTION\_POINT\_MAP](../Topic/END_CONNECTION_POINT_MAP.md)|Marca el final de las entradas del mapa de puntos de conexión.|
