# Agregar un proveedor OLE DB ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Utilice este asistente para agregar un proveedor OLE DB ATL al proyecto.  Un proveedor OLE DB ATL se compone de clases de origen de datos, sesión, comando y conjunto de filas.  El proyecto debe haberse creado como aplicación COM ATL.  
  
### Para agregar un proveedor OLE DB ATL a un proyecto  
  
1.  En la Vista de clases, haga clic con el botón secundario del mouse en el proyecto.  En el menú contextual, haga clic en **Agregar** y, después, en **Agregar clase**.  
  
2.  En la carpeta de Visual C\+\+, haga doble clic en el icono **Proveedor OLE DB ATL** o selecciónelo y haga clic en **Abrir**.  
  
     Se inicia el Asistente para proveedores OLE DB ATL.  
  
3.  Defina la configuración como se describe en [Asistente para proveedores OLEDB ATL](../../atl/reference/atl-ole-db-provider-wizard.md).  
  
4.  Haga clic en **Finalizar** para cerrar el asistente, que insertará el código de proveedor OLE DB recién creado en el proyecto.
