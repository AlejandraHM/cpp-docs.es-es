# <a name="adding-an-atl-ole-db-provider"></a>Agregar un proveedor OLE DB de ATL

Use este asistente para agregar un proveedor OLE DB de ATL a un proyecto. Un proveedor OLE DB ATL consta de un origen de datos, sesión, comandos y las clases de conjunto de filas. El proyecto se debe haber creado como una aplicación de COM de ATL.

### <a name="to-add-an-atl-ole-db-provider-to-your-project"></a>Para agregar un proveedor OLE DB ATL al proyecto

1. En la vista de clases, haga clic en el proyecto. En el menú contextual, haga clic en **agregar** y, a continuación, haga clic en **Agregar clase**.

2. En la carpeta de Visual C++, haga doble clic en el **proveedor OLE DB ATL** icono o selecciónelo y haga clic en **abierto**.

     Se abre el Asistente para proveedores OLE DB ATL.

3. Definir la configuración como se describe en [el Asistente para proveedores OLE DB ATL](../../atl/reference/atl-ole-db-provider-wizard.md).

4. Haga clic en **finalizar** para cerrar el asistente, que se insertará el código del proveedor OLE DB recién creado en el proyecto.
