# Agregar un componente de p&#225;ginas Active Server ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Para agregar un objeto de la biblioteca ATL \(Active Template Library\) a un proyecto, éste debe haberse creado como aplicación COM ATL, o como aplicación MFC compatible con ATL.  Se puede usar el [Asistente para proyectos ATL](../../atl/reference/atl-project-wizard.md) para crear una aplicación ATL, seleccionar **Agregar compatibilidad ATL a MFC** en el [cuadro de diálogo Agregar clase](../../ide/add-class-dialog-box.md) o [agregar un objeto ATL a una aplicación MFC](../../mfc/reference/adding-atl-support-to-your-mfc-project.md) para implementar la compatibilidad con ATL en una aplicación MFC.  
  
 Los componentes de páginas Active Server forman parte de la arquitectura de Internet Information Services \(IIS\), que proporciona las siguientes características avanzadas de desarrollo Web:  
  
-   Se pueden incrustar componentes ASP en las páginas HTML para crear contenido dinámico e independiente del explorador web.  
  
-   Se pueden usar páginas ASP para proporcionar conectividad con bases de datos basada en estándares.  
  
-   También se pueden usar las características de control de errores de ASP para las aplicaciones Web.  
  
### Para agregar un componente de páginas Active Server ASP al proyecto  
  
1.  En el **Explorador de soluciones** o en la [Vista de clases](http://msdn.microsoft.com/es-es/8d7430a9-3e33-454c-a9e1-a85e3d2db925), haga clic con el botón secundario del mouse en el nombre del proyecto al que desee agregar el componente de páginas Active Server ATL.  
  
2.  En el menú contextual, haga clic en **Agregar** y, después, en **Agregar clase**.  
  
3.  En el cuadro de diálogo [Agregar clase](../../ide/add-class-dialog-box.md), en el panel Plantillas, haga clic en **componente de páginas Active Server ATL** y después en **Abrir** para mostrar el [Asistente para componentes de páginas Active Server ATL](../../atl/reference/atl-active-server-page-component-wizard.md).
