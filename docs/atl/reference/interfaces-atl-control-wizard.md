# Interfaces, Asistente para controles ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta página del asistente identifica las interfaces que admite el control.  De forma predeterminada, las interfaces admitidas suelen ser las utilizadas por la mayoría de los contenedores.  
  
> [!NOTE]
>  Si seleccionó **Control mínimo** en la ficha [Opciones](../../atl/reference/options-atl-control-wizard.md), no aparecerán interfaces de forma predeterminada en el cuadro de lista **Compatible**.  
  
 **No compatible**  
 Indica las interfaces con las que no es compatible el control actualmente.  
  
 **Compatible**  
 Indica las interfaces con las que el control es compatible actualmente.  
  
|Botón de transferencia|Descripción|  
|----------------------------|-----------------|  
|**\>**|Agrega a la lista **Compatible** el nombre de interfaz seleccionado actualmente en la lista **No admitida**.|  
|**\>\>**|Agrega a la lista **Compatible** todos los nombres de interfaz disponibles en la lista **No admitida**.|  
|**\<**|Quita el nombre de interfaz seleccionado actualmente en la lista **Compatible**.|  
|**\<\<**|Quita todos los nombres de interfaz seleccionados actualmente en la lista **Compatible**.|
