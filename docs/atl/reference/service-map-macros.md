# Service Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros definen mapas y entradas del servicio.  
  
|||  
|-|-|  
|[BEGIN\_SERVICE\_MAP](../Topic/BEGIN_SERVICE_MAP.md)|Marca el principio de un mapa de servicio ATL.|  
|[END\_SERVICE\_MAP](../Topic/END_SERVICE_MAP.md)|Marca el final de un mapa de servicio ATL.|  
|[SERVICE\_ENTRY](../Topic/SERVICE_ENTRY.md)|Indica que el objeto es compatible con un identificador de servicio concreta|  
|[SERVICE\_ENTRY\_CHAIN](../Topic/SERVICE_ENTRY_CHAIN.md)|Indica a [IServiceProviderImpl:: QueryService](../Topic/IServiceProviderImpl::QueryService.md) para encadenar al objeto especificado.|
