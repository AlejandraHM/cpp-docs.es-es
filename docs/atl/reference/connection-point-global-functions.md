# Connection Point Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas funciones proporcionan compatibilidad para los puntos de conexión y los mapas del receptor.  
  
> [!IMPORTANT]
>  Las funciones enumeradas en la tabla siguiente no se pueden utilizar en las aplicaciones que se ejecutan en [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlAdvise](../Topic/AtlAdvise.md)|Crea una conexión entre el punto de conexión de un objeto y el receptor de un cliente.|  
|[AtlUnadvise](../Topic/AtlUnadvise.md)|finaliza la conexión establecida con `AtlAdvise`.|  
|[AtlAdviseSinkMap](../Topic/AtlAdviseSinkMap.md)|Recomienda o las entradas de los unadvises en un mapa de receptor de eventos.|
