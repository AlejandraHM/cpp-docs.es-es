# Window Class Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

estas macros definen utilidades de la clase de ventana.  
  
|||  
|-|-|  
|[DECLARE\_WND\_CLASS](../Topic/DECLARE_WND_CLASS.md)|Permite especificar el nombre de una nueva clase de ventana.|  
|[DECLARE\_WND\_SUPERCLASS](../Topic/DECLARE_WND_SUPERCLASS.md)|Permite especificar el nombre de una clase de ventana existente en la que una nueva clase de ventana está basada en.|  
|[DECLARE\_WND\_CLASS\_EX](../Topic/DECLARE_WND_CLASS_EX.md)|Permite especificar los parámetros de una clase.|
