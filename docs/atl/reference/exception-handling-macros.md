# Exception Handling Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Estas macros proporcionan compatibilidad para el control de excepciones.  
  
|||  
|-|-|  
|[\_ATLCATCH](../Topic/_ATLCATCH.md)|Instrucciones para controlar los errores que aparecen en `_ATLTRY`asociado.|  
|[\_ATLCATCHALL](../Topic/_ATLCATCHALL.md)|Instrucciones para controlar los errores que aparecen en `_ATLTRY`asociado.|  
|[\_ATLTRY](../Topic/_ATLTRY.md)|marca una sección de código guardada donde un error podría aparecer posiblemente.|
