# What Is AtlAxWin100?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`AtlAxWin100` es el nombre de una clase de ventana que ayuda proporcionan funcionalidad que hospeda ATL.  Cuando se crea una instancia de esta clase, el procedimiento de ventana automáticamente utilizará la API que hospeda para crear un objeto host asociado a la ventana y carga con el control que se especifica como el título de la ventana.
