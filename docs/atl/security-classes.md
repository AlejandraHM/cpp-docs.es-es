# Security Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Estas clases son contenedores para las clases y los objetos de seguridad comunes de Win32.  
  
-   La clase de[CAccessToken](../atl/reference/caccesstoken-class.md) Esta es un contenedor para un token de acceso.  
  
-   La clase de[CAcl](../atl/reference/cacl-class.md) Esta es un contenedor para una estructura de **ACL** \(lista de control de acceso\).  
  
-   La clase de[CDacl](../atl/reference/cdacl-class.md) Esta es un contenedor para una estructura de **DACL** \(lista de control de acceso discrecional \(DACL\)\).  
  
-   La clase de[CPrivateObjectSecurityDesc](../atl/reference/cprivateobjectsecuritydesc-class.md) This representa un objeto de descriptor de seguridad privado del objeto.  
  
-   La clase de[CSacl](../atl/reference/csacl-class.md) Esta es un contenedor para una estructura de **SACL** \(lista de control de acceso del sistema\).  
  
-   La clase de[CSecurityAttributes](../atl/reference/csecurityattributes-class.md) Esta es un contenedor fino para la estructura de **SECURITY\_ATTRIBUTES** .  
  
-   La clase de[CSecurityDesc](../atl/reference/csecuritydesc-class.md) Esta es un contenedor para la estructura de **SECURITY\_DESCRIPTOR** .  
  
-   La clase de[CSid](../atl/reference/csid-class.md) Esta es un contenedor para una estructura de `SID` \(identificador de seguridad\).  
  
-   La clase de[CTokenGroups](../atl/reference/ctokengroups-class.md) Esta es un contenedor para la estructura de **TOKEN\_GROUPS** .  
  
-   La clase de[CTokenPrivileges](../atl/reference/ctokenprivileges-class.md) Esta es un contenedor para la estructura de **TOKEN\_PRIVILEGES** .
