# Data Transfer Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Las clases siguientes admiten tipos diferentes de transferencia de datos:  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) admite la Transferencia de datos uniforme con formatos estándar que recupere y establezca datos.  Controla las notificaciones de los datos administrar conexiones para advertir a los destinatarios.  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) Permitir un moniker asincrónico para enviar y recibir información sobre la descarga de datos asincrónica en el objeto.
