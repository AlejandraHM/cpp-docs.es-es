# <a name="catstr"></a>CATSTR
Concatena los elementos de texto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>Comentarios  
 Cada elemento de texto puede ser una cadena literal, una constante precedida por un `%`, o la cadena devuelta por una función de macro.
