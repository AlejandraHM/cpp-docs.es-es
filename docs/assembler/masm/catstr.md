# CATSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Concatena elementos.  
  
## Sintaxis  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## Comentarios  
 Cada elemento de texto puede ser una cadena literal, una constante precedida por `%`, o la cadena devuelta por una función de macro.
