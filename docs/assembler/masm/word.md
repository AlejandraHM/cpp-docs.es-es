# <a name="word"></a>WORD
Asigna y, opcionalmente, inicializa una palabra (2 bytes) de almacenamiento para cada `initializer`. También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[[name]] WORD initializer [[, initializer]]...  
```
