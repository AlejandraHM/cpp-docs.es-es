# DT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente 10 bytes de almacenamiento para cada `initializer`.  También puede utilizar como un especificador de tipo en un tipo es válido.  `DT` es un sinónimo de [TBYTE](../../assembler/masm/tbyte.md).  
  
## Sintaxis  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
