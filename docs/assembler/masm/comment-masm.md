# <a name="comment-masm"></a>COMMENT (MASM)
Trata todos *texto* entre o en la misma línea que los delimitadores como un comentario.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
