# COMMENT (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

trata todo el *texto* entre o en la misma línea que los delimitadores como comentario.  
  
## Sintaxis  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
