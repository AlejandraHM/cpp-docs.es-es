# <a name="codesize"></a>@CodeSize
0 para **MINÚSCULO**, **pequeño**, **COMPACT**, y **Plano** modelos y 1 para **medio**,  **GRANDES**, y **ENORME** modelos (equivaler numérico).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
@CodeSize  
  
```
