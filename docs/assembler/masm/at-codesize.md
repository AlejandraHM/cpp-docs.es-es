# <a name="codesize"></a>@CodeSize

0 para **MINÚSCULO**, **pequeño**, **COMPACT**, y **planos** modelos y 1 para **medio**,  **GRANDES**, y **ENORME** modelos (equivalen numérico).

## <a name="syntax"></a>Sintaxis

> @CodeSize
