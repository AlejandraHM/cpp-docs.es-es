# <a name="operator-gt"></a>Operador GT
Devuelve true (-1) si *expression1* es mayor que *expression2*, o devuelve un valor falso (0) si no lo está.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 GT expression2  
```
