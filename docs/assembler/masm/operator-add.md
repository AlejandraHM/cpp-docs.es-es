# <a name="operator-"></a>operador +
El primer operador devuelve *expression1* más *expression2*. El segundo operador invierte el signo de *expresión*.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
      expression1 + expression2  
-expression  
```
