# <a name="label-masm"></a>LABEL (MASM)
Crea una nueva etiqueta asignando el valor del contador de la ubicación actual y el especificado `type` a *nombre*.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
