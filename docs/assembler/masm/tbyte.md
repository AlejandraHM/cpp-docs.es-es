# <a name="tbyte"></a>TBYTE
Asigna y, opcionalmente, inicializa 10 bytes de almacenamiento para cada `initializer`. También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[[name]] TBYTEinitializer [[, initializer]]...  
```
