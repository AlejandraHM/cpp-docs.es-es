# <a name="tbyte"></a>TBYTE

Asigna e inicializa opcionalmente 10 bytes de almacenamiento para cada `initializer`. También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.

## <a name="syntax"></a>Sintaxis

> [[nombre]] TBYTEinitializer [[, inicializador]]...
