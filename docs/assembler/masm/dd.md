# DD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente una doble palabra \(4 bytes\) de almacenamiento para cada `initializer`.  `DD` es un sinónimo de [dword](../../assembler/masm/dword.md).  
  
## Sintaxis  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## Comentarios  
 También puede utilizar como un especificador de tipo en un tipo es válido.
