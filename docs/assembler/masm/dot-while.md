# <a name="while"></a>.WHILE
Genera código que se ejecuta el bloque de `statements` mientras `condition` siga siendo verdadera.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
