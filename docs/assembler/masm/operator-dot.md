# <a name="operator-"></a>Operador .
El primer operador devuelve *expresión* más el desplazamiento de *campo* dentro de su estructura o unión. El segundo operador devuelve el valor en la ubicación señalada por *registrar* más el desplazamiento de *campo* dentro de su estructura o unión.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
