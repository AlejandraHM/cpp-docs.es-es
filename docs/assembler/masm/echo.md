# <a name="echo"></a>ECHO

Muestra *mensaje* al dispositivo de salida estándar (de forma predeterminada, la pantalla).

## <a name="syntax"></a>Sintaxis

> ECHO *mensaje*

## <a name="remarks"></a>Comentarios

Igual que [% OUT](../../assembler/masm/percent-out.md).
