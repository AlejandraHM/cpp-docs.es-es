# ECHO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Muestra *el mensaje* al dispositivo de salida estándar \(de forma predeterminada, la pantalla\).  
  
## Sintaxis  
  
```  
  
ECHO   
message  
  
```  
  
## Comentarios  
 Igual que [%OUT](../../assembler/masm/percent-out.md).
