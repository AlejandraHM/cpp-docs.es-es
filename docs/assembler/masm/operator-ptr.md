# operator PTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

El primer operador fuerza *la expresión* que se tratará como si `type`especificado.  el segundo operador especifica un puntero a `type`.  
  
## Sintaxis  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
