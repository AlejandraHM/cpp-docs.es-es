# <a name="operator-ptr"></a>Operador PTR

El primer operador las fuerzas el *expresión* a tratarse como si tuviera especificado *tipo*. El segundo operador especifica un puntero a *tipo*.

## <a name="syntax"></a>Sintaxis

> *tipo* PTR *expresión*

> [[*distancia*]] PTR *tipo*
