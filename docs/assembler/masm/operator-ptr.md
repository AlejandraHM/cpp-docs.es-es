# <a name="operator-ptr"></a>Operador PTR
El primer operador se fuerza la *expresión* tratará como si tuviera especificado `type`. El segundo operador especifica un puntero a `type`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
