# <a name="ml-nonfatal-error-a2107"></a>Error recuperable A2107 de ML
**no puede tener salto lejano implícita o llamada a un nivel próximo etiqueta**  
  
 Se intentó realizar un modo implícito saltar ahora o llamar a una etiqueta casi en otro segmento.
