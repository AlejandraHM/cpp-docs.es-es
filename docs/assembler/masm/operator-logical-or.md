# <a name="operator-"></a>operador ||
Lógico `OR`. Utilizan solo en [. IF](../../assembler/masm/dot-if.md), [. MIENTRAS](../../assembler/masm/dot-while.md), o [. Repita](../../assembler/masm/dot-repeat.md) bloquea y se evalúa en tiempo de ejecución, no en tiempo de ensamblado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 || expression2  
```
