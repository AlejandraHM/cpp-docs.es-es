# <a name="operator-lowword"></a>Operador LOWWORD
Devuelve los 16 bits inferiores de *expresión*. Expresiones MASM son valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
LOWWORD expression  
```
