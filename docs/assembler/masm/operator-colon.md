# operator :
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Reemplaza el segmento predeterminado *de la expresión* con `segment`.  `segment` puede ser un registro de segmento, un nombre de grupo, un nombre de segmento, o una expresión del segmento.  *la expresión* debe ser una constante.  
  
## Sintaxis  
  
```  
  
segment  
: expression  
```
