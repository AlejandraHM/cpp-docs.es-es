# <a name="operator-"></a>Operador :
Invalida el segmento predeterminado de *expresión* con `segment`. La `segment` puede ser un registro de segmento, nombre del grupo, nombre de segmento o expresión de segmento. El *expresión* debe ser una constante.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
segment  
: expression  
```
