# ML Nonfatal Error A2031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**debe ser el índice o el registro base**  
  
 Se intentó utilizar un registro que no era un registro de base o de índice en una expresión de memoria.  
  
 Por ejemplo, las expresiones siguientes producen este error:  
  
```  
[ax]  
[bl]  
```
