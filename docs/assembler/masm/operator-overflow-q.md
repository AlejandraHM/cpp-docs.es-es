# <a name="operator-overflow"></a>Operador OVERFLOW?
Estado del indicador de desbordamiento. Utilizan solo en [. IF](../../assembler/masm/dot-if.md), [. MIENTRAS](../../assembler/masm/dot-while.md), o [. Repita](../../assembler/masm/dot-repeat.md) bloquea y se evalúa en tiempo de ejecución, no en tiempo de ensamblado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
OVERFLOW?  
  
```
