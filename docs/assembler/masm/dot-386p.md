# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Habilitar el ensamblado de todas las instrucciones \(privilegiado incluida para el procesador 80386; deshabilita el ensamblado de instrucciones presentadas con procesadores posteriores.  
  
## Sintaxis  
  
```  
  
.386P  
  
```  
  
## Comentarios  
 También permite 80387 instrucciones.
