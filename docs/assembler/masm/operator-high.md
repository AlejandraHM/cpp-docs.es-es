# <a name="operator-high"></a>Operador HIGH
Devuelve los 8 bits de los 16 bits inferiores de alta *expresión*. Expresiones MASM son valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
HIGH expression  
```
