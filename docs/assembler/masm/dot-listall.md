# <a name="listall"></a>.LISTALL
Inicia el listado de todas las instrucciones.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
.LISTALL  
  
```  
  
## <a name="remarks"></a>Comentarios  
 Equivalente a la combinación de [. LISTA](../../assembler/masm/dot-list.md), [. LISTASI](../../assembler/masm/dot-listif.md), y [. LISTMACROALL](../../assembler/masm/dot-listmacroall.md).
