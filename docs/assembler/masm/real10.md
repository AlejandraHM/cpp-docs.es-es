# <a name="real10"></a>REAL10
Asigna y, opcionalmente, inicializa un número de punto flotante de 10 bytes para cada *inicializador*.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
