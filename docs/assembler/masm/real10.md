# <a name="real10"></a>REAL10

Asigna e inicializa, opcionalmente, un número de punto flotante de 10 bytes para cada *inicializador*.

## <a name="syntax"></a>Sintaxis

> inicializador de REAL10 nombre [[, inicializador]]...
