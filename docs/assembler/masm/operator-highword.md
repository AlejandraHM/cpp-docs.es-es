# <a name="operator-highword"></a>Operador HIGHWORD

Devuelve los 16 bits superiores de 32 bits inferiores de *expresión*. Expresiones MASM son valores de 64 bits.

## <a name="syntax"></a>Sintaxis

> Expresión HIGHWORD
