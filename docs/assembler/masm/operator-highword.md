# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve los bits height 16 de los bits de punto en 32.  Las expresiones de MASM son valores de 64 bits.  
  
## Sintaxis  
  
```  
  
HIGHWORD expression  
```
