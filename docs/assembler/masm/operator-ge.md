# <a name="operator-ge"></a>Operador GE
Devuelve true (-1) si *expression1* es mayor o igual que *expression2*, o devuelve un valor falso (0) si no lo está.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 GE expression2  
```
