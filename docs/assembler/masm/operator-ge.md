# operator GE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve true \(– 1\) si *expression1* es mayor o igual que *expression2*, o devuelve false \(0\) si no es.  
  
## Sintaxis  
  
```  
  
expression1 GE expression2  
```
