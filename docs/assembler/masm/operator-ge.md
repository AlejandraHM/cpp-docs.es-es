# <a name="operator-ge"></a>Operador GE

Devuelve true (-1) si *expression1* es mayor o igual a *expression2*, o devuelve false (0) si no lo está.

## <a name="syntax"></a>Sintaxis

> Expresión1 GE Expresión2
