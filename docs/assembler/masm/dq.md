# DQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente 8 bytes de almacenamiento para cada `initializer`.  También se puede utilizar como especificador de tipo siempre que un tipo es válido.  `DQ` es un sinónimo de [QWORD](../../assembler/masm/qword.md).  
  
## Sintaxis  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
