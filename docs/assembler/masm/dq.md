# <a name="dq"></a>DQ

Asigna e inicializa opcionalmente 8 bytes de almacenamiento para cada `initializer`. También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal. `DQ` es un sinónimo de [QWORD](../../assembler/masm/qword.md).

## <a name="syntax"></a>Sintaxis

> [[nombre]] Inicializador de calidad de datos [[, inicializador]]...
