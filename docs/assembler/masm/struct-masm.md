# <a name="struct-masm"></a>STRUCT (MASM)

Declara un tipo de estructura tener especificado *declaraciones de campo*. Cada campo debe ser una definición de datos válido. Igual que [STRUC](../../assembler/masm/struc.md).

## <a name="syntax"></a>Sintaxis

> *nombre* STRUCT [[*alineación*]] [[único y]]<br/>
> *declaraciones de campo*<br/>
> *nombre* finaliza
