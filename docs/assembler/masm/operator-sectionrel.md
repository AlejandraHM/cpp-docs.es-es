# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve la diferencia de relativa de la sección de la expresión en relación con la sección que contiene el destino en el ejecutable final.  
  
## Sintaxis  
  
```  
SECTIONREL expression  
```  
  
## Comentarios  
 SECTIONREL solo está disponible con la emisión del objeto COFF.
