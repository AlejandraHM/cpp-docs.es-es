# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**no se puede abrir el archivo abierto: nombre de archivo**  
  
 El ensamblador no pudo abrir un origen, incluyen, o archivo de salida.  
  
 Uno de los siguientes puede ser una causa:  
  
-   El archivo no existe.  
  
-   El archivo está en uso por otro proceso.  
  
-   El nombre de archivo no es válido.  
  
-   Un archivo de sólo lectura con el nombre del archivo de salida ya existe.  
  
-   La unidad actual está completa.  
  
-   El directorio actual es la raíz y está lleno.  
  
-   El dispositivo no se puede escribir.  
  
-   la unidad no está lista.
