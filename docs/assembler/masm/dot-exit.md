# <a name="exit"></a>.EXIT
Genera código de terminación.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
.EXIT [[expression]]  
```  
  
## <a name="remarks"></a>Comentarios  
 Devuelve opcional *expresión* Shell.
