# ML Nonfatal Error A2004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**conflicto de tipos de token: identificador**  
  
 La directiva de [EXTERNDEF](../../assembler/masm/externdef.md) o de [ETIQUETA](../../assembler/masm/label-masm.md) se utilizó en una variable, un símbolo, una estructura de datos, o una etiqueta que se define en el mismo módulo pero con otro tipo.
