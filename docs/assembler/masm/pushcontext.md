# PUSHCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

guarda la parte o todo el `context`actual: el registro de segmento supone, los indicadores de valores de base, en la lista y cref, o valores de procesador\/coprocesador.  `context` puede ser **ASSUMES**, `RADIX`, **EL ENUMERAR**, **CPU**, o **TODO**.  
  
## Sintaxis  
  
```  
  
PUSHCONTEXT context  
```
