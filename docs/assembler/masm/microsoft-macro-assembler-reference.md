# Microsoft Macro Assembler Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Microsoft Macro Assembler \(MASM\) proporciona varias ventajas sobre el ensamblado en línea.  MASM contiene un lenguaje de macros que tiene características tales como procesamiento de cadenas de bucle, aritméticos y de texto.  MASM también permite un mayor control sobre el hardware ya que admite los conjuntos de instrucciones de la 386, 486 y Pentium.  Mediante el uso de MASM, también puede reducir la sobrecarga de tiempo y memoria.
