# operator &lt;= (MASM Run Time)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Es menor o igual que.  Sólo se utiliza dentro de [.IF](../../assembler/masm/dot-if.md), de [.WHILE](../../assembler/masm/dot-while.md), o de los bloques de [.REPEAT](../../assembler/masm/dot-repeat.md) y se evalúa en tiempo de ejecución, no en tiempo de ensamblado.  
  
## Sintaxis  
  
```  
  
expression1 <= expression2  
```
