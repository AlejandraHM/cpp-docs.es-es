# SUBSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve una subcadena *de textitem*, desde *la posición*.  *El textitem* puede ser una cadena literal, una constante precedida por `%`, o la cadena devuelta por una función de macro.  
  
## Sintaxis  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
