# <a name="if2"></a>IF2
**IF** bloque se evalúa en cada fase de ensamblado si **opción: SETIF2** es **TRUE**.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
IF2   
expression  
  
```
