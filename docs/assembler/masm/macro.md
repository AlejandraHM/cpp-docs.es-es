# <a name="macro"></a>MACRO

Marca un bloque de macro llamado *nombre* y establece *parámetro* marcadores de posición para los argumentos se pasan cuando se llama a la macro.

## <a name="syntax"></a>Sintaxis

> *nombre* MACRO [[*parámetro* [[: REQ |: =*predeterminada* |: VARARG]]]]...<br/>
> *Instrucciones*<br/>
> ENDM [[*valor*]]

## <a name="remarks"></a>Comentarios

Devuelve una función de macro *valor* a la instrucción de llamada.
