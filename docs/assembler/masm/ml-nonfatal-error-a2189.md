# <a name="ml-nonfatal-error-a2189"></a>Error recuperable A2189 de ML
**combinación no válida con alineación de segmento**  
  
 La alineación especificada por el **alinear** o `EVEN` directiva era mayor que la alineación de segmento actual según lo especificado por el **segmento** directiva.
