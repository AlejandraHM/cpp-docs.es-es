# @@:
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Define una etiqueta de código reconocible solo entre *label1* y *label2*, donde se inicia el código o la etiqueta *label1* anterior de `@@:` , y *label2* es final del código o la siguiente etiqueta de `@@:` .  Vea [@B](../../assembler/masm/at-b.md) y [@F](../../assembler/masm/at-f.md).  
  
## Sintaxis  
  
```  
  
@@:  
  
```
