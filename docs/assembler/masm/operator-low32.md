# <a name="operator-low32"></a>Operador LOW32
Devuelve los 32 bits inferiores de *expresión*. Expresiones MASM son valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
LOW32 expression  
```
