# <a name="repeat"></a>REPEAT

Marca un bloque que se repetirá *expresión* veces. Igual que [REPT](../../assembler/masm/rept.md).

## <a name="syntax"></a>Sintaxis

> Repita *expresión*<br/>
> *Instrucciones*<br/>
> ENDM
