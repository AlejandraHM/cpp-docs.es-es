# <a name="includelib-masm"></a>INCLUDELIB (MASM)
Se notifica al vinculador que debe vincularse con el módulo actual *nombrebiblioteca*.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
INCLUDELIB libraryname  
```  
  
## <a name="remarks"></a>Comentarios  
 El *nombrebiblioteca* deben incluirse entre corchetes angulares si incluye una barra diagonal inversa, punto y coma, mayor-signo, menor-que símbolos, las comillas simples o comillas dobles.
