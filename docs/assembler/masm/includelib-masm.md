# <a name="includelib-masm"></a>INCLUDELIB (MASM)

Notifica al vinculador que el módulo actual debe vincularse con *nombrebiblioteca*.

## <a name="syntax"></a>Sintaxis

> INCLUDELIB nombrebiblioteca

## <a name="remarks"></a>Comentarios

El *nombrebiblioteca* debe ir entre corchetes angulares si incluye una barra diagonal inversa, punto y coma, mayor-signo, menor-que los símbolos, comillas simples o dobles comillas.
