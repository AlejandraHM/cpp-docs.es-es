# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Informa al vinculador que el módulo actual se debe vincular con *el libraryname*.  
  
## Sintaxis  
  
```  
  
INCLUDELIB libraryname  
```  
  
## Comentarios  
 *El libraryname* se debe incluir entre corchetes angulares si incluye una barra diagonal inversa, punto y coma, mayor símbolos, que símbolo, comilla simple, o comillas.
