# <a name="ifidn"></a>IFIDN]
Ensamblado de concesiones si los elementos de texto son idénticos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>Comentarios  
 Si **I** es dada, la comparación distingue entre mayúsculas y minúsculas. Vea [IF](../../assembler/masm/if-masm.md) para consultar la sintaxis completa.
