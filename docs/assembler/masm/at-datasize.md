# <a name="datasize"></a>@DataSize

0 para **MINÚSCULO**, **pequeño**, **medio**, y **planos** modela, 1 para **COMPACT** y  **GRANDES** modelos y 2 para **ENORME** modelo (equivalen numérico).

## <a name="syntax"></a>Sintaxis

> @DataSize
