# <a name="ifnb"></a>IFNB
Concede al ensamblado si *textitem* no está en blanco.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
IFNB textitem  
```  
  
## <a name="remarks"></a>Comentarios  
 Vea [IF](../../assembler/masm/if-masm.md) para consultar la sintaxis completa.
