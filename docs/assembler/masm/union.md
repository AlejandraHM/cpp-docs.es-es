# UNION
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Declara una combinación de uno o más tipos de datos.  *los fielddeclarations* deben ser definiciones de datos válidas.  Omita el*nombre*  de [EXTREMOS](../../assembler/masm/ends-masm.md)en las definiciones anidada de **UNION** .  
  
## Sintaxis  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
