# QWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente 8 bytes de almacenamiento para cada `initializer`.  También se puede utilizar como especificador de tipo siempre que un tipo es válido.  
  
## Sintaxis  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
