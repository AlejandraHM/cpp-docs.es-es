# <a name="sbyte-masm"></a>SBYTE (MASM)
Asigna y, opcionalmente, inicializa un byte con signo de almacenamiento para cada `initializer`. También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
