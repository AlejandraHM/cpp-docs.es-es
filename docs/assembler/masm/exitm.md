# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Finaliza la extensión del bloque actual de repetición o la macro y comienza ensamblado de la siguiente instrucción fuera del bloque.  
  
## Sintaxis  
  
```  
  
EXITM [[textitem]]   
```  
  
## Comentarios  
 En una función de macro, *el textitem* es el valor devuelto.
