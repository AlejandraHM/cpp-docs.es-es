# <a name="align-masm"></a>ALIGN (MASM)
Alinea a la siguiente variable o instrucción en un byte que es un múltiplo de `number`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
ALIGN [[  
number  
]]  
  
```
