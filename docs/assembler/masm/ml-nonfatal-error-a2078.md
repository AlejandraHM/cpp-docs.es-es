# <a name="ml-nonfatal-error-a2078"></a>Error recuperable A2078 de ML
**instrucción no permite lejano direccionamiento indirecto**  
  
 Un bucle o salto condicional no puede tomar un operando de memoria. Se le debe proporcionar una etiqueta o una dirección relativa.
