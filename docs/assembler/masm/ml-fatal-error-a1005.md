# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**límite de código ensamblador: tabla de macro del nombre de parámetro completa**  
  
 Demasiados parámetros, valores locales, o etiquetas de macro eran definido para una macro.  No hay ningún sitio en la tabla del nombre de la macro.  
  
 Defina más corto o menos nombres, quite las macros innecesarias.
