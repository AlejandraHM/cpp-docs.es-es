# <a name="operator-"></a>Operador ;;
Trata *texto* como un comentario en una macro que sólo aparece en la definición de macro. No se muestra la lista *texto* donde se expande la macro.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
;;  
text  
  
```
