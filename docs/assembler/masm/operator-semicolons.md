# operator ;;
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Trata *el texto* como comentario en una macro que sólo aparece en la definición de macro.  la lista no muestra a *texto* donde se expande la macro.  
  
## Sintaxis  
  
```  
  
;;  
text  
  
```
