# <a name="for-masm"></a>FOR (MASM)
Marca un bloque que se repetirá una vez para cada `argument`, con el actual `argument` reemplazando `parameter` en cada repetición.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```  
  
## <a name="remarks"></a>Comentarios  
 Igual que [IRP](../../assembler/masm/irp.md).
