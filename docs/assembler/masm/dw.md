# DW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente una palabra \(2 bytes\) de almacenamiento para cada `initializer`.  También puede utilizar como un especificador de tipo en un tipo es válido.  `DW` es un sinónimo de [word](../../assembler/masm/word.md).  
  
## Sintaxis  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
