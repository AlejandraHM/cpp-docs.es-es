# <a name="operator-lt"></a>Operador LT
Devuelve true (-1) si *expression1* es menor que *expression2*, o devuelve un valor falso (0) si no lo está.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 LT expression2  
```
