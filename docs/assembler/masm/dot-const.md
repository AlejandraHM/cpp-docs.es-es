# .CONST
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cuando se utiliza con [.MODEL](../../assembler/masm/dot-model.md), empiece al segmento de datos constantes \(con el nombre de segmento CONST\).  
  
## Sintaxis  
  
```  
  
.CONST  
  
```  
  
## Comentarios  
 este segmento tiene el atributo de sólo lectura.
