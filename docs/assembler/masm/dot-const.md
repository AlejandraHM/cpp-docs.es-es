# <a name="const"></a>.CONST
Cuando se utiliza con [. MODELO](../../assembler/masm/dot-model.md), inicia un segmento de datos constantes (por segmento nombre CONST).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
.CONST  
  
```  
  
## <a name="remarks"></a>Comentarios  
 Este segmento tiene el atributo de sólo lectura.
