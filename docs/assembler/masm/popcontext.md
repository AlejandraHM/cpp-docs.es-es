# <a name="popcontext"></a>POPCONTEXT

Restaura todo o parte del actual `context` (guardado por el [PUSHCONTEXT](../../assembler/masm/pushcontext.md) directiva). El `context` puede ser **asume**, `RADIX`, **enumerar**, **CPU**, o **todas**.

## <a name="syntax"></a>Sintaxis

> Contexto POPCONTEXT
