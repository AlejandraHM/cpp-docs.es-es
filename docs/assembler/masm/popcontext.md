# POPCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restablece la parte o todo el `context` actual \(guardado con la directiva de [PUSHCONTEXT](../../assembler/masm/pushcontext.md) \).  `context` puede ser **ASSUMES**, `RADIX`, **EL ENUMERAR**, **CPU**, o **TODO**.  
  
## Sintaxis  
  
```  
  
POPCONTEXT context  
```
