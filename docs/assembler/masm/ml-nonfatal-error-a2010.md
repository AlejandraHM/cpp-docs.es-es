# <a name="ml-nonfatal-error-a2010"></a>Error recuperable A2010 de ML

**expresión de tipo no válido**

El operando situado a [THIS](../../assembler/masm/operator-this.md) o [PTR](../../assembler/masm/operator-ptr.md) no era una expresión de tipo válido.
