# <a name="ml-nonfatal-error-a2047"></a>Error recuperable A2047 de ML

**cadena (null) vacía**

Una cadena formada por un par de comillas y sin caracteres dentro de delimitador.

Una cadena ser válido, debe contener de 1 a 255 caracteres.
