# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Se utiliza para operandos multimedia de 256 bits con Intel avanzaron instrucciones de AVX \(advanced vector extensions\).  
  
## Sintaxis  
  
```  
YMMWORD  
```  
  
## Comentarios  
 `YMMWORD` está pensado para representar el mismo tipo que `__m256` para intrínsecos de AVX.
