# <a name="ymmword"></a>YMMWORD

Se usa para los operandos multimedios de 256 bits con las instrucciones de extensiones de Vector avanzadas de Intel (AVX).

## <a name="syntax"></a>Sintaxis

> YMMWORD

## <a name="remarks"></a>Comentarios

`YMMWORD` está diseñado para representar el mismo tipo que `__m256` para las funciones intrínsecas AVX.
