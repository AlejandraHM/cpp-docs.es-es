# <a name="ml-nonfatal-error-a2206"></a>Error recuperable A2206 de ML
**Falta el operador de expresión**  
  
 No se puede evaluar una expresión porque le falta un operador. Este mensaje de error también puede ser un efecto secundario de un error en un programa anterior.  
  
 La siguiente línea generará este error:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
