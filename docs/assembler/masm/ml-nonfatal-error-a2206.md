# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**falta un operador en la expresión**  
  
 Una expresión no se evalúa porque falta un operador.  Este error puede ser también un efecto secundario de un error del programa anterior.  
  
 la siguiente línea generará este error:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
