# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**el tipo de lenguaje debe especificar**  
  
 Una definición de procedimiento o un prototipo no se aplicó un tipo de lenguaje.  
  
 Un tipo de lenguaje debe declararse de cada definición de procedimiento o prototipo si no se especifica un tipo de idioma predeterminado.  Establecen un tipo de idioma predeterminado mediante la directiva de [.MODEL](../../assembler/masm/dot-model.md) , **LANG OF TABLE OPCIÓN**, u opciones de línea de comandos **\/Gc** o **\/Gd**de ML.
