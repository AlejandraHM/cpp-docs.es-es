# @data
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

El nombre del grupo predeterminado de los datos.  Se evalúa como DGROUP para todos los modelos excepto **Plana**.  Se evalúa como **Plana** bajo el modelo de memoria de **Plana** \(macro de texto\).  
  
## Sintaxis  
  
```  
  
@data  
  
```
