# <a name="data"></a>@data
El nombre del grupo de datos predeterminado. Se evalúa como DGROUP para todos los modelos excepto **FLAT**. Se evalúa como **FLAT** en el **FLAT** el modelo de memoria (macro de texto).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
@data  
  
```
