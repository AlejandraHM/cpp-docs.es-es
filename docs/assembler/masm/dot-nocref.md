# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Suprime la lista de símbolos en el archivo de la tabla y del explorador de símbolo.  
  
## Sintaxis  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Comentarios  
 Si se especifican varios nombres, sólo se omiten los nombres especificados.  Igual que [.XCREF](../../assembler/masm/dot-xcref.md).
