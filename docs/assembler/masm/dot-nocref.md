# <a name="nocref"></a>.NOCREF

Suprime la lista de símbolos en el archivo de tabla y el Explorador de símbolos.

## <a name="syntax"></a>Sintaxis

> . NOCREF [[nombre [[, nombre]]...]]

## <a name="remarks"></a>Comentarios

Si se especifican, se suprimen sólo los nombres especificados. Igual que [. XCREF](../../assembler/masm/dot-xcref.md).
