# <a name="byte-masm"></a>BYTE (MASM)
Asigna y, opcionalmente, inicializa un byte de almacenamiento para cada `initializer`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Comentarios  
 También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.
