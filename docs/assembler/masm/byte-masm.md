# BYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente un byte de almacenamiento para cada `initializer`.  
  
## Sintaxis  
  
```  
[[name]] BYTE initializer [[, initializer]] ...  
```  
  
## Comentarios  
 También puede utilizar como un especificador de tipo en un tipo es válido.
