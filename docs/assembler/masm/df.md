# DF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asignan e inicializan opcionalmente 6 bytes de almacenamiento para cada `initializer`.  `DF` es un sinónimo de [FWORD](../../assembler/masm/fword.md).  
  
## Sintaxis  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## Comentarios  
 También se puede utilizar como especificador de tipo siempre que un tipo es válido.
