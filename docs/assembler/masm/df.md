# <a name="df"></a>DF
Asigna y, opcionalmente, inicializa 6 bytes de almacenamiento para cada `initializer`. `DF`es un sinónimo de [FWORD](../../assembler/masm/fword.md).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Comentarios  
 También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.
