# <a name="operator-eq"></a>Operador EQ
Devuelve true (-1) si *expression1* es igual a *expression2*, o devuelve un valor falso (0) si no es así.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 EQ expression2  
```
