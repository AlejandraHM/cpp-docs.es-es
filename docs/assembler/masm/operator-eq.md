# operator EQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve true \(– 1\) si *expression1* es *expression2*, o devuelve false \(0\) si no lo hace.  
  
## Sintaxis  
  
```  
  
expression1 EQ expression2  
```
