# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**el operando debe ser tipos de registro o campo**  
  
 El operando que sigue al operador de [ANCHO](../../assembler/masm/operator-width.md) o de [Mscara](../../assembler/masm/operator-mask.md) no era válido.  
  
 El operador de [ANCHO](../../assembler/masm/operator-width.md) toma un operando que es el nombre de un campo o un registro.  El operador de [Mscara](../../assembler/masm/operator-mask.md) toma un operando que es el nombre de un campo o tipo de registro.
