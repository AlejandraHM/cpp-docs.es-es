# <a name="ml-nonfatal-error-a2066"></a>Error recuperable A2066 de ML
**tamaño de modo y el segmento de CPU incompatible**  
  
 Se intentó abrir un segmento con un **USE16**, **USE32**, o **PLANO** atributo que no es compatible con la CPU especificada, o para cambiar a una CPU de 16 bits en 32 bits segmento.  
  
 El **USE32** y **PLANO** atributos deben ir precedidos de la.386 o directiva de procesador mayor.
