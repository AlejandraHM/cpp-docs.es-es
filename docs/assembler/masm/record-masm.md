# <a name="record-masm"></a>RECORD (MASM)

Declara un tipo de registro que consta de los campos especificados. *FieldName* nombra el campo, *ancho* especifica el número de bits, y *expresión* ofrece su valor inicial.

## <a name="syntax"></a>Sintaxis

> *NOMBREDEREGISTRO* registro *fieldname*:*ancho* [[= *expresión*]] [[, *fieldname*:*deancho* [[= *expresión*]]]]...
