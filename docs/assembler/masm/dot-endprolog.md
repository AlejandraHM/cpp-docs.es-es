# <a name="endprolog"></a>.ENDPROLOG
Marca el final de las declaraciones del prólogo.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
.ENDPROLOG  
```  
  
## <a name="remarks"></a>Comentarios  
 Es un error utilizar cualquiera de las declaraciones del prólogo fuera de la región entre [PROC](../../assembler/masm/proc.md) marco y. ENDPROLOG.  
  
 Para obtener más información, consulte [MASM para x64 (ml64.exe)](../../assembler/masm/masm-for-x64-ml64-exe.md).
