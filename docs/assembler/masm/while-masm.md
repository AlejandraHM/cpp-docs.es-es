# <a name="while-masm"></a>WHILE (MASM)

Se repite el ensamblado del bloque *instrucciones* mientras *expresión* siga siendo verdadera.

## <a name="syntax"></a>Sintaxis

> MIENTRAS *expresión*<br/>
> *Instrucciones*<br/>
> ENDM
