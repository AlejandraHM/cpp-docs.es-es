# <a name="public-masm"></a>PUBLIC (MASM)
Hace que cada variable, una etiqueta o un símbolo absoluto especificado como *nombre* disponibles para todos los demás módulos en el programa.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
