# <a name="operator-le"></a>Operador LE
Devuelve true (-1) si *expression1* es menor o igual que *expression2*, o devuelve un valor falso (0) si no lo está.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 LE expression2  
```
