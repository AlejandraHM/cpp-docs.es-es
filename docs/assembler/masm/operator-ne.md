# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve true \(– 1\) si *expression1* no es igual a *expression2*, o devuelve false \(0\) si lo está.  
  
## Sintaxis  
  
```  
  
expression1 NE expression2  
```
