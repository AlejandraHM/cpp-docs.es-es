# <a name="ml-nonfatal-error-a2074"></a>Error recuperable A2074 de ML
**no se puede obtener acceso a la etiqueta en los registros de segmento de**  
  
 Se intentó obtener acceso a una etiqueta a través de un registro de segmentos que no se supone que su segmento o el grupo.
