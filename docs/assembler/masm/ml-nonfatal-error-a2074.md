# ML Nonfatal Error A2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**no puede tener acceso a la etiqueta a través de los registros de segmentos**  
  
 Se realizó un intento de obtener acceso a una etiqueta a través de un registro de segmentos que no se suponía al segmento o grupo.
