# <a name="ml-nonfatal-error-a2038"></a>Error recuperable A2038 de ML
**Falta un operando para el operador de macro**  
  
 El ensamblador encontró el final de la lista de parámetros de la macro inmediatamente después de la [!](../../assembler/masm/operator-logical-not-masm-run-time.md) o [ % ](../../assembler/masm/operator-percent.md) operador.
