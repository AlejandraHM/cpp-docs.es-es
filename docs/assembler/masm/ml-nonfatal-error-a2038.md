# ML Nonfatal Error A2038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**falta un operando del operador macro**  
  
 El ensamblador encontró el final de la lista de parámetros de una macro inmediatamente después del operador de [\!](../../assembler/masm/operator-logical-not-masm-run-time.md) o de [%](../../assembler/masm/operator-percent.md) .
