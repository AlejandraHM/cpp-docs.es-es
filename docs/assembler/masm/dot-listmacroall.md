# <a name="listmacroall"></a>.LISTMACROALL
Inicia el listado de todas las instrucciones de macros.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
.LISTMACROALL  
  
```  
  
## <a name="remarks"></a>Comentarios  
 Igual que [. LALL](../../assembler/masm/dot-lall.md).
