# <a name="forc"></a>FORC
Marca un bloque que se repetirá una vez por cada carácter de `string`, con la sustitución de caracteres actual `parameter` en cada repetición.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## <a name="remarks"></a>Comentarios  
 Igual que [IRPC](../../assembler/masm/irpc.md).
