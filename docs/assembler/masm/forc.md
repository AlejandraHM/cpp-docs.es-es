# FORC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marca un bloque que se repetirá una vez por cada carácter de `string`, con el carácter actual reemplazando `parameter` en cada repetición.  
  
## Sintaxis  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## Comentarios  
 Igual que [IRPC](../../assembler/masm/irpc.md).
