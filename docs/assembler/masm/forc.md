# <a name="forc"></a>FORC

Marca un bloque que se repetirá una vez por cada carácter de *cadena*, con la sustitución de caracteres actual *parámetro* en cada repetición.

## <a name="syntax"></a>Sintaxis

> FORZADA *parámetro*, \< *cadena*><br/>
> *Instrucciones*<br/>
> ENDM

## <a name="remarks"></a>Comentarios

Igual que [IRPC](../../assembler/masm/irpc.md).
