# <a name="dword"></a>DWORD
Asigna y, opcionalmente, inicializa una palabra doble (4 bytes) de almacenamiento para cada `initializer`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Comentarios  
 También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.
