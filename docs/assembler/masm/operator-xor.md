# <a name="operator-xor"></a>Operador XOR
Devuelve el resultado de un bit a bit **XOR** operación para *expression1* y *expression2*.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 XOR expression2  
```
