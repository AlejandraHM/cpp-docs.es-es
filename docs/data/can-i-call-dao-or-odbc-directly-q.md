# &#191;Se puede llamar directamente a objetos DAO u ODBC?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Como es habitual en MFC, si necesita un control más preciso, puede llamar directamente a DAO o a ODBC, además de obtener acceso a estas tecnologías a través de las clases.  MFC pretende simplificar la programación para Windows, pero no le impide el acceso a las API subyacentes.
