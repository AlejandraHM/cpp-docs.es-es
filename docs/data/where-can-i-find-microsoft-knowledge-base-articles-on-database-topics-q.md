# &#191;D&#243;nde encontrar art&#237;culos de Microsoft Knowledge Base sobre bases de datos?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Para encontrar artículos de Knowledge Base sobre temas de ODBC de MFC, busque la palabra clave MFCDATABASE en Knowledge Base.  
  
 Para encontrar artículos de Knowledge Base sobre temas de DAO de MFC, busque la palabra clave MFCDAO en Knowledge Base.
