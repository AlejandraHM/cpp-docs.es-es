# Dise&#241;ar y crear una vista de registros (acceso a datos MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Puede crear la clase de vista de registros con el [Asistente para aplicaciones MFC](../mfc/reference/database-support-mfc-application-wizard.md).  Si utiliza un asistente para aplicaciones, este crea la clase de vista de registros y un recurso de plantilla de cuadro de diálogo \(sin controles\).  Debe utilizar el editor de cuadros de diálogo de Visual C\+\+ para agregar controles al recurso de plantilla de cuadro de diálogo.  Por otra parte, si utiliza **Agregar clase**, debe crear primero el recurso de plantilla de cuadro de diálogo en el editor de cuadros de diálogo y, a continuación, crear la clase de vista de registros.  
  
 Esta información se aplica a `CRecordView` y `CDaoRecordView`.  
  
#### Cómo crear la vista de registros con el asistente para aplicaciones MFC  
  
1.  Consulte [Compatibilidad con bases de datos, Asistente para aplicaciones MFC](../mfc/reference/database-support-mfc-application-wizard.md).  
  
#### Cómo diseñar el formulario  
  
1.  Consulte [Editor de cuadros de diálogo](../mfc/dialog-editor.md).  
  
#### Cómo crear la clase de vista de registros  
  
1.  Consulte [Agregar un consumidor ODBC de MFC](../mfc/reference/adding-an-mfc-odbc-consumer.md).
