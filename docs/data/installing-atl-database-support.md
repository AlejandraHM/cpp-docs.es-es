# Instalar compatibilidad con bases de datos ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El programa de instalación instala todos los componentes necesarios para la compatibilidad con la programación de OLE DB, como parte de la instalación de línea base.  Las interfaces OLE DB se instalan como parte del SDK de MDAC.  Las plantillas OLE DB, que proporcionan compatibilidad con la programación de OLE DB en Visual C\+\+ .NET, se instalan como parte de la biblioteca ATL.  El programa de instalación de Visual C\+\+ .NET instala tanto el SDK de MDAC \(versión 2.8\) como la biblioteca ATL.
