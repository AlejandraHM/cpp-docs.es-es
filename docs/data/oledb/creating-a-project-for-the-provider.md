# <a name="creating-a-project-for-the-provider"></a>Crear un proyecto para un proveedor
#### <a name="to-create-a-project-in-which-the-ole-db-provider-will-reside"></a>Para crear un proyecto en el que residirá el proveedor OLE DB  
  
1.  Desde el **archivo** menú, haga clic en `New`y, a continuación, haga clic en **proyecto**.  
  
     Aparecerá el cuadro de diálogo **Nuevo proyecto** .  
  
2.  En el panel tipos de proyecto, haga clic en la carpeta de proyectos de Visual C++. En el panel Plantillas, haga clic en **proyecto ATL**.  
  
3.  En el **nombre** cuadro, escriba un nombre para el proyecto y, a continuación, haga clic en **Aceptar**.  
  
     Aparece el Asistente para proyectos ATL.  
  
4.  En el Asistente para proyectos ATL, elija **biblioteca de vínculos dinámicos (DLL)** para **tipo de servidor**.  
  
5.  Haga clic en **Finalizar**.
