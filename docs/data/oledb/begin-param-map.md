# BEGIN_PARAM_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marca el principio de las entradas del mapa de parámetros.  
  
## Sintaxis  
  
```  
  
BEGIN_PARAM_MAP(  
x  
 )  
  
```  
  
#### Parámetros  
 *x*  
 \[in\] El nombre de la clase de registro de usuario.  
  
## Comentarios  
 Los parámetros son utilizados por [comandos](https://msdn.microsoft.com/en-us/library/ms724608.aspx).  
  
## Ejemplo  
 Vea el ejemplo para la macro de [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md) .  
  
## Requisitos  
 **Encabezado:** atldbcli.h
