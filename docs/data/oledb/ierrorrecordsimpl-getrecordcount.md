# <a name="ierrorrecordsimplgetrecordcount"></a>IErrorRecordsImpl::GetRecordCount
Devuelve el número de registros en el objeto de registro de OLE DB.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
      STDMETHOD(GetRecordCount )(ULONG *pcRecords);  
```  
  
#### <a name="parameters"></a>Parámetros  
 Vea [IErrorRecords::GetRecordCount](https://msdn.microsoft.com/en-us/library/ms722724.aspx) en el *referencia del programador OLE DB*.
