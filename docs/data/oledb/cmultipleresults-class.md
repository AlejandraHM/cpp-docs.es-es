# CMultipleResults (Clase)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Si desea un comando de controlar varios conjuntos de resultados, utilice `CMultipleResults` para el argumento *TMultiple*de la plantilla de `CCommand` .  
  
## Sintaxis  
  
```  
class CMultipleResults  
```  
  
## Comentarios  
 Para controlar varios conjuntos de resultados, [CCommand](../../data/oledb/ccommand-class.md) debe heredar de esta clase.
