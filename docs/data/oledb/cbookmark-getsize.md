# CBookmark::GetSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Recupera el tamaño del búfer del marcador.  
  
## Sintaxis  
  
```  
  
virtual DBLENGTH GetSize( ) const throw( );  
  
```  
  
## Valor devuelto  
 El tamaño del búfer en bytes.  
  
## Requisitos  
 **Encabezado:** atldbcli.h
