[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Un consumidor de plantillas OLE DB se puede crear con o sin el Asistente para consumidores OLE DB ATL, aunque se recomienda el uso del asistente.  Un consumidor puede funcionar en cualquier aplicación compatible con ATL.
