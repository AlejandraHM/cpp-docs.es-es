# <a name="cenumeratoraccessormntype"></a>CEnumeratorAccessor::m_nType
Una variable que indica si la fila describe un origen de datos o un enumerador.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
USHORT m_nType;  
  
```
