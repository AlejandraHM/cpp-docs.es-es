# CEnumeratorAccessor::m_nType
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Una variable que indica si la fila describe un origen de datos o un enumerador.  
  
## Sintaxis  
  
```  
  
USHORT m_nType;  
  
```
