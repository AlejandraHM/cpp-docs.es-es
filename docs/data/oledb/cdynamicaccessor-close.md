# CDynamicAccessor::Close
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Desata todas las columnas, libera la memoria asignada, y libera el puntero de interfaz de [IAccessor](https://msdn.microsoft.com/en-us/library/ms719672.aspx) en la clase.  
  
## Sintaxis  
  
```  
  
void Close( ) throw( );  
  
```  
  
## Requisitos  
 **Encabezado:** atldbcli.h
