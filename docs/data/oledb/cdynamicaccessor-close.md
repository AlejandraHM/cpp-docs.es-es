# <a name="cdynamicaccessorclose"></a>CDynamicAccessor::Close
Desenlaza todas las columnas, libera la memoria asignada y libera el [IAccessor](https://msdn.microsoft.com/en-us/library/ms719672.aspx) puntero de interfaz en la clase.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
void Close() throw();  
  
```
