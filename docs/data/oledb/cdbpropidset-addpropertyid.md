# CDBPropIDSet::AddPropertyID
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Agrega un identificador de propiedad del conjunto de identificadores de la propiedad.  
  
## Sintaxis  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### Parámetros  
 `propid`  
 \[in\] El identificador de la propiedad que se debe agregar al conjunto de identificadores de la propiedad.  
  
## Requisitos  
 **Encabezado:** atldbcli.h
