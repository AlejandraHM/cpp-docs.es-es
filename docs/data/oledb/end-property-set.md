# <a name="endpropertyset"></a>END_PROPERTY_SET
Marca el final de un conjunto de propiedades.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### <a name="parameters"></a>Parámetros  
 `guid`  
 [in] El GUID de la propiedad.
