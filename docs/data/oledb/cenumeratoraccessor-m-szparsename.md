# CEnumeratorAccessor::m_szParseName
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cadena que pasar a [IParseDisplayName](http://msdn.microsoft.com/library/windows/desktop/ms680604) para obtener un moniker para el origen de datos o el enumerador.  
  
## Sintaxis  
  
```  
  
WCHAR m_szParseName[129];  
  
```  
  
## Comentarios  
 Vea [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) en *la referencia del* programador para obtener más información.  
  
## Requisitos  
 **Encabezado:** atldbcli.h
