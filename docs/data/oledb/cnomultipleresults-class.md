# CNoMultipleResults (Clase)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Utilizado como un argumento de plantilla \(*TMultiple*\) a [CCommand](../../data/oledb/ccommand-class.md) para crear un comando optimizado que controla un único conjunto de resultados.  
  
## Sintaxis  
  
```  
class CNoMultipleResults  
```  
  
## Comentarios  
 Si desea un comando de controlar varios conjuntos de resultados, utilice [CMultipleResults](../../data/oledb/cmultipleresults-class.md) en su lugar.  
  
## Requisitos  
 **Header:** atldbcli.h
