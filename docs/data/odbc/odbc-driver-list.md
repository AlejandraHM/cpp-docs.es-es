# Lista de controladores ODBC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ proporciona controladores ODBC para las siguientes bases de datos:  
  
-   SQL Server  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   dBASE  
  
-   Paradox  
  
-   Microsoft ODBC para Oracle  
  
-   Archivos de texto  
  
 Para obtener información sobre los controladores ODBC de Microsoft y de otras compañías, incluido el paquete de controladores ODBC, póngase en contacto con los servicios de soporte técnico de Microsoft que atienden a su país.
