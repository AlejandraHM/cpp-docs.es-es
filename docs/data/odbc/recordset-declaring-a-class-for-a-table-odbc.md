# Conjunto de registros: Declarar una clase para una tabla (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Este tema es aplicable a las clases ODBC de MFC.  
  
 La clase de conjunto de registros más común abre una sola tabla.  Para declarar una clase de conjunto de registros para una sola tabla, use el [Asistente para consumidores ODBC de MFC](../../mfc/reference/adding-an-mfc-odbc-consumer.md) de **Agregar clase** y elija las columnas que desee nombrando el miembro de datos de campo correspondiente al conjunto de registros.  
  
 Otros usos para los conjuntos de registros:  
  
-   Combinar dos o más tablas.  
  
-   Contener los resultados de una consulta predefinida.
