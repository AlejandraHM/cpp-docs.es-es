# <a name="work-with-odbc-and-drivers"></a>Trabajar con ODBC y controladores

ODBC permite escribir aplicaciones que no son dependientes de un solo sistema de administración de bases de datos (DBMS). Puede escribir una aplicación que puede operar en varios DBMS para el que los usuarios tienen el controlador ODBC específicos para DBMS correcto.
