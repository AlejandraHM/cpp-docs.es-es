# &#191;Cu&#225;ndo se deben utilizar las clases de base de datos?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Muchas aplicaciones requieren el almacenamiento de datos en una base de datos y muchas otras aplicaciones pueden beneficiarse del uso de una base de datos.  Las bases de datos proporcionan un depósito flexible de datos a los que pueden tener acceso, en muchos casos, varios usuarios y varias aplicaciones.  Las bases de datos pueden almacenar grandes cantidades de datos y proporcionar acceso rápido a datos para consultas y actualizaciones.
