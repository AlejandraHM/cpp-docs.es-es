# swap_ranges (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cambie los elementos de un intervalo con elementos de otro, como \- intervalo ordenados.  
  
## Sintaxis  
  
```  
template<class _FwdIt1, class _FwdIt2> inline  
    _FwdIt2 swap_ranges(_FwdIt1 _First1, _FwdIt1 _Last1,  
        _FwdIt2 _First2);  
```  
  
## Comentarios  
 Esta función se comporta igual que la función `swap_ranges`STL.  Para obtener más información, vea [swap\_ranges](../Topic/swap_ranges.md).
