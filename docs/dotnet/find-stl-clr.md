# find (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Busque la posición de la primera aparición de un elemento en un intervalo con un valor especificado.  
  
## Sintaxis  
  
```  
template<class _InIt, class _Ty> inline  
    _InIt find(_InIt _First, _InIt _Last, const _Ty% _Val);  
```  
  
## Comentarios  
 Esta función se comporta igual que la función `find`STL.  Para obtener más información, vea [find](../Topic/find%20\(STL\).md).
