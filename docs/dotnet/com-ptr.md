# com::ptr
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un contenedor para un objeto COM que se puede utilizar como miembro de una clase CLR.  El contenedor también automatiza la administración de la duración del objeto COM, liberando referencias son propiedad del objeto cuando se llama al destructor.  Análogo a [CComPtr Class](../atl/reference/ccomptr-class.md).  
  
## Sintaxis  
  
```  
#include <msclr\com\ptr.h>  
```  
  
## Comentarios  
 [com::ptr \(Clase\)](../dotnet/com-ptr-class.md) se define en \<el archivo de msclr\\com\\ptr.h\> .
