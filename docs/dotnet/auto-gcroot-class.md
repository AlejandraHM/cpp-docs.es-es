# auto_gcroot (Clase)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Administración de recursos automática \(como [auto\_ptr \(Clase\)](../standard-library/auto-ptr-class.md)\) que se puede utilizar para insertar un identificador virtual en un tipo nativo.  
  
## Sintaxis  
  
```  
template<typename _element_type>  
class auto_gcroot;  
```  
  
#### Parámetros  
 `_element_type`  
 El tipo administrado que se va a insertar.  
  
## Requisitos  
 **Archivo de encabezado** \<msclr\\auto\_gcroot.h\>  
  
 msclr de**Namespace**
