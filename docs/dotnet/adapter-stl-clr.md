# <a name="adapter-stlclr"></a>adapter (STL/CLR)
El encabezado STL/CLR `<cliext/adapter>` especifica dos clases de plantilla (`collection_adapter` y `range_adapter`) y la función de plantilla `make_collection`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
#include <cliext/adapter>  
```  
  
## <a name="remarks"></a>Comentarios  
  
|Clase|Descripción|  
|-----------|-----------------|  
|[collection_adapter (STL/CLR)](../dotnet/collection-adapter-stl-clr.md)|Encapsula la colección de la biblioteca de clases Base (BCL) como un intervalo.|  
|[range_adapter (STL/CLR)](../dotnet/range-adapter-stl-clr.md)|Ajusta el intervalo como una colección de BCL.|  
  
|Función|Descripción|  
|--------------|-----------------|  
|[make_collection (STL/CLR)](../dotnet/make-collection-stl-clr.md)|Crea un adaptador de intervalo mediante un par de iteradores.|  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** \<cliext/adaptador >  
  
 **Namespace:** cliext
