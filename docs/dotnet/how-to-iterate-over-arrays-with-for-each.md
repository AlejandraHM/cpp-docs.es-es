# C&#243;mo: Iterar por matrices con for each
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Este tema muestra cómo utilizar la palabra clave de [for each, in](../dotnet/for-each-in.md) en diferentes tipos de matrices.  
  
## Ejemplo  
 Este ejemplo muestra cómo utilizar `for each` en la matriz de tipos de referencia.  Tenga en cuenta que si cualquier dimensión de una matriz dimensional múltiples es cero, el bucle de `for each` no iterará sobre la matriz.  
  
```  
// for_each_arrays.cpp  
// compile with: /clr  
using namespace System;  
ref struct MyClass {  
   void Test() { Console::WriteLine("in MyClass"); }  
};  
  
ref struct MyClass2 {  
   void Test() { Console::WriteLine("in MyClass2"); }  
};  
  
int main() {  
   array<MyClass ^> ^ MyArray = gcnew array<MyClass ^>(2);  
  
   int i = 0;  
   for each ( MyClass ^ c in MyArray ) {  
      Console::Write("{0} = ", i++);  
      c -> Test();  
   }  
  
   Console::WriteLine();  
  
   array< MyClass2 ^, 2 > ^ MyArray2 = gcnew array< MyClass2 ^, 2 >(2, 2);  
   i = 0;  
   for each ( MyClass2 ^ c in MyArray2 ) {  
      Console::Write("{0} = ", i++);  
      c -> Test();  
   }  
  
   array< MyClass2 ^, 2 > ^ MyArray3 = gcnew array< MyClass2 ^, 2 >(2, 0);  
   i = 0;  
   for each ( MyClass2 ^ c in MyArray3 ) {  
      Console::Write("{0} = ", i++);  
      c -> Test();  
   }  
}  
```  
  
  **0 \= en MyClass**  
**1 \= en MyClass**  
**0 \= de MyClass2**  
**1 \= de MyClass2**  
**2 \= de MyClass2**  
**3 \= de MyClass2**   
## Ejemplo  
 En este ejemplo se muestra para cada que recorre en iteración <xref:System.Collections.ArrayList>, que implementa <xref:System.Collections.IEnumerable>.  
  
```  
// for_each_arrays_2.cpp  
// compile with: /clr  
using namespace System;  
using namespace System::Collections;  
  
int main() {  
   int retval = 0;  
  
   ArrayList ^ arr = gcnew ArrayList();  
   arr->Add(10);  
   arr->Add(20);  
   arr->Add(30);  
  
   for each ( int c in arr )  
      retval += c;  
   Console::WriteLine(retval);  
}  
```  
  
  **60**   
## Ejemplo  
 En este ejemplo se muestra cómo recorrer en iteración una matriz de matrices.  
  
```  
// for_each_arrays_3.cpp  
// compile with: /clr  
using namespace System;  
  
#define ARRAY_SIZE 2  
  
int main() {  
   int i, j;  
  
   // Declares an array of managed arrays of Int32.  
   array< array< Int32 > ^ > ^ IntArray = gcnew array<array< Int32 > ^>(ARRAY_SIZE);  
  
   for (i = 0 ; i < ARRAY_SIZE ; i++) {  
      IntArray[i] = gcnew array< Int32 >(ARRAY_SIZE);  
         for ( int j = 0 ; j < ARRAY_SIZE ; j++ )   
            IntArray[i][j] = i + 10;  
   }  
  
   for (i = 0 ; i < ARRAY_SIZE ; i++)  
      for (int j = 0 ; j < ARRAY_SIZE ; j++)  
         Console::WriteLine("IntArray[{0}] = {1}", i, IntArray[i][j]);  
   Console::WriteLine();  
  
   for each (array<Int32> ^ xyz in IntArray)  
      for each ( int c in xyz )  
         Console::WriteLine(c);  
}  
```  
  
  **IntArray \[0\] \= 10**  
**IntArray \[0\] \= 10**  
**IntArray \[1\] \= 11**  
**IntArray \[1\] \= 11**  
**10**  
**10**  
**11**  
**11**
