# utility (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Incluya el encabezado `<cliext/utility>` de STL\/CLR para definir la clase de plantilla `pair` y varios que admiten funciones de plantilla.  
  
## Sintaxis  
  
```  
#include <utility>  
```
