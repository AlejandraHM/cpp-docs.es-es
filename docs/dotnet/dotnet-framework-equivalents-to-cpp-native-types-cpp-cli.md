# <a name="net-framework-equivalents-to-c-native-types-ccli"></a>Equivalentes de .NET Framework para tipos nativos de C++ (C++/CLI)
La siguiente tabla muestra las palabras clave para los tipos integrados de Visual C++, que son alias de tipos predefinidos en el **System** espacio de nombres.  
  
|Tipo de Visual C++|Tipo de .NET Framework|  
|-----------------------|-------------------------|  
|**bool**|**System.Boolean**|  
|**firmado char** (consulte [/j](../build/reference/j-default-char-type-is-unsigned.md) para obtener más información)|**System.SByte**|  
|**unsigned char**|**System.Byte**|  
|**wchar_t**|**System.Char**|  
|**doble** y **long double**|**System.Double**|  
|**float**|**System.Single**|  
|**int**, **firmado int**, **largo**, y **long con signo**|**System.Int32**|  
|**int sin signo** y **largo sin signo**|**System.UInt32**|  
|**__int64** y **firmado __int64**|**System.Int64**|  
|**unsigned __int64**|**System.UInt64**|  
|**corto** y **firmados de resumen**|**System.Int16**|  
|**unsigned short**|**System.UInt16**|  
|**void**|**System.Void**|
