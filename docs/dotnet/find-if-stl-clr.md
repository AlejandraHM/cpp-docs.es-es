# find_if (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Busque la posición de la primera aparición de un elemento de un intervalo que cumple una condición especificada.  
  
## Sintaxis  
  
```  
template<class _InIt, class _Pr> inline  
    _InIt find_if(_InIt _First, _InIt _Last, _Pr _Pred);  
```  
  
## Comentarios  
 Esta función se comporta igual que la función `find_if`STL.  Para obtener más información, vea [find\_if](../Topic/find_if.md).
