# <a name="null-directive"></a>NULL (Directiva)
La directiva de preprocesador nula es un único signo de número (**#**) solo en una línea. No tiene efecto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
#  
  
```
