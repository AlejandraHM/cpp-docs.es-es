# rename_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específicos de C\+\+**  
  
 Tiene la misma funcionalidad que el atributo [rename\_namespace](../preprocessor/rename-namespace.md), pero se utiliza en las bibliotecas de tipos que se usan la directiva \#import con el atributo [auto\_search](../preprocessor/auto-search.md).  
  
## Sintaxis  
  
```  
rename_search_namespace("NewName")  
```  
  
#### Parámetros  
 `NewName`  
 Nuevo nombre del espacio de nombres.
