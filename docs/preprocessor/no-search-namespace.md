# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específicos de C\+\+**  
  
 Tiene la misma funcionalidad que el atributo [no\_namespace](../preprocessor/no-namespace.md), pero se utiliza en las bibliotecas de tipos que se usan la directiva \#import con el atributo [auto\_search](../preprocessor/auto-search.md).  
  
## Sintaxis  
  
```  
no_search_namespace  
```
