# omp
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Toma una o más directivas de OpenMP, junto con cualquier cláusula directiva opcional.  
  
## Sintaxis  
  
```  
  
#pragma omp directive  
```
