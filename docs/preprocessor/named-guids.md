# named_guids
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específicos de C\+\+**  
  
 Indica al compilador que defina e inicialice las variables GUID antiguas con el formato **LIBID\_MyLib**, **CLSID\_MyCoClass**, **IID\_MyInterface** y **DIID\_MyDispInterface**.  
  
## Sintaxis  
  
```  
named_guids  
```
