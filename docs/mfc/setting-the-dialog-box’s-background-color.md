# Configurar el color de fondo de un cuadro de di&#225;logo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Puede establecer el color de fondo de los cuadros de diálogo controlando los mensajes de `WM_CTLCOLOR` para la ventana del cuadro de diálogo.  Color establece utilizan sólo para el cuadro de diálogo especificado.
