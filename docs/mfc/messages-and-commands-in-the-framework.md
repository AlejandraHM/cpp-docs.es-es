# Mensajes y comandos en el marco
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Las aplicaciones escritas para Microsoft Windows son “mensaje controlado”. En respuesta a eventos como los clics, las pulsaciones de tecla, los movimientos de la ventana, etc., Windows envían mensajes a la ventana correspondiente.  Los mensajes de Windows del proceso de aplicaciones de .NET framework tienen como de cualquier otra aplicación para Windows.  Pero el marco también proporciona algunos mejoras que hacen mensajes de procesamiento más fáciles, más fáciles de mantener, y los encapsularon mejor.
