# <a name="using-csliderctrl"></a>Usar CSliderCtrl

El [CSliderCtrl](../mfc/reference/csliderctrl-class.md) clase representa un control deslizante, que también se denomina una barra de seguimiento. Un "control deslizante" es una ventana que contiene un control deslizante y graduación opcionales marcas. Cuando el usuario mueve el control deslizante, con el mouse o las teclas de dirección, el control deslizante envía mensajes de notificación para indicar el cambio.

Controles deslizantes son útiles cuando desee que el usuario seleccione un valor discreto o un conjunto de valores consecutivos en un intervalo. Por ejemplo, podría usar un control deslizante para permitir al usuario establecer la velocidad de repetición del teclado moviendo el control deslizante hasta una marca de graduación determinado.

Mueve el control deslizante en un control deslizante en incrementos que se especifican al crearlo. Por ejemplo, si especifica que el control deslizante debe tener un intervalo de cinco, el control deslizante solo puede ocupar seis posiciones: una posición en el lado izquierdo del control deslizante y una posición para cada incremento en el intervalo. Normalmente, cada una de estas posiciones se identifica mediante una marca de graduación.
