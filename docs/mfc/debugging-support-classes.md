# Depurar las clases de depuraci&#243;n
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC proporciona clases siguientes para ayudarle a aislar los problemas de asignación de memoria dinámica de depuración.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Proporciona un destino para los volcados de diagnóstico.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Estructura que proporciona las instantáneas de uso de memoria.  También se utiliza para comparar instantáneas anteriores y posteriores de memoria.
