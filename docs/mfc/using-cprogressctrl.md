# Usar CProgressCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Puede utilizar el control de progreso para indicar el progreso de una operación larga.  Es un rectángulo que se llena gradualmente con el color de resaltado del sistema a medida que la operación.  
  
 El control de progreso se representa en MFC por la clase [CProgressCtrl](../mfc/reference/cprogressctrl-class.md).  
  
 Cuando crea inicialmente el control de progreso, especifique el tamaño y posición, ventana primaria \(normalmente un cuadro de diálogo\), e identificador  Mediante el parámetro de `dwStyle` , también puede especificar varias estilos de ventana para el control y los estilos de cómo rellena.
