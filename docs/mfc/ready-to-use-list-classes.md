# Clases de lista preparadas para usar
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Las clases siguientes son clases listas para usar de la lista.  
  
 [CObList](../mfc/reference/coblist-class.md)  
 Almacena punteros a objetos de la clase `CObject` u objetos de clases derivadas de `CObject` en una lista vinculada.  
  
 [CPtrList](../mfc/reference/cptrlist-class.md)  
 Almacena punteros a `void` \(punteros genéricos\) en una lista vinculada.  
  
 [CStringList](../mfc/reference/cstringlist-class.md)  
 Almacena los objetos de `CString` en una lista vinculada.
