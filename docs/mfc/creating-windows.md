# Crear ventanas
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El marco de trabajo crea automáticamente la mayoría de las ventanas que necesita en un programa de base.  [Creación de documentos y vistas](../mfc/document-view-creation.md) muestra cómo el marco de trabajo crea las ventanas de marco asociadas a los documentos y las vistas.  Pero para fines especiales puede crear dispone de las ventanas \(incluidos poseer las ventanas secundarias de ventanas o de vistas de cuadro — además de ventanas proporcionadas por el marco.
