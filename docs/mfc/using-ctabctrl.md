# Usar CTabCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un “control de ficha” es análogo a los divisores de un bloc de notas o carpetas etiquetadas en un archivo.CAB.  Utilice el control de ficha, representado por la clase [CTabCtrl](../mfc/reference/ctabctrl-class.md), para mostrar varias páginas de información o controles a un usuario, uno a la vez, en un formato que sugiere una relación del mismo nivel o lógica entre cada página.  
  
 Para obtener más información sobre los controles de ficha, vea [Controles tab](http://msdn.microsoft.com/library/windows/desktop/bb760550) en [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)].
