# Clases de plantillas para matrices, listas y mapas
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Estas clases de colección son plantillas cuyos parámetros determinan los tipos de los objetos almacenados en agregados.  `CArray`, `CMap`, y las funciones globales auxiliares de uso de las clases de `CList` que deben ser personalizadas normalmente.  Para obtener más información sobre estas funciones auxiliares, vea [Aplicaciones auxiliares de la clase de colección](../mfc/reference/collection-class-helpers.md).  Las clases escritas de puntero son contenedores de otras clases de la biblioteca de clases.  Mediante estos contenedores, se da de alta la comprobación de tipos del compilador para ayudarle a evitar errores.  Para obtener más información sobre cómo utilizar estas clases, vea [Colecciones](../mfc/collections.md).  
  
 Estas clases proporcionan plantillas que puede utilizar para crear matrices, listas, y los mapas mediante cualquiera se escriben tienen como.  
  
 [CArray](../mfc/reference/carray-class.md)  
 Clase de plantilla para crear matrices de tipos arbitrarios.  
  
 [CList](../mfc/reference/clist-class.md)  
 Clase de plantilla para crear listas de tipos arbitrarios.  
  
 [CMap](../mfc/reference/cmap-class.md)  
 Clase de plantilla para crear mapas con la clave y los tipos de valor arbitrarios.  
  
 [CTypedPtrArray](../mfc/reference/ctypedptrarray-class.md)  
 Clase de plantilla para las matrices tipo\-seguras de punteros.  
  
 [CTypedPtrList](../mfc/reference/ctypedptrlist-class.md)  
 Clase de plantilla para las listas tipo\-seguras de punteros.  
  
 [CTypedPtrMap](../mfc/reference/ctypedptrmap-class.md)  
 Clase de plantilla para los mapas tipo\- seguros con punteros.
