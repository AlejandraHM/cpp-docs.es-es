# Servidores: Problemas de la interfaz de usuario
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Una aplicación de servidor tiene varias características que se deben agregar a la interfaz de usuario para proporcionar los elementos OLE a las aplicaciones contenedoras.  Para obtener más información sobre los menús y los recursos adicionales que se deben agregar a una aplicación de servidor, vea [Menús y recursos: Adiciones de Servidor](../mfc/menus-and-resources-server-additions.md).
