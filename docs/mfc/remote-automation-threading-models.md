# Modelos de subprocesos de automatizaci&#243;n remota
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La automatización remota admite los modelos de un único subproceso y apartamento\- roscados de COM.  En plataformas de 32 bits y de 64 bits, que son todas que Visual C\+\+ admite, el modelo de subprocesos se debe establecer en “apartamento.” Para obtener más información sobre cómo y dónde hacerlo, vea [Crear programas que utilizan la automatización remota](../mfc/creating-programs-that-use-remote-automation.md).
