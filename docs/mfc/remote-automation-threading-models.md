# <a name="remote-automation-threading-models"></a>Modelos de subprocesos de automatización remota
Automatización remota es compatible con el subproceso único y los modelos de subprocesamiento controlado de COM. En plataformas de 32 bits y 64 bits, que todo lo que son compatibles con Visual C++, el modelo de subprocesamiento debe establecerse en "apartamento". Para que obtener más información acerca de cómo y dónde se hace esto, vea [crear programas que utilizan la automatización remota](../mfc/creating-programs-that-use-remote-automation.md).
