# <a name="using-the-dialog-editor-to-add-controls"></a>Usar el editor de cuadros de diálogo para agregar controles

Cuando se crea un recurso de plantilla de cuadro de diálogo con el [editor de cuadro de diálogo](../windows/dialog-editor.md), arrastre controles desde una paleta de controles y colocarlos en el cuadro de diálogo. Esto agrega las especificaciones de ese tipo de control para el recurso de plantilla de cuadro de diálogo. Cuando se construcción un objeto de cuadro de diálogo y llamar a su `Create` o `DoModal` función miembro, el marco de trabajo crea un control de Windows y lo coloca en la ventana de cuadro de diálogo en la pantalla.

En su lugar puede [crear controles a mano](../mfc/adding-controls-by-hand.md) si desea. Esto supone más trabajo.
