# Disabling Guides
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Se pueden usar teclas especiales en conjunción con el mouse para deshabilitar el efecto de ajuste de las guías.  Con la tecla ALT se deshabilita el efecto de ajuste de la guía seleccionada.  Si se desplaza una guía con la tecla MAYÚS, se evita que los controles ajustados se desplacen con ella.  
  
### Para deshabilitar el efecto de ajuste de las guías  
  
1.  Arrastre el control mientras mantiene presionada la tecla **ALT**.  
  
### Para mover guías sin mover los controles ajustados  
  
1.  Arrastre la guía mientras mantiene presionada la tecla **MAYÚS**.  
  
### Para desactivar las guías  
  
1.  En el menú **Formato**, elija **Configuración de las guías**.  
  
2.  En el [cuadro de diálogo Configuración de las guías](../mfc/guide-settings-dialog-box.md), en **Guías de diseño**, seleccione **Ninguna**.  
  
    > [!NOTE]
    >  También puede hacer doble clic en la barra de la regla para obtener acceso al cuadro de diálogo **Configuración de las guías**.  
  
 \-O bien\-  
  
-   En el menú **Formato**, haga clic en **Alternar guías**.
