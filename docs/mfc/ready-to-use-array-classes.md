# Clases de matriz listas para usar
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Las clases siguientes son clases listas para usar de la matriz.  
  
 [CByteArray](../mfc/reference/cbytearray-class.md)  
 Almacena elementos de **byte** escrito en una matriz.  
  
 [CDWordArray](../mfc/reference/cdwordarray-class.md)  
 Almacena elementos de `DWORD` escrito en una matriz.  
  
 [CObArray](../mfc/reference/cobarray-class.md)  
 Almacena punteros a objetos de la clase `CObject` u objetos de clases derivadas de `CObject` en una matriz.  
  
 [CPtrArray](../mfc/reference/cptrarray-class.md)  
 Almacena punteros a `void` \(punteros genéricos\) en una matriz.  
  
 [CUIntArray](../mfc/reference/cuintarray-class.md)  
 Almacena elementos de **UINT** escrito en una matriz.  
  
 [CWordArray](../mfc/reference/cwordarray-class.md)  
 Almacena elementos de **palabra** escrito en una matriz.  
  
 [CStringArray](../mfc/reference/cstringarray-class.md)  
 Almacena los objetos de `CString` en una matriz.
