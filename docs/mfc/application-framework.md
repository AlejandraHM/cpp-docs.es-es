# <a name="application-framework"></a>Marco de aplicación
El núcleo de la biblioteca (Microsoft Foundation Classes) es una encapsulación de una gran parte de la API de Windows en formato de C++. Clases de biblioteca representan windows, cuadros de diálogo, contextos de dispositivo, objetos GDI comunes como pinceles y lápices, controles y otros elementos de Windows estándares. Estas clases proporcionan una cómoda interfaz de funciones miembro de C++ a las estructuras de Windows que encapsulan. Para obtener más información acerca del uso de estas clases, consulte [temas del objeto de ventana](../mfc/window-objects.md).  
  
 Pero la biblioteca MFC también proporciona un nivel de funcionalidad adicional para la aplicación depende de la encapsulación de C++ de la API de Windows. Este nivel es un marco de trabajo de aplicación para Windows que proporciona la mayor parte de la interfaz de usuario común espera de programas para Windows, incluidas las barras de herramientas, barras de estado, impresión, vista previa de impresión, la base de datos compatibilidad y soporte técnico de ActiveX. [Usar las clases para escribir aplicaciones para Windows](../mfc/using-the-classes-to-write-applications-for-windows.md) explica con detalle el marco de trabajo.
