# <a name="ole-classes"></a>Clases de OLE
Las clases OLE funcionan con las otras clases de framework de aplicación para facilitar el acceso a la API de ActiveX, lo que proporciona los programas de una manera fácil de proporcionar la potencia de ActiveX a los usuarios. Con ActiveX, puede:  
  
-   Crear documentos compuestos, lo que permite a los usuarios crear y editar documentos que contengan datos creados por varias aplicaciones, tales como texto, gráficos, hojas de cálculo, sonidos, u otros tipos de datos.  
  
-   Crear objetos OLE que se pueden incrustar en documentos compuestos.  
  
-   Usar funciones OLE de arrastrar y colocar para copiar datos entre aplicaciones.  
  
-   Usar la automatización para controlar un programa con otro.  
  
-   Crear controles ActiveX y contenedores de controles ActiveX (anteriormente denominados controles OLE y contenedores de controles OLE, respectivamente).
