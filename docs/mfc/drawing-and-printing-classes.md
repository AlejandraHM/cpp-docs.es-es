# <a name="drawing-and-printing-classes"></a>Clases de dibujo e impresión

En Windows, toda la salida gráfica se dibuja en un área de dibujo virtual llama a un contexto de dispositivo (DC). MFC proporciona clases para encapsular los distintos tipos de controladores de dominio, así como las encapsulaciones de las herramientas de dibujo, como mapas de bits, pinceles, lápices y paletas de Windows.
