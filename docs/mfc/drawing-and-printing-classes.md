# Clases de dibujo e impresi&#243;n
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En Windows, dibujan a la salida todos gráfica en un área de gráfico virtual denominada contexto \(DC\) del dispositivo.  MFC proporciona clases para encapsular los diferentes tipos de Para, así como las encapsulaciones para las herramientas de dibujo de Windows como mapas de bits, pinceles, tablas, y lápices.
