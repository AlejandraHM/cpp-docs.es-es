# Seguimiento: Implementar el seguimiento en la aplicaci&#243;n OLE
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los rastreadores proporcionan una interfaz gráfica que permitan a los usuarios interactuar con los elementos de OLE de cliente.  Utilización de estilos de seguimiento, los elementos de OLE de cliente se pueden mostrar con bordes tramados, los controladores de tamaño, o una variedad de otros efectos visuales.  En este artículo se describe:  
  
-   [Cómo implementar la traza en el código se](../mfc/how-to-implement-tracking-in-your-code.md).  
  
-   [Efecto de banda elástica y rastreadores](../mfc/rubber-banding-and-trackers.md).  
  
 El artículo también trata el uso de estilos con los rastreadores.  Además, crea varias referencias al ejemplo OLE [OCLIENT](../top/visual-cpp-samples.md)MFC.
