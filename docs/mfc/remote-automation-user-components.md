# <a name="remote-automation-user-components"></a>Componentes de usuario de automatización remota
Debe asegurarse de que cada equipo cliente contiene el programa de cliente y los archivos DLL necesarios de soporte técnico. También necesitará asegurarse de que la aplicación de servidor y cualquier soporte de archivos DLL necesarios están presentes en el equipo del servidor. Por último, debe asegurarse de que el programa de servidor está registrado en cada equipo cliente antes de poder ejecutar el Administrador de RAC para configurar la conexión. Si el programa se registra automáticamente (como la mayoría), sólo tendrá que ejecutar el programa de servidor en el equipo cliente para su registro. En su defecto, tendrá que ejecutar un archivo de registro que proporciona o editar manualmente el registro.
