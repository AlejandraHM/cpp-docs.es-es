# Clases de ventana, cuadro de di&#225;logo y control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La clase `CWnd` y sus clases derivadas encapsulan `HWND`, un identificador de una ventana de Windows.  `CWnd` se puede usar en solitario o como base para derivar nuevas clases.  Las clases derivadas proporcionadas por la biblioteca de clases representan diferentes tipos de ventanas.  
  
 [CWnd](../mfc/reference/cwnd-class.md)  
 La clase base para todas las ventanas.  Puede utilizar una de las clases derivadas de `CWnd` o derivar dispone de clases directamente desde.
