# Contenedores de controles ActiveX
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Puede usar controles ActiveX desarrollados en Visual C\+\+ en otras aplicaciones, como admiten la contención de controles ActiveX.  Varias aplicaciones de Microsoft, empezando por versiones mostradas, contención admiten el control ActiveX.  
  
> [!NOTE]
>  La lista siguiente no es una lista completa de aplicaciones que los controles ActiveX admiten pero representan el conjunto utilizado con mayor frecuencia en pruebas:  
  
 Aplicaciones que incluyen admiten la contención de controles ActiveX:  
  
-   Internet Explorer 3.x y mayor  
  
-   Visual Basic 4.x y mayor  
  
-   Visual C\+\+ 4.x y mayor  
  
-   Access 95 y mayor  
  
-   Excel 97 y mayor  
  
-   Word 97 y mayor  
  
-   Access 97 y mayor  
  
-   FrontPage 97 y mayor  
  
-   PowerPoint 97 y mayor  
  
-   Visual Interdev 97 y mayor  
  
 Los siguientes son aplicaciones que no es de Microsoft que contención admiten el control ActiveX:  
  
-   PowerBuilder  
  
-   Delphi  
  
-   Generador de C\+\+  
  
-   Complemento de NCompass para Netscape Navigator
