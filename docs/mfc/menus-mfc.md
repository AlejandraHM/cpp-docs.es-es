# <a name="menus-mfc"></a>Menús (MFC)
MFC proporciona dos elementos para ayudarle a trabajar con menús:  
  
-   Clase [CMenu](../mfc/reference/cmenu-class.md) para manipular los menús del programa en tiempo de ejecución. Usar la documentación de `CMenu` y el ejemplo para aprender a usar `CMenu` eficazmente.  
  
-   Un mecanismo para actualizar menús y botones de barra de herramientas: habilitación o deshabilitación sobre la marcha para satisfacer las condiciones del programa actual.  
  
 Visual C++ también proporciona un [editor de menús](../windows/menu-editor.md) para crear y editar recursos de menú del programa.
