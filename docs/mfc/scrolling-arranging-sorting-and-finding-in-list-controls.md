# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Desplazar, organizar, ordenar y buscar en los controles de lista
Controles de lista ([CListCtrl](../mfc/reference/clistctrl-class.md)) son desplazables de forma predeterminada. Para obtener más información, consulte [posición de desplazamiento](http://msdn.microsoft.com/library/windows/desktop/bb774736) en el SDK de Windows y la [desplazamiento](../mfc/reference/clistctrl-class.md#scroll) función miembro.  
  
 Puede llamar a `CListCtrl` funciones de miembro para organizar los elementos de lista en el control, ordenar los elementos y buscar elementos concretos. Para obtener más información, consulte [organizar, ordenar y buscar](http://msdn.microsoft.com/library/windows/desktop/bb774736) en el SDK de Windows y la [CListCtrl](../mfc/reference/clistctrl-class.md) miembros [organizar](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems) , y [FindItem](../mfc/reference/clistctrl-class.md#finditem).
