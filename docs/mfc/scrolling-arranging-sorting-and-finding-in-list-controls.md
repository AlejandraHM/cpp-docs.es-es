# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Desplazar, organizar, ordenar y buscar en los controles de lista

Los controles de lista ([CListCtrl](../mfc/reference/clistctrl-class.md)) son desplazable de forma predeterminada. Para obtener más información, consulte [posición de desplazamiento](/windows/desktop/Controls/using-list-view-controls) en el SDK de Windows y la [desplazamiento](../mfc/reference/clistctrl-class.md#scroll) función miembro.

Puede llamar a `CListCtrl` las funciones miembro para organizar los elementos de lista en el control, ordenar los elementos y buscar elementos concretos. Para obtener más información, consulte [organizar, ordenar y buscar](https://msdn.microsoft.com/library/windows/desktop/bb774736) en el SDK de Windows y la [CListCtrl](../mfc/reference/clistctrl-class.md) miembros [organizar](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems) , y [FindItem](../mfc/reference/clistctrl-class.md#finditem).
