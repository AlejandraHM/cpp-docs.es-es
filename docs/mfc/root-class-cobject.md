# Clase ra&#237;z: CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La mayoría de las clases de la biblioteca de \(MFC\) de la clase de la Microsoft foundation class se derivan de una clase única basada en la raíz de la jerarquía de clases.  `CObject` proporciona varias funciones útiles para todas las clases derivadas de ella, con poca sobrecarga.  Para obtener más información sobre `CObject` y sus funciones, vea [Mediante CObject](../mfc/using-cobject.md).  
  
 [CObject](../mfc/reference/cobject-class.md)  
 La última clase base de las clases MFC.  Admite la serialización de datos y cómo obtener información en tiempo de ejecución sobre una clase.  
  
 [Recursos](../mfc/reference/cruntimeclass-structure.md)  
 Estructura utilizada para determinar la clase exacta de un objeto en tiempo de ejecución.
