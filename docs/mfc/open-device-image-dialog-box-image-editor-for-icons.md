# Cuadro de di&#225;logo Abrir imagen de &lt;dispositivo&gt; (Editor de im&#225;genes para iconos)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Use este cuadro de diálogo para abrir imágenes de dispositivo.  En él se enumeran las imágenes de dispositivo existentes en el recurso actual \(imágenes que forman parte del recurso\).  
  
 **Imágenes actuales**  
 Muestra una lista de las imágenes incluidas en el recurso.  Seleccione el tipo de imagen que desee abrir.  
  
## Requisitos  
 None
