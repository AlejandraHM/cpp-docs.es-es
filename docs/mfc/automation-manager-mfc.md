# <a name="automation-manager-mfc"></a>Administrador de Automation (MFC)
AUTMGR32. EXE debe copiarse al directorio del sistema de Windows de cada máquina que se pretende proporcionar objetos de automatización remota. Suele ser C:\WINDOWS\SYSTEM32.  
  
 Si desea habilitar las devoluciones de llamada desde el servidor al cliente, este archivo ejecutable debe copiarse también en el directorio del sistema de cada equipo cliente.  
  
 Cuando se ejecuta el Administrador de automatización, aparecerá una pequeña ventana en el equipo del servidor que contiene información de estado resumida. Si desea ocultarla, consulte el artículo Q138067 en Microsoft Knowledge Base.
