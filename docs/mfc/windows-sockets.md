# Windows Sockets
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Esta familia de casos cubre la implementación de MFC de Windows Sockets.  MFC proporciona dos clases a las aplicaciones de red de programación admiten con Windows Sockets API.  La clase [CAsyncSocket](../mfc/reference/casyncsocket-class.md) encapsula el Windows Sockets API de una en una, dando a desarrolladores avanzados de red la mayoría de energía y la flexibilidad.  La clase [CSocket](../mfc/reference/csocket-class.md) proporciona una interfaz simplificada para serializar los datos entre un objeto de [CArchive](../mfc/reference/carchive-class.md) .
