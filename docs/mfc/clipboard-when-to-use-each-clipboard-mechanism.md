# Portapapeles: Cu&#225;ndo usar cada mecanismo del portapapeles
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Siga estas instrucciones al utilizar el portapapeles:  
  
-   Ahora utilice el mecanismo OLE del portapapeles para habilitar funciones nuevas en el futuro.  Mientras el portapapeles estándar API se mantiene, el mecanismo OLE es el futuro de transferencia de datos.  
  
-   Utilice el mecanismo de portapapeles OLE si escribe una aplicación OLE o desea ninguna de las características de OLE, como arrastrar y colocar.  
  
-   Utilice el mecanismo OLE del portapapeles si está proporcionando formatos de OLE.
