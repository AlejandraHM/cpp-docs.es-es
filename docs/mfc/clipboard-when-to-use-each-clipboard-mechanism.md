# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>Portapapeles: Cuándo usar cada mecanismo del portapapeles
Siga estas instrucciones para poder usar el Portapapeles:  
  
-   Usar el mecanismo del Portapapeles de OLE ahora para habilitar las nuevas capacidades en el futuro. Mientras que la API del Portapapeles estándar se mantendrá, el mecanismo OLE es el futuro de la transferencia de datos.  
  
-   Usar el mecanismo del Portapapeles de OLE si está escribiendo una aplicación OLE o desea alguna de las características OLE, como arrastrar y colocar.  
  
-   Usar el mecanismo del Portapapeles de OLE si está proporcionando formatos OLE.
