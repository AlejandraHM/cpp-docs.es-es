# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037: Aplicaciones MFC 2.1 multiproceso

Esta nota técnica descrita originalmente las limitaciones de los programas multiproceso con MFC 2.1, proporcionó originalmente con Visual C++ 1.0 para Windows NT. MFC 3.0 admite multithreading directamente y está documentado. Ver esa referencia para obtener más información.
