# Asignar mensajes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cada clase de.NET framework que puede recibir mensajes o comandos tiene su propio “mapa de mensajes”. El marco de trabajo usa mapas de mensajes para conectar mensajes y los comandos al controlador funcionan.  Cualquier clase derivada de la clase `CCmdTarget` puede tener un mapa de mensajes.  Otros casos explican mapas de mensajes con detalle y se describe cómo utilizarlos.  
  
 A pesar de nombre “mapa de mensajes”, los mapas de mensajes controlan los mensajes y los comandos — las tres categorías de mensajes incluidos en [Categorías de mensaje](../mfc/message-categories.md).
