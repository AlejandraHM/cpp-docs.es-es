# Usar CDateTimeCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El control selector de fecha y hora \([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)\) implementa un método intuitivo y reconocible de escribir o de seleccionar una fecha concreta.  La interfaz principal del control es similar en funcionalidad a un cuadro combinado.  Sin embargo, si el usuario expande el control, un control de calendario mensual aparece \(de forma predeterminada\), lo que especifique una fecha determinada.  Cuando se elige una fecha, el control de calendario mensual automáticamente desaparece.  
  
> [!NOTE]
>  Para utilizar las clases de **CDateTimePicker** y de `CMonthCalCtrl` en el proyecto, debe incluir AFXDTCTL.H, normalmente en el archivo de STDAFX.H del proyecto.
