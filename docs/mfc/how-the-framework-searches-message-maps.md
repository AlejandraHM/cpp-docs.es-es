# C&#243;mo busca el marco los mapas de mensajes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El marco busca la tabla de mensaje\- mapa para coincide con los mensajes entrantes.  Una vez que se escribe una entrada de mensaje\- mapa para cada mensaje que desea una clase para controlar y escribir controladores correspondientes, el marco de trabajo llama a los controladores automáticamente.  Los temas siguientes explican búsqueda de mensaje\- mapa:

-   [Mapas derivados de mensajes](../mfc/derived-message-maps.md)  
  
-   [Intervalos de asignación de mensajes, de id. de comando, o de id. de control un controlador](../mfc/handlers-for-message-map-ranges.md)
