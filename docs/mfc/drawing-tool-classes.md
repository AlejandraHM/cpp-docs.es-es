# <a name="drawing-tool-classes"></a>Clases de herramientas de dibujo
Estas clases encapsulan las herramientas de dibujo que se utilizan para dibujar en un contexto de dispositivo.  
  
 [CGdiObject](../mfc/reference/cgdiobject-class.md)  
 La clase base para las herramientas de dibujo de GDI.  
  
 [CBrush](../mfc/reference/cbrush-class.md)  
 Encapsula un pincel GDI que puede seleccionarse como el pincel actual en un contexto de dispositivo. Pinceles se usan para rellenar interiores de los objetos que se va a dibujar.  
  
 [CPen](../mfc/reference/cpen-class.md)  
 Encapsula un lápiz GDI que puede seleccionarse como el lápiz actual en un contexto de dispositivo. Lápices se usan para dibujar las líneas de borde de objetos.  
  
 [CFont](../mfc/reference/cfont-class.md)  
 Encapsula una fuente GDI que puede seleccionarse como la fuente actual en un contexto de dispositivo.  
  
 [CBitmap](../mfc/reference/cbitmap-class.md)  
 Encapsula un mapa de bits GDI, proporcionando una interfaz para manipular los mapas de bits.  
  
 [CPalette](../mfc/reference/cpalette-class.md)  
 Encapsula una paleta de colores GDI para su uso como una interfaz entre la aplicación y un dispositivo de salida de color como una presentación.  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 Muestra y administra la interfaz de usuario para cambiar el tamaño y mover objetos rectangulares.
