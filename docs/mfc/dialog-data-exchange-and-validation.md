# Intercambio y validaci&#243;n de datos de cuadros de di&#225;logo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

El diálogo intercambio de datos \(DDX\) es una manera fácil de inicializar los controles en el cuadro de diálogo y para recopilar la entrada de datos de usuario.  Validación de datos \(DDV\) de diálogo es una manera fácil de validar la entrada de datos en un cuadro de diálogo.  Para aprovechar DDX y de DDV en los cuadros de diálogo, utilice [Asistente para agregar variables miembro](../ide/add-member-variable-wizard.md) para crear miembros de datos y establecer sus tipos de datos y especificar reglas de validación.
