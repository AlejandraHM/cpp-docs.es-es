# Clases OLE DB
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Compatibilidad de OLE DB en MFC consta de actualmente la clase [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md).  `COleDBRecordView` muestra los registros de una base de datos en controles, con una vista de formulario directamente conectada a un objeto de [CRowset](../data/oledb/crowset-class.md) .  Para obtener más información sobre las plantillas de consumidor OLE DB, vea [Lista de plantillas de consumidor OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
