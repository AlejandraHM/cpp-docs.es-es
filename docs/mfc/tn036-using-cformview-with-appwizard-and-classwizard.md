# TN036: Usar CFormView con AppWizard y ClassWizard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Esta nota técnica descrita cómo modificar un AppWizard generó la aplicación para que utiliza `CFormView` en lugar de `CView` predeterminado como la clase principal de la vista.  Esto se admite directamente con esta versión de Visual C\+\+.
