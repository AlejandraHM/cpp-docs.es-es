# Controlar mensajes de Windows en un cuadro de di&#225;logo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los cuadros de diálogo son ventanas, por lo que pueden controlar mensajes de Windows si proporciona las funciones adecuadas de controlador.  Cuando se crea la clase de diálogo con el asistente para la clase add, el asistente agrega un mensaje vacío asignado a la clase.  Utilice la ventana Propiedades para asignar los mensajes o comando de Windows que desee la clase para controlar.  
  
 Vea [Mensajes de Windows a la clase de diálogo se](../mfc/mapping-windows-messages-to-your-class.md) para obtener más información.
