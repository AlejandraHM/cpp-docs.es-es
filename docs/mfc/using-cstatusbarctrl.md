# <a name="using-cstatusbarctrl"></a>Usar CStatusBarCtrl

Puede usar el control de barra de estado ([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)) para crear una ventana de control que refleja los diversos tipos de información de estado acerca de la aplicación. La ventana de estado puede dividirse en partes que muestren más de un tipo de información.
