# Usar una plantilla de cuadro de di&#225;logo en memoria
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En lugar de utilizar los métodos especificados en [Tabla de creación del diálogo](../mfc/creating-a-dialog-class-with-code-wizards.md), puede crear uno adecuada del cuadro de diálogo indirectamente de una plantilla de cuadro de diálogo en memoria.  Para obtener más información, vea la clase [CDialog](../mfc/reference/cdialog-class.md) en *la referencia de MFC*.
