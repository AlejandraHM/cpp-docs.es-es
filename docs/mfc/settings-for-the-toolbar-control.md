# Configuraci&#243;n del control Toolbar
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los botones de una barra de herramientas pueden mostrar un mapa de bits, una cadena, o ambas.  De forma predeterminada, el tamaño de la imagen se establece en las dimensiones de 16 por 15 píxeles.  Todos los botones tienen el mismo ancho, de forma predeterminada 24 por 22 píxeles.  El alto de una barra de herramientas está determinado por el alto de los botones, y el ancho de una barra de herramientas es el mismo que el ancho del área de cliente de la ventana primaria, también de forma predeterminada.  
  
 Una barra de herramientas puede tener características integradas de personalización, como un cuadro de diálogo sistema\- definido de personalización, que permiten al usuario inserte, eliminar, o reorganiza los botones de la barra de herramientas.  Una aplicación determina si las características de personalización disponibles al usuario y controla la extensión a la que el usuario puede personalizar la barra de herramientas.  Para obtener más información sobre cómo personalizar la barra de herramientas, vea la clase [CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md) en *la referencia de MFC*.
