### Tareas Diálogo\-relacionadas  
  
|Tarea|Aplique a. . .|  
|-----------|--------------------|  
|[Cree una nueva clase CDialog\-derivada](../mfc/creating-your-dialog-class.md) para administrar el cuadro de diálogo.|Cada cuadro de diálogo.|
