# Inicializaci&#243;n de OLE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Antes de que una aplicación puede utilizar los servicios del sistema de OLE, debe inicializar archivos DLL de OLE del sistema y comprobar que los archivos DLL son la versión correcta.  La función de **AfxOleInit** inicializa los archivos DLL de OLE del sistema.
