# CMFCVisualManagerOfficeXP Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`CMFCVisualManagerOfficeXP` da a la aplicación un aspecto de Microsoft Office XP.  
  
## Sintaxis  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## Members  
 Todos los miembros de esta clase son funciones virtuales que son derivadas de antecesores de esta clase, [CMFCVisualManager Class](../../mfc/reference/cmfcvisualmanager-class.md).  
  
## Comentarios  
 Utiliza la clase de `CMFCVisualManagerOfficeXP` para cambiar la apariencia visual de la aplicación para ser similar al de Microsoft Office XP.
