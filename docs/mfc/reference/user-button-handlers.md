# Controladores del bot&#243;n de usuario
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Las entradas de mapa siguiente corresponden a los prototipos de función.  
  
|Entrada de asignación|Prototipo de función|  
|---------------------------|--------------------------|  
|ON\_BN\_CLICKED \( \<id.\>, \<memberFxn\> \)|memberFxn vacío de afx\_msg \(\);|  
|ON\_BN\_DISABLE \( \<id.\>, \<memberFxn\> \)|memberFxn vacío de afx\_msg \(\);|  
|ON\_BN\_DOUBLECLICKED \( \<id.\>, \<memberFxn\> \)|memberFxn vacío de afx\_msg \(\);|  
|ON\_BN\_HILITE \( \<id.\>, \<memberFxn\> \)|memberFxn vacío de afx\_msg \(\);|  
|ON\_BN\_PAINT \( \<id.\>, \<memberFxn\> \)|memberFxn vacío de afx\_msg \(\);|
