# Inicializaci&#243;n y terminaci&#243;n del motor de bases de datos DAO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Al utilizar los objetos DAO de MFC, el motor de base de datos de DAO debe inicializar y después se finaliza antes de que la aplicación o DLL salga.  Dos funciones, `AfxDaoInit` y `AfxDaoTerm`, realizan estas tareas.  
  
### Inicialización y finalización del motor de base de datos de DAO  
  
|||  
|-|-|  
|[AfxDaoInit](../Topic/AfxDaoInit.md)|Inicializa el motor de base de datos de DAO.|  
|[AfxDaoTerm](../Topic/AfxDaoTerm.md)|Finaliza el motor de base de datos de DAO.|
