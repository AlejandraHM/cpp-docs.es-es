# DHTML (Mapas de eventos)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Las macros siguientes se pueden utilizar para controlar los eventos de DHTML.  
  
## Macros de mapa de eventos DHTML  
 Las macros siguientes se pueden utilizar para controlar los eventos de [CDHtmlDialog](../../mfc/reference/cdhtmldialog-class.md)\- clases derivadas de DHTML.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_EVENT\_MAP](../Topic/BEGIN_DHTML_EVENT_MAP.md)|Marca el inicio del mapa de eventos DHTML.|  
|[BEGIN\_DHTML\_EVENT\_MAP\_INLINE](../Topic/BEGIN_DHTML_EVENT_MAP_INLINE.md)|Marca el inicio del mapa de eventos DHTML.|  
|[DECLARE\_DHTML\_EVENT\_MAP](../Topic/DECLARE_DHTML_EVENT_MAP.md)|Declara el mapa de eventos DHTML.|  
|[DHTML\_EVENT](../Topic/DHTML_EVENT.md)|Se utiliza para controlar un evento en el documento nivel para un único elemento HTML.|  
|[DHTML\_EVENT\_AXCONTROL](../Topic/DHTML_EVENT_AXCONTROL.md)|Utilizado para controlar un evento que se activa por un control ActiveX.|  
|[DHTML\_EVENT\_CLASS](../Topic/DHTML_EVENT_CLASS.md)|Se utiliza para controlar un evento en el documento nivel para todos los elementos HTML con un tipo determinado de CSS.|  
|[DHTML\_EVENT\_ELEMENT](../Topic/DHTML_EVENT_ELEMENT.md)|Se utiliza para controlar un evento en el nivel de elemento.|  
|[DHTML\_EVENT\_ONAFTERUPDATE](../Topic/DHTML_EVENT_ONAFTERUPDATE.md)|Utilizado para controlar el evento de **onafterupdate** de un elemento HTML.|  
|[DHTML\_EVENT\_ONBEFOREUPDATE](../Topic/DHTML_EVENT_ONBEFOREUPDATE.md)|Utilizado para controlar el evento de **onbeforeupdate** de un elemento HTML.|  
|[DHTML\_EVENT\_ONBLUR](../Topic/DHTML_EVENT_ONBLUR.md)|Utilizado para controlar el evento de **onblur** de un elemento HTML.|  
|[DHTML\_EVENT\_ONCHANGE](../Topic/DHTML_EVENT_ONCHANGE.md)|Utilizado para controlar el evento de `onchange` de un elemento HTML.|  
|[DHTML\_EVENT\_ONCLICK](../Topic/DHTML_EVENT_ONCLICK.md)|Utilizado para controlar el evento de **onclick** de un elemento HTML.|  
|[DHTML\_EVENT\_ONDATAAVAILABLE](../Topic/DHTML_EVENT_ONDATAAVAILABLE.md)|Utilizado para controlar el evento de **ondataavailable** de un elemento HTML.|  
|[DHTML\_EVENT\_ONDATASETCHANGED](../Topic/DHTML_EVENT_ONDATASETCHANGED.md)|Utilizado para controlar el evento de **ondatasetchanged** de un elemento HTML.|  
|[DHTML\_EVENT\_ONDATASETCOMPLETE](../Topic/DHTML_EVENT_ONDATASETCOMPLETE.md)|Utilizado para controlar el evento de **ondatasetcomplete** de un elemento HTML.|  
|[DHTML\_EVENT\_ONDBLCLICK](../Topic/DHTML_EVENT_ONDBLCLICK.md)|Utilizado para controlar el evento de **ondblclick** de un elemento HTML.|  
|[DHTML\_EVENT\_ONDRAGSTART](../Topic/DHTML_EVENT_ONDRAGSTART.md)|Utilizado para controlar el evento de **ondragstart** de un elemento HTML.|  
|[DHTML\_EVENT\_ONERRORUPDATE](../Topic/DHTML_EVENT_ONERRORUPDATE.md)|Utilizado para controlar el evento de **onerrorupdate** de un elemento HTML.|  
|[DHTML\_EVENT\_ONFILTERCHANGE](../Topic/DHTML_EVENT_ONFILTERCHANGE.md)|Utilizado para controlar el evento de **onfilterchange** de un elemento HTML.|  
|[DHTML\_EVENT\_ONFOCUS](../Topic/DHTML_EVENT_ONFOCUS.md)|Utilizado para controlar el evento de **onfocus** de un elemento HTML.|  
|[DHTML\_EVENT\_ONHELP](../Topic/DHTML_EVENT_ONHELP.md)|Utilizado para controlar el evento de `onhelp` de un elemento HTML.|  
|[DHTML\_EVENT\_ONKEYDOWN](../Topic/DHTML_EVENT_ONKEYDOWN.md)|Utilizado para controlar el evento de **onkeydown** de un elemento HTML.|  
|[DHTML\_EVENT\_ONKEYPRESS](../Topic/DHTML_EVENT_ONKEYPRESS.md)|Utilizado para controlar el evento de **onkeypress** de un elemento HTML.|  
|[DHTML\_EVENT\_ONKEYUP](../Topic/DHTML_EVENT_ONKEYUP.md)|Utilizado para controlar el evento de **onkeyup** de un elemento HTML.|  
|[DHTML\_EVENT\_ONMOUSEDOWN](../Topic/DHTML_EVENT_ONMOUSEDOWN.md)|Utilizado para controlar el evento de **onmousedown** de un elemento HTML.|  
|[DHTML\_EVENT\_ONMOUSEMOVE](../Topic/DHTML_EVENT_ONMOUSEMOVE.md)|Utilizado para controlar el evento de `onmousemove` de un elemento HTML.|  
|[DHTML\_EVENT\_ONMOUSEOUT](../Topic/DHTML_EVENT_ONMOUSEOUT.md)|Utilizado para controlar el evento de **onmouseout** de un elemento HTML.|

|[DHTML\_EVENT\_ONROWENTER](../Topic/DHTML_EVENT_ONROWENTER.md)|Utilizado para controlar el evento de **onrowenter** de un elemento HTML.|  
|[DHTML\_EVENT\_ONROWEXIT](../Topic/DHTML_EVENT_ONROWEXIT.md)|Utilizado para controlar el evento de **onrowexit** de un elemento HTML.|  
|[DHTML\_EVENT\_ONSELECTSTART](../Topic/DHTML_EVENT_ONSELECTSTART.md)|Utilizado para controlar el evento de **onselectstart** de un elemento HTML.|  
|[DHTML\_EVENT\_TAG](../Topic/DHTML_EVENT_TAG.md)|Se utiliza para controlar un evento en el documento nivel para todos los elementos con una etiqueta HTML determinada.|  
|[END\_DHTML\_EVENT\_MAP](../Topic/END_DHTML_EVENT_MAP.md)|Marca el final del mapa de eventos DHTML.|  
  
## Macros de mapa del evento de dirección URL  
 Las macros siguientes se pueden utilizar para controlar los eventos de [CMultiPageDHtmlDialog](../../mfc/reference/cmultipagedhtmldialog-class.md)\- clases derivadas de DHTML.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_URL\_EVENT\_MAP](../Topic/BEGIN_DHTML_URL_EVENT_MAP.md)|Marca el inicio mapa de evento de varias páginas de DHTML y dirección URL.|  
|[BEGIN\_EMBED\_DHTML\_EVENT\_MAP](../Topic/BEGIN_EMBED_DHTML_EVENT_MAP.md)|Marca el inicio de un mapa incrustado de eventos DHTML.|  
|[BEGIN\_URL\_ENTRIES](../Topic/BEGIN_URL_ENTRIES.md)|Marca el inicio de un mapa de entrada de eventos de la dirección URL.|  
|[DECLARE\_DHTML\_URL\_EVENT\_MAP](../Topic/DECLARE_DHTML_URL_EVENT_MAP.md)|Declara mapa de evento de varias páginas de DHTML y dirección URL.|  
|[END\_DHTML\_URL\_EVENT\_MAP](../Topic/END_DHTML_URL_EVENT_MAP.md)|Marca el final mapa de evento de varias páginas de DHTML y dirección URL.|  
|[END\_EMBED\_DHTML\_EVENT\_MAP](../Topic/END_EMBED_DHTML_EVENT_MAP.md)|Marca el final de un mapa incrustado de eventos DHTML.|  
|[END\_URL\_ENTRIES](../Topic/END_URL_ENTRIES.md)|Marca el final de un mapa de entrada de eventos de la dirección URL.|  
|[URL\_EVENT\_ENTRY](../Topic/URL_EVENT_ENTRY.md)|Asigna a un recurso de dirección URL o HTML a una página en un diálogo de varias páginas.|
