# Agregar un controlador de mensajes MFC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Se puede [utilizar la ventana Propiedades para agregar un controlador de mensajes](../../mfc/reference/mapping-messages-to-functions.md) \(una función miembro que controla mensajes de Windows\) a una clase y asignar mensajes de Windows al controlador de mensajes.  También se puede agregar [un controlador de eventos para cualquier control de cuadro de diálogo](../../mfc/adding-event-handlers-for-dialog-box-controls.md).  
  
 Mediante el uso de la ventana Propiedades para definir funciones de control de mensajes y eventos, se puede actualizar automáticamente la tabla de envío de mensajes \(o mapa de mensajes\) y el archivo de encabezado de clase.  
  
> [!NOTE]
>  Es posible agregar un controlador de mensajes a una clase ATL desde la ventana Propiedades; sin embargo, parte de los resultados puede variar.  Para obtener más información, vea el tema de ATL [Agregar un controlador de mensajes ATL](../../atl/adding-an-atl-message-handler.md).
