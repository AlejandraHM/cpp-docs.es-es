##  <a name="setdefaultwinxpcolors"></a>CMFCVisualManagerOffice2003::SetDefaultWinXPColors  
 Especifica si el administrador visual debe usar colores de tema de Windows XP nativos o colores obtenido de [GetSysColor](http://msdn.microsoft.com/library/windows/desktop/ms724371).  
  
```  
static void SetDefaultWinXPColors(BOOL bDefaultWinXPColors = TRUE);
```  
  
### <a name="parameters"></a>Parámetros  
 [in] `bDefaultWinXPColors`  
 Especifica si el administrador visual utilizará colores nativos de Windows XP.  
  
### <a name="remarks"></a>Comentarios  
 Si `bDefaultWinXPColors` es `TRUE`, el administrador visual utilizará los colores de Windows XP nativos como azul, oliva o silver. De lo contrario, el administrador visual utilizará los colores obtenidos de `GetSysColor`. El administrador visual utiliza elementos visuales como `COLOR_3DFACE`, `COLOR_3DSHADOW`, `COLOR_3DHIGHLIGHT`, `COLOR_3DDKSHADOW`, y `COLOR_3DLIGHT`.  
  
 De forma predeterminada, un `CMFCVisualManagerOffice2003` objeto utiliza nativos colores de tema de Windows XP.
