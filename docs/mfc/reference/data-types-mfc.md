# Tipos de datos (MFC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

En este tema se enumeran los tipos de datos más utilizadas de la biblioteca Microsoft Foundation Class.  La mayoría de los tipos de datos son los mismos que los del kit de desarrollo de software \(SDK\) de plataforma, mientras que otros son únicos en MFC.  
  
 Para obtener información sobre los tipos de datos utilizados en [!INCLUDE[winSDK](../../atl/includes/winsdk_md.md)] y MFC, vea [Tipos de datos de Windows](http://msdn.microsoft.com/library/windows/desktop/aa383751).  
  
 Los tipos de datos únicos en la biblioteca Microsoft Foundation Class incluyen:  
  
-   Valor de**posición**A utilizado para indicar la posición de un elemento en una colección; utilizado por las clases de colección de MFC.  
  
-   Puntero de 32 bits de**LPCRECT**A una estructura \(no modificable\) constante de `RECT` .
