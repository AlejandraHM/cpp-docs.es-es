# Funciones de mapa de bits grises o interpoladas
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Funciones de mapa de bits grises**  
  
 MFC proporciona dos funciones para dar a un mapa de bits la apariencia de un control deshabilitado.  
  
 ![Comparación de versiones de icono grises y originales](../../mfc/reference/media/vcgraybitmap.png "vcGrayBitmap")  
  
|||  
|-|-|  
|[AfxDrawGrayBitmap](../Topic/AfxDrawGrayBitmap.md)|Dibuja una versión gris de un mapa de bits.|  
|[AfxGetGrayBitmap](../Topic/AfxGetGrayBitmap.md)|Copia una versión gris de un mapa de bits.|  
  
 **Funciones de mapa de bits interpoladas**  
  
 MFC también proporciona dos funciones para reemplazar el fondo de un mapa de bits por un patrón interpolado.  
  
 ![Comparación de versiones de icono interpoladas y originales](../../mfc/reference/media/vcditheredbitmap.png "vcDitheredBitmap")  
  
|||  
|-|-|  
|[AfxDrawDitheredBitmap](../Topic/AfxDrawDitheredBitmap.md)|Dibuja un mapa de bits con un fondo interpolado.|  
|[AfxGetDitheredBitmap](../Topic/AfxGetDitheredBitmap.md)|Copia un mapa de bits con un fondo interpolado.|
