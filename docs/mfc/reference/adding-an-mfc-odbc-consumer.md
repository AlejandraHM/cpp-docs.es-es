# Agregar un consumidor ODBC de MFC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Un consumidor ODBC MFC está formado por una clase de conjunto de registros de ODBC y los enlaces de datos necesarios para el acceso a un origen de datos.  
  
### Para agregar un consumidor ODBC MFC  
  
1.  En la Vista de clases, haga clic con el botón secundario del mouse en el proyecto.  En el menú contextual, haga clic en **Agregar** y, después, en **Agregar clase**.  
  
2.  En la carpeta Visual C\+\+, haga doble clic en el icono **Consumidor ODBC MFC** o bien selecciónelo y haga clic en **Abrir**.  
  
     Se abre el Asistente para consumidores ODBC MFC.  
  
3.  Defina la configuración que describa el [Asistente para consumidores ODBC MFC](../../mfc/reference/mfc-odbc-consumer-wizard.md).  
  
4.  Haga clic en **Finalizar** para cerrar el asistente, tras lo cual se insertará en el proyecto el nuevo código de consumidor de ODBC.
