# <a name="adding-an-mfc-odbc-consumer"></a>Agregar un consumidor ODBC de MFC
Un consumidor ODBC de MFC está formada por un ODBC recordset clase y enlaces de datos necesarios para tener acceso a un origen de datos.  
  
### <a name="to-add-an-mfc-odbc-consumer"></a>Para agregar un consumidor ODBC de MFC  
  
1.  En la vista de clases, haga clic en el proyecto. En el menú contextual, haga clic en **agregar** y, a continuación, haga clic en **Agregar clase**.  
  
2.  En la carpeta Visual C++, haga doble clic en el **consumidor ODBC de MFC** icono o selecciónelo y haga clic en **abiertos**.  
  
     Se abre el Asistente para consumidores de ODBC de MFC.  
  
3.  Defina la configuración como se describe en [Asistente para consumidores ODBC de MFC](../../mfc/reference/mfc-odbc-consumer-wizard.md).  
  
4.  Haga clic en **finalizar** para cerrar el asistente, que se insertará el código de consumidor ODBC recién creado en el proyecto.
