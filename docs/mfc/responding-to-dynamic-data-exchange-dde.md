# Responder al intercambio din&#225;mico de datos (DDE)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La ventana de marco puede responder a las solicitudes de intercambio de datos dinámicos de \(DDE\) abrir los archivos del administrador de archivos \(si la extensión de archivo está registrado o está asociado a la aplicación\).  Vea [Registro de shell](../mfc/special-cwinapp-services.md).
