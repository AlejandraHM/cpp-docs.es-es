# SDK de REST de C++ (nombre c&#243;digo &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Con el SDK de REST de C\+\+ \(nombre de código "Casablanca"\) podrá escribir con mayor facilidad código de C\+\+ moderno y asincrónico que se conecta con servicios basados en la nube.  
  
 El proyecto Casablanca ya no se distribuye con Visual Studio.  La documentación y las fuentes están alojadas en CodePlex y se actualizan ocasionalmente.  Para obtener información más detallada sobre este proyecto, vea [C\+\+ REST SDK \(nombre en clave “Casablanca”\)](http://casablanca.codeplex.com/).
