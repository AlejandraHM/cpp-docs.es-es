# Caracteres de finalizaci&#243;n de nueva l&#237;nea
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.2** Si la última línea de una secuencia de texto requiere un carácter de línea nueva de terminación  
  
 Las funciones de secuencia reconocen la nueva línea o el final del archivo como carácter de terminación de una línea.
