# clock (Funci&#243;n) (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1** La era de la función `clock`  
  
 La era de la función `clock` comienza \(con un valor de 0\) cuando el programa C empieza a ejecutarse.  Devuelve el tiempo medido en 1\/**CLOCKS\_PER\_SEC** \(que es igual a 1\/1000 para Microsoft C\).
