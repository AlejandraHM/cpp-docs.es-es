# <a name="clock-function-c"></a>clock (Función) (C)
**ANSI 4.12.2.1** La era de la función `clock`  
  
 La era de la función `clock` comienza (con un valor de 0) cuando el programa C empieza a ejecutarse. Devuelve el tiempo medido en 1/**CLOCKS_PER_SEC** (que es igual a 1/1000 para Microsoft C).
