# <a name="remainders"></a>Restos

**ANSI 3.3.5** El signo del resto en la división de enteros

El signo del resto es igual que el signo del dividendo. Por ejemplo,

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
