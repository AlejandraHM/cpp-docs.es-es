# Bits por car&#225;cter
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** Número de bits de un carácter  
  
 El número de bits de un carácter se representa mediante la constante de manifiesto **CHAR\_BIT**.  El archivo LIMITS.H define **CHAR\_BIT** como 8.
