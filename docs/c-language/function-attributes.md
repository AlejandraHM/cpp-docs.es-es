[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específicos de Microsoft**  
  
 El elemento *attribute\-seq* opcional no terminal permite seleccionar una convención de llamada para cada función.  También se pueden especificar funciones como `__fastcall` o `__inline`.
