# Caracteres significativos con vinculaci&#243;n externa
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** El número de caracteres significativos con vinculación externa  
  
 Los identificadores declarados como `extern` en los programas compilados con Microsoft C tienen 247 caracteres significativos.  Puede modificar este valor predeterminado por un número menor mediante la opción \/H \(restringir longitud de nombres externos\).
