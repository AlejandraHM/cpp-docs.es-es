# Firma de campos de bits
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.2.1** Si un campo `int` “plain” se trata como un campo de bits **signed int** o como un campo de bits unsigned int  
  
 Los campos de bits pueden ser signed \(con signo\) o unsigned \(sin signo\).  Los campos de bits normales se tratan como valores con signo.
