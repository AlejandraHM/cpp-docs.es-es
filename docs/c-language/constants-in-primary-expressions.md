# Constantes en expresiones primarias
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un operando constante tiene el valor y el tipo del valor constante que representa.  Una constante de caracteres tiene el tipo `int`.  Una constante entera es de tipo `int`, **long**, `unsigned int` o `unsigned long`, dependiendo del tamaño del entero y del modo en que se especifique el valor.  Para obtener más información, vea [Constantes](../c-language/c-constants.md).
