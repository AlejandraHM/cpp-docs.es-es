# Posici&#243;n de archivo en el modo Append
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Indica si el marcador de posición de archivo de una secuencia de modo Append se coloca inicialmente al principio o al final del archivo  
  
 Cuando un archivo se abre en modo Append, el indicador de posición de archivo señala inicialmente al final del archivo.
