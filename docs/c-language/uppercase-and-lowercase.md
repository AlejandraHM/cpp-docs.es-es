# <a name="uppercase-and-lowercase"></a>Mayúsculas y minúsculas
**ANSI 3.1.2** Indica si la distinción de mayúsculas y minúsculas es significativa  
  
 Microsoft C trata los identificadores de una unidad de compilación distinguiendo entre mayúsculas y minúsculas.  
  
 El vinculador de Microsoft distingue entre mayúsculas y minúsculas. Debe especificar todos los identificadores de manera coherente en cuando a mayúsculas y minúsculas.
