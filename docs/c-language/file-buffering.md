# Almacenamiento en b&#250;fer de archivo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Las características del almacenamiento en búfer de archivos  
  
 Los archivos de disco a los que se accede a través de funciones de E\/S estándar se almacenan en búfer.  De forma predeterminada, el búfer contiene 512 bytes.
