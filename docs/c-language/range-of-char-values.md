# Intervalo de valores char
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.2.1.1** Si **char** "plain" tiene el mismo intervalo de valores que **signed char** o `unsigned char`  
  
 Todos los valores de caracteres con signo van de –128 a 127.  Todos los valores de caracteres sin signo van 0 a 255.  
  
 La opción del compilador \/J cambia el valor predeterminado de **signed** a `unsigned`.
