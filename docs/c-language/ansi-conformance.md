# <a name="ansi-conformance"></a>Conformidad con ANSI
Microsoft C se ajusta a la norma del lenguaje C como se establece en la edición 9899:1990 del estándar ANSI C.  
  
 Las extensiones de Microsoft para el estándar ANSI C se indican en el texto y la sintaxis de este libro así, como en la referencia en línea. Dado que las extensiones no forman parte del estándar ANSI C, su uso puede limitar la portabilidad de programas entre sistemas. De manera predeterminada, las extensiones de Microsoft están habilitadas. Para deshabilitar las extensiones, especifique la opción del compilador /Za. Con /Za, todo el código no ANSI genera errores o advertencias.
