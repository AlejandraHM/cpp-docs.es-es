# <a name="summary-of-string-literals"></a>Resumen de literales de cadena
*string-literal*:  
 **'** *s-char-sequence* opt **'**  
  
 **L'** *s-char-sequence* opt **'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 cualquier miembro del juego de caracteres de origen excepto las comillas dobles ("), la barra diagonal inversa (\\) o el carácter de nueva línea *escape-sequence*
