# Caracteres multibyte
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.1.2** Estados de cambio para caracteres multibyte  
  
 Los caracteres multibyte se usan en algunas implementaciones, como en Microsoft C, para representar caracteres de otros idiomas no representados en el conjunto de caracteres base.  Sin embargo, Microsoft C no admite codificaciones dependientes del estado.  Por consiguiente, no hay estados de cambio.  Para obtener más información, vea [Caracteres multibyte y anchos](../c-language/multibyte-and-wide-characters.md).
