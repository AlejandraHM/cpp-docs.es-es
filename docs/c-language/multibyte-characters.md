# <a name="multibyte-characters"></a>Carácter multibyte

**ANSI 2.2.1.2** Estados de cambio para caracteres multibyte

Los caracteres multibyte se usan en algunas implementaciones, como en Microsoft C, para representar caracteres de otros idiomas no representados en el conjunto de caracteres base. Sin embargo, Microsoft C no admite codificaciones dependientes del estado. Por consiguiente, no hay estados de cambio. Para más información, vea [Caracteres multibyte y anchos](../c-language/multibyte-and-wide-characters.md).
