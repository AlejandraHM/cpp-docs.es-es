# Mayor tama&#241;o de matriz
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.3.4, 4.1.1** Tipo de entero necesario para contener el tamaño máximo de una matriz \(es decir, el tamaño de **size\_t**\)  
  
 El typedef de `size_t` es `unsigned int` en la plataforma x86 de 32 bits.  En las plataformas de 64 bits, el typedef de `size_t` es **unsigned \_\_int64**.
