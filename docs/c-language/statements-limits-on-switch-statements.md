# Instrucciones: L&#237;mites en las instrucciones switch
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** El número máximo de valores **case** en una instrucción `switch`  
  
 Microsoft C no limita el número de valores **case** en una instrucción **switch**.  El número solo está limitado por la memoria disponible.
