# <a name="significant-characters-without-external-linkage"></a>Caracteres significativos sin vinculación externa
**ANSI 3.1.2** El número de caracteres significativos sin vinculación externa  
  
 Los identificadores son significativos hasta 247 caracteres. El compilador no limita el número de caracteres que se pueden usar en un identificador; simplemente omite cualquier carácter más allá del límite.
