# <a name="registers-availability-of-registers"></a>Registros: Disponibilidad de los registros
**ANSI 3.5.1** Indica hasta qué punto pueden colocarse realmente los objetos en los registros por medio del especificador de clase de almacenamiento register  
  
 El compilador no admite las solicitudes de usuario para las variables de register. En su lugar, crea sus propias opciones al optimizar.
