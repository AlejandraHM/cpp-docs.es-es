# IBox::Value (Propiedad)
Devuelve el valor que se almacenó originalmente en este objeto.  
  
## Sintaxis  
  
```cpp  
property T Value {T get();}  
```  
  
```csharp  
  
```  
  
#### Parámetros  
 `T`  
 Tipo del valor al que se le ha aplicado la conversión boxing.  
  
## Valor de propiedad y valor devuelto  
 Devuelve el valor que se almacenó originalmente en este objeto.
