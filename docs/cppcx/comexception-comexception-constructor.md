# COMException::COMException (Constructor)
Inicializa una nueva instancia de la clase COMException.  
  
## Sintaxis  
  
```cpp  
COMException( int hresult )  
```  
  
#### Parámetros  
 hresult  
 Valor HRESULT de error representado por la excepción.
