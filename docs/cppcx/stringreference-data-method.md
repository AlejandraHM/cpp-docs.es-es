# StringReference::Data (M&#233;todo)
Devuelve el contenido de este `StringReference` como matriz de valores char16.  
  
## Sintaxis  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## Valor devuelto  
 Matriz de caracteres de texto UNICODE char16.
