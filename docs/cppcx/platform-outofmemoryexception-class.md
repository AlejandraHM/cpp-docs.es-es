# Platform::OutOfMemoryException (Clase)
Se produce cuando la memoria es insuficiente para completar la operación.  
  
## Sintaxis  
  
```cpp  
public ref class OutOfMemoryException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## Comentarios
