# Agile::~Agile (Destructor)
Destruye la instancia actual de la clase Agile.  
  
## Sintaxis  
  
```cpp  
  
~Agile();  
```  
  
## Comentarios  
 Este destructor también libera el objeto representado por el objeto Agile actual.
