# <a name="platformdetails-namespace"></a>Platform::Details (Espacio de nombres)
Este espacio de nombres solo está previsto para uso interno y no para utilizarlo en el desarrollo.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
  
namespace Platform {  
   namespace Details {  
}}  
```
