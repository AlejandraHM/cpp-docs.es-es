# String::GetHashCode (M&#233;todo)
Devuelve el código hash de esta instancia.  
  
## Sintaxis  
  
```cpp  
  
virtual override int GetHashCode()  
```  
  
## Valor devuelto  
 Código hash de esta instancia.
