# <a name="platformoperationcanceledexception-class"></a>Platform::OperationCanceledException (Clase)

Se produce cuando se anula una operación.

## <a name="syntax"></a>Sintaxis

```cpp
public ref class OperationCanceledException : COMException,    IException,    IPrintable,    IEquatable
```

### <a name="remarks"></a>Comentarios

Para obtener más información, consulta la clase [COMException](../cppcx/platform-comexception-class.md) .

**Servidor mínimo admitido:** Windows Server 2012

**Espacio de nombres:** Plataforma

**Metadatos:** platform.winmd
