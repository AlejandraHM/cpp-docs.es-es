# <a name="platformoperationcanceledexception-class"></a>Platform::OperationCanceledException (Clase)
Se produce cuando se anula una operación.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
public ref class OperationCanceledException : COMException,    IException,    IPrintable,    IEquatable  
```
