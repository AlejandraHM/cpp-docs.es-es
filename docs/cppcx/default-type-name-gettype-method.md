# default::(type_name)::GetType (M&#233;todo)
Devuelve un Platform::Type^ que representa el tipo actual.  
  
## Sintaxis  
  
```cpp  
  
Platform::Type^ GetType()  
```  
  
## Valor devuelto  
 Objeto [Platform::Type](../cppcx/platform-type-class.md)^ que representa el objeto actual.
