# Platform::ObjectDisposedException (Clase)
Se produce cuando se realiza una operación en un objeto desechado.  
  
## Sintaxis  
  
```cpp  
public ref class ObjectDisposedException : COMException,    IException,    IPrintable,    IEquatable  
```
