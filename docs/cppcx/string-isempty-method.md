# String::IsEmpty (M&#233;todo)
Indica si el objeto String actual está vacío.  
  
## Sintaxis  
  
```cpp  
  
bool IsEmpty()  
```  
  
## Valor devuelto  
 `true` si el objeto String actual es `null` o la cadena vacía \(L""\); de lo contrario, `false`.
