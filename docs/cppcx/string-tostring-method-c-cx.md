# String::ToString (M&#233;todo) (C++/CX)
Devuelve un objeto String cuyo valor es igual al de la cadena actual.  
  
## Sintaxis  
  
```cpp  
  
String^ String::ToString()  
```  
  
## Valor devuelto  
 Un objeto String cuyo valor es igual al de la cadena actual.
