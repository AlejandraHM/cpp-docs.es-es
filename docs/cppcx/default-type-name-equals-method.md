# default::(type_name)::Equals (M&#233;todo)
Determina si el objeto especificado es igual al objeto actual.  
  
## Sintaxis  
  
```cpp  
  
bool Equals(  
    Object^ obj  
)  
```  
  
## Parámetros  
 obj  
 Objeto que se va a comparar.  
  
## Valor devuelto  
 Es `true` si los objetos son iguales; de lo contrario, es `false`.
