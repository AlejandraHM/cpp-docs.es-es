# Platform::ClassNotRegisteredException (Clase)
Se produce cuando una clase COM no se ha registrado.  
  
## Sintaxis  
  
```cpp  
public ref class ClassNotRegisteredException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## Comentarios  
 Para obtener más información, consulta la clase [COMException](../cppcx/platform-comexception-class.md).
