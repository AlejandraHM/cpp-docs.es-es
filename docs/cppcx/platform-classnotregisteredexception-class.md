# <a name="platformclassnotregisteredexception-class"></a>Platform::ClassNotRegisteredException (Clase)
Se produce cuando una clase COM no se ha registrado.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
public ref class ClassNotRegisteredException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
### <a name="remarks"></a>Comentarios  
 Para obtener más información, consulta la clase [COMException](../cppcx/platform-comexception-class.md) .  
  
### <a name="requirements"></a>Requisitos  
 **Cliente mínimo admitido:** Windows 8  
  
 **Servidor mínimo admitido:** Windows Server 2012  
  
 **Espacio de nombres:** Platform  
  
 **Metadatos:** platform.winmd
