# SizeT::SizeT (constructor)
Inicializa una nueva instancia de SizeT con el valor especificado.  
  
## Sintaxis  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## Parámetros  
 value1  
 Un valor sin signo de 32 bits.  
  
 value2  
 Puntero a un valor sin signo de 32 bits.
