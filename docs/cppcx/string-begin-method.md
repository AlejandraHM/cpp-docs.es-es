# String::Begin (M&#233;todo)
Devuelve un puntero al principio de la cadena actual.  
  
## Sintaxis  
  
```cpp  
  
char16* Begin()  
```  
  
## Valor devuelto  
 Un puntero al principio de la cadena actual.
