# StringReference::Length (M&#233;todo)
Devuelve el número de caracteres de la cadena.  
  
## Sintaxis  
  
```cpp  
unsigned int Length() const  
```  
  
## Valor devuelto  
 Un entero sin signo que especifica la cantidad de caracteres de la cadena.
