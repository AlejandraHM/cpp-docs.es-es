# Platform::OutOfBoundsException (Clase)
Se produce cuando una operación intenta tener acceso a datos que están fuera del intervalo válido.  
  
## Sintaxis  
  
```cpp  
public ref class OutOfBoundsException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## Comentarios  
 Para obtener más información, consulta la clase [COMException](../cppcx/platform-comexception-class.md).
