# IntPtr::ToInt32 (M&#233;todo)
Convierte el valor IntPtr actual en un entero de 32 bits.  
  
## Sintaxis  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## Valor devuelto  
 Un entero de 32 bits.
