# StringReference::GetHSTRING (M&#233;todo)
Devuelve el contenido de la cadena como `__abi_HSTRING`.  
  
## Sintaxis  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## Valor devuelto  
 `__abi_HSTRING` que contiene los datos de tipo String.
