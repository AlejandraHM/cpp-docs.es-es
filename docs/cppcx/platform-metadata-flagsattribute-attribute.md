# Platform::Metadata::FlagsAttribute (Atributo)
Indica que una enumeración se puede tratar como un campo de bits, es decir, un conjunto de marcas.  
  
## Sintaxis  
  
```cpp  
  
public ref class Flags abstract : Attribute  
```
