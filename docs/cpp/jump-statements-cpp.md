# <a name="jump-statements-c"></a>Instrucciones de salto (C++)
Una instrucción de salto de C++ realiza una transferencia de control local inmediata.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
