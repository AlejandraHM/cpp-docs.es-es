# Instrucciones de salto (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Una instrucción de salto de C\+\+ realiza una transferencia de control local inmediata.  
  
## Sintaxis  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
