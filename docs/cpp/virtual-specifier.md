# virtual (Especificador)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La palabra clave [virtual](../cpp/virtual-cpp.md) solo se puede aplicar a las funciones de miembro de clase no estáticas.  Eso significa que el enlace de las llamadas a la función se aplazará hasta el momento de la ejecución.  Para obtener más información, vea [Funciones virtuales](../cpp/virtual-functions.md).
