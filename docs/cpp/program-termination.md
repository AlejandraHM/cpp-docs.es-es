# Finalizaci&#243;n del programa
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

En C\+\+, hay varias maneras de salir de un programa:  
  
-   Llamar a la función [exit](../cpp/exit-function.md).  
  
-   Llamar a la función [abort](../cpp/abort-function.md).  
  
-   Ejecutar una instrucción [return](../cpp/return-statement-in-program-termination-cpp.md) desde **main**.
