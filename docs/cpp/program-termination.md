# <a name="program-termination"></a>Finalización del programa
En C++, hay varias maneras de salir de un programa:  
  
-   Llame a la [salir](../cpp/exit-function.md) función.  
  
-   Llame a la [anular](../cpp/abort-function.md) función.  
  
-   Ejecutar un [devolver](../cpp/return-statement-in-program-termination-cpp.md) instrucción desde **principal**.
