# _com_error::operator =
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específicos de Microsoft**  
  
 Asigna un objeto `_com_error` existente a otro.  
  
## Sintaxis  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### Parámetros  
 `that`  
 Un objeto `_com_error`.
