# <a name="comerroroperator-"></a>_com_error::operator =
**Específicos de Microsoft**  
  
 Asigna un objeto `_com_error` existente a otro.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### <a name="parameters"></a>Parámetros  
 `that`  
 Objeto `_com_error`.
