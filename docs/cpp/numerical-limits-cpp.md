# L&#237;mites num&#233;ricos (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los dos archivos include estándar, LIMITS.H y FLOAT.H, definen los límites numéricos, o los valores mínimo y máximo que puede contener una variable de un tipo determinado.  Estos mínimos y máximos se pueden mover a cualquier compilador de C\+\+ que use la misma representación de datos que ANSI C.  El archivo include LIMITS.H define los [límites numéricos de los tipos enteros](../cpp/integer-limits.md) y FLOAT.H define los [límites numéricos de los tipos de punto flotante](../cpp/floating-limits.md).
