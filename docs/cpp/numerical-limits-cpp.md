# <a name="numerical-limits-c"></a>Límites numéricos (C++)
El estándar de dos archivos de inclusión, \<limits.h > y \<float.h >, definen los límites numéricos o valores mínimo y máximo que puede contener una variable de un tipo determinado. Estos mínimos y máximos se garantiza que sean portables a cualquier compilador de C++ que usa la misma representación de datos que ANSI C. El \<limits.h > Incluir archivo define la [límites numéricos de los tipos enteros](../cpp/integer-limits.md), y \<float.h > define el [límites numéricos de los tipos de flotante](../cpp/floating-limits.md).
