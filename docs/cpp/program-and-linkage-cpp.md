# Programa y vinculaci&#243;n (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un programa consta de una o más unidades de traducción vinculadas entre sí.  La ejecución \(conceptualmente\) comienza en la unidad de traducción que contiene la función **main**.  \(Para obtener más información sobre unidades de traducción, vea [Fases de traducción](../preprocessor/phases-of-translation.md) en la *Referencia del preprocesador*\). Para obtener más información sobre la función **main**, vea [Inicio del programa: la función main](../cpp/main-program-startup.md).
